    殿外虎视眈眈，殿内大战以起。<br/>
<br/>
    来自魂殿地煞分殿的天级护法，抬手一掌便朝叶晨拍去，身为九星斗宗，在叶晨率先发难的瞬间他便判断出了他的修为。<br/>
<br/>
    “五星斗宗？呵呵，不错的实力，但你以为地煞殿的护法，都是人级吗？”<br/>
<br/>
    鼓动着体内磅礴的斗气，高大黑袍人掌心刹那涌起层层空间波动，直接徒手迎上叶晨的攻击，剑掌相撞间，其手掌前方空间顿如凝固了般，发出了如金属撕扯的尖唳之音，随即道道凹陷透明的褶皱纹路浮现当空，而凹陷空间内，魔渊剑尖亦停在了高大黑袍人的掌心之前，不得寸进。<br/>
<br/>
    “哼，小子，看你也有魂殿之令，想来也是我地煞殿的人，若你现在缴械投降，本护法可以保证你安然无恙，甚至坐上被你杀死的人级护法的位置，如何！”<br/>
<br/>
    挡住叶晨一剑，高大黑袍人的话锋顿时一改，变化之快，让已经闪到一旁观战的另外两名魂殿护法皆都看得面面相觑，忍不住暗道：以天级护法首座，林厉的心性，谁惹到他基本都是一个死字，今天这是怎么了？转性啦？<br/>
<br/>
    正想着，他们便突然发现，一抹银白寒光顿从那血色剑尖一闪而来，旋即，林厉施展凝聚的空间防御，刹那发出了无数密密麻麻的透明崩裂痕迹，而且那剑尖之上，亦是沾染上了一抹猩红。<br/>
<br/>
    “原来如此！”<br/>
<br/>
    两个一胖一瘦的黑袍人对视一眼，旋即异口同声道：“叶老弟，我们魂殿的制度地位自古都是有能者居之，既然林厉护法说你能当上人级之位，那就自然不会有假，所以呀，即便杀了自己人，但只要你够强能得到地煞殿主的肯定，那就不会出事。”<br/>
<br/>
    “住嘴，这里哪轮到你们说话。”<br/>
<br/>
    被队友在背后插了一刀，高大黑袍人林厉，脸色顿时难看起来，而也正在此时，右手掌心传来的刺痛，亦让他神色大变不已。<br/>
<br/>
    很快，他就发现被挡住的长剑，正以肉眼可见的速度穿透着他施展的空间防御，且还在不断前行，朝着他的掌心刺来。<br/>
<br/>
    此子实属天骄，不好应付。<br/>
<br/>
    林厉心中做出利弊判断，随即张开说道：“我承认你很强，但若想挑战我还是不够看的。本护法再给你一次机会，速速停手，我保你当上地级护法，甚至天级。”<br/>
<br/>
    “真的？”<br/>
<br/>
    “绝无虚假。”<br/>
<br/>
    叶晨终于说话了，而林厉听到他似乎很感兴趣，嘴角顿时露出笑意道：“只要你老实听话，以后在地煞殿本护法一定罩着你，至于那个极品炉鼎，我们也可以共享。”<br/>
<br/>
    “天级护法，炉鼎共享？哈哈，你可真会说话，但我只想你死。”<br/>
<br/>
    叶晨的眼神越来越冷，手掌微紧间，股股浓稠如墨的黑色气息，顿从魔渊剑内不断弥漫而出，旋即盘旋在叶晨头顶后，瞬息涌入至他体内。<br/>
<br/>
    霎时，一股巨大的心脏跳动声音响彻整座大殿，随即怪异无比的斗气波动从叶晨体内疯狂肆意，使其长发无风自动飘舞，眉间更是浮现出了一道血痕，而且斗气修为也从五星斗宗直接飙升到了六星、七星、八星直至九星巅峰才算彻底止步，但其眉间仍有大量的黑色气息弥漫汹涌，仿佛无穷无尽。<br/>
<br/>
    一时之间，大殿之内的所有人，都震惊的看着叶晨，眼中满是惊骇。<br/>
<br/>
    可还没等他们反应过来到底发生了什么，那被叶晨刺出的魔渊剑，百年瞬息洞穿了林厉的空间防御，直朝其掌心一剑刺去。<br/>
<br/>
    “啊！”<br/>
<br/>
    忍不住痛叫一声，林厉猛地抽回手掌退到一旁后，顿时震怒的看着手掌出现的一个血洞，鲜血不断自内流淌涌出。<br/>
<br/>
    运起斗气封住破碎的经脉，林厉举目盯着叶晨，阴森而道：“小子，你惹怒我了，本护法已经给过你机会，但是你不珍惜，那也就别怪我痛下杀手了。”<br/>
<br/>
    眼中的怒火几欲狂涌，林厉扯去宽大黑袍，露出一身强壮块大的腱子肉道：“不过是靠着秘法强行提升了境界而已，难道你以为，自己就是九星斗宗了吗？小子，本护法现在就让你看看，什么叫做，巅峰斗宗。”<br/>
<br/>
    说罢，凌厉浑身壮硕肌肉不断颤动，凌空飞起，随即股股肉眼可见的磅礴斗气，顿如猛龙出海般瞬息自其体内疯狂四溢。<br/>
<br/>
    刹那间，云岚宗门正殿内的桌椅不断被斗气卷飞而起，旋即于空纷纷炸裂成无数碎木坠落，直至接触地面，亦全都碎成了细小木屑，盖得地上厚厚的一层。<br/>
<br/>
    与此同时，殿外高空忽然下起雨，而当那些雨点滑落之际，亦变成了一道道炫白的晴天霹雳，自空急速坠落轰向大殿。<br/>
<br/>
    “奔腾引雷诀！”<br/>
<br/>
    林厉狂吼一声，体表瞬间弥漫道道细小雷蛇窜动，旋即高空落下的无数粗壮雷电冲至大殿屋顶，如雨点般，瞬间就将整个宽阔屋顶蒸发得一干二净，携带着莫大威能直袭叶晨而去。<br/>
<br/>
    见此一幕，另外两名天级护法，顿时惊叫道：“这是低阶中级斗技，叶兄，扛不住还是认怂吧。”<br/>
<br/>
    看似嘲讽的话语，却又夹杂提醒之意，林厉闻言，顿时怒目看向两人，想着事后，一定要好好教训一番。<br/>
<br/>
    可他还没想好要教训方案，一道冷冽的声音顿时响彻在其耳旁：“就这？也是巅峰斗宗？”<br/>
<br/>
    音落，叶晨微微闭上双眼，霎时股股熟悉的力量如遇旧主般，瞬间在其掌心疯狂凝聚起来，紧接着，道道时空之力犹如实质性的流光线条，尽数在周空领域暴露在叶晨精神力下。<br/>
<br/>
    很奇妙的感觉，又或许是曾经拥有，如今靠着吸收了剑神传承神力内魔气的魔渊，进入到入魔状态，修为暴涨至九星斗宗的叶晨，不但对空间掌控力变得极强，就连周围时间，都似任其拿捏一般。<br/>
<br/>
    “时空流转，逆！”<br/>
<br/>
    再次喊出熟悉的魂技名称，一时间，在强大的灵魂和精神力支撑下，叶晨猛地一把握住了密布在周空领域的流光线条，旋即只见宗门大殿内的一切都如静止了般，所有人都不会动了，而那疾驰劈来的万千雷霆，亦是在头顶变得极其缓慢异常。<br/>
<br/>
    叶晨抓着时空流光线条，漫步在布满雷霆的大殿中自由穿梭行动，但很快，他便屋内的人，其实不是完全静止不动，只是变得极其缓慢而已，相当于静止状态。<br/>
<br/>
    “咳咳！”<br/>
<br/>
    一声咳嗽突然自动咳出，叶晨皱了皱眉后，亦是在自己嘴角摸到了血迹，这时他才发现此刻自己的魂力、精神力和斗气，都在以极快的速度疯狂流逝，短短几秒钟，他的所有状态就十不存一了。<br/>
<br/>
    而随着头顶雷霆攻击的缓慢下降，那种要被掏空的感觉，更是让他感到吃力。<br/>
<br/>
    对此，叶晨不由笑了笑，暗道以目前的修为还不足够施展如此逆天的技能，当下还是速战速决算了。<br/>
<br/>
    想罢，他便举剑刺穿了天级护法林厉的心脏，旋即将他推向了原本自己的位置。<br/>
<br/>
    当这一切做完后，叶晨便松开了手中光华四溢多彩的时空线条。<br/>
<br/>
    刹那间，周方时空恢复正常，半空雷霆轰然落下，而林厉也在众目睽睽之下，被那漫天雷霆轰成虚无，诡异的死在了自己的斗技手中。<br/>
<br/>
    一时间，这般惊骇的场面，让所有人都觉得难以置信，大家不知道发生了什么，在那短短的几秒钟，叶晨的所作所为无人知晓，而林厉，在所有人眼中，就死得挺突然地。<br/>
<br/>
    “赢了？”<br/>
<br/>
    虽然有些莫名其妙，但眼见林厉命死身散，云山一时凝聚的强大斗气，顿时溃散下沉一地。<br/>
<br/>
    而大殿边缘观战的另外两名天级护法，则有些汗毛直立，心肝具颤。<br/>
<br/>
    他们原本想着让叶晨和林厉斗个两败俱伤，好让自己上位，可如今，这算盘好像打错了，而且地煞分殿以后，将会引来一位更强的护法首座。<br/>
<br/>
    “咕噜。”<br/>
<br/>
    忍不住咽了咽口水，两名黑袍护法对视一眼后，顿时紧张地朝叶晨拱手恭敬道：“叶老弟，哦不，是叶兄。如今真相已明，我等也就先行告辞了，不过你的情况，我们还是会向地煞殿主禀告的，请您体谅一二。”<br/>
<br/>
    “滚吧，想说什么都可以，但炉鼎二字，我希望你们从未听说过。”<br/>
<br/>
    “是是，叶兄放心，我们可不知道您刚才说了什么字。”<br/>
<br/>
    话罢，两人手捏一道法诀，笼罩住云岚宗的鸟笼屏障不由消散而逝，当下，他们便逃也似的飞奔出大殿，头都不敢回一下。<br/>
<br/>
    “叶晨，没想到你居然这么强。”<br/>
<br/>
    云山看着离去的两个护法，嘴角的笑意始终愈发明显，他觉得自己赌叶晨赌对了，而且还是大对特对，有他在，云岚宗何愁不能壮大。<br/>
<br/>
    想到这，云山不由仰头大笑起来，然而就在这时，殿内空气中忽然弥漫起阵阵淡淡白烟，旋即一声娇呼下，美杜莎身旁的月媚便突然娇躯一软，晕倒在了地上。<br/>
<br/>
    而美杜莎的情况也不怎么好，紫色面纱下的俏脸满是虚落之感，旋即双眼发困间，亦是无力瘫坐在了身后的木椅上。<br/>
<br/>
    “何人胆敢偷袭我云岚宗。”<br/>
<br/>
    云山身躯摇晃，脚步虚浮地看向殿外，可才刚说完一句话，他就被一柄长剑架在了脖子上。<br/>
<br/>
    发现情况不对的叶晨，亦是因为消耗过度，只能堪堪扶住云韵靠在殿内的石柱上，面露警惕的看着一拥而入的上百人群。<br/>
<br/>
    其中之人修为，不乏斗灵斗王，而为首的老者，亦是有着九星斗皇巅峰的强大境界。<br/>
<br/>
    虽然大殿内的斗宗不少，可如今身中怪药之力，不管是云山还是美杜莎，都已经无法催动斗气了，而唯一百毒不侵的叶晨，亦是因为控制时空变得有心无力，只能暂且观望。<br/>
<br/>
    “哈哈哈，没想到今日，可以将云岚宗一网打尽。”<br/>
<br/>
    面部有疤的强壮男子，眼眉微微轻佻，随即举目看向云山道：“你就是云岚宗的老宗主吧，闻名不如一见呀，果然有强者之风，就连那三个斗宗强者，都奈何不了你，但可惜，你们云岚宗，还是落到了帝国手中。”<br/>
<br/>
    “竖子休得猖狂，如此卑劣手段，汝也敢叫嚣。”<br/>
<br/>
    云山气得不行，他万万没想到，刚赶走魂殿的人，帝国就趁虚而入，时机把握得如此之好，想来一定是早有准备，该死。<br/>
<br/>
    “老云何必动怒，你们云岚宗该有此劫。”<br/>
<br/>
    为首的麻衣老者抚须看着没了牙齿（斗气）的云山，顿时摇摇头道：“你我好歹也是同个时代的强者，老夫也不愿伤你，可云岚宗，怕是没有存在的必要了。”<br/>
<br/>
    “你！”<br/>
<br/>
    云山脸色黝黑，说不出话来，只好将目光转向叶晨，但得到的，却只是微微摇头示意罢了。<br/>
<br/>
    很快，整个云岚山上，空无一人，所有宗门弟子长老，接都被锁链困住，连成了一条长长的队伍，朝着加玛帝都走去。
