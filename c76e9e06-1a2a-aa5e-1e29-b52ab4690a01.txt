　　第441章
<br/>　　李乐天骑马行在草原上，身上锦袍，座下高头大马，凛冽的风吹过，长发轻扬。
<br/>　　他身后跟着十几名亲卫，表情肃穆，眼神干净，气质硬得像钢铁。
<br/>　　虽然同样是傀儡，但李乐天和李行哉不同。从长安逃出来时，他身后便跟着几千死士，这都是看着他坚守长安城的士兵。而后来在将起义军赶出长安的过程中，李乐天也是冲锋在前，悍不畏死，即便在将军府，也有很多少壮派是支持他的。
<br/>　　所以，不能简单说他是被将军府控制，准确来说，双方是合作关系。
<br/>　　李乐天骑马在草原上奔驰，抒发着心中的豪情。如果顺利的话，他此刻早该坐上那张龙椅，然后整理江山，清楚匪患。
<br/>　　但事情不太顺利，很多人都跳了出来，要和自己争帝位，所以才有现在竞赛谁能杀死林少羽的事情发生。
<br/>　　而在所有竞争者之中，相府支持的李行哉是最有竞争力的。对于自己这个六弟，李乐天的心情很复杂：嫉妒、鄙视以及血脉里斩不断的牵挂。
<br/>　　而就在昨夜，发生了一件事，有人潜入李乐天的大帐，想要刺杀他。虽然刺客没有得手，被李乐天一剑砍了脑袋。
<br/>　　刺客的身份已经确认，是相府的门客。那么，自己的六弟，究竟知不知道这件事？
<br/>　　于是，李乐天的心情很糟糕，今天借着骑马，出来透透气。
<br/>　　“殿下……”
<br/>　　一匹黑马奔来，李乐天的亲卫同时握紧长矛，将李乐天护在当中。
<br/>　　马上一个老者，黑发，白须，脸上堆满周围，目光却如实质一般沉重。
<br/>　　帝国大将军，尉迟离。
<br/>　　但就算对方是帝国将军，李乐天的亲卫也没有放松防备，握着兵器，紧盯着尉迟离不放。假若李乐天此时肯下达命令的话，他们会毫不犹豫的执行，合力围攻尉迟离。
<br/>　　如果李行哉在现场，看到眼前这一幕，毫无疑问会羡慕得流口水。这就是手里有人的好处，可以摆脸子，使性子，什么低调，耍心机完全用不着。
<br/>　　李乐天挥挥手，让手下人散开，尉迟离这才有机会上前。
<br/>　　“殿下，刚得到的消息，程大雷来了并州。”
<br/>　　“他……怎么来了？”
<br/>　　李乐天皱起了眉头，关于程大雷的情报，他手中也掌握很多，知道他在凉州有一股势力。据说，这程大雷和正义教的逆贼还有联系，但具体什么联系，他也不知道。
<br/>　　此时此地，他来并州做什么。
<br/>　　“消息准确么？”李乐天。
<br/>　　“千真万确，柳轻名带来的消息，程大雷一共三个人，正向杀虎口而来。”
<br/>　　“尉迟将军什么打算？”李乐天问。
<br/>　　“这贼人为祸天下多时，早就该除掉了。他躲在凉州，没时间收拾他，今天既然来了……”预尉迟离挥挥手：“就把他留在这里。”
<br/>　　“原该如此。”李乐天挥挥手：“这件事尉迟将军处理就是。”
<br/>　　尉迟离拍了拍手，守在远处的三个人快马奔到。等接近尉迟离时，从马上跃下来，奔驰的骏马也同时停住。
<br/>　　李乐天饶有兴致的看着这一幕，心中却是震惊不已。将军府招揽江湖人士为门客，当然，李乐天也有自己的门客。三人能够同时将马勒住，单单这手骑术已经算得上难得的好手。
<br/>　　“只有三个人，是不是少了些？”李乐天。
<br/>　　“殿下放心，他们都是江湖上的高手。”尉迟离指着这三人：“苍南子，善于寻人，豢养的猎犬能找到五十里内任何一人。白壮，原先是一个猎户，可以空手打死黑熊，张北峰，原先是屠户，双臂力大无穷。”
<br/>　　“屠户也能杀人么？”李乐天笑笑，目光落到这三人身上。
<br/>　　苍南子是个道士，年纪已经不小。白壮生得虎背熊腰，而张北峰身上肌肉紧绷，在冬季，身上也只穿了一件单衣。
<br/>　　“殿下放心便是，这程大雷的人头定能呈到殿下面前。”
<br/>　　“那我就拭目以待。”李乐天微微点头：“若真能杀了那恶徒，你们也算为国为民做了一件好事。”
<br/>　　与此同时，京州兵大营。
<br/>　　屈九江跪在一个老者面前，老者留着三撇胡须，皱纹堆积在一起，看上去，像随时都有可能一命呜呼。
<br/>　　“三年前，你败在他的手中，如此，耽误了你十年前程，现在去吧，把浪费的十年拿回来。”
<br/>　　“是！”屈九江长身而起，退出大帐。
<br/>　　李行哉愣愣的坐在一旁，道：“那程大雷可是极其可恶，丞相只派一人去，会不会太冒险了些。”
<br/>　　老者摇摇头，面带笑意道：“殿下放心，我已派了帮手，这件事必然万无一失。”
<br/>　　“帮手，怎样的帮手，我可是了解程大雷那恶贼，一般人，怕是对付不了他。”李行哉。
<br/>　　“老夫也知道贼人可恶，不若，殿下派一个人随行？”
<br/>　　李行哉沉默下来，崔相这句话问到他的难处。现在他是双手空空，一个能用的人也没有。如今李行哉，甚至有些怀念福德勒在身边的时候了。起码，他能算自己的朋友。
<br/>　　他深吸一口气：“我对那贼人是恨之入骨，罢了，这次我就亲自出马，定要砍掉他的脑袋。”
<br/>　　“殿下，殿下。”崔相笑着道：“那程大雷自然有人处理，他们只有三个人，用不着殿下出手。殿下还有更重要的事情要做？”
<br/>　　李行哉一顿，目光看向崔相。
<br/>　　崔相眯在一起的双目缓缓睁开，道：“逆贼已经被逼近杀虎口，他们才是殿下要解决的大事。”
<br/>　　程大雷来到并州的消息，在极短的时间便传开了。骑黑牛，背斧头，面目丑陋，一行三人……这几个特征，就可以准确的锁定程大雷。
<br/>　　相府、将军府，扬州兵等等大大小小的势力，都派出人手，要在解决林少羽之前，先解决掉程大雷。
<br/>　　各路诸侯派出的高手，足足有五六百人，他们自然不会合作，却又一个共同的目的，就是杀掉程大雷。
<br/>　　一时间，并州草原上，风声鹤唳，草木皆兵，展开了对程大雷的追击战。
<br/>(本章完)
