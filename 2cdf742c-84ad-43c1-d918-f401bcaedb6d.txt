　　云丹宗宗主慕容丹，被一名神秘年轻人诛杀的消息，犹如一阵飓风般，传遍了整个云丹城，乃至整个云丹星域！
<br/>　　此事一经传开，顿时引起了巨大轰动，云丹宗之内更是人心惶惶。
<br/>　　直到一连过去数日，叶尘杀了慕容丹之后，并没有继续找云丹宗的麻烦，云丹宗弟子这才纷纷松了一口长气。
<br/>　　客栈之中，
<br/>　　“叶前辈，虽然师尊死在你的手上，不过一切都是他咎由自取，我还是要谢谢您，没有迁怒云丹宗其他人！”
<br/>　　冷蝉衣由衷的感谢道。
<br/>　　叶尘摆了摆手，淡淡道：
<br/>　　“实不相瞒，我其实曾经受过你们云丹宗一位前辈的恩惠，所以才手下留情，今后只要你们不招惹我，这一次的事情，我可以当做没有发生过！”
<br/>　　冷蝉衣微微一愣，旋即连忙躬身点头，道：
<br/>　　“是是！叶前辈放心，云丹宗日后绝对不敢再与前辈为敌！”
<br/>　　叶尘又道：
<br/>　　“其实这一次请你过来，是有一件事相托！”
<br/>　　“门外的那些我的同伴，我想让他们在云丹宗定居，希望你以后能代为多加照顾！”
<br/>　　冷蝉衣顿时再次一愣，眼眸中透着一抹惊愕之色，
<br/>　　“叶前辈就这么放心？”
<br/>　　按理说，叶尘虽然不再追究，可毕竟和云丹宗有了过节，如今竟然还敢把自己的同伴留在云丹城，实在令冷蝉衣感觉匪夷所思。
<br/>　　叶尘淡淡一笑，
<br/>　　“怎么？难道你对我怀恨在心，等到我走之后，就会报复在我的同伴身上吗？”
<br/>　　冷蝉衣连忙摇头，道：
<br/>　　“当然不会！既然叶前辈如此信任我，那我冷蝉衣可以对天发誓，一定会确保他们的周全！”
<br/>　　叶尘点了点头，他对冷蝉衣的人品还是很放心，
<br/>　　“既然如此，我也不能让你白干……”
<br/>　　说着，叶尘右手一翻，取出一枚玉简，递到冷蝉衣的跟前，道：
<br/>　　“这里面，是我用神念刻箓的丹术心得，以及一些适合你修炼的功法，你拿回去仔细研习，会对你的丹术和修为，有一些帮助！”
<br/>　　冷蝉衣闻言，顿时大喜，当即便跪在了地上，双手接过玉简，恭恭敬敬的磕了三个头，
<br/>　　“多谢叶前辈！”
<br/>　　叶尘堂而皇之的受了冷蝉衣三拜之后，这才随手一挥，将她从地上托起。
<br/>　　他这种做法，已经不亚于收徒，受对方三拜，也是理所应当。
<br/>　　他之所以将自己的丹术心得，传授给冷蝉衣，是报前世云丹仙子的传承之恩，将这份人情还给云丹宗。
<br/>　　当然，除此之外，冷蝉衣也的确是一个丹道天才，不应该被埋没。
<br/>　　……
<br/>　　一晃眼，
<br/>　　叶尘又在云丹城待了三日，将身上的伤势彻底养好之后，修为也已经稳定在元婴境一重境界。
<br/>　　灵溪和灵韵两姐妹，在冷蝉衣的帮助下，已经成为云丹宗的内门弟子，而灵山等人也各自找到了适合自己的职业，算是在云丹宗彻底定居了下来。
<br/>　　这一日，叶尘留下了一封辞别信和一些灵石，然后启动逍遥仙珠，离开了云丹星域。
<br/>　　按照修真界的方位，云丹星域位于整个中央大星域的最南端，而迷罗星海则位于中央大星域的西北角，中间隔着数万个星域，即便是用逍遥仙珠一直马不停蹄的赶路，也需要两三个月的时间。
<br/>　　好在如今，离迷罗星海正式开启，还有一年左右的时间，所以叶尘的时间，倒也十分充足。
<br/>　　于是，接下来的日子里，叶尘一边赶路，一边潜心修炼，偶尔路过一些妖兽横行的星域，便出手猎杀一些高级妖兽，既可以磨炼修为，又可以换取灵石，再用灵石购买药材，炼制一些自己需要的丹药。
<br/>　　一晃眼，
<br/>　　两个多月过去了，
<br/>　　眼看离迷罗星海越来越近，而叶尘的修为，也已经从元婴境一重，提升到了元婴四重！
<br/>　　就这，叶尘为了避免体内真元过于浮躁，还刻意压制着自己的修为提升，否则以吞天神功的的强悍，再加上吞天鼎、噬血妖珠以及造化仙珠等法宝，他如果肆意提升修为的话，只怕早已经达到元婴巅峰境界了。
<br/>　　不过，如今唯一让叶尘感到郁闷的是，自从上次一战，他的紫琼剑被毁，他便没有了攻击类的法宝。
<br/>　　他这一路走了，倒也走过不少拍卖场，可惜最高也就是圣器级别，根本入不了叶尘的法眼。
<br/>　　没办法，即便是强者如云的中央大星域，仙器也是极为稀少的存在。
<br/>　　像那云丹宗，如此强的底蕴，也不过只有镇天印，这么一件仙器法宝而已。
<br/>　　寻常的修真者，得到仙器级别的法宝，肯定都宝贝供起来，又有谁舍得拍卖出去？
<br/>　　因此，这一路下来，叶尘依旧没能找到一件，令他满意的攻击法宝。
<br/>　　这一天，
<br/>　　叶尘来到某个六级星域，走在某座城池的街道之上，忽然听到旁边传来一阵议论之声，
<br/>　　“哎，你们听说了吗？据说有人一个无名星球上，发现了灭世仙君的遗迹！”
<br/>　　“之前不是都有类似的传闻吗？也不知道是真是假？”
<br/>　　“听说连咱们南斗星域四大家族的高手，也全都赶了过去，这一次多半应该是真的！”
<br/>　　“嘿嘿，就算是真的又如何？灭世仙君的遗迹，又岂是那么容易闯的？不知道又要死多少人呢！”
<br/>　　……
<br/>　　听到众人的议论声，叶尘顿时眉头微微一挑，
<br/>　　“灭世仙君？好像是两千多年前，一位凶名赫赫的人物，虽然修为并非顶级，但是他的那把灭世魔刀，据说是一件足以媲美神器的存在，连渡劫帝君都可以斩杀……等等，灭世魔刀？”
<br/>　　想到这里，叶尘顿时双眸一亮，脸上不由得露出一抹喜色，
<br/>　　还真是，踏破铁鞋无觅处，得来全不费工夫！
<br/>　　他之前，还正在为没有像样的兵器法宝犯愁，若是能得到这灭世魔刀，他的战力必定可以提升一倍都不止！
<br/>　　“去看看！”
<br/>　　反正时间还很充足，叶尘当机立断，决定先前往那人口中的地方，去那位灭世仙君的遗迹中，碰一碰运气，说不定真的能够得到自己想要的东西呢？
