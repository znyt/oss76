<br/>  第二天，清早。<br/>　　<br/>　　  城民们起来，推开门，小心翼翼的看了看外面。<br/>　　<br/>　　  昨天夜里，县衙里发生了那么大的动静，谁又能听不到？<br/>　　<br/>　　  街坊邻居们聚在一起，开始闲聊。<br/>　　<br/>　　  “惯例又来了啊！”<br/>　　<br/>　　  “谁说不是呢？咱这新奉县虽然现今是富阳伯爵府的领地，但真正说了算的还是山匪。”<br/>　　<br/>　　  “之前那几任县令上任都想清剿山匪，最后的下场如何？死的死，退的退！”<br/>　　<br/>　　  “徐允他也是妥协了，还勾结青崖口山匪，不然他能坐稳这县令之位？不过昨天直接被赶了出去。”<br/>　　<br/>　　  “你们说那个伯爵府少爷，他真的是个败家子？败家子能有这么狠辣的手段？”<br/>　　<br/>　　  “不管是不是也好不了，听昨晚那动静不小，肯定是青崖口山匪来报复了，还想当领主？哪是那么容易的。”<br/>　　<br/>　　  “走，我们去瞧瞧？”<br/>　　<br/>　　  “不好了，不好了！”这时有人大喊。<br/>　　<br/>　　  “咋了，老李头？看把你吓成这样！”<br/>　　<br/>　　  “京筑观，又一座京筑观！”<br/>　　<br/>　　  老李头忙着说道。<br/>　　<br/>　　  “你说啥，什么意思？”<br/>　　<br/>　　  “你们去看就知道了，就在县城中的广场上。”<br/>　　<br/>　　  闻言，人们都是好奇的跑了过去，说是广场，其实是一片空地，只是位于县城中心，人们习惯这样叫而已。<br/>　　<br/>　　  此刻，在这片空地上竟然又摆满了人头，一个个人头堆积成冢，其中石勇的人头赫然在最上边。<br/>　　<br/>　　  “这是？”看到这一幕，所有的人都是大惊失色。<br/>　　<br/>　　  “是山匪，青崖口的山匪！”<br/>　　<br/>　　  “那个不是青崖口的二当家石勇吗？”<br/>　　<br/>　　  “这么说来昨晚他们前去报复，是完全失败了！”<br/>　　<br/>　　  “狠啊，这足足有一百多人吧，全被砍了头！”<br/>　　<br/>　　  “新奉县的规矩第一次被人破了，看来这天是要变了啊！”<br/>　　<br/>　　  人们议论纷纷，有惊有恐，心中都是觉得不可思议，新奉县的山匪有多凶悍，大家都是清楚的。<br/>　　<br/>　　  可至从伯爵府的人到来，接二连三的重挫山匪，青崖口三大当家如今已经死了两个……<br/>　　<br/>　　  在此刻，谁也不敢小视这位伯爵府的少爷了。<br/>　　<br/>　　  外界有多大的震动，王康却是没功夫理会了，他此刻正钻在一间屋子里炼制着，火药！<br/>　　<br/>　　  火药就是他对付山匪的大杀器。<br/>　　<br/>　　  新奉县的山匪之所以难以清剿，就是因为找不到他们真正的老巢。<br/>　　<br/>　　  狡兔三窟，明知道就在这一片，但就是找不到，因而没办法彻底除掉。<br/>　　<br/>　　  所以王康就准备炼制火药，找不到你，我就直接炸山，看你们还如何存活。<br/>　　<br/>　　  而且在之后的新奉县建设中，也自然少不了火药，开山修路更是不可或缺。<br/>　　<br/>　　  难道还真以人工开挖吗？<br/>　　<br/>　　  先不说需要雇佣多少人，就算是有这么多人，等凿开了山又不知何年何月，他可等不了这么长时间。<br/>　　<br/>　　  所以，火药是必不可少的。<br/>　　<br/>　　  火药的炼制并不难，所需的主要材料也只有三种，硝石，硫磺和木炭。<br/>　　<br/>　　  这三种原料也并不难找，硫磺本质是一种矿石，而在新奉县最不缺的就是石头，硫磺石虽然不常见，但还是能够找到。<br/>　　<br/>　　  至于硝就更简单了，农家的土房子墙角表层一般会有一层白色物质，这就是硝石，<br/>　　<br/>　　  而且在粪池旁边会产生大量的硝土，也可以提炼出硝。<br/>　　<br/>　　  木炭自不用提了。<br/>　　<br/>　　  首先第一步就是要制取硫磺，王康在院中挖了一个土坑，又将硫磺石放至在一个素罐里加热融化。<br/>　　<br/>　　  这般之后，取出其上层硫磺溶液，倒入一个模型，待冷却之后，取出，行成黄色脆性晶体，这个就是硫磺。<br/>　　<br/>　　  三种原料齐全，就该配制火药了。<br/>　　<br/>　　  这个其实更简单，只要将硫磺，硝石，木炭按照一定的配比混合就可。<br/>　　<br/>　　  这样，黑火药就制成了。<br/>　　<br/>　　  王康用油纸紧密的包裹起来，加了引线，又放置在干燥的房间内。<br/>　　<br/>　　  “这就是你说的大杀器？”李清曼好奇的问道。<br/>　　<br/>　　  王康在炼制的时候，她一直在身边看着，还给打了下手。<br/>　　<br/>　　  “对！”王康点头。<br/>　　<br/>　　  “它真有你说的那么大的威力？”<br/>　　<br/>　　  “你很快会看到的。”<br/>　　<br/>　　  王康冷声道：“我要让青崖口彻底的灰飞烟灭！”<br/>　　<br/>　　  而后王康把周青和刘进叫来，他们两个的身手是伯爵府私兵里最好的，所以这一项任务也只能交给他们完成。<br/>　　<br/>　　  因为在火药爆炸的瞬间，会有很大的威力，若是跑不及时，很可能会被炸伤，甚至炸死。<br/>　　<br/>　　  王康指着两个油布包，对着二人镇重道：“这是一个大杀器，你们每人带一个，去青崖口！”<br/>　　<br/>　　  “青崖口你们已经都侦查过了，你们从两个方向，把油布包放置好，把上面的引线点燃。”<br/>　　<br/>　　  “然后，你们就赶紧跑！”<br/>　　<br/>　　  王康镇重道：“用最快的速度远离，越远越好，要保证爆炸引起的坍塌，不会威胁到你们，明白吗？”<br/>　　<br/>　　  “明白！”见得王康如此镇重，两人都是重重的点头。<br/>　　<br/>　　  “去吧！我等着你们的好消息！”<br/>　　<br/>　　  而后两人便带着火药包离开了。<br/>　　<br/>　　  就在此时，青崖口老巢里，石坚正在发着疯！<br/>　　<br/>　　  “砰！砰！”<br/>　　<br/>　　  所有的石桌，石椅被他砸了稀碎，所有的山匪都是大气不敢喘。<br/>　　<br/>　　  “王康，王康！我跟你不共戴天！”<br/>　　<br/>　　  他真的是要疯掉了，原本去找王康报仇，结果仇没报了，把二弟都是搭了进去，一百多号人又被砍了头！<br/>　　<br/>　　  青崖口山匪至从成立也没遭受过这么大的折损，尤其是两个兄弟的死亡，更是让他心痛如绞。<br/>　　<br/>　　  看着暴怒的石坚，徐允摇了摇头，他已经劝说过多次，可这两人就是不听。<br/>　　<br/>　　  现在好了吧，又搭进去一个，他现在对青崖口山匪已经不报希望了，死了这么多人，实力大减，还怎么对抗王康？<br/>　　<br/>　　  看来得联系其他两股山匪了，徐允暗暗想到。<br/>　　<br/>　　  “不行，我要去黑风寨！”这时石坚也冷静下来，此刻他也明白，现在的他根本就没实力复仇了。<br/>　　<br/>　　  就剩下这点人，给王康塞牙缝都不够。<br/>　　<br/>　　  “石大当家的，我觉得你现在该担心的是，那个王康会不会主动来清剿你。”徐允开口道。<br/>　　<br/>　　  “他敢？他敢来我就让他有来无回！”听了这话石坚又是跳起。<br/>　　<br/>　　  “他连我的老巢都寻不到何来清剿，我现在就去黑风寨，联合绿林众，我们三大山匪共同出击，我看那个王康如何能够抗住？”<br/>　　<br/>　　  “好，既然石大当家的有此决心，徐某愿当说客，我们共同……”<br/>　　<br/>　　  “轰！”<br/>　　<br/>　　  徐允的话还没说完，一声巨响蓦然响起。<br/>　　<br/>　　  “轰！”<br/>　　<br/>　　  紧接着又是一声，<br/>　　<br/>　　  地面震颤，整个山洞剧烈的摇晃起来，无数的条石从洞顶落下！<br/>　　<br/>　　  整个山洞在快速的坍塌……<br/>　　<br/>　　  【作者题外话】：第二更，投投票吧，感谢<br/><br/><br/>
