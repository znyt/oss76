　　“是是！多谢叶前辈不杀之恩！”
<br/>　　金老听到叶尘这话，哪里敢说半个不字，而且相比于魂飞魄散、连尸身都没有留下的冥沧山，他的这个处罚，已经是很轻了！
<br/>　　“咔嚓！”
<br/>　　金老右手作掌，以真元化刀，狠狠斩落在自己的左手手臂之上，那条手臂顿时便应声而断！
<br/>　　金老咬着牙，一手抓住自己的断臂，冲着叶尘又躬身行了一礼之后，转身拔腿就走，一转眼便消失不见了。
<br/>　　……
<br/>　　替冥沧荨彻底解决了麻烦之后，叶尘并没有在玄冥城多待，只停留了半日，便动用逍遥仙珠离开了南斗星域，准备继续朝迷罗星海的方向进发了。
<br/>　　原本，冥沧荨提出陪叶尘一同前往的，不过却被叶尘拒绝了。
<br/>　　这丫头虽然是难得的天才人物，但是修为还太浅。
<br/>　　迷罗星海乃是修真界十大绝地之一，即便是前世的叶尘，都不敢保证能够绝对全身而退，更何况如今的他，还远远没有达到前世的修为。
<br/>　　连他到时候都得如履薄冰，怎么能再带一个拖油瓶？
<br/>　　不过，
<br/>　　在离开之前，他在冥家特地设下一个小型的秘密传送法阵，并用逍遥仙珠做了标记，将来可以随时回到这里。
<br/>　　而就在叶尘，离开南斗星域的同时，
<br/>　　远在数十个星域外，八大仙门之一的擎苍宗之中。
<br/>　　一处奢华大气的宫殿之中，
<br/>　　“轰！”
<br/>　　一名身穿金色法袍、神态霸气十足的青年男子，忽然拍案而起，脸上浮现出一抹怒色。
<br/>　　那桌子，竟是用上等玄铁铁打造，硬度丝毫不弱于圣器，可是在那青年男子的随手一掌之下，竟然出现了一道清晰可见的掌印！
<br/>　　甚至，
<br/>　　那青年男子这一掌拍落之后，连同整个大殿，都跟着一阵剧烈晃动。
<br/>　　这也足以看出，这青年男子的掌力，有多么的恐怖！
<br/>　　原来，这青年男子，便是被称为，擎苍宗千年以来的第一天才，绝无敌！
<br/>　　而此时在绝无敌面前，正跪着一个人，赫然正是当初那个，被叶尘赶跑的韩月公子。
<br/>　　“那小子当真这么说？他要在迷罗星海等着我？问我敢不敢去？”
<br/>　　绝无敌的声音，就犹如他的人一般，仿佛金属的撞击声，透着一股难以形容的霸气，和凌厉无比的煞气，让人不自觉的，就想要臣服。
<br/>　　韩月公子此时，早已经没有了昔日的傲慢与高冷，在绝无神的面前，犹如一只温顺无比的小猫咪，可怜巴巴的点了点头，道：
<br/>　　“绝师兄！那小子还说了，您要是不敢去找他，等他从迷罗星海出来，就会亲自踏上咱们擎苍宗！”
<br/>　　“混账！！”
<br/>　　绝无敌听到这话，顿时怒气冲天，再次一掌拍落在那玄铁打造的桌子上。
<br/>　　轰隆！
<br/>　　那堪比圣器的玄铁桌子，愣是被绝无敌，一掌生生轰成了稀巴烂。
<br/>　　“这个叫叶尘的小子，到底是从哪里冒出来的？竟然敢如此挑衅于我？是谁给了他这么大的胆子？！！”
<br/>　　绝无敌怒气冲天，扬天咆哮了一阵之后，猛然一把抓住脚下的韩月公子，两眼一瞪，寒声道：
<br/>　　“韩月！你小子该不是为了让我给你报仇，故意编出这么一个故事诓我吧？”
<br/>　　也难怪绝无敌会怎么想！
<br/>　　要知道，绝无敌可是整个修真界，千年都不世出的修炼天才。
<br/>　　在年轻一代之中，能够和他并驾齐驱，够资格向他叫板的天骄，绝对不会超过十个。
<br/>　　而这十个人之中，绝无敌可以肯定，绝对没有这个叫叶尘的。
<br/>　　甚至，
<br/>　　即便是排名前五十的天骄人物，绝无敌也都知道的一清二楚，同样没有这号人物。
<br/>　　难道是恶作剧？
<br/>　　又或者，这个名字，只是他的化名？
<br/>　　绝无敌百思不得其解，
<br/>　　韩月公子听到这话，顿时吓得浑身一哆嗦，他刚刚的确有些夸大其词，但大体上也没有胡编乱造，
<br/>　　“绝师兄，我就是吃了熊心豹子胆，也不敢诓您啊！那小子真是这么说的，您要是不信的话，可以把祁老叫过来对质，问问他，我到底有没有骗您！”
<br/>　　绝无敌见韩月连这种话都说出来了，知道他应该没有胆子诓自己，眉头顿时一皱，继续又问道：
<br/>　　“那小子长什么模样？你且说说看！”
<br/>　　“他……”
<br/>　　韩月公子正要开口，将叶尘的模样，向绝无敌描述出来，整个人却忽然微微一滞，继而全身开始微微颤抖起来，神情也开始变得扭曲，仿佛有什么东西，要从他的体内窜出来似得！
<br/>　　“嗯？”
<br/>　　绝无敌猛然察觉到情况不对，当即毫不犹豫，一脚将韩月公子踢飞了出去。
<br/>　　轰！
<br/>　　就在韩月公子飞出的那一瞬间，整个人竟然犹如人形炸弹一般，忽然爆炸开来，化成了一团赤红色的火焰！
<br/>　　“啊！！”
<br/>　　随着韩月公子一声凄厉的惨叫，几乎一转眼的功夫，他整个人就被那诡异的火焰，彻底烧成了一团灰烬，死的不能再死了。
<br/>　　而就在韩月公子被焚烧殆尽之后，那团火焰却并没有立即消散，居然在空中幻化成了一行字：
<br/>　　“绝无敌，我必杀你！”
<br/>　　绝无敌先是微微一愣，继而两眼通红无比，在强大的怒气之下，连头发都根根倒立起来，
<br/>　　轰！
<br/>　　绝无敌一掌拍出，便将那团火焰组成的大字，拍散于无形，
<br/>　　“叶尘！叶尘！我不管你到底是何方神圣，你死定！死定了！！啊！！”
<br/>　　咆哮之声自大殿之中发出，整个大殿都跟着簌簌颤抖，似乎随时都要倒塌一般。
<br/>　　甚至整个擎苍宗，所有弟子都听到了这道可怕的声音，顿时个个一脸骇然之色。
<br/>　　“这……好像是绝无敌大师兄的声音！太可怕了！”
<br/>　　“谁这么大胆，竟然将大师兄气得如此程度？”
<br/>　　“不管这人是谁，只怕都活不久了！”
<br/>　　……
<br/>　　而就在韩月公子，被五昧真火焚烧成灰烬的同时，此时还正在空间通道中赶路的叶尘，也顿时有所察觉，
<br/>　　“看来绝无敌，应该已经看到我给他留的字了吧？以那家伙的性格，肯定会不惜一切代价追过来，到时候，迷罗星海便是你的葬身之地！”
