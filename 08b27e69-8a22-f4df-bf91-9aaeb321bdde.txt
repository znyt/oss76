　　崔钰并不知道秦琼与自己的顶头上司谈了一些什么，但是从其表现来看，显然是被说服了。
<br/>　　这位大唐翼国公跟着李世民南征北战，每战必然争先，对大唐的忠诚可以说是毋庸置疑，能把他说服的话，显然并不是一件容易的事情。
<br/>　　不过老秦似乎并不想提起这件事情，对崔钰一直是讳深莫测，顾左右而言他，老崔对此也没有什么好办法，最后只能不了了之。
<br/>　　而在大唐的南海，李世民已经苏醒，不过因为过于悲痛秦琼的死，所以老头子的精神状态一直不怎么好。
<br/>　　李佑这几天一直陪在老头子身边，听着老头子坐在那里给自己讲古，讲当初秦琼如何如何，这些事情虽然李小五早就已经听的耳朵都快要磨出茧子，但考虑到老头子的心情，还是耐着性子听了下去。
<br/>　　老尉迟似乎也对秦琼的死无法接受，把自己关在帐篷里面好几天，等出来的时候人似乎精神了不少，但却有些神叨。
<br/>　　长孙无忌一直在处理大食俘虏，数万人的战俘根本不是他们能带走的，而放任自留也不是个办法，所以最终这些战俘全部都被炮决掉了。
<br/>　　大概十天之后，李世民的精神终于算是彻底恢复了回来，至少表面上看是这样。
<br/>　　老头子在恢复后的第一件事就是上船，两支舰队合二为一，一路向北而去，在靠近大唐的近海完成补给之后，由神叨了的尉迟恭和尉迟宝琪带着四艘战舰保护长孙皇后归国，另外二十艘钢铁战舰在李世民的督促下直接转道马六甲，越过海峡直扑大食。
<br/>　　在李世民的字典里面，此时已经没有了忍一时风平浪静，退一步海阔天空。
<br/>　　在从李佑的口中得知李承乾已经派了程咬金与候君集去了西域之后，李二更是没了顾忌，直催着李佑加快速度。
<br/>　　曾几何时，李世民还是皇帝，那个时候不管干什么都会有人弹劾、谏言，那时候的老李特别羡慕李承乾，因为那小子可以不顾一切的折腾，反正到了最后有自己这个当老子的给他擦屁股。
<br/>　　现如今，老李已经不是皇帝，他终于可以现实当年自己的梦想：想干啥就干啥！
<br/>　　至于说事后惹出来麻烦，大不了由那个小混蛋去擦屁股好了，这叫六月的帐还的快，也让那个小混蛋知道知道当年自己这个当爹的有多么痛苦。
<br/>　　时间就在无聊的航行中一点点度过，不过好在天竺那地方此时已经是大唐的地盘，老李的舰队倒也不缺补给，每日的伙食也不再那么单调，可以算是无聊中唯一的宽慰之处。
<br/>　　半个多月之后，老头子的舰队越过天竺与食的边境，出现在大食海域，庞大的钢铁战船，高高飘扬的大唐军旗，就好像一条巨大的鲨鱼进入了沙丁鱼群。
<br/>　　无数的大食渔船拼了命的想要逃走，但最终都被钢铁战舰碾压成了齑粉，无数的大食渔民祈祷着本国的舰队能够来救他们，但他们并不知道，自己国家的舰队已经在南海之上全军覆没，再也不可能回来救他们。
<br/>　　而随着老头子的战舰逐步深入，大食国的朝堂开始陷入慌乱。
<br/>　　东面，与大唐天竺道的边境之上，大唐有数十万精锐在枕戈待旦；西面，茫茫大海之上有二十艘大唐的钢铁战船在耀武扬威；在北面还有一支大唐军团横亘在大食与拜占庭之间。
<br/>　　怎么办？如何应对？无数的大食人开始后悔当初的决定，为什么好死不死的鬼迷心窍要去招惹身边的这个邻居，早知道这个邻居如此好战，就应该早早投降才是。
<br/>　　……
<br/>　　程咬金早在数日之前已经得到了来自长安的消息，秦琼的死讯几乎让老程崩溃。
<br/>　　大唐朝野上下，与老程交好的人数并不多，除了牛进达这个当初一起劫皇杠的兄弟之外，就是秦琼这个小时候一起长大的死党。
<br/>　　现在儿时的死党竟然先一步走了，这让老程如何受得了这样的打击，若不是有两个儿子在他身边，老程几乎当场被一口瘀血憋死过去。
<br/>　　这一日，老程终于从悲痛中恢复过来，立刻下达了整军的命令，所有随他出征的将领全部集中到了他的帅帐之中。
<br/>　　“诸将，想必你们已经都知道了大食人在南海干了什么吧？”老程的脸上没有了以往嘻嘻哈哈，玩世不恭的表情，取而代之的是嗜血的狰狞。
<br/>　　“末将知晓，一切听从大将军安排！”帐中众将也都是憋了一肚子的火，早就等着老程下达最后总攻的命令。
<br/>　　“好，既然如此，各部今日停止操练，回去休整一日，待明日一早，随本将出征，不将大食踏平，吾等誓不回师！”
<br/>　　“是！末将等遵令！”
<br/>　　没有战前动员，也没有精彩的演说，老程办事就是这么直接，就是这么不讲道理，作为他的属下，你可以有自己的见解，但是却没有必要说出来，因为老程根本不会听，尤其是在眼下这个时候。
<br/>　　不过，这一次跟着老程出来的都是当初的老将，哪个跟秦琼没有一点关系，知道老秦战死的消息之后，个个气的义愤填膺，恨不能马上就带兵杀进大食。
<br/>　　便是这样，驻扎在天竺道与大食边境的数十万大军动了起来，普通士兵每人六十发子弹，五个铁菠萝一夜之间全部配发到位；发射距离达到万余步的贞观炮被掀起炮衣；火箭弹部队全部被打散分配到各战斗小组。
<br/>　　几乎是一夜之间，原本平静无波的唐军大营变的杀气腾腾。
<br/>　　次日一早，随着阵阵号角之声，数十万大军依次出营，兵分几路杀进了大食境内。
<br/>　　驻守在边境的大食人几乎都疯了，他们完全没有想过大唐竟然在没有任何预兆的情况下突然对他们发起攻击，等到他们做好准备，列队想要出营的时候，无数的炮弹已经如火流星般砸进了人群。
<br/>　　几乎是瞬间，大食人的军营便被大火覆盖，数不清的士兵因为慌不择路被大火围困，最后被烧死，能逃出来的只有不足一半。
