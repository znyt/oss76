　　“这是什么？”
<br/>　　“一件坐骑灵器，居然可以抵挡住夜牧的刀锋？”
<br/>　　北邙宫之中的无数老祖，看到这夜牧出手打出的刀锋，居然是在战龙宝车的面前，如此轻而易举的便是被破，一个个的脸色都是惊讶到了极点。
<br/>　　夜牧老祖，可已经不是那种刚刚超越生死的人物了，他的境界，已经是超越了碎命之境，达到了生死台阶的第二重，不死之境的人物。
<br/>　　他打出的刀锋，别说是让一名生死境武者来抵挡，就算是一位碎命之境的武者，想要抵挡，都需要付出极大的代价才行。
<br/>　　可现在居然是被战龙宝车，轻而易举的给抵挡住。
<br/>　　“天级灵器！这宝车也是一件天级灵器！”

<br/>　　一些老祖看向楚风眠的眼神，都是无比的嫉妒。
<br/>　　他们这些人，可都是北邙学院的老祖，超越生死的人物，就算是放在九域的任何一处，地位都是高高在上，备受尊崇的。
<br/>　　而楚风眠却不过是生死境的武者，居然就有着如此强大的灵器。
<br/>　　他们这些老祖，一般来说也就只能拥有一些寻常的天级灵器，像是祖龙至尊剑，战龙宝车这等品级的天级灵器，他们都没有资格得到。
<br/>　　可楚风眠仅仅是生死境的人物，北邙学院的一位弟子而已，居然不仅仅有着龙族至宝，祖龙至尊剑，如今更是还拥有这样一座战龙宝车。
<br/>　　这让那些老祖，如何不嫉妒，尤其是那些夜家老祖，现在都是恨不得赶紧将楚风眠杀了，夺取楚风眠身上的一切宝藏。
<br/>　　“夜牧，你说你还想杀了我？可笑，生死之境又如何？在我的面前，你不过就是一只蝼蚁罢了，给我灭！”
<br/>　　楚风眠看着夜牧老祖，突然大声笑了起来。
<br/>　　在他的座下，战龙宝车上的八尊远古战龙，突然咆哮一声，猛然之间冲着那夜牧老祖已经是撞了过去。
<br/>　　“轰隆！”
<br/>　　这速度之快，刹那之间，一道虹光，便是贯穿了夜牧老祖的胸口。
<br/>　　一条远古远古，咆哮一声，猛然一爪已经是轰击过去，这夜牧老祖的胸口，都被硬生生的打碎，被撞出去了数十公里之远。
<br/>　　一撞之下。
<br/>　　就连身为不死之境的夜牧，都是直接重伤。
<br/>　　那站在楚风眠身边的邓灵副院长，表情已经是目瞪口呆了，眼前发生的这一切种种，简直是全部超出了他的意料之外了。
<br/>　　不止是陆千刃，就连夜牧老祖，在楚风眠的手中，都是毫无还手之力，一个小小的弟子，都足以暴打超越生死的老祖。
<br/>　　若不是亲眼所见这一切的话，只怕是传出去，在九域之中流传，都没有任何人会相信。
<br/>　　“好恐怖的威力！这座宝车，到底是什么来历？”
<br/>　　“八尊远古战龙拉车，罗伞顶盖！这是九魔城的至宝，天魔宝车？”
<br/>　　一尊老祖，突然大声喝道。
<br/>　　似乎是突然之间，看出了这战龙宝车的真正身份。
<br/>　　在九域之中，这样一座宝车，绝不可能是默默无闻的存在，在回想一下这种种，突然之间大喊了起来。
<br/>　　“什么？是天魔宝车？”
<br/>　　“这不是九魔城的至宝，一直以来身为九魔城魔主的座驾吗？怎么会落入到了此子的手中！”
<br/>　　“据说这一次，九魔城的小魔主，也是进入到了归墟天之中，只怕是带这天魔宝车进入的，最后却是死在了此子的手中，被此子得到！”
<br/>　　“好运！此子的运气简直逆天，这等宝物，都被此子得到了！”
<br/>　　一尊尊老者，看着楚风眠，每个人的眼光都不一样，有感慨，有兴奋，有贪婪，有杀意。
<br/>　　“此子的实力，实在是太恐怖了，控制这宝车，就算是不死之境的人物都不是他的对手，这还是一尊生死境的武者？”
<br/>　　楚风眠在一招轰击夜牧老祖之后，便是安静的坐在战龙宝车之上。
<br/>　　在这一刻，那些本来叫嚣着要轰杀楚风眠的老祖，一个个都沉默了。
<br/>　　楚风眠若是仅仅只是生死境的实力，倒也好说，对他们这些老祖来说，生死境的武者，都是蝼蚁一般的人物，他们想要如何拿捏都可以。
<br/>　　可是现在，楚风眠展现出来的实力，已经是跟他们这些老祖，都没有任何的区别了，这种实力，足以是跟他们平起平坐。
<br/>　　就算是在想要对付楚风眠，也不是一件容易之事。
<br/>　　无数老祖看着楚风眠，一个个目光闪烁，都是在思考这什么。

<br/>　　楚风眠若是实力不济的话，他们也不介意卖给夜家一个人情，一起出面来对付楚风眠。

<br/>　　对他们来说，这也并非是一个明智的选择。
<br/>　　“楚风眠，放肆，你还不跪下！来到这北邙宫，在这无数老祖的面前，你居然还敢嚣张！”

<br/>　　“是吗？”
