　　他性格有点闷，比较耿直，平时话不多，只要开口，便一定是认定了要去做的。
<br/>　　叶少阳拍拍他的肩膀，表示安慰。
<br/>　　大家一起静静地望着这两座牌位，半晌，叶少阳道：“你们知道我现在最怕什么吗？”
<br/>　　不等他们开口，他接着说道：“我最怕的是……这里的牌位会越来越多。”
<br/>　　大家听出了他的意思，一时心悸，都沉默不语。
<br/>　　叶少阳叹了口气，喃喃说道：“从昨晚回来，我一直在想，我们为什么要帮阴司打这一仗，我们牺牲了秋莹，但是得到了什么？”
<br/>　　大伙都不开口。
<br/>　　老郭道：“这一战之后，捉鬼联盟的意义不同了，以前我们纯粹是为了自己而战，而今，我们当了阴司的英雄，不免也跟他们绑在一条船上了。”随即瞥了叶少阳一眼道：“我有个大胆的想法，咱们……有可能是被阴司给利用了。”
<br/>　　叶少阳点头，“军师之前跟我讨论过了，阴司找我们打这一仗，一来也是他们真的腾不出人手——本来还说岳家军会来接应，结果也没来，但最重要的是把我们拉入战争，我们跟太阴山结了仇，同时也等于宣布了与阴司的关系……”
<br/>　　吴嘉伟接过话头道：“这也没什么，我虽然不服阴司，但遇到这种事，一定也是要站在阴司这边的，否则万一太阴山统一了鬼域，我们难道要向他们臣服？”
<br/>　　叶少阳道：“是这个道理。”
<br/>　　天降大任于斯世人。
<br/>　　叶少阳突然想起了这句话。
<br/>　　中午，大家一起吃饭，吃完老郭回去，打造了一个更好的用来摆放牌位和贡品的柜子，傍晚时候搬过来，然后叫上叶少阳一行人去家里吃饭。
<br/>　　“为什么要去家里吃？”叶少阳不解。
<br/>　　老郭白了他一眼道：“你是不是连日子都不知道了，今天祭灶啊，我们这传统是吃面条，你们几个单身汉难道要凑在一起过节啊，走都去我家吧。”
<br/>　　祭灶就是小年，有些地方的传统是吃饺子，有些地方要吃面条。
<br/>　　四宝道：“我反对，我不是单身汉。不过看你们可怜，勉为其难陪你们一起吧。”
<br/>　　叶少阳道：“你就拉倒吧，你倒是想去人家家里过节，可人家谁认识你啊，更不要说你还是个和尚。”
<br/>　　四宝扫兴地摸了摸光头，“看透不说透，还能做朋友。”
<br/>　　到了老郭家，郭大嫂已经在忙着做菜了，看到他们进屋，也没太客气，招呼小鱼给他们倒茶。
<br/>　　小鱼从房间里出来，大方地招呼。
<br/>　　半年没见，小鱼长高了不少，出落地更加漂亮。
<br/>　　叶少阳看到她，猛然心悸，她长得……更像鱼玄机了。
<br/>　　不知道鱼玄机在修罗界怎么样了？
<br/>　　小鱼跟他们虽然很久没联系，但之前因为改命，跟他们成天在一起相处，彼此熟的不能再熟，相处并没障碍，只有四宝，一个劲问她有没有交男朋友之类的，挨了小鱼不少白眼。
<br/>　　说是小年吃面条，但也不能光吃面条，郭大嫂做了不少菜，一样样端上来，然后一大家人在一起喝酒，郭大嫂一直抱怨老郭天天不沾家，不管家里的事，老郭也不敢说话，只一个劲跟他们喝酒。
<br/>　　这顿饭，大家都吃得很尽兴，叶少阳看着他们一家三口其乐融融的样子，心中暗暗做了个决定，以后凡是拼命的行动，绝不叫上老郭。
<br/>　　吃完饭，郭大嫂切了水果跟他们吃，一边跟他们聊家常，吃完水果，叶少阳三人也就告辞了（瓜瓜不方便在郭大嫂面前现身，反正也不吃饭，压根就没来），老郭送他们下去打车，四个人都喝了不少酒，慢慢往小区门口走，也当是散步醒酒了。
<br/>　　叶少阳突然想起一个人，问吴嘉伟：“怎么好久都没你堂姐吴晓寻的消息了，她最近忙啥呢。”
<br/>　　“她……前段时间嫁人了。”
<br/>　　“嫁人？”叶少阳吓了一跳，“我怎么不知道啊！”
<br/>　　“那时候我们都被困在星月奴的幻境里，她没联系上，后来我出来才知道。”
<br/>　　叶少阳摇了摇头，“居然就嫁人了，也不说一声，改天一定要她请喝酒！对了新郎是什么人？”
<br/>　　吴嘉伟有点尴尬地说道：“是个写网络小说的。”
<br/>　　叶少阳跟四宝一起叫起来：“啥？”
<br/>　　网络小说叶少阳是看过的，有时候微信上也能看到推送的相关新闻，说是一天要写几千字，没节假日没保险，连双休都没有，新闻上说还有一些过劳猝死的，感觉比他们法师还辛苦些，让他们吃惊的是，这职业跟法术界风马牛不相及。
<br/>　　“就是……一个文人吧。所以她不打算请你们喝酒了，她结婚了，就淡出法术界了，最近好像鼓捣淘宝店什么的。”
<br/>　　淡出法术界……
<br/>　　叶少阳心中突然感慨起来，想当初，吴晓寻一个劲缠着自己，要带着她一起行动，要加入捉鬼联盟，眨眼之间却嫁给了一个网络作家，淡出了法术界。
<br/>　　她并不是第一个，这么多年来，他们都见过不少法师隐退的事，就说老郭，如果不是被他下山之后找到，硬拉进了捉鬼联盟，也就隐退了。
<br/>　　只是事情发生在身边人的身上，不免有些感慨。
<br/>　　“不管如何都是自己的选择，应该祝福。”叶少阳笑着看了看吴嘉伟，“告诉她，她的选择是好的，好好过日子吧。”
<br/>　　四宝伸手搂着两人的肩膀，笑道：“只有我们这些老人家，还坚持在第一线。”
<br/>　　走在前面的老郭回头白了他一眼，“还有我这被逼要经常给你们擦屁股的。”
<br/>　　三人相视大笑。
<br/>　　叶少阳借着酒劲，大声唱起一首歌：“来忘掉错队，来怀念过去，曾共同患难过的日子总有乐趣……”
<br/>　　他这个一向跟不上潮流的家伙，会唱的歌不多，十几岁的时候在网吧看《古惑仔》，觉得这首主题曲很酷，就学会了。
<br/>　　没想到四宝也会唱，搂着他们两个脖子接着唱起来，吴嘉伟不太记得歌词，有点羞涩地跟着他们唱，叶少阳把老郭也抓过来，四个人这样搂着脖子，在小区里没羞没臊地高歌起来，引来一片骂声，连保安都惊动了，灰溜溜逃走。
<br/>(本章完)
