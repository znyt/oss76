　　“一万功勋值？没问题。”
<br/>　　白萱微微惊异了一下，随后笑着开口道。
<br/>　　“等到虚空海中去，寻找一件宝物，都足以抵得上一万功勋值了，这点小事，不用在意。”
<br/>　　说着，白萱便是拿出身份牌来，给楚风眠划了一万功勋值。
<br/>　　一万功勋值，虽然数量不小，但是对白萱这等地榜前十的武者来说，还是可以拿得出来的。
<br/>　　“好，那多谢白萱师姐了。”
<br/>　　楚风眠接过功勋值开口道。
<br/>　　“恩。”
<br/>　　白萱点了点头道。
<br/>　　“这几日，你先准备一下，我去联系一下其他人，等过几日，我会他通过玉符来联系你的。”
<br/>　　说完，白萱的身形，便是直接飞走离开。
<br/>　　“这白萱的话，不知道可信多少，不过这虚空海，我倒是没去过，这一次去看看，倒也不错。”
<br/>　　楚风眠心中自言自语道。
<br/>　　他当然没有傻到，关于白萱的一切话都相信，不过楚风眠感觉的出，这其中的话，应该是真的七七八八。
<br/>　　虚空海中，也有着无数奇遇，许多强者，都会选择在虚空中决斗，其中输的一方的灵器财富，许多都会被虚空乱流卷入到虚空海中去。
<br/>　　这一次就算是那颜古龙帝的洞府是假的，去一趟虚空海，也肯定会有着不小的收获的。
<br/>　　“在此之前，可以先去圣山了，在前往虚空海之前突破神力境，我也会多几分把握，一旦我真正踏入神力境的话，击败那白萱，应该不成问题。”
<br/>　　楚风眠心中暗自想到。
<br/>　　白萱的实力，楚风眠的灵识探察，如今现在表现出的，至少也要达到生死境二重左右的地步。
<br/>　　这还是她有所隐瞒的，白萱真正的实力，至少要达到生死境三重。
<br/>　　跟现在的楚风眠相比，楚风眠对付她，虽然不落下风，但是却无法击败她。
<br/>　　但是若是楚风眠能够真正突破神力境的话，楚风眠有自信，击败这白萱。
<br/>　　倒时候就算前往虚空海的时候，遇到些什么意外，楚风眠也可以从容不迫的化解。
<br/>　　“走。”
<br/>　　楚风眠身形一动，直接便是飞行起来，向着无数山脉的最核心，那最大的一座圣山，飞了过去。
<br/>　　这支撑北邙学院修炼的圣泉，就位于圣山之下，所以在圣山上修炼，灵气最为浓郁，要超越一等山脉的数十倍。
<br/>　　至于比起这等六等山脉，七等山脉，只怕是要浓郁千倍万倍。
<br/>　　不过从圣山中修炼的代价，也是极为高昂的，第一次前去修炼，需要一万功勋值，而接下来之后，则是每次都需要五万功勋值才可以进入其中修炼。
<br/>　　每次修炼时间，不得超过七日，所以多数只有在即将突破的时候，才会积攒功勋值，进入其中修炼。
<br/>　　楚风眠一路破空而行，大摇大摆的走入到了圣山之中。
<br/>　　“哦？一个外门弟子，居然去了圣山？”
<br/>　　“就算是第一次进入圣山，只需要一万功勋值，也不是一个外门弟子可以拿得出来的吧。”
<br/>　　一道道目光，看着楚风眠进入圣山，纷纷冷笑道。
<br/>　　没有足够的功勋值，只怕是楚风眠，会直接被扔出去。
<br/>　　“这圣山，乃是禁地，没有事不许靠近，速速离去。”
<br/>　　楚风眠踏入到圣山之中，便是一尊看守圣山的长老，打量了楚风眠一眼修为，眼中露出几分不屑道。
<br/>　　“看你是这一届的新人吧，不懂规矩正常，这圣山不许随意靠近的，你还是速速离去吧。”
<br/>　　楚风眠看着这长老，直接开口道。
<br/>　　“我是来进入圣山修炼的。”
<br/>　　说着楚风眠，便是将身份牌拿出去，直接递给了那看守圣山的长老。
<br/>　　“什么？你是来修炼的？”
<br/>　　圣山长老听着楚风眠的话，有些惊讶，来到圣山修炼的弟子中，内门弟子都是极少的，只有种子弟子，甚至是核心弟子，才能经常来到圣山修炼。
<br/>　　一些内门弟子，积攒数年，也许才能攒够换取一次圣山修炼的机会。
<br/>　　还从未有外门弟子，前来圣山修炼的。
<br/>　　不是外门弟子没有资格，所有北邙学院的弟子，只要是缴纳足够的功勋值，都可以换取圣山修炼的资格。
<br/>　　外门弟子，也是一样。
<br/>　　只是一般外门弟子，一个任务做完，也就得到几十点功勋值，攒够一万点，简直是天方夜谭。
<br/>　　守山长老不由的提醒道。
<br/>　　“这圣山修炼的兑换，就算你的第一次来兑换，也需要一万功勋值，你可知道？”
<br/>　　楚风眠点了点头道。
<br/>　　“自然。”
<br/>　　守山长老半信半疑之间，接过楚风眠的身份牌，注入灵力，其中上面便是显现出了数字。
<br/>　　“居然真的有一万功勋值？你那里来的？”
<br/>　　看着上面浮现出的数字，守山长老吃惊的问道。
<br/>　　“这圣山修炼的资格，我应该可以兑换吧。”
<br/>　　楚风眠看向那守山长老，开口道。
<br/>　　“第一次进入圣山修炼，需要一万功勋值，没错吧。”
<br/>　　这关于圣山修炼的事，楚风眠还是从拓跋鼎哪里听来的，不过拓跋鼎自己，都没有进入过圣山修炼。
<br/>　　所以如今楚风眠也不清楚，这倒是对不对。
<br/>　　“不错，当然可以。”
<br/>　　守山长老倒是微微平复了下心中的惊讶，拿起了一个牌子，递给了楚风眠。
<br/>　　“这圣山修炼，从现在开始起，一共七日时间，你在七日之后，必须要离开，不然就会有人进入圣山中，抓你出来。”
<br/>　　“不要因为贪恋修炼，忘记时间，一旦是超过时间，也会受到宗门的触发。”
<br/>　　“其中的无数洞府，你自己挑选，不过有人修炼的洞府，不许去打扰，挑选一些没人的洞府便是。”
<br/>　　守山长老提醒了一下，便是打开了禁制，楚风眠接过那牌子，直接便是走了进去。
<br/>　　一入圣山，楚风眠便是感觉到，其中的灵气，简直是浓郁到了不像话的地步。
<br/>　　楚风眠随意吸一口的灵气，都要抵得上数十枚九鼎丹的药力，他大手一挥，足以是在楚风眠的手中，凝聚出了无数晶莹丹药来。
<br/>　　这每一枚丹药的效果，都要超越九鼎丹，七日时间，单单楚风眠要是从这里凝聚丹药，也可以凝聚出数亿，数十亿的丹药来。
<br/>　　不过这一次的修炼机会，可是不易，楚风眠当然不会浪费这种时间，他身形一动，直接向着圣山上走去。
