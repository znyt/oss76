
<br/>　　<br/>　　    王冬至跟车里的人交代完，就推开院门进了家中，这就是王惊蛰和她生活了二十多年的家。<br/>　　<br/>　　    赊刀人都是一脉单传，没有什么门派也没有山头，历代赊刀一脉就居住在了朝歌云梦山下的王村，因为云梦山上就是王氏一门的祖坟地，千年来所有赊刀人都被埋葬在此了。<br/>　　<br/>　　    王冬至回到家里后就简单的收拾了下，然后拿着几根香又从村里的小店中买了些酒，拎着就往村后的云梦山去了，王家的祖坟地从外表看其实很普通，坟头都随意的分散在了山坡上，只有中间一块立着个挺大的坟冢，坟前竖着块墓碑，落款是王诩。<br/>　　<br/>　　    王冬至拎着东西来到老坟地，穿过几座坟头之后到了中间的坟冢前，点上长香插在地上，然后“噗通”一声就跪下了，恭恭敬敬的磕了三个头。<br/>　　<br/>　　    “不好意思了老祖宗，过几天可能要打扰你们安息了······”<br/>　　<br/>　　    良水城中。<br/>　　<br/>　　    几天过去，王惊蛰的气色已经好了不少，伤势肯定还在，不过已经处于稳步上升中了，内伤比伤筋动骨一百天还要严重，可不是短时间就能恢复如初的。<br/>　　<br/>　　    其实王惊蛰很享受这种在小城里无忧无虑生活的日子，这些年他一直都在外游荡，很少在一座城市里呆过十天半个月的，如今一停下来后顿时感觉整个人都焕发了新生，因为身边有个挺会照顾人的女人。<br/>　　<br/>　　    王惊蛰跟茅小草又出来遛弯的时候，于是问了个压在心里很多天的疑问：“是什么信念支持着你跟我呆在了这个小城里，是爱心么，是助人为乐么，还是别的什么？”<br/>　　<br/>　　    “是我闲的”茅小草翻了翻白眼说道。<br/>　　<br/>　　    王惊蛰不死心的继续说道：“不是吧？你要是真闲的，完全可以自己一个人去游山玩水啊，这里也没什么可玩的，你还得每天照顾我的起居给我做饭，我觉得这不只是道德或者情义上的问题，而是·····”<br/>　　<br/>　　    “啪”茅小草打了个响指，截断了王惊蛰声情并茂，喷着唾沫星子的感慨，叹了口气说道：“你不要说的好像我已经和你要共度余生似的，我的余生很长，有没有你都不好说，你扮演着什么角色也没准，我现在只是因为我之前说过要和你仗剑走天涯这个承诺，才没有把你给扔下的，这么说OK不？”<br/>　　<br/>　　    王惊蛰沉闷了片刻，寻思着说道：“就没有点别的什么嘛？”<br/>　　<br/>　　    小草姑娘忽然探着身子，把脸伸了过去，跟王惊蛰几乎都要脸贴着脸了：“你该不会是要问我，对你有没有好感吧？”<br/>　　<br/>　　    王惊蛰弱弱的问道：“有没有这个可能？”<br/>　　<br/>　　    “哈哈······”茅小草夸张的捂着嘴笑了半天，说道：“我对你有没有好感我还不知道，但我就想问问你，你该不会是喜欢上我了吧？”<br/>　　<br/>　　    王惊蛰呐呐的张着嘴，吭哧了半天，一棒子也没放出个屁来，喜欢或者爱这种词对于一个很少接触过陌生女性的男人来讲，吐口还是很难的。<br/>　　<br/>　　    “唉，小傻鸟你过来，我跟你说句话”茅小草勾了勾手指。<br/>　　<br/>　　    王惊蛰：“？？？”<br/>　　<br/>　　    茅小草咬着嘴唇说道：“你如果喜欢我，那我建议你······半途而废不好，我建议你喜欢我到终老”<br/>　　<br/>　　    “为啥啊？”王惊蛰懵懂的说道：“你要是不喜欢我的话，凭啥让我喜欢你到终老啊”<br/>　　<br/>　　    茅小草无力的放下手，幽幽的在心里腹诽了一句：“这家伙好像是钢铁侠做的，真是个标准的钢铁直男啊！”<br/>　　<br/>　　    两人正探讨着人生哲理的时候，身后陈三岁家的院门打开，他照例又推着轮椅出来了，轮椅上依旧倒着那个女人的尸体，浑身上下都被包裹的还是那严实，没有一点皮肤露出来。<br/>　　<br/>　　    王惊蛰和茅小草让到一旁，皱着眉头看着一人一尸从自己身边经过，陈三岁仿佛没有看到他俩似的，眼神默然的就从他俩面前擦身而过，眼睛都没有往这边瞄一下。<br/>　　<br/>　　    王惊蛰轻声说道：“这才叫不是半途而废呢，人都死了也没放弃，真不知道他会坚持到什么时候，真打算一辈子就守着一具尸体过下去么？”<br/>　　<br/>　　    “如果那个女人泉下有灵，也该欣慰了，一个女人一辈子都有个这么守护她的男人，此生不枉然了······”<br/>　　<br/>　　    两人刚感慨了几句，眼睛还看着推着轮椅的陈三岁时，忽然之间从那具女尸耷拉在轮椅旁边的袖子里，一个黑色的虫子突然就爬了出来，速度极快的掉到椅子上然后落向了地面。<br/>　　<br/>　　    茅小草脸色“唰”的一下就变了，惊异的说道：“是尸蟞？”<br/>　　<br/>　　    尸蟞是一种专门生活在坟地里的虫子，基本是以动物或者人的内脏为食，但是如果尸蟞爬进活人的体内，会生生的把人的内脏给啃食干净了，只不过尸蟞已经多少年都没有在出现过了，据说只有在一些古老的墓葬中有过，现实社会里基本没人在看到过了。<br/>　　<br/>　　    王惊蛰随手就从身上掏出一张符纸掐在手指上，盯着那只尸蟞，这玩意不可能让它钻进人群里，不然就会出大麻烦了。<br/>　　<br/>　　    王惊蛰刚要出手，茅小草见机按了他一下摇了摇头示意别动，此时已经走出很远的陈三岁，仿佛脑后长了眼睛一样，他迈出去的右脚突然踩到了一块石子上，朝后略一抬脚，石子顿时速度极快的斜着飞了出去。<br/>　　<br/>　　    “噗！”一声闷响，那只已经爬到了旁边墙上的尸蟞，直接被陈三岁踢飞的石子给钉死在了墙上，随后他若无其事的推着轮椅继续往前走，就仿佛那一脚根本就不是他踢的一样。<br/>　　<br/>　　    王惊蛰和茅小草对视一眼，眼中全是莫名的惊诧，那尸蟞外面都是黑色的甲壳，寻常人就是拿个锤子都不一定能砸得死，没想到却被陈三岁一脚石子给钉死了。
