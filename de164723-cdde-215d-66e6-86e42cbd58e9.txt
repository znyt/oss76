　　漆黑的天地，几乎没有一丝光亮。沧桑的世界，笼罩着一片无穷的浓雾，寂静而空廖，仿佛亘古永存的苍茫。
<br/>　　这就是幽冥界。
<br/>　　幽冥界中，三生石旁，刚要离开的元始突然停住了前进的脚步。他抬头看向不远处的一片虚空。
<br/>　　“嗯？”
<br/>　　元始皱了皱眉，心念一动，大手向着不远处的一片虚空之中径直抓去，大手伸入虚空之中，等到出来，手中便多了一本书。书上面写道：
<br/>　　“生死簿！”
<br/>　　“生死簿，冥书？”元始低语。
<br/>　　生死簿，为极品先天灵宝，为冥书，有时亦称人书，乃天地人三书之一，记载这洪荒天地苍生的寿命，只要是大罗金仙以下，没有超脱命运长河者，其寿命都有记载。
<br/>　　“生死簿，死亡法则？一丝命运法则？是个好东西。”元始呢喃道。
<br/>　　“不过，生死簿都出现了，那么判官笔呢？”元始疑惑道。
<br/>　　之后，他再次用神痕道眼搜寻判官笔，不过，也许是机缘不够，或者自己与判官笔无缘，再或者它已经被人找到了，元始并没有发现它。
<br/>　　既然没有发现，那便算了。元始没有再次停留，而后直接转身，离开了这个正在孕育之中的幽冥界。
<br/>　　……
<br/>　　“咳咳！……咳！……”
<br/>　　身影闪过，一个黑衣少年倒飞出数里之外，跌了几个跟头之后，慢慢狼狈地爬了起来，只见这个少年相貌普通，脸色坚毅，此刻头发散乱，嘴角鲜血直流，身上出现多处残破，模样极为凄惨。
<br/>　　“师兄，你没事吧？”
<br/>　　这时，一个白衣少年飞了过来，扶住狼狈的男子，急声问道，模样极为焦急，小脸之上冷汗直冒。
<br/>　　“师兄，你怎么样？”
<br/>　　此时，又有两个女子飞了过来，对着男子关心地问道。
<br/>　　“咳咳，我……我没事，你……你们先走，不……用管我。”
<br/>　　男子对着身边的师弟师妹说道。
<br/>　　“师兄，我们怎么可能抛下你不管呢，今天，就算是死，我们也一定会留下来的。”
<br/>　　绿衣裙女子急声说道。
<br/>　　“是啊，师兄，今天就算是死在这里，我们也不会抛下你的。”另外两人也急声道。
<br/>　　就在此时，虚空中出来一阵嚣张的笑声：
<br/>　　“哈哈哈！你们都不用争了，今日就全部留下吧！哈哈哈！”
<br/>　　“糟了，你们赶紧走！”
<br/>　　黑衣男子对着三人急忙喊道。
<br/>　　“哈哈，急什么，都留下吧。”
<br/>　　只见从不远处虚空之上飞来一个长着金色狮子头，下身却是人形的粗狂妖兽。这妖兽身披金甲，手提一柄金色长刀，脚穿一双金色飞云靴，对着下面四人狂笑道。他全身气势逼人，一声大笑，周边吹起狂风，飞沙走石，树倒石崩。此妖一身修为不下于大罗中期。
<br/>　　“糟糕！”
<br/>　　四人变色，同时极为警惕地看着对方。
<br/>　　“哈哈，四个小辈，识相的赶紧将东西交了出来，你狮爷高兴了给你们的痛快，否则——，嘿嘿！”
<br/>　　狮子头妖兽看着下方的四人，心中一阵火热，他确定自己刚刚没有看错，他们一定是得到了一件了不得的宝贝，否则不会闹出那么大的动静。
<br/>　　那动静已经堪比一株顶级先天灵根出世了，虽然他只是匆匆一瞥，不知道那是什么东西，但他知道那东西一定了不起。一想到那东西马上就会属于自己了，他心中就升起一阵阵小激动。
<br/>　　“师妹，你赶紧带着小师弟与小师妹离开此处，我来拦住对方。”
<br/>　　黑衣少年对着一旁的绿衣少女急声说道。
<br/>　　“师兄，他虽然是大罗金仙，但我们也不差，我们是不会走的，今天就留下来帮助你了。”绿衣少女不想看着黑衣少年冒险，决定留下来。
<br/>　　“是啊，师兄，以前一直是我们给你惹麻烦，今天我们是不会后退的。”白衣少女道。
<br/>　　“我们四人不一定会败，师兄，我们留下来帮你。”白衣少年道。
<br/>　　“你——你们……”
<br/>　　少年看着眼前的师妹师弟，眼中闪过晶莹泪花，既有些生气又有些感动。但，最终他还是欣慰地笑了笑，道：
<br/>　　“好，我们今天就一起战斗吧！”
<br/>　　说完，少年手持青色长枪，全身气势大涨，遥指狮子头妖兽，一声气势已经到达太乙金仙顶峰了。
<br/>　　同时，绿衣少女取出一把黑色长琴，琴身上刻着各种四大凶兽的图文，琴弦七根，一根不同于一根，都是顶级神材，有的是神龙筋，有的是神兽皮，一眼看上去就知道此琴不凡。
<br/>　　女子盘腿坐下，将琴身放于眼前，伸手快速一抹琴弦，全身气势如仙如魔。
<br/>　　白衣青年只手一翻，手中便已出现了一柄银色细剑，剑身银亮，剑光绚烂，剑柄玉身，端是一柄无双好剑。
<br/>　　灵剑在手，少年气势变得凌厉异常，剑意璀璨，煌煌如日。
<br/>　　最后白衣少女取出一条白色绫条，挥舞起来，一步一物，绫条飞舞，如九天仙女临尘，绫条之上，散发出凌厉的刃气，挥动之间，刃光森寒，杀气逼人。
<br/>　　毫无疑问，此四人就是元始那出来历练的四大记名弟子青玄、玉玲珑、白衣与风晓月。
<br/>　　但此时，他们明显是遇上了麻烦。
<br/>　　“呦呦哟！几个小辈，还想翻天，哈哈哈！”
<br/>　　狮子头妖兽看着眼前四人的动作，明显是想要与自己动手的节奏，四个太乙就想与他一个大罗动手，是不是脑袋坏了。他摇头大笑。
<br/>　　“看刀！”
<br/>　　狮子妖兽右手持刀，刀锋一转，脸色一变，却是变得严肃起来，显然，他要认真了。在看了眼前四人手上的宝贝后，四剑宝贝都是先天灵宝，不可小觑，他心中便已认真起来。
<br/>　　他知道，这四人不是有什么奇遇就是有强大的背景。他更倾向于后者，奇遇？奇遇是那么好遇到的吗？更可况一遇就是四件。
<br/>　　狮子将刀光对准四人，全身妖气弥漫，滚滚而来，他单手劈刀，刀光凌空而下，威压逼人，砍向四人。
<br/>　　“一剑耀世！”
<br/>　　白衣手持灵剑凌空而起，遥指狮子，亮光闪烁，如同一个太阳，炙热而明亮，光亮瞬间照遍虚空，剑光快如闪电，让人看不见痕迹，但却又能清晰地感受到这其中的巨大威胁。
<br/>　　剑光与刀光相撞，虚空震荡，发生巨大爆炸，两者相互抵消，消散于虚空。
<br/>　　“二件飞仙！”
<br/>　　一剑不消，紧接着第二剑便已出现。
<br/>　　同时，一道道青色枪芒遮天蔽日般的向狮子妖兽轰来，枪芒如同一道道青色亮光，连成一片枪网，布满整个虚空，让人无处可躲。
<br/>　　狮子头妖兽终非一般人，他看见这遮天般的枪芒，一点也不着急，而是抬手挥刀，刀光凝练如实质，一道金色天幕出现在虚空，挡住的那些凶狠的枪芒。
<br/>　　此时，一道道虚幻的飞刀攻入他的脑海之中，直斩他的神识，让他防不胜防。却是玉玲珑的琴音发威了。
<br/>　　天龙八音，琴音凝物，专攻人的神识。
<br/>　　“不好！”
<br/>(本章完)
