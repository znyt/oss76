　　叶少阳这样安慰自己，心头仍然体会到了一种深深的孤独。这个时候，他多么希望捉鬼联盟的小伙伴们在身边，哪怕一个人也行，至少能跟自己讨论一下，而不是像现在这样，自己一个人独自茫然地前进。
<br/>　　北京城外，一座并不出名的寺庙里。
<br/>　　禅房原本该是最安静的地方，但是现在却一点也不安静。
<br/>　　一个和尚站在窗前，望着外面的天空。不断有炮火声从远处传来，不过个把小时，声音仿佛比之前更加近了一点。
<br/>　　一个身穿布衣、面色刚毅冷峻的年轻人匆匆穿过寺院，一把推开了禅房的门，看见那和尚还站在窗前，急忙说道：“宝爷，你怎么还不收拾东西，城里打仗了，估计很快就会打到我们这儿来！”
<br/>　　四宝一只手撑着窗台，百般无奈地说道：“打仗跟咱们有什么关系啊！打死了最好，说不定死了就能回到我们的世界了。”
<br/>　　那冷峻少年说道：“你想什么呢，我们要是死了，可能就永远都回不去了！”
<br/>　　四宝叹了口气，望着他说道：“我也就是发个牢骚，大伟，我们不走。我们就在北京城呆着，我们在这等少阳。”
<br/>　　被他称作“大伟”的，正是吴嘉伟。他们两个来到这个世界也有一周了，比叶少阳幸运的是，他们俩穿越的时候就在同一片树林里，很快就遇到了，当时也跟叶少阳一样，一脸懵逼，不知道发生了什么事，还以为是被东皇钟的余波卷到了青冥界的某个地方。
<br/>　　两人从树林里出来，找到一个小村子，两人好奇地进村，找人询问，结果回答是北京城外，两人当场就懵了。
<br/>　　然后四宝发现他们的装束不对，村子里房舍的建筑风格也很奇怪，总之处处都透着一种跟当代完全不同的气息。
<br/>　　最后两人总算发现了问题所在，找人打听了一下日期，才得知他们所在的是1922年，起初两人不信，后来去找了本黄历，这才确定自己的确是穿越了……跟叶少阳一样，两人起初的那种震惊和郁闷，无以复加，好在他们是两个人，互相可以商量。
<br/>　　在发现回不去之后，两人很快就缓过来，商量应该做什么，怎么做。
<br/>　　虽然他们不知道自己是怎么来到这个世界的，但是他们相信，一定是跟徐福有关，最重要的是，他们不认识徐福，想来想去，他们两个应该是被“误伤”——徐福针对的应该是叶少阳，他们八成是因为一起站在了东皇钟上，才被跟叶少阳一起吸了过来。
<br/>　　也就是说，叶少阳八成也在这个世界。两人自然就把目标定成了寻找叶少阳，但却不知道上哪去找，在周边寻访了两天，完全没有消息，两人开始怀疑，叶少**本就没跟他们穿越到一个地方……两人饿得不行，跟鬼似的到处游荡，找了这个寺庙住。
<br/>　　这寺庙没人，但是房间不错，被褥之类的居然也都可以用，两人在周围打听才知道这寺庙里的和尚，因为躲避战乱去了南方，两人就不客气地住下了，并且在地窖里找到了一些埋在里面的红薯。估计是不方便带走留下的。
<br/>　　两人这几天吃的都是红薯，各种吃饭都尝试了一遍，四宝虽然是和尚，现在一想到红薯就想吐。
<br/>　　吴嘉伟听了四宝的话，不解地说道：“留在北京城有什么用，少阳未必会来。”
<br/>　　四宝说道：“假如少阳跟我们一样在这个世界，那么留在北京，机会一定比在小城市多。”
<br/>　　吴嘉伟不解地说道：“这话怎么说？”
<br/>　　“你想过没有，我们在找少阳，少阳很可能也在找我们。”见吴嘉伟点头，四宝继续说道，“从人的潜意识来说，如果漫无目的找人的话，肯定是去大城市，因为大城市人多，方便打听。而且只有北京上海这几个大城市。他从不能随便去个我们不知道的小城市找我们吧？我们也是一样。”
<br/>　　吴嘉伟缓缓点头，说道：“这说起来也算是一种默契。可是，就算是少阳也来了北京城，我们又怎么找到他？”
<br/>　　“等战乱过去，我来想办法。”四宝摸着光秃秃的脑门说道。
<br/>　　吴嘉伟皱眉说道：“想什么办法，你在这个世界，不也是个陌生人么，难道贴寻人启事不成？”
<br/>　　四宝在他细皮嫩肉的脸上捏了一把说道：“你还真猜对了一半，不过有一种东西，比寻人启事作用更大，不过……我也得先好好筹划一下，先等打仗结束吧。”
<br/>　　“谁知道这要等到什么时候。”吴嘉伟道，“打仗又不是我们能决定的。”
<br/>　　四宝鄙夷地看了他一眼，“就说你这个人吧，除了修行还是修行，一点不关心历史，今年是1922年，这个时期发生了什么大事，你肯定不知道了。”
<br/>　　吴嘉伟震惊地看着他，说道：“难道你知道？”
<br/>　　“我当然知道，免费说给你听听，1922年四月，第一次直奉战争爆发，现在是东北王张大元帅和直系吴司令在打仗，到五月初，吴司令出奇兵，绕道卢沟桥……击败东北军，不过是五月五日，张大元帅退守东北，直系入主北京城……所以最多到五月五号，就太平了。当然这都是历时书上写的，万一写的不对，我也没办法了。”
<br/>　　吴嘉伟听了之后，惊为天人，傻傻地看着他，说道：“你居然知道这么详细的历史？”突然一拍脑门，说道：“我想起来了，你是盗墓贼！怪不得！”
<br/>　　盗墓贼要了解各个时代的墓葬，对于时代背景、朝代更替之类的，也会有所涉猎，肯定比普通人要懂历史的多。
<br/>　　四宝瞪了他一眼说道：“跟盗墓没啥关系，实话跟你说，我师父在出家之前，就是北洋时期某个军阀手下的一个军官，因为杀人太多，后来解放后为了避祸才出的家。他对民国历史了如指掌，平时没事就跟我说这些，他禅房里又有很多关于民国历史方面的书。
<br/>　　我小时候不爱看什么经书，然后你懂得。不是我吹，如果考民国历史，一百分的卷子我至少能考九十五分。”
