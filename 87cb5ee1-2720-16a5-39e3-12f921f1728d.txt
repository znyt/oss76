<br/>　　这杀戮剑术，楚风眠已经练就大成，如今正在向这大圆满冲刺，相传杀戮剑术修炼到极限，足以唤出远古杀神来，加持力量在身上。

<br/>　　现在的楚风眠，远远做不到这唤出真正的远古杀神来，不过他却是可以将远古杀神的一部分力量，凝聚出来，加持在他的身上。
<br/>　　在斩杀了秦皇大帝，吞噬了他的精血之后，楚风眠的杀戮剑术便是已经有了更新一重的领悟。
<br/>　　这远古杀神的力量，加持在楚风眠的身上，全部融入到了楚风眠的身躯之中去。

<br/>　　顿时一瞬间，他手中的剑动了。
<br/>　　楚风眠这一次的杀戮剑术，居然是有着远古杀神的力量加持，单单在力量上强大无数倍，一瞬间便是变的庞大无比。
<br/>　　同时在楚风眠的身躯之中，一千二百九十六滴远古战龙精血，也是同时爆发出来了其中的力量。
<br/>　　这两股力量，互相融合，居然是化为了一道巨大无比的血龙。

<br/>　　一尊尊圣者，看着楚风眠爆发出来的血气，都是惊讶的说不出话来。
<br/>　　这血气，实在是太过浓厚了，不知道得是斩杀了多少圣者，才会拥有。
<br/>　　死在楚风眠的手中，如今少说也要数十尊了，正因如此，楚风眠才有着如此雄厚的血气。

<br/>　　顿时那九妖古阵，仿佛都被撼动了一样，晃动了起来，一股股力量，开始变的虚弱无比。
<br/>　　“该死！”
<br/>　　夏祖圣者看着这一幕，大声怒喝一声，一锤胸口，无数精血便是直接喷了出来。
<br/>　　这些精血喷出，便是开始燃烧起来，熊熊燃烧精血的力量，全部加持在了夏祖圣者的身上。
<br/>　　“九妖合一！灭杀天地！”
<br/>　　九妖古阵上的力量，九道合一，同时加持在了夏祖圣者的身上，突然猛然一拳，对着楚风眠砸了过来。
<br/>　　他如今燃烧精血，也要彻底击败了楚风眠，这一拳打出，想要跟楚风眠的凝聚出来的血龙抗衡。
<br/>　　“噗！”
<br/>　　可是下一刻，却是极为凄惨的景象。
<br/>　　这夏祖圣者，在血龙的力量之下，直接是被镇飞了出去，一大口鲜血吐了出去，被这血龙的力量，一下子硬生生震的吐血。

<br/>　　这一幕，让那些心中本来还贪婪的向着，要夺取楚风眠身上奇遇的圣者，一个个都面如死灰。

<br/>　　楚风眠看了一眼那九妖古阵，手一抓，便是直接抓了过去，要将这九妖古阵，直接收取。

<br/>　　看着楚风眠的举动，另外那四尊巅峰圣者也忍不住了，一个个站了出来，无数武技，冲着楚风眠轰击而来。

<br/>　　“苍天霸主！”

<br/>　　秦莽老祖，王风老祖，大供奉谷焚，镇虚老祖四人，同时出手。
<br/>　　如今他们也再也不顾及自己的身份了，四尊赫赫有名的强者，居然是一齐联手，要对付楚风眠。
