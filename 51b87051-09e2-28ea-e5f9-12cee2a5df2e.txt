　　第349章
<br/>　　一柄横空出世的剑，拦住了薛问婷的杀机。薛问婷一怔，下意识道：“宋哥哥。”
<br/>　　宋游渠冲他点点头，目光落在程大雷身上。
<br/>　　不知不觉间，十几个人已经将骑牛背斧的程大雷围在当中。
<br/>　　“这位先生，这黑牛是你的么？”
<br/>　　程大雷本来就是山贼，所以无所谓做贼心虚。但光着身子，的确有些心虚。现在心里想的是，如何拖延时间，把衣服先穿上。
<br/>　　“这个很难看出来么，干嘛？”
<br/>　　宋游渠仰起头：“出个价吧，这头牛我要了。”
<br/>　　“呵，我怕你出不起。”程大雷已经把衣服拿起来。
<br/>　　“放心，这世上我买不起的东西不多。”
<br/>　　“嘿，你还不知道你面前的是谁吧，肯让你出价是给你脸了，如果在别的地方，宋公子看上的东西自有人送到府上。”钟武才道。
<br/>　　“呦，那倒要请教了？”程大雷已经穿上裤子，心中长长的松口气。
<br/>　　“呔，你与我仔细了。”钟武才大声道：“你面前是凉州牧宋伯康宋大人的公子……”
<br/>　　钟武才一一说来，每一个都来头不小，宋游渠是西北凉州牧宋伯康的公子，薛问婷乃是丞相的孙女，以相府的权势薛问才比一个郡主的身份不低。
<br/>　　至于其他人，大都也是凉州城里的纨绔子弟。
<br/>　　“武才！”宋游渠板起脸：“说这些做什么，仗势欺人么？”
<br/>　　他目光转向程大雷，道：“这位壮士莫怕，我们只是在此地郊游，相中这头黑牛，你若出个价来，价格定是会让你满意的。”
<br/>　　“如果我不愿卖呢？”
<br/>　　宋游渠一怔，脸色沉下来。周围人握着手中的兵器，看程大雷的目光像看一只待宰的羔羊。
<br/>　　“你若敬酒不吃吃罚酒，那我只能说你不走远了。”钟武才冷笑道。
<br/>　　程大雷此刻把皂匹靴穿好，轻轻叹了口气。
<br/>　　“好吧，我也没时间和你们扯淡了。你们想不想知道，我是那个？”
<br/>　　“不管你是谁，我们宋公子看上的东西，就是宋公子的。”
<br/>　　“宋哥哥，干脆杀了他，让他在我面前放肆。”薛问婷。
<br/>　　“好啦，好啦。”程大雷挥挥手：“恭喜你们，你们现在被绑架了，男人在左边，女人站右边，不男不女站中间。”
<br/>　　程大雷说罢，周围人哄堂大小。钟武才突然止住笑容：“小贼，你是吃了雄心豹子胆不成，只有一个人，就敢打劫我们？”
<br/>　　“动手。”宋游渠冷喝。
<br/>　　十几人一起动手，各样兵器同时刺向牛背上的程大雷。大家都是有身份的体面人，所用的武器自然以剑居多。剑也是颇有讲究，怎样的剑鞘、怎样剑颚，上面在镶嵌怎样宝石……也是大家经常在一起攀比的。
<br/>　　程大雷在牛背上长身跃起，手中的大袍子已经披在身上。薛问婷下意识抬起头，宛若看到一片黑云升空，笼罩众人头顶。
<br/>　　她还未反应过来，已被一只大手捏住手腕，微微用力，手中的剑易主。
<br/>　　然后是阿飞快剑第二式，爆梨花，连环快剑刺伤众人手腕，众人手中的兵器铛铛落地。
<br/>　　宋游渠捂着出血的虎口，目光看向程大雷。他一开始就隐隐觉得这人不简单，所以才不肯用强。但对方不简单到这种程度，还是令他感受到前所未有的震惊。
<br/>　　几人同时受伤，伤的还都是虎口，这需要怎样的速度和力度控制。
<br/>　　“你究竟是谁？”宋游渠咬着牙问。
<br/>　　程大雷一粒一粒系上扣子，问：“你们既然敢来琴川，来之前就没有打听过，在琴川有一个人是绝对不能惹的么？”
<br/>　　宋游渠一惊：“是你！”
<br/>　　程大雷微微点头：“是我。”
<br/>　　……
<br/>　　琴川关。
<br/>　　今日城门的守卫名叫崔二，他刚加入蛤蟆寨不久，因为有膀子力气，为人又机警，领了个什长的职位，负责把守城门。
<br/>　　当然，琴川关的防御不是很紧，来来往往的商队，借贷粮食的流民，也不可能每天都关着城门。
<br/>　　可这并不代表琴川关的防御是松弛的，十八家山寨连在一起，只要有敌来犯，烽火台燃起，那么琴川关瞬间就可以从和平状态进入战斗状态。
<br/>　　那时，这座城关就会在所有人面前亮出獠牙。
<br/>　　因为就算在如今，城内的练兵也从未停止。
<br/>　　远远就看到程大雷骑着黑牛过来，崔二立刻抖擞精神，挺胸抬头收腹。
<br/>　　“大当家，您回来了，呦，这么多人是干嘛的？”
<br/>　　崔二注意到，程大雷身后还跟着十几匹马，都是上好的高头大马。同时，还有十几个人被绳子串在一起，跌跌撞撞被程大雷牵着走。
<br/>　　“崔二呐，今天是你值班。”程大雷把绳子丢给他：“正好，把这些人都押到牢里，我待会儿去看他们。”
<br/>　　“大当家知道我？”崔二脸上泛起红光。
<br/>　　程大雷白了他一眼，我谁不知道，我有系统的好不好。
<br/>　　“这些人是谁呐？”
<br/>　　“他们……”程大雷顿了顿：“肉票。”
<br/>　　“都是大当家一个人绑来的？”崔二咂舌：“大当家果真有本事呐，一个人就绑了这么多人。”
<br/>　　宋游渠一行人惭愧得恨不得将头埋在土里。
<br/>　　不久之后，宋游渠一行人出现在琴川关的牢房。就算如今的琴川关已经欣欣向荣，但牢房也绝对谈不上干净。如天下所有牢房一样，这里阴暗，潮湿，令人恐怖。
<br/>　　树大有枯枝，现在琴川关多了，难免也有为非作歹之徒。当然，对于一个山贼窝子来讲，所有人都为非作歹。这里的犯人，指的是扰乱山寨秩序的人。
<br/>　　程大雷亲自主持成立了治安队，日夜在山寨巡逻，但凡有不法之徒，便被丢进牢里严惩。
<br/>　　在程大雷的铁腕手段下，琴川关的治安还是相当不错的，所以如今的牢房除了关着两个喝酒误事的兄弟外，就只有宋游渠一伙。
<br/>　　他们被绑在木桩上，这里应该算牢房的审讯室，周围烧着火盆，墙上挂着各种刑具。
<br/>　　“宋哥哥，他会杀了我们么？”薛问婷胆战心惊的问。
<br/>　　宋游渠没有回答，对于长安一地的王孙贵胄来说，无论如何，程大雷这个名字都是一个恐怖的存在。
