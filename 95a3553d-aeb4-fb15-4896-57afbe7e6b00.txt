　　第319章
<br/>　　程大雷一双眼睛，睁得从未像今日一般大，张开的大口，恨不得将林少羽吞下去。
<br/>　　良久，他才冷静下来，笑着挥挥手。
<br/>　　“你们快别扯淡了。”
<br/>　　“大当家，实不相瞒，这正义教就是以你为尊，如今信徒十万，只要您一声令下，他们便唯您马首是瞻。”林少羽。
<br/>　　“住口！”
<br/>　　程大雷猛地大喝，终于搬出杀千屠万的魄力，他一双虎目紧紧盯着林少羽，慢慢松弛下来。
<br/>　　“秦蛮，军师，你们先回避一下，有些话我想和少羽单独谈谈。”
<br/>　　秦蛮和徐神机还不知发生什么事，晕头晕脑的离开了，房间内只剩下程大雷和林少羽、林冲三人。
<br/>　　程大雷此刻安静下来，开口问道：“仔细说说，具体什么事？”
<br/>　　林少羽怔了怔，抬头道：“我父是被明帝杀死的，身为人子，杀父之仇不得不报。”
<br/>　　程大雷点点头：“我可以理解，继续？”
<br/>　　“但明帝身在大内，又有鱼龙卫保护，我纵然天下第一，也杀他不得。”
<br/>　　“然后，你想有更多的人？”
<br/>　　林少羽点点头：“连年战火，便是富庶如江南，也挡不住官府层层剥削。许多人都过得生不如死，只要你给他们一个希望，他们便愿意追随你。”
<br/>　　“这也可以理解。”程大雷再问：“然后呢？”
<br/>　　“然后便有了正义教，在扬州一地传教，大半年时间，已有十余万信徒。与朝廷打了几仗，我们都赢了。”
<br/>　　“你出息了。”程大雷微微点头，语气却听不出夸赞。
<br/>　　“多亏大当家教导得好……”
<br/>　　“我……”程大雷有些无语。
<br/>　　林少羽重重点头：“我跟在大当家身边时间最长，也学了很多东西。做起事情来，我也会试着多想想。”
<br/>　　程大雷无语的看着林少羽，时过境迁，自己不是初来时的自己，林少羽也不是当初的愣头青。现在高飞豹都是顶级山贼了，更何况林少羽。但程大雷从来没想过，自己竟成了林少羽的老师。自己是个好老师么？程大雷觉得不是，可在林少羽眼里，很显然是的。
<br/>　　能在半年时间内，拉起一支人马，虽然最后兵败，但足见林少羽的本事。
<br/>　　程大雷忽然一顿，这件事情不对，凭林少羽自己做不到。
<br/>　　“是不是有其他人帮你？”
<br/>　　林少羽避开程大雷的目光。
<br/>　　“你还想瞒我，你的本事我是知道的，就算你现在已了不得，但也没有了不得到这种程度？”
<br/>　　“确实还有一人……”林少羽道：“我碰到了郭烦人，大当家你还记得他么？”
<br/>　　程大雷当然记得，那个举着『看破人间』白幡，带着一名叫红尘了的小道士行走人间的烦人道长。仔细算起来，他对程大雷还有救命之恩，只是程大雷想不明白，郭烦人怎么会和林少羽碰在一起。
<br/>　　“我们逃到江南时，碰到正在传教的烦人道长，道长是有些神通的，没有他，的确成不了这么大事。”
<br/>　　程大雷点点头，对这郭烦人恨得牙痒痒，你自己作死，何必害我兄弟。
<br/>　　“我们和烦人道长一样，都愿意奉大当家为首，大当家在青牛山挡住十万戎族铁骑，护了多少百姓。只要您一声令下，定有万万人拜在您的旗下。大当家，这天下就是您的。”
<br/>　　程大雷越听眉头皱得越紧，他刚得到正义教的消息时，还说这正义教的神主必然是个了不得的人物。却是没想到，正义教的领袖竟然是自己。
<br/>　　那古怪的神明，极其相似的旗帜……程大雷本该想到些什么的。
<br/>　　只能说，上苍是个爱看玩笑的家伙。
<br/>　　他拍拍林少羽的肩膀：“别想那么多了，好好养伤，以后世上没什么正义教了，你依旧是我兄弟。”
<br/>　　“大当家，你不和我们做事么，虽然这次兵败，但大正义教的根基未失。江南，西北，哪怕是长安，都有我们的弟兄。我们这次来，就是请大当家出山，一呼百应，争一争这天下。”
<br/>　　程大雷盯着林少羽：“你觉得我像争天下的人么，你仔细看看我这张脸。”
<br/>　　林少羽和林冲同时点点头：“像。”
<br/>　　“呃……快别扯淡了。”程大雷挥挥手。
<br/>　　“大当家，您真以为，我做这么多事，只是为了报仇么？”
<br/>　　程大雷一怔，见林少羽一动不动的盯着自己。
<br/>　　“大当家，你不觉得这天下有些事情不对么？”
<br/>　　程大雷再次沉默。
<br/>　　“戎族屡屡犯边，帝国十战十败，为富者不忍，有权者横行，天下盗匪作孽，遍视帝国十三州一百单八城，多少人活在水深火热中。”林少羽问：“帝国该是这样的么？”
<br/>　　“大当家，你还记得您第一次见我时说得话么？”林少羽目光看向窗外，似乎想起那久远的记忆：“您告诉我说，您有一个理想，愿耕者有其田，愿人人安居乐业，读圣贤书，行仁义事，愿用一双手救万民于水火。”
<br/>　　“正是因为这个理想，我才愿意拉住大当家的手，也才一步步走到今日。”
<br/>　　说到最后，林少羽的语气已经有些责怪。
<br/>　　程大雷默然无语，这些话他当然记得，记得的原因是，他觉得自己这番话说得很漂亮，但不觉得这么做很漂亮。
<br/>　　真去这么做，很傻的好不好。程大雷没有兼济天下的忍心，也没有逐鹿中原的壮志。他不过是个小小的山贼，能快活过完此生也就足够。
<br/>　　时过境迁，很多事情已经改变。程大雷刚才这个想法是不对的，有些事的确改变了，但有些事永远不会改变。程大雷依旧是那个小富即安的逍遥山贼，林少羽是个想成为大侠的男人，到现在他也相信正义，善良，太平盛世之类的美好字眼。
<br/>　　可程大雷自己不相信啊，大哥，你清醒一下好不好，我当初是忽悠你的，那些话我一个字都不相信。
<br/>　　林少羽撑着身子，目光灼灼盯着程大雷，纵然身受重伤，他整个人也如一支火炬。
<br/>　　不燃烧成灰烬，他不会熄灭。
<br/>　　世上最无奈的一件事：有些话，说的人不信，听的人却信了。
<br/>(本章完)
