　　洪荒之中，大战持续了千年的时间，它还在继续，一点也没有停下来的意思。
<br/>　　太华山劫魔殿，魔气森寒，煞气惊人，劫气弥漫。阴森森一片，不见丝毫日光，这里就是一片魔域。
<br/>　　魔域之中，随着大战的持续，万劫魔尊的实力在一天天的增加。之前在巫族对战祖巫之时，靠的是他强大的肉身之力，那时他还没有修炼过《万劫魔典》，法力还是之前的法力。直到来到太华山后，他才散去法力，转修劫气。如今在这大劫之中，他的法力很快便恢复了一大半。这样，用不了多久，他便能再次恢复到巅峰，而且还会实力大增。
<br/>　　此时，万劫魔尊并没有在修炼，而是在炼制一枚黑色小旗。要说这小旗的材料，还真是不凡。炼制用的火焰，便是《万劫魔典》独有的劫魔焰，劫魔焰是炼制魔器最好的神焰，经过劫魔焰的加持，魔器更具魔性，还会留有劫魔焰的特性，威力更胜。炼制用的器材，是他这些年收集的顶尖器材，最后他甚至用上了一具混沌魔神尸体。
<br/>　　终于，经过九九八十一天时间的祭炼，他炼制出了一枚黑色小旗。
<br/>　　轰隆隆！
<br/>　　这黑色小旗刚一成形，便散发出无尽凶威，爆发出强烈的吞噬之力，吞噬着大殿之中的劫煞之气，鲸吞般的吞噬，无尽的劫气被吞噬，但万劫魔尊毫不理会。
<br/>　　不知过了多久，这枚小旗才停了下来，之后，它散发着黑色的邪恶气息，小旗之上，一张巨大的饕鬄鬼头，散发着魔性之力，另一面，则是一朵黑色的火焰，散发着灼热的光芒。
<br/>　　万劫魔尊注入法力，摇了摇黑色小旗，顿时，小旗变得巨大无比，一面黑色大旗出现在眼前，之后大旗一晃，无尽虚空之中无尽劫气被吞噬，大旗如同一个滔天凶兽，吞噬诸天无尽生灵。
<br/>　　天地之间的那些被杀死的妖兽魂魄，此时仿佛受到某种牵引一般，纷纷毫无意识的向着太华山飘来。之后，这些妖兽魂魄被大旗一口吞噬，不管什么，它都来之不拒。仔细感受，可以感受到，随着吞噬魂魄与劫气的增多，黑色大旗的威力在慢慢增大。
<br/>　　显然，这是一面邪恶的旗帜，它能够吞噬天地间一切负面能量与生灵的魂魄，而且它还是一枚可以进化的灵宝，它可以通过吞噬这些魂魄来进化。
<br/>　　劫魔炼魂旗！
<br/>　　这是万劫魔尊炼制的第一件灵宝，它能够通过吞噬生灵魂魄来进化，而劫气等负面能量也只是附带的。随着它吞噬的魂魄越多，它便会越强。
<br/>　　“哈哈哈！好！吞噬生灵魂魄，我喜欢！哈哈！血尊者！前来见本尊！”万劫魔尊大喜，之后传音唤手下前来。
<br/>　　“尊主！”过了不久，便见一个身穿血红袍子的人来到万劫魔尊面前，对着他单膝跪地道。
<br/>　　这人是万劫魔尊之前收下的一个顶尖大罗金仙高手，因为修炼血道魔功，功力非凡，被万劫魔尊收服，作为己用。
<br/>　　“这枚旗帜乃本尊所炼，名为‘劫魔炼魂旗’，它需要吞噬无尽魂魄，如今，外面正在大战，不用本尊提醒，想必你也知道该怎么做了吧？”万劫魔尊淡淡看了血尊者一样，魔性的双眼中充满震慑之力，让人不敢与他对视。
<br/>　　“属下知道，尊主！”之后，血尊者接过黑色小旗，带着几个手下，便朝着大战之地潜了过去。
<br/>　　小旗想要进化，无非就是吞噬魂魄，如今巫、妖两族大战，死伤无数，巫族没有元神，想要留下魂魄十分困难，就算留下了，也十分弱小，不用多久，便会消散。但妖族不同，妖族大神通者，他们的魂魄同样强大，就算陨落，留下魂魄，一时三刻也不会消散。
<br/>　　这枚劫魔炼魂旗如今正好派上用处，吞噬了那些魂魄后，想必劫魔炼魂旗便能进化了。
<br/>　　……
<br/>　　且不管血尊者如何带领几个手下在两族大战的战场上收割吞噬妖族高手魂魄，却说东方大地昆仑山，三清圣地，玉虚宫中，元始此刻闭关已经到了关键时刻。
<br/>　　只见他此刻气息缥缈若仙，如梦如幻，似虚似实，似真似假，若隐若现。一会儿如同一尊混沌魔神，气势亘古沧桑，神威如狱，一会儿又如同一尊飞仙，腾云驾雾，飞上九天。一会儿又如同一尊魔头，魔性入骨，魔威震天，屠尽苍生。一会儿又如同一位智者，智慧之光散发，饱读诗书，全身充满智慧的气息……
<br/>　　各种姿态，人生百态，此刻全部在他身上显现。
<br/>　　忽然，他全身一震，各种姿态，全部消失不见，他的全身只剩下了最初了姿态，不，应该是说，他们全部回归了最初。
<br/>　　元始一袭白衣，坐在悟道古茶树下，像是睡着了，面容平和，不被外物干扰，他的精气神都提升到了顶峰，三者合一。渐渐地，他感觉自己像是于这片天地融合在了一起，感觉到了水的温暖，大地的脉搏，草木的呼吸……
<br/>　　这是一种很玄妙的感觉，融入到天地之间，像是成了天地的一部分，他清晰地感受到，天地之间像是有一种特别的规律，万事万物都在沿着这种规律跳动。
<br/>　　元始心神放开，精神空灵，这是悟道的最好状态，这种状态称之为天人合一，人即是天，天即是人，在这种状态下修炼感悟，效果最好。
<br/>　　心如止水，身若空明，这一刻，他仿佛回到了过去，再一次经历了过去的种种，一直到了现如今，他还没停下，一直走下去，他又来到了未来。那一刻，他成圣了，他开创了阐教，收弟子。
<br/>　　后来，三清分家，封神大战，弟子的背叛，他怒而出手，联合西方两圣，欺负自己的兄弟。再后来，佛教东渡……
<br/>　　最后，纪元轮回，天地覆灭，重演混沌，他仿佛看到了盘古神尊的出现，更是看到了三千魔神的重演，他还是没有超脱，就要随着这世界一起回归混沌，归于寂灭。他非常不甘，想要召唤时间长河，回溯过去。
<br/>　　但，随着时间的延长，他的意识在时间长河之中慢慢变得模糊，就快要消散。或许不久，他便会消散在时间之中。
<br/>　　但是，在最后的时刻，他仿佛听到了有人在呼唤自己，一时之间，他的眼睛一下子睁开，光芒璀璨，照亮了整个宇宙。
<br/>　　……
