　　这名老者，身穿宽大的银灰色长袍，长袍之上，绣着一条条飞舞的神龙。
<br/>　　老者脸盘宽大，面容刚毅，面色不怒自威，最主要的是，留着长长的银灰色胡须。
<br/>　　“见过龙帝！”龙腾大步走进大殿，向主座之上的老者，躬身行礼。
<br/>　　“见过龙帝！”
<br/>　　“见过龙帝！”
<br/>　　苏莫和司空炎两人，亦是不敢怠慢，急忙向老者行礼，这位可是吞虚龙帝，龙族目前的掌控者，无论是实力还是地位，都不是人德宫主所能比拟的。
<br/>　　当然，虽为龙帝，但是，吞虚龙帝的修为，同样是高级神王，还没有达到至强者的地步。
<br/>　　“免礼！”吞虚龙帝面带威严，目光打量着苏莫和龙腾，暗暗点头。
<br/>　　以他的眼力，自然能看出苏莫和司空炎均是非常不凡，一个修炼五行道统，一个修炼火属性道统。
<br/>　　“炎空、屠霄，你们两人的天赋，本帝都已经知晓，本帝也是爱才之人，只要你们愿意效忠龙族，便可脱去奴仆的身份，成为龙族的外族族人！”吞虚龙帝没有拐弯抹角，直接言明了自己的意图。
<br/>　　“敢问龙帝，如何才算是效忠！”苏莫恭敬抱拳，沉声问道，若是效忠龙族，要将自己的命运都交给龙族掌控，那他是一百个不愿意。
<br/>　　当然，若是没有附加的条件，他自然愿意成为龙族外族族人。
<br/>　　“效忠龙族，就是完全听命于龙族，不得背叛，违者杀无赦！”吞虚龙帝言语淡漠，但是，自有一股冰冷萧杀之意，在大殿之中涌动。
<br/>　　苏莫和司空炎闻言，互相对视一眼，随即纷纷向吞虚龙帝躬身行礼。
<br/>　　“我们愿意！”苏莫和司空炎朗声说道，只是口头上的效忠，对于他们来说，自然是没有什么影响。
<br/>　　当然，外族族人，对于龙族而言，也只是底层而已，肯定不能被龙族高度重视。
<br/>　　“很好！”吞虚龙帝点了点头，随即看向龙腾，道：“龙腾，你尽快带他们去奴务府撤去他们的奴印，他们从今以后，便是我族的外族族人，待遇与本族同层次的族人相当！”
<br/>　　“是，我回去之后，立刻去办，并带他们办理入族事务！”龙腾恭敬的说道。
<br/>　　“嗯！你们去吧！”吞虚龙帝摆了摆手，并没有多谈的意思，虽然司空炎和苏莫，均是万古天骄，甚至是比一般的万古天骄更加的强大，但也不会让他太过在意。
<br/>　　毕竟，龙族的万古天骄不少，而且，司空炎和苏莫修为太低，还不足以他重视。
<br/>　　今日接见两人，已经是最大的看重了。
<br/>　　“是！”苏莫三人行礼，随即纷纷告退，离开了大殿。
<br/>　　走出吞虚龙府之后，龙腾面带微笑，道：“从今以后，你们不在是龙族的奴仆，可以自由的出入龙都城以及天龙神朝各地！”
<br/>　　“如此最好！”苏莫点了点头，这的确算是一件好事，正好他准备离开龙族呢。
<br/>　　这样的话，他就保留了龙族外族族人的身份，随时可以回来。
<br/>　　“总算摆脱了这种尴尬的身份！”司空炎笑了起来，奴仆的身份，让他感觉颇为丢人。
<br/>　　随后，三人去了奴务府，奴务府显然已经提前受到了消息，立刻将苏莫和司空炎的奴印彻底消除了。
<br/>　　随后，龙腾又带着苏莫和司空炎，去了龙族的族务府，为两人办理了加入龙族的手续，领取了身份牌和属于两人自己的府邸。
<br/>　　两人的府邸，比龙腾的府邸还要小，显得颇为寒碜，占地不足百丈，只有一座微型的宫殿，以及两座更小的偏殿。
<br/>　　以此，也可以看出，外族族人，在龙族之中，地位还是比较低下的。
<br/>　　不过，司空炎和苏莫两人，并未太过在意，这比做奴仆已经好的多了。
<br/>　　苏莫的府邸之中，大厅之中，苏莫和龙腾、司空炎，相对而坐。
<br/>　　“苏莫，还没有恭喜你，修为再次提升！”龙腾笑着说道，他自然早就发现了，苏莫修为达到了三品高级的地步。
<br/>　　“想必你们的修为，也差不多了！”苏莫沉声说道。
<br/>　　他的感知力敏锐，无论是龙腾还是司空炎，身上的气息，均是有隐隐的勃发之意，想必距离突破不远了。
<br/>　　“的确快了，我的修为，有把握在一个月之内提升！”司空炎笑着说道，
<br/>　　“那你要追上我了！”龙腾面带轻笑，他的修为虽然也快了，但是，一个月之内突破是远远做不到的。
<br/>　　随即，龙腾的面色，很快便郑重了下来，变得无比郑重。
<br/>　　“苏莫，我有一个想法，你看看是否可行？”龙腾沉声说道。
<br/>　　“什么想法？”苏莫疑惑的问道。
<br/>　　龙腾沉吟了片刻，凝声道：“这一段时间，我是在族中极力表现，天赋也得到了很多龙族高层的赞赏，但这对于你交给我的任务而言，还远远不够，我的想法，是由你亲自出手，助我一臂之力，一劳永逸。”
<br/>　　“亲自出手？如何出手？”苏莫惊疑不定的问道，司空炎亦是有些疑惑，目光紧紧的盯着龙腾。
<br/>　　“是这样的，龙族中有很多宝塔，其中一座名为祖龙塔，你若是能取得塔顶的祖龙精血，给予我融合，大事可期！”龙腾郑重的说道。
<br/>　　“祖龙精血？”苏莫闻言惊讶。
<br/>　　“不错，祖龙精血，洪荒祖龙留下的一滴精血！”
<br/>　　龙腾重重的点了点头，继续道：“据闻，当年洪荒祖龙留下这滴精血之时，曾经留下话语，龙族后辈之中谁能融合这滴精血，将来就是龙族之尊，他的继承者！”
<br/>　　“这么厉害？”司空炎目瞪口呆，融合了这滴精血，就能成为祖龙的继承者，成为龙族之尊，这还得了？
<br/>　　“具体这滴精血，有多厉害没有人知晓，因为从来没有人能得到。”
<br/>　　龙腾面带唏嘘之色，道：“这精血对于其它种族来说是宝物，对于龙族之人来说乃是圣物，龙族历经无数代，上上下下，所有天才之人，无不想要夺取祖龙精血，但是从未有人成功。”
<br/>　　“所以，你想让我取得这滴祖龙精血？”苏莫的面色，亦是郑重了下来，此事不是小事，他要仔细斟酌一番。
<br/>　　“嗯，以你的天赋，有希望！”龙腾点头道。
<br/>　　“我是一个外族之人，不是真正的龙族，若是真的取得了这滴精血，会不会闹出大乱子？”苏莫面色凝重的问道。
<br/>　　“精血在你身上，你绝对护不住，因为这是龙帝都眼馋的宝物，所以，你若是取得了祖龙精血，第一时间就要让我融合！”龙腾告诫道。
<br/>　　“你觉得此事可行？”苏莫沉声问道，若是他真的去做了，并且全力以赴，怕是身份都会暴露。
<br/>　　“可行！”
<br/>　　龙腾郑重的点了点头，道：“就算你的天赋和实力，震动了整个龙族，也不会立刻暴露你的身份，而一旦我融合了祖龙精血，必定一飞冲天，即便暂时无法掌控龙族，也会在龙族之中拥有无上的地位。”
<br/>　　此事，是他这段时间仔细斟酌的结果，唯有这个办法，才能让他短时间内，在龙族拥有崇高的地位，从而可以动用龙族的力量。
<br/>　　“既然你认为可行，那我便去试上一试！”苏莫深深的吸了口气，微微的点了点头。
<br/>　　祖龙精血，这还得了，那洪荒祖龙可是太古至强者之中，最巅峰的几人之一，其一滴精血定然极为不凡。
