　　一周后，两国之间的焦灼升级，国王的失踪让明成国更加的肆无忌惮。
<br/>　　同时在上堂河的对峙，却出乎意料的不了了之，原因很简单，明成国在发现山洞内的飞船损坏后，一怒之下就攻向了王城！
<br/>　　对此，整个王城的兵力都出动，在城墙外与明成国的军队打了起来。
<br/>　　一战打了三天三夜，没让明成国进入王城，但却死了不少天运国的士兵。
<br/>　　并且由于城门被堵，天运国的王城就这么被孤立，东西运不进去，人出不出来，彻底被围困。
<br/>　　与此同时，一则极其糟糕的消息，也传遍整个王城！
<br/>　　明成国出兵了，已有大批军队到达天运国边境！
<br/>　　接到消息的天运国各个将领及大臣，集体失眠焦灼万分，共同商议讨论了半天。
<br/>　　国王不在，他们只能自己先想办法应对！
<br/>　　国师安然无恙了几天，胆子也大了起来，在国师府把酒当歌，夜夜笙箫，根本对这些事不闻不问。
<br/>　　有人去请，他将人打了一顿。
<br/>　　大臣们群集往前国师府，则将人全部打了一顿。
<br/>　　嚣张跋扈，在王城名声极臭却无人敢惹。
<br/>　　王城的兵力光是抵挡明成国攻城的那批队伍了，其他城市的兵力自发出动，全体赶往边境，与边关的军队合二为一。
<br/>　　大战一触即发，但大将军却不在，他被困在了王城。
<br/>　　这一情况让很多边境的将领无奈绝望，没有一个统战，这仗该如何打？
<br/>　　“派出一名高手潜入王城吧，先将大将军救出来，这里需要他！”
<br/>　　“没错，统战不在，这仗我们必输。”
<br/>　　“王城到底什么情况，明成国只是一小支兵力潜入，比起边境的几万大军根本没得看，竟然被攻城？”
<br/>　　“这也太督查不力了！”
<br/>　　“而且王城的士兵不是集训过么？怎么会是被人围困的战力！”
<br/>　　“有人来报，王城在请求支援！来源是国师府！”
<br/>　　“支援他妈，国王不在王城，就几个大臣和一群脑子有坑的将军，还不如弃了！”
<br/>　　“就是，我们这里的兵力要是调走，就要被明成国攻陷国土！不能走！”
<br/>　　“我说这国师当了这么久有点作用么？个人战力这么高，但却从没见他为了国家出力。”
<br/>　　几人各抒己见，也愤愤不平。
<br/>　　对比王城那里的软弱，边境的这些人反而更血气方刚。
<br/>　　一天后，第一批明成国的军队，已经踏入了天运国的国土，他们装备精良，一举就杀上了天运国的第一道防线！
<br/>　　边境的几个将领一商量，没什么战术直接开打！
<br/>　　两方打的如火如荼，兵荒马乱，战火纷飞。
<br/>　　这时候的楚涵，则是终于带着国王一路奔走到了这里。
<br/>　　到达此处的国王很惊讶：“我来到了边境？为什么你带我来边境？”
<br/>　　楚涵理所当然的说道：“你没看到打仗了？”
<br/>　　国王不解：“打仗是将军领兵上战场，我的杀敌本领可能不行。”
<br/>　　楚涵不屑一笑：“自己的江山不自己打？”
<br/>　　一句话，就直接让国王镇住。
<br/>　　楚涵有些好奇：“话说你们家掌控了天运国几代君王之位了？”
<br/>　　国王想了想，回道：“从建国开始，至今已经十几代了。”
<br/>　　楚涵点了点头：“也该被灭了。”
<br/>　　国王大怒：“你说什么！”
<br/>　　楚涵一点不虚的继续：“当一国之王一点侵略性都没有的时候，这个国家就是在走向衰落。”
<br/>　　国王抿嘴，不出声。
<br/>　　楚涵这时候一脚踢在了她屁·股上：“愣着干什么，换衣服上去干啊！”
<br/>　　训练的这一周内，楚涵本性暴露，动不动就是一脚踹上去，国王也被踹习惯了，这也是狼牙初期时那些人竟然遭遇的事。
<br/>　　国王虽有怨念，但也知道楚涵句句都一针见血。
<br/>　　她一声不吭的换上衣服，经历了一周后大事小事都不在话下，更不在乎换衣服这种小事，此时她心中所想之宏大，已经容不下这不起眼的细节。
<br/>　　楚涵别开眼，忽的心中有点异样和愧疚。
<br/>　　好好的国王，都被他折腾成啥样了……
<br/>　　终于有机会梳洗干净换上王袍的国王，恢复了往日的容貌和端庄，只是这与生俱来的气质之中，还夹杂着后天被楚涵培养的凶狠。
<br/>　　楚涵看了眼很满意，就差威严了。
<br/>　　国王定了定神，而后抬起脚步，手握长剑就这么一身王袍的走上了战场！
<br/>　　楚涵跟在她的身后，手拎修罗战斧，一脸的傲世笑容。
<br/>　　两人一前一后，在这战场上尤其明显！
<br/>　　正在大战的双方人马统统一惊，由于是从侧面走入，左边的天运国等人眼珠子都快掉了下来，这不是他们国王吗，怎么跑这儿来了？
<br/>　　右边的明成国士兵更是一脸懵逼，天运国的国王就这么只带了一个人送上了门？
<br/>　　“抓国王！”
<br/>　　一声大喝响起，下一秒全体的明成国士兵就这么冲了上来，呼啦啦像是成片的乌鸦。
<br/>　　相当于新人类五阶的国王本身体能就还可以，只是战力不行，但在这一周中被楚涵一对一的强化训练中。
<br/>　　将国王的全是潜能都激发了出来，生死边缘走了不知道多少遭，不仅将五阶的战力训练了个淋漓尽致。
<br/>　　国王还在训练中突破了六阶！
<br/>　　于是这时候，大批的普通士兵冲过来，国王抬手便是犀利一剑。
<br/>　　这剑也不简单，本来是分配给狼牙精良成员所剩下的一把，出自阴阳谷。
<br/>　　于是一剑劈开，当场就有十几名士兵中剑身亡！
<br/>　　甚至还有一两个，被她拦腰斩断！
<br/>　　血流了一地，凶残的一幕刺激着无数人的眼球。
<br/>　　国王自己似乎也惊讶了一下，没想到自己竟然这么强，紧接着，她就好似被楚涵激发了某个不正常的特性，竟然单枪匹马的扬剑杀了过去！
<br/>　　她憋屈了太久，一直隐忍，一直弱小。
<br/>　　今天终于有一天能够亲手杀敌，国王觉得很爽！
<br/>　　楚涵在身后都没料到这一发展，他呆了两秒后就追了上去，一路杀到人群中间，紧跟着国王的步伐以防万一。
<br/>　　让她杀只是培养气势，树立君威。
<br/>　　没想到这女人竟然还杀上瘾了……
