　　叶尘展露出实力之后，云天青等人已经见识过，并不觉得太惊奇，而其他四大宗门高手，则全都纷纷露出一脸凝重之色。
<br/>　　“不用再浪费时间了，你们一起上吧！”
<br/>　　叶尘一剑斩了夏侯长老之后，狂笑一声，抡起手中紫琼剑，便直接朝云天青等人冲了过去。
<br/>　　“来得好！布阵！”
<br/>　　云天青大喝一声，连同四大长老同时出手，再次施展出了五毒大阵。
<br/>　　嗖嗖嗖！
<br/>　　五具毒兽傀儡，瞬间将叶尘包围在了其中。
<br/>　　轰轰轰！
<br/>　　这五毒大阵的确不凡，五人联手之下，再次将叶尘困于大阵之中。
<br/>　　叶尘虽然游刃有余，可是这傀儡毒兽，防御实在是高，他若是不动用底牌，一时半刻根本无法摆脱这五具傀儡的纠缠。
<br/>　　看到这一幕，周围的众人纷纷暗暗摇头，
<br/>　　叶尘的表现，的确出乎他们意料之外，
<br/>　　单单是对抗药王宗五大顶级高手，而完全不落下风，就已经是天榜前十的实力了。
<br/>　　如此年轻就有如此实力，的确称得上惊艳。
<br/>　　可惜，
<br/>　　一人之力，终究有限，再怎么努力，恐怕也难以改变结果。
<br/>　　祝明飞也不由得暗暗摇头，祝小白和苏蔓同样脸色十分不好看。
<br/>　　很显然，已经没有人看好叶尘，可以击败五大高手。
<br/>　　“桀桀！小子！就算你能从万毒窟逃出来又如何？你终究破不了我们的五毒大阵，我们就算是干耗，也能把你耗死！”
<br/>　　“乖乖跪下投降，然后自废修为，或许还可以饶你一命！”
<br/>　　“不错！把你所修炼的功法交出来，报出你的师门，或可留你一命！”
<br/>　　……
<br/>　　云天青等人纷纷叫嚣了起来。
<br/>　　他们虽然用五毒大阵困住了叶尘，其实心里同样惊骇不已。
<br/>　　要知道，他们五个人，可是药王宗里面，除了云千秋之外的最强者！
<br/>　　五人联手也才只堪堪将对方困住，而且对方的修为明明连金丹境都没有达到。
<br/>　　这意味着什么？
<br/>　　意味着，这个世俗少年所修炼的功法，必定无比逆天！
<br/>　　别说云天青等人动了心思，就连其余四大宗门的高手，眼中也纷纷露出炙热之色。
<br/>　　不料，
<br/>　　叶尘听到这话，顿时冷冷一笑，
<br/>　　“怎么？又想打我功法的主意？就凭你们这几个废物，也配！”
<br/>　　说完这话的同时，叶尘脚下猛然一顿，然后整个人冲天而起，竟突然拔高了近千米的高度。
<br/>　　“不好！这小子想逃跑！快追！”
<br/>　　其中一名长老见状，当即飞身追了上去，其余三名长老也不甘落后，几乎同时追出。
<br/>　　唯独云天青，隐隐感觉到一股不妙的感觉，可是眼看其他四人，都已经追了出去，他自然也不能落于人后。
<br/>　　然而，
<br/>　　下一刻，
<br/>　　九天之上，却忽然传来一道冰冷的声音，
<br/>　　“逆生百转，第一转！”
<br/>　　轰！
<br/>　　已经冲到数千米高空的叶尘，气息陡然提升了一倍！
<br/>　　紫色剑芒也变得越发璀璨，朝那名率先追上来的药王宗长老，狠狠一剑斩落了下来！
<br/>　　“天地一剑！”
<br/>　　“什么！”
<br/>　　那名长老顿时脸色狂变，怎么也没有想到，原本都没有还手之力的叶尘，竟然可以突然爆发出，如此强大的实力！
<br/>　　然而，
<br/>　　他此时反应过来之后，再想要躲避，已然是来不及了！
<br/>　　“不！”
<br/>　　紫琼剑斩落下来，竟然只一剑，便将那头蜈蚣傀儡，生生斩成了两半！
<br/>　　而且去势不减，连同那名长老，同样瞬间被轰成了肉渣。
<br/>　　药王宗的功法，本身战斗力不强，全都依仗傀儡术和五毒阵法，这才能叶尘死死困住。
<br/>　　如今五去其一，而叶尘的实力又暴涨了一倍！
<br/>　　此消彼长之下，五人的围攻之势顿时荡然无存。
<br/>　　其后跟过来的三名长老，看到同伴被叶尘一剑斩杀，顿时全都一慌。
<br/>　　“快退！”
<br/>　　三人大吼了一声，便要转身逃窜，然而叶尘又岂会给他们机会？
<br/>　　轰轰轰！
<br/>　　直接又是一连三剑斩出！
<br/>　　配合上快如迅雷的身法，已经诡异莫测的空间法则奥义。
<br/>　　嘭嘭嘭！
<br/>　　五人单打独斗，远远不是叶尘的对手，几乎毫无任何反抗之力，就已经被叶尘全部轻松斩杀。
<br/>　　下方的药王宗众人，和四大宗门的高手们，以及祝明飞兄弟俩，早已经彻底看傻了眼。
<br/>　　任谁也没有想到，
<br/>　　之前明明已经处于劣势的叶尘，竟然在转眼之间，就完成了惊天大逆转！
<br/>　　而且，
<br/>　　还是当着云千秋，和其他四大宗门宗主的面，一举斩杀了四大长老！
<br/>　　原来，
<br/>　　方才叶尘突然冲出包围，看似想要逃跑，原来不过是一招以退为进，诱敌深入，最后趁其不备，一举斩杀！
<br/>　　竟然连云千秋这样的天榜强者，都没有反应过来，完全没有来得及搭救，四大长老就已经命归黄泉。

<br/>　　叶尘诱杀了四大长老之后，当即又顺势朝最后赶到的云天青，扑杀了过去！
<br/>　　云天青顿时差点吓尿，当即飞速暴退，同时冲着下方的云千秋高呼，
<br/>　　“太上长老救命啊！”
<br/>　　轰！
<br/>　　璀璨的剑芒，再次狠狠落下！
<br/>　　云天青的脸色，瞬间变得煞白无比，
<br/>　　“去！”
<br/>　　在云天青的全力操控之下，他的那具毒蛇傀儡，义无反顾的挡了过去。

<br/>　　在那道璀璨的剑芒面前，巨蛇傀儡是如此的脆弱。
<br/>　　几乎如同菜刀切西瓜一般，数据就被斩成了两段！

<br/>　　轰隆隆！

<br/>　　一道苍老的声音陡然响起，
<br/>　　“小子！休要猖狂！”
<br/>　　轰！
<br/>　　下方竟然又出现了一道，丝毫不逊色的青色刀芒，和叶尘的惊天剑芒，狠狠撞击在了一处。

<br/>　　轰隆！
<br/>　　刀芒和剑芒撞击在一处，周围空间仿佛要炸裂开了一边，似乎连时空，都被这股强大的力量所扭曲。

<br/>　　轰！
<br/>　　云天青听到这话的同事，眼前陡然出现了一个硕大的金色拳影，嘴角才刚刚露出的一抹笑容，瞬间凝固当场。
