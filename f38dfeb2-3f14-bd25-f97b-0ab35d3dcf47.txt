　　只见巫人王脸色黑紫，眸中闪烁出凶狠的光芒，身上气息不断的鼓荡，如同滔天骇浪，冲天而起。
<br/>　　他的目光死死的盯着苏莫，仿佛是一头太古凶兽，要将苏莫一口吞下。
<br/>　　这已经是巫人王在极力的压制自己的气势了，不然的话，以他武帝境九重的修为境界，威势足以惊天动地。
<br/>　　但是，巫人王心中怒火已经彻底沸腾，怎么也压不住自己的气势。
<br/>　　“巫人王，你有何不服？”大祭司淡淡的问道，他对于巫人王的所为，没有丝毫动怒，仿佛早已料到会有此一遭。
<br/>　　“大祭司！”
<br/>　　巫人王闻言立刻向大祭司抱拳，朗声问道：“我的考核表现虽然不如巫尘，但也是全部通过了，应该同样有成为少司的资格吧？”
<br/>　　“不错！”大祭司点了点头，通过考核便有成为少司的资格。
<br/>　　只不过，他只会选取成绩最好的人，若是别人不服，那就只能挑战了。
<br/>　　“所以我不服，我要和巫尘一战，生死一战，取而代之！”巫人王大声说道，声音中充满了决绝。
<br/>　　任何人敢阻挡他获得少司之位，就要死，必须死！
<br/>　　此战，不是他死，就是巫尘亡！
<br/>　　此战，他不成功便成仁！
<br/>　　当然，他对于自己的实力有绝对的自信，同阶一战他不相信巫尘是他的对手。
<br/>　　大祭司点了点头，随即看向苏莫，道：“巫尘，巫人王向你挑战，你若想坐实了少司的位置，只能应战！”
<br/>　　苏莫闻言面现沉吟之色，他知道这种战斗是同阶一战，巫人王的修为会被压制在和他同样的境界。
<br/>　　对于此战，他并不畏惧，但是也没有必胜的把握。
<br/>　　因为巫人王太强了，首先，对方不仅是有特殊体质，还有双生战魂。
<br/>　　更主要的是，就算对方的修为被压制在和他相同的境界，但对方毕竟是武帝境九重的武者，一身的武学，或者是兵器，估计都是尊级的级别。
<br/>　　当然，这还不是苏莫最担心的事情。
<br/>　　他最大的忌惮，是巫天御和巫长远等曾经去过苍穹世界的人，因为他一旦肆无忌惮的使用战魂，一定会被识破身份。
<br/>　　但是，为了少司之位，他又不能退缩。
<br/>　　沉吟了一番，苏莫面色凝重的向大祭司说道：“大祭司，我可以和他一战，但是有一个要求。”
<br/>　　“哦？什么要求？”大祭司闻言微微有些意外。
<br/>　　“此战，不能公开进行，只能由大祭司和少量族中高层观战！”苏莫沉声说道。
<br/>　　这就是他现在能想到的办法，因为只要巫天御、巫长远等人看不见他出手，他暴露的风险就能大大降低。
<br/>　　承然，巫天御等人当初回到巫族之后，可能将苏莫的情况全部都告诉了大祭司等巫族高层。
<br/>　　但是，只要不是亲眼所见，就很难认出他来。
<br/>　　毕竟，他的一切都已经改变，肉身改变、玄力改变，武魂已经进化为了战魂，气息与威势和当初都不可同日而语了。
<br/>　　只要他谨慎一些，不让巫天御等去过苍穹世界的人，感受到他的战魂之力，其他人不可能认出他得身份。
<br/>　　“不公开进行？这是为何？”大祭司闻言一怔，其他人亦是面露疑惑之色，不明白苏莫为何有这个要求。
<br/>　　两人的比试，公开与否，对于结果好像没有什么影响吧！
<br/>　　“这一战胜负难料，或许我会陨落，不想被所有族人看到这个场景！”苏莫淡淡的说道，他找了一个这种不是借口的借口。
<br/>　　众人闻言讶然，巫尘居然没有信心战胜巫人王？
<br/>　　不过，众人转念一想也就释然了，巫人王有双生战魂，特殊体质，战力超绝，巫尘没有信心也在情理之中。
<br/>　　巫人王闻言冷笑一声，他自从修炼武道以来，不仅是在族中，纵横数十个星辰，同阶之中，还没有人能打败他。
<br/>　　就算是上一届的少司巫凌，也只是在千招之后，才能勉强略胜他半招，但想要彻底打败他也不可能！
<br/>　　“可以！”大祭司略一沉吟，便点了点头，这个请求很简单，他自然不会拒绝。
<br/>　　只不过，无论在哪里比试，此战中都不会有人陨落。
<br/>　　因为巫尘和巫人王都是超绝的天才，大祭司不会坐视任何一人陨落，不然是巫族巨大的损失。
<br/>　　“巫人王，赌注先拿来吧！”这时，苏莫转头看向巫人王，朗声说道。
<br/>　　一万块下品灵晶，这可是巨富，他自然不会放弃。
<br/>　　“哼！但愿你有命想用！”巫人王冷笑一声，随即手臂一挥，一小堆灵晶向苏莫飞射而去。
<br/>　　此地有这么多人在场，他当然不会赖账！
<br/>　　苏莫脸色冷漠的收起一万下品灵晶，并未多言。
<br/>　　“我带你二人进入星空！”
<br/>　　大祭司的声音响起，而后一股无形的伟力，瞬间将苏莫和巫人王两人拉到了其身边。
<br/>　　下一刻，大祭司裹挟着苏莫和巫人王消失不见。
<br/>　　而后，看台上一直默默观看的四祭司，以及其他的核心长老，纷纷身形一动，爆射长空，瞬间消失无踪。
<br/>　　所有高层都跟随着大祭司，前往了天冥星之外的星空之中。
<br/>　　广场上的众人，看着消失的一众族中高层，均是心中暗叹。
<br/>　　他们也很想看看这巅峰一战，但他们实力太低，哪有本事跟的上大祭司。
<br/>　　就算是巫源等普通长老，也是跟不上，就算追上去了，等赶到之后，估计战斗已经结束了。
<br/>　　众人无奈的叹息，但也没有人离开，因为众人都想知道结果。
<br/>　　要不了多久，大祭司一定会和众核心长老返回的，那时候就能知道结果了。
<br/>　　“巫尘死定了！”
<br/>　　这个想法在很多人心中升起，诚然，在之前的三关考核之中，巫尘表现出了不可思议的天赋和悟性，但战斗力想超越巫人王，这根本不可能！
<br/>　　巫人王是谁？
<br/>　　巫族的绝世天才，战斗力堪称妖孽，因为同阶之中从未败过，自认为是人中之王，故改名——巫人王。
<br/>　　巫人王的战斗力，别说是在天冥星，就算是在周围十几颗星辰，都是最顶级的存在。
<br/>　　上一届的少司选拔，同为绝世天才的巫凌，若不是有一件至宝，怎么可能略胜巫人王半招？
<br/>　　即便是靠着那件至宝，巫凌苦战上千招，也只能略胜巫人王半招，依旧无法击败巫人王。
<br/>　　由此可见巫人王的战斗力，有多么的恐怖！
<br/>　　所以，巫尘对上巫人王，几乎必败无疑了。
<br/>　　巫天御、巫源和巫穹三人，面上都是露出冷笑之色。
<br/>　　这刚刚崛起的巫尘，还未绽放出应有的光芒，就要成为巫人王的踏脚石了！
<br/>　　巫天御本来还想等到选拔结束之后，约战巫尘呢！现在看来也不用了！
