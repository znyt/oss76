　　罗子雄胸有成竹，这个林凡虽然是生物掌控者，但他不过是一个控鼠的，上不了台面，有自己的血月巨狼对付，足够了。
<br/>　　够他狠狠喝一壶。
<br/>　　不能浪费时间，要是等最强新生回来，事情就麻烦了。
<br/>　　罗子雄看着仙灵孔雀，他的七八个小伙伴，既然有三人已经被这只孔雀打趴在地。
<br/>　　另外四人也是非常的狼狈，不是在进攻，而是在苦苦坚守。
<br/>　　那孔雀一张长长尖尖的嘴，比世人任何兵器都厉害，啄向人的时候，能吓破人胆。
<br/>　　好几把长剑被它的嘴一啄，既然成了两半。
<br/>　　“好一直超等仙灵孔雀，真是不凡，怪不得大小姐看上了你”
<br/>　　“孔雀，我劝你不能挣扎了，乖乖的跟我走，我给你换一个好主人”
<br/>　　青枭发出清脆的声音：“丑八怪，泥奏凯，你丑到本孔雀了”
<br/>　　丑八怪？
<br/>　　罗子雄顿时一脸错愕，他不管是对自己的实力，还是对自己的容貌，都非常的自信。
<br/>　　既然被一只鸟给嫌弃丑？
<br/>　　你是孔雀就可以随意嫌弃别人吗，罗子雄勃然大怒：“等我抓到你，非拔光你的毛不可”
<br/>　　“住手”
<br/>　　当罗子雄准备朝着仙灵孔雀走去的时候，身后一道声音传来。
<br/>　　嗯？
<br/>　　还有碍事的？
<br/>　　他转身，看到一个二百多斤的胖子，带着一丝畏惧的看着他，道：“喂，我不管你是谁，可我劝你不要乱来”
<br/>　　“你也想拦我吗”
<br/>　　东方袁惊慌的咽了咽口水，他看到林凡被那只血月巨狼纠缠住，一时半刻根本脱不开身。
<br/>　　“我不拦你，我给你钱”
<br/>　　“钱？”罗子雄一楞，这什么套路。
<br/>　　东方袁道：“你立刻带着你的人离开这里，我就给你一百万”
<br/>　　“呵呵...”罗子雄饶有兴致的看着这个胖子，就跟看傻子一样。
<br/>　　“二百万”
<br/>　　他继续用看傻子一样眼神看着他。
<br/>　　“五百万总行了吧”
<br/>　　罗子雄不言不语，双腿猛然发力，身形猛然跃出，拖出一排排虚影，眨眼来到东方袁的面前。
<br/>　　啪的一声脆响，东方袁直接嘴角带血的滚出去。
<br/>　　“死胖子，不想死给劳资滚远点，废物，还他妈拿钱砸我，你怎么不砸五个亿给我，呸...”
<br/>　　“我给你一千万，不能在多了”
<br/>　　东方袁全身跟散架了一样，但依然带着畏惧的又喊了一声，不是说有钱能使鬼推磨吗，钱在古学院真的不好使？
<br/>　　罗子雄扬了扬手，吓的胖子捂着脸，冷汗涔涔。
<br/>　　“滚出古学院当你的废物公子哥吧”
<br/>　　罗子雄不在搭理东方袁，骤然朝着仙灵孔雀冲去，单手凭空一拍，犹如平地惊雷，轰鸣声不绝而耳。
<br/>　　“丑八怪，敢偷袭本孔雀”
<br/>　　青枭漂亮绝伦的一双眼眸精光一亮，轻盈的抽身退开，原地留下一道手掌印记。
<br/>　　“丑八怪，看招”
<br/>　　只见青枭骤然飞到空中，全身闪烁九彩色光芒，美的惊艳绝伦，顾盼生辉间神圣不可侵犯。
<br/>　　“暴风羽刃”
<br/>　　咻咻咻...
<br/>　　仙林孔雀通体九彩霞光下，一片片发光发亮的羽毛散落，变成了见血封喉的羽刃，朝着罗子杰大范围的席卷而去。
<br/>　　卧靠...
<br/>　　罗子雄惊呆了，一片疾驰而来的飞羽射来，空气中便是一道爆响声，这说明什么，说明这羽毛携带了惊人的杀伤力。
<br/>　　这羽毛要是落在人的身上，一只手臂都能给你卸下来。
<br/>　　躲，唯有躲，绝对不能硬接，否则非变成筛子不可，全身插满羽毛。
<br/>　　电光火石间，说时迟那时快，罗子雄连忙朝着一边闪躲，蹭蹭蹭...一根根羽毛跟钉子一样的钉在了地板上。
<br/>　　羽毛攻击还没完，成批成批的飞羽朝着罗子雄席卷，他最后狼狈的跟狗打滚一样，疯狂躲避，全身都是虚汗。
<br/>　　“老大...老大...”
<br/>　　羽毛攻击总算结束，可罗子雄一个放眼望去，他的几个小伙伴，身上都插满了羽毛，血淋淋，触目惊心。
<br/>　　“该死，你这只死鸟，劳资非抓到你不可”
<br/>　　罗子雄彻底怒了，体内元力疯狂的爆涌而出，在古学院中，除了天才班的高手，他几乎可以立于不败之地，今天居然在这只孔雀身上差点翻了船。
<br/>　　霎时间，罗子雄一脚踏出，骤然朝着空中的孔雀抓去。
<br/>　　青枭漂亮绝伦的眼眸一丝狡黠骂道：“白痴，大笨蛋，看我回羽”
<br/>　　咻咻咻...
<br/>　　罗子雄感觉身后有异动，一个扭头，差点三魂不见了七魄，那些之前飞出去插在地面的羽毛咻咻咻的又飞回来了。
<br/>　　朝着他身后刮来。
<br/>　　啊..啊...啊...
<br/>　　好几个身上插满羽毛的学员痛苦的惨叫起来，那些该死的羽毛自动拔出，飞回去。
<br/>　　如此惊艳的一幕，让罗子雄顾不上朝着仙灵孔雀攻击，那些羽毛一根都沾不得。
<br/>　　他又狗打滚的闪躲疾驰而来的飞羽。
<br/>　　堂堂二阶三段的修武者，被这些飞羽弄的狼狈不堪。
<br/>　　“哇...青枭好厉害”
<br/>　　东方袁看傻眼了，看着它身上飞出大批大批的羽毛，美奂美轮，可更惊奇的是，这些飞出去的羽毛，还能在回来。
<br/>　　飞回来的羽毛，又重新的接在了青枭的身上，灿灿生辉，散发九彩色的光芒。
<br/>　　东方袁觉得自己跟林凡有点多管闲事。
<br/>　　根本不需要他们来帮忙好吗，这些人还不够仙灵孔雀自己打的，而他自己还白白挨了对方一个大耳光，嘴角现在还溢出血，痛死了。
<br/>　　“丑八怪，你那么弱，还想来抓本孔雀，傻逼...”青枭眼中满是轻蔑鄙夷，明明高高在上，神圣不可侵犯的样子，却满嘴脏话。
<br/>　　形成了巨大的反差萌。
<br/>　　罗子雄艰难的躲避回羽，但身上还是出现了不少血痕，只要被那些羽毛轻轻的擦边，身上就会多出一条伤痕。
<br/>　　好可怕的招式，简直逆天，羽毛飞出去，还能回来，变态。
<br/>　　不愧是超等生物，罗子雄暗暗的吃惊，不服不行。
<br/>　　自己带来的小伙伴都倒在了地上，失去战斗力，只能看他了。
<br/>　　“丑八怪，你快来抓我啊，本孔雀还没玩够呢”
<br/>　　“玩”罗子雄一脸日了狗，紧接着一股怒火不由得从两肋一下窜了上来，脸上暴起了一道道青筋。
<br/>　　“死鸟，等我把你抓住，我非扒光你的毛”
<br/>　　青枭嘻嘻一笑：“你一个丑八怪，还想拔本孔雀的毛，大傻逼，你还是丑死我算了”
<br/>　　“劳资今天非抓到你不可”
<br/>　　罗子雄勃然大怒，他的声音由低到高，渐渐地咆哮起来，脸色涨红，进而发青，脖子涨得像要爆炸的样子。
<br/>　　他堂堂生物掌控者，被一只鸟给狠狠的羞辱了，今天要是不将这只仙灵孔雀抓住，他在古学院也就不用混了。
<br/>　　“血月巨狼，给我合体”一道仰天咆哮冲天而起。
