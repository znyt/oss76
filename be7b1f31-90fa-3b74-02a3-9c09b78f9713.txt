　　第622章
<br/>　　万军从中取上将首级，如探囊取物相仿。
<br/>　　所有人都目瞪口呆，心道：如这样的恶贼，天下又有谁制得住。
<br/>　　程大雷擒回陆长迢，心中也是长松口气。刚才的行动看似轻描淡写，但他所承受的风险又有谁知道。
<br/>　　眼下总算是成功了，程大雷看着陆长迢道：“陆将，怎样，我们聊聊。”
<br/>　　一天之内，被劫持为人质两次，他恨不得找个石头当场撞死。此刻他心中忍不住想，自己是否不该针对程大雷。柳追风和莫将难两个，明显比程大雷好对付多。
<br/>　　“呃……哈哈。”陆长迢干笑两声：“程当家打算怎么聊，一切都好商量。”
<br/>　　“让你的手下放开一条路，我们兄弟要回家，你不惹我，我也不想在此地惹是生非。”
<br/>　　“好说，好说。”陆长迢挥舞着双手：“都听见了没有，快与程当家散开一条路。”
<br/>　　手下士兵面面相觑，在西面与程大雷放开一条路。陆长迢被押着往外走，程大雷离开蛤蟆城已经足够久，便不想在此地拖延时间。
<br/>　　“程当家……”莫将难突然道：“大家份属江湖同道，还请程当家也容我们兄弟一块儿走，山高路远俺们记程当家的恩情。”
<br/>　　程大雷一怔，诧异的看着莫将难，心中只有一句话想说：我从未见过如此厚颜无耻之人。
<br/>　　程大雷根本懒得搭理他，令白元飞押着陆长迢往外走。柳追风与周桃也缀在后面，想着趁机溜出来。
<br/>　　这个时候，谁也没想到，莫将难突然冲白元飞出手，白元飞猝不及防，慌忙避开。如此，竟让莫将难将陆长迢抢走。
<br/>　　白元飞气得咬牙切齿，这才是终日打雁，却被雁啄了眼。
<br/>　　“将军，放我一条生路。”莫将难道。
<br/>　　“好，好。”陆长迢从虎口脱险，忙不迭的向外走，他可不愿再次落在程大雷手中。
<br/>　　见到陆长迢脱险，东方童急忙下令，四面兵丁围杀过来。
<br/>　　事情已经发生，程大雷也没有别的法子，如今只有破釜沉舟，向外冲杀一条路走。
<br/>　　他、白元飞还有小白狼，每个人都有不弱的身手，如今三人骑在坐骑上，人人抖索精神，一阵风似得向外冲撞。
<br/>　　四周人虽然多，但却被三人硬生生闯开一条生路。在他们身后，还跟着柳追风和周桃，二人想着浑水摸鱼，趁机溜出去。
<br/>　　程大雷杀威赫赫，匹夫剑淅沥沥往下滴血，周围兵丁也是混口饭吃，又有谁愿意拼了性命不要，去拦阻程大雷。
<br/>　　他们不敢攻击程大雷，当先竟散开一条路。压力却又全都在柳追风和周桃身上，柳追风伤痕累累，大家都抱着痛打落水狗的心思。
<br/>　　事情和柳追风想的不一样，他没能占到程大雷便宜，反而引来许多士兵的追杀。
<br/>　　周围密密麻麻都是想杀自己的人，尤其是东方童，想起刘凯惨死的模样，恨得咬牙切齿。此刻仇人就在眼前，他挥舞着兵器冲过来，要将柳追风的脑袋斩成两半。
<br/>　　“程当家，程当家……”
<br/>　　程大雷回过头，见柳追风不停唤他的名字。
<br/>　　“求程当家千万救她一条性命，来世柳某当牛做马报答程当家的恩情。”
<br/>　　柳追风已经知道自己逃生无望，东方童是绝对不会让她走的，此刻，他只想拼着一条命保全周桃。
<br/>　　程大雷难免心中感慨：此人竟痴情如此。
<br/>　　不过感慨是感慨，程大雷可没打算多管闲事。他不记挂周桃害自己的心思就不错了，现在如此危险的场合，程大雷又怎么会多管闲事。
<br/>　　看到程大雷的眼神，柳追风便知道他心中想什么。程大雷定然是无利不起早的角色，他拼命将身周的士兵击退，口中道：“我柳氏一门愿投靠程当家，只求……”
<br/>　　柳追风门人弟子不少，至少有上千人，也是一股不小的势力。他今天也是走窄了，不然不会落到这步田地。
<br/>　　可话是如此说，程大雷并不觉得柳追风死后，他的门人弟子会听自己的。
<br/>　　此刻，一名士兵举长枪扎向周桃后背，周桃根本没能力防御。关键时刻，程大雷还是忍不住出了一剑，将这长矛击开。
<br/>　　柳追风体内似乎多了一股莫名的精神，他握着剑，脸上湿漉漉的也分不清是血还是汗。在最后关头，他强撑着往外冲。
<br/>　　程大雷在前方开路，闯开一条通道，柳追风缀在后面，也快要突出重围。他猛地一推周桃，口中爆喝一声：“走啦。”
<br/>　　回过身，硬生生扑进兵丁的长矛。这一刻他如疯如魔，宛若从地狱中冲出的恶鬼，在做最后的困兽之斗。
<br/>　　扶风一剑柳追风，号称能斩断纷飞的落叶，此刻，大概是他一生实力的巅峰。剑法抛弃了飘逸花俏，只想着杀人。
<br/>　　可他终究斩不断一个情字，东方童拍马过来，大刀狠狠挥下。咔嚓一声，将柳追风的脑袋斩成两瓣。
<br/>　　杀了柳追风，东方童忍不住哈哈大笑，口中咆哮道：“刘凯，你的仇兄弟与你报了。”
<br/>　　程大雷此刻已突出重围，毕竟此刻场上能拦得住他的人不多，或者说几乎没有。回头看着这一幕，口中发出一声长叹。
<br/>　　“走啦，走啦。”程大雷口中连喊几声，又看了周桃一眼：“走啊。”
<br/>　　如果周桃愿意随程大雷一起走，程大雷也不介意用手中剑帮她一帮。谁料周桃突然转身，竟扑向了东方童。
<br/>　　在接近东方童时，她的身子高高跃起，手中匕首亮了出来，想要为东方童报仇。
<br/>　　并不是每个人都能如程大雷一样跳起来的，身在空中无有借力之处，如果此刻有人攻击，根本无法闪避。
<br/>　　兵丁一起亮出了长矛，扎向空中的周桃。瞬间，有七八柄长矛扎穿了她的身体，血顺着她的肚子汩汩流出来。
<br/>　　身体噗通落在地上，倒在一片血泊之中。东方童握着刀，一刀扎穿了她的背心，周桃连一声喊都来不及发出，便在此地香消玉殒。
<br/>　　程大雷轻轻叹一声，冲看呆的白元飞和小白狼道：“走啦，走啦。”
<br/>(本章完)
