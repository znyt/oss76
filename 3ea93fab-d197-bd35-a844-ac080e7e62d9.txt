　　京城，某军方秘密监狱。
<br/>　　李霄力被带回京城后，就被直接送到了这所监狱。
<br/>　　一间面积最多不超过二十平方米的房间内，整个房间里除了那扇门之外，不存在其他的出入口，房间里依靠换气扇来完成换气。
<br/>　　房间里的摆设十分简单，只有两把椅子、一张桌子和一盏200w的白炽灯。
<br/>　　李霄力双手被塑料捆在椅子把手上，脑袋耷拉着，整个人毫无精神的坐在椅子上。李霄力身体不断地往前倾，当前倾到一定角度的时候，又被绑在他身上的皮带给拉了回来。
<br/>　　李霄力被关在这个房间里超过48个小时了，48个小时内，没有水、没有实物、不能上厕所，更别提睡觉了。倒不是李霄力不想睡，而是李霄力根本就没办法谁，被一盏200w的白炽灯挂在脑袋上照啊照的，根本不可能睡得着。
<br/>　　在这样的疲劳轰炸下，李霄力整个人的精神状态极差，早就没有了杭山市“力哥”的风采，整张脸油腻腻的，在灯光下油亮油亮的，头发也是腻在了一起，一撮一撮的，此时李霄力是要多狼狈就有多狼狈，哪里还是那个杭山市百分之三十未婚女性的最佳择偶对象？
<br/>　　房间门，被人从外面推开，罗昊从房间外走了进来。
<br/>　　在这个超过48个小时的时间内，罗昊只来过这个房间两趟，这是第三趟来到这个房间。
<br/>　　罗昊径直走到房间里那张桌子背后，双手撑在桌子上，盯着面前昏昏欲睡的李霄力，问道：“李霄力，你还是什么都不说吗？你别指望着你背后的人会来救你，我告诉你这里是中国，不是北美洲，就算是你背后的人想要救你，也要掂量掂量她有没有这个本事！”
<br/>　　“你想要知道什么？”
<br/>　　李霄力用他那双疲惫的眼睛看了眼罗昊，最终还是松了口。罗昊说得不错，这里是中国，而且还是被军方所控制的监狱，就算玛丽科娃的能量再大都未必能从这里把自己捞出去，至于电影里经常会出现的一队装备精良的雇佣兵来劫狱，那就更加不可能存在了。
<br/>　　雇佣兵来劫狱？别闹了，装备再精锐的雇佣兵来这里也不过是白送人头。
<br/>　　“‘毒皇后’玛丽科娃！”罗昊盯着李霄力说道：“把你知道的，跟她有关的事情全都说出来！”
<br/>　　在听到罗昊说出“‘毒皇后’玛丽科娃”的时候，李霄力的神情发生了一丝变化，李霄力终于明白了自己为什么会把特种部队给招来了，他们不是冲自己而来，而是冲着他背后的“毒皇后”玛丽科娃而来的。
<br/>　　“‘毒皇后’玛丽科娃我知道的很少，当年我帮助警方打掉了我原来老大的势力后，拿了奖励带着卧底时自己偷偷藏起来的资源，我就逃到了国外，准备在国外避避风头然后再回国发展，毒皇后就是在那个时候找到我的。我就是这样成了毒皇后的下线，每次跟我联系的都是毒皇后手下的人，我根本就没有见过她，至于你问得那些问题，我根本就不知道，对于毒皇后这个女人，我知道的可能还不如你们多。”
<br/>　　李霄力知道自己是隐瞒不下去了，就把自己所知道的一切全都给交代了，其中还包括‘crystal-ball’这种全新的人工合成型毒品，以及自己把钱鑫和陆岑两个人送到北美洲给玛丽科娃用作进行“crystal-ball”改进版试验的事情。
<br/>　　不过，有一件事情李霄力却是隐瞒了下来，那就是布克尔特让他盯着佟千雪的事情。知道这件事情的刘三和钱鑫全都已经死了，目前知道这件事情的就只有李霄力一个人，李霄力知道这可能是自己最后的保命符。
<br/>　　罗昊盯着李霄力看了一会儿，随后转身离开了房间，没过多久，就有两个武警战士走进房间，解开了李霄力身上的皮带和捆在他双手手腕上的塑料扎带，将他从房间里带走。
<br/>　　从三次对李霄力的盘问中罗昊能感觉到李霄力对玛丽科娃这个女人知道的的确不多，事实真的很有可能如李霄力走进所说的那样，他对于玛丽科娃而言就是个微不足道的小角色，根本算不上什么，所以玛丽科娃对她根本不会透露什么重要的东西，就算是李霄力这边出了什么问题，直接就可以丢弃掉。
<br/>　　不过李霄力虽然接触不到玛丽科娃的核心，但是对于罗昊来说，也不是没有什么收获，至少他知道了李霄力跟玛丽科娃是有关系的，那家“维伦恩”生物制药公司也是玛丽科娃的产业，还有那个由“维伦恩”生物制药公司所生产，用以改善男性-性-功能的“crystal-ball”也是一种全新的人工合成型毒品！
<br/>　　有了这些东西，罗昊可以跟国际刑警组织进行联系，对“维伦恩”生物制药公司采取行动。
<br/>　　……
<br/>　　北美洲，玛丽科娃所居住的庄园。
<br/>　　玛丽科娃抱着她的白色波斯猫坐在沙发中正在看一部奥斯卡获奖电影，曾经玛丽科娃也想过自己有一天也会成为一部奥斯卡级电影中的女主角，并且站在那个万人瞩目的领奖台上，捧起那个被视为电影界最高荣誉的小金人。
<br/>　　白色波斯猫舒服的趴在玛丽科娃的身上，张嘴打了个哈欠，舒服的眯着眼睛。
<br/>　　玛丽科娃放在茶几上的手机突然响了起来，玛丽科娃拿起手机，看了眼上面的电话号码，接通了电话，说道：“什么事情？”
<br/>　　“李霄力被中国警方抓了。”霍斯曼德说道。
<br/>　　“被抓了？”玛丽科娃皱了皱眉，问道：“怎么回事？”
<br/>　　“具体怎么回事我也不是很清楚，传来的消息只说李霄力被抓了，而且其中似乎还有军方参与……”
<br/>　　霍斯曼德在电话中欲言又止。
<br/>　　“军方？”玛丽科娃那张精致的脸上露出思索之色，左手在波斯猫身上轻轻挠着，说道：“他们的目标不是李霄力，是冲我来的。他们应该是通过李霄力在国内售卖‘crystal-ball’的事情，猜出了一些什么。”
<br/>(本章完)
