　　慕清霄嘴角抽了抽，不明白这小妮子究竟哪里来的自信。
<br/>　　望着怀中的露出欢笑的佳人，他也不忍再去责备，只要她没事并且开心就好，其他都已经无所谓了。
<br/>　　相拥片刻，回过神来，小医仙抬起臻首，美眸看向四周，道：“这里是……天焚炼气塔？”
<br/>　　“既然毒体已经封印，那就没必要留在这里了，暂时先出去吧。”
<br/>　　慕清霄饶有深意的瞥了一眼岩浆池。
<br/>　　想要开启其下的陀舍古帝洞府，需要远古八族手中的陀舍古帝碎片。
<br/>　　陀舍古帝是斗气大陆最后一名斗帝，本体是排名第一的异火，自命‘帝炎’吸收所有的异火聚灵之后而成，人们称之为‘陀舍古帝’。
<br/>　　自从远古以后，很少有人突破至斗帝，当年的斗帝强者，似乎都在一夜中消失，现只能从一些古籍中查到些残留讯息。
<br/>　　大陆上似乎遗留下一座在虚无空间的斗帝洞府，是少有的保存完好的斗帝遗迹。
<br/>　　其中，所遗留着可助人突破至斗帝阶别的本源帝气和帝品雏丹，陀舍古帝玉便是开启陀舍古帝洞府的钥匙。
<br/>　　陀舍古帝玉共有八块碎片，远古八族各有一块。
<br/>　　如今萧没落已久，但萧炎手中依旧拥有一块，至于其他的七块碎片则在其余七族手中。
<br/>　　慕清霄想要突破渡劫，陀舍古帝洞府就是一条捷径，真正的捷径。
<br/>　　“仙儿，等迦南学院的事情处理好，我们就去中州逛一逛。”
<br/>　　“嗯。”
<br/>　　开心的点着小脑袋，小医仙挽着慕清霄的手臂，犹如一只黄鹂鸟，叽叽喳喳的说着五年间的事情。
<br/>　　前脚刚踏出天焚炼气塔，慕清霄与小医仙脚步一顿，因为前面已经有两道身影拦住了去路。
<br/>　　来者是两名白发苍苍的老者，正是内院大长老苏千与内院的柳长老。
<br/>　　此时，柳长老手中正提着一名昏死过去的青年。
<br/>　　“白山。”
<br/>　　小医仙一眼就认出了柳长老手中的青年，先前比赛中，青年的狂妄自大与心狠手辣可是给她留下了很深刻的印象。
<br/>　　让她不明白的是，为何柳长老会提着白山来此。
<br/>　　苏千在看到慕清霄后，眼底闪过一抹唏嘘，微微弯腰，语气中颇为恭敬，道：“苏千见过尊驾。”
<br/>　　柳长老也微微行礼，牵强一笑。
<br/>　　“没想到白山此子心狠手辣，今日的比赛给小医仙姑娘留下了不好的回忆，学院没有及时制止实属无奈，特将此子带来赔罪，任由姑娘处置。”
<br/>　　闻言，小医仙有些懵，目光不由的看向慕清霄，美眸中闪烁着好奇。
<br/>　　慕清霄则轻轻一笑，笑声中含有一丝讥讽。
<br/>　　迦南学院又如何，在绝对的力量前，不是照样得低头，若是他的实力不敌紫袍斗尊，就算出现，恐怕也只会饮恨于此，
<br/>　　迦南学院此举，将强者为尊的道理充分的体现了出来。
<br/>　　而且，这群老家伙脸皮一个比一个厚，居然将责任全部推脱到一名蝼蚁身上。
<br/>　　听到慕清霄讥讽的笑声，苏千与柳长老脸上火辣辣的，自从学院创立以来，他们何时如此低声下气过。
<br/>　　柳长老厚着脸皮干咳一声，道：“为了表达歉意，学院特地准备了一些丹药与灵草给小医仙姑娘疗伤，希望阁下忘了刚才一番不好的回忆。”
<br/>　　说着，柳长老已经将白山如死狗一样丢在地上，并且恭恭敬敬的将一枚纳戒送了过去。
<br/>　　要不是关键时刻，柳长老想将白山扒皮抽筋的冲动都有了。
<br/>　　若非这小子心狠手辣，哪里会搞出这样的破事，害的学院高层一起丢脸，甚至要面临被斗圣抹杀的风险。
<br/>　　慕清霄接过纳戒，扫了一眼其中的灵药，各种玉瓶少说几百，其中都是高阶疗伤丹药，甚至利于修炼的丹药。
<br/>　　最多的就属灵草，堆积成了一个小草堆。
<br/>　　各种灵草泛着各种奇异的光晕，大多数都是能给小医仙修炼用的，可见迦南学院这回确实是下了血本。
<br/>　　见慕清霄接过纳戒，柳长老稍稍松了口气，忐忑的心情总算是平静了一些。
<br/>　　唯独心中肉疼的紧，纳戒中存放的可有学院五分之一的高阶疗伤丹药，灵草更是将迦南学院储存库中的毒草全部搜刮了去。
<br/>　　仅仅是一枚纳戒，就算放在中州，都有无数强者为此杀的血流成河。
<br/>　　“小医仙姑娘，白山体内气旋已经被封住，如何处置，仅凭姑娘一言，若是姑娘怕脏了手，可由老夫代劳。”
<br/>　　柳长老乘此机会顺着栏杆往上爬，早已摩拳擦掌，只要小医仙一句话，立刻就将地上的孽畜料理了。
<br/>　　小医仙瞥了眼地上昏死的白山，微不可查的摇头，顿时体会到一种身为弱者的悲哀，收回目光，轻轻拉了拉慕清霄的袖子，柔声道：“清霄哥哥，我们走吧。”
<br/>　　闻言，慕清霄溺爱的拿起她的柔荑，将纳戒戴在她手中，温柔一笑道：“那就走吧。”
<br/>　　说罢，完全不理会两人，直接跃过苏千与柳长老，朝着内院的居住区迈步而去。
<br/>　　没等两人走出多远，柳长老手起掌落，斗宗气息毫无保留的倾泻而出，白山的身躯瞬间就爆成了一团血雾。
<br/>　　自此，世间在没有白山这号人物。
<br/>　　“念在你们诚意不错的份上，此事就此揭过……”
<br/>　　耳边传来慕清霄平静的声音，柳长老紧绷的心弦总算是松懈下来，与苏千对视一眼，皆是仰天长叹。
<br/>　　这个差事，是他们有史以来做的最幸苦的一件事情。
<br/>　　虽然只有简简单单的几句话，但谁也不知道，他们心间顶着多强的压力。
<br/>　　斗圣对他们来说，本身就是一种无形的威压，几乎将两人压的喘不过气来。
<br/>　　因为他们不知道，会不会因为一句话惹来后者的不喜。
<br/>　　牺牲白山，解除迦南学院面临的空前危机，对于任何一名学院高层来说，都是一件喜事，虽然做的有些不光彩。
<br/>　　苏千随手一挥，就将空气中的血腥味驱散。
<br/>　　“走吧，既然事情已经解决，先向其他长老知会一声，毕竟学院秩序不能乱。”
