药房里寂静无声，陆子羡轻搂着夏夕绾，夏夕绾短暂的晕眩过后所有的知觉才慢慢的恢复了过来。
<br/>
<br/>        这个曼陀罗花的毒太强了，纵然她从小被药材浸泡，是百毒不侵的身体也差点没抵抗住，如果换成别人，早毒发身亡了。
<br/>
<br/>        这一次的试毒并不成功，她必须进行第二次的试毒。
<br/>
<br/>        纤长的羽捷颤了颤，夏夕绾想要睁开眼，但是视线里还是黑暗的。
<br/>
<br/>        这时她就感觉一只大手按上了她的香肩，头顶响起了陆子羡的声音，“你还好么？”
<br/>
<br/>        随着知觉的回笼夏夕绾这才意识到自己靠在陆子羡怀里了，她吓了一跳，她什么时候到他怀里去了，她迅速伸手想要推开他。
<br/>
<br/>        但是她身上一点力气都使不上来，两只小手抵上他的胸膛，倒有点欲拒还迎的意思。
<br/>
<br/>        这时耳畔就响起了一道低沉冷戾的嗓音，“你们真是让我看了一出好戏！”
<br/>
<br/>        夏夕绾迅速抬头，她的视线已经渐渐恢复了光明，只见门边伫立着一道高大挺拔的身躯，陆寒霆来了！
<br/>
<br/>        陆寒霆身上覆着一层森冷的寒霜，那双深邃的狭眸染着几分可怕的猩红直直的射了过来，那目光像是淬了毒的冰，夏夕绾第一次看到他这样可怕的眼神，她已经从他的眼神里看到了浓浓的杀机。
<br/>
<br/>        “陆先生…”
<br/>
<br/>        夏夕绾叫了一声，但是这时陆寒霆已经迈开长腿走了过来，他剪裁如刀锋的西裤带出强大而凛冽的气场，走过来，一把拽住了陆子羡的衣领，一个结实的拳头就砸了上去。
<br/>
<br/>        陆子羡受了一拳，摔在了地上，嘴角立刻沁出了血迹。
<br/>
<br/>        陆寒霆又一个拳头砸了过去，他从喉头里逼出森然的嗓音，“陆子羡，知道她是什么人么恩？她是我太太，是你的嫂子！”
<br/>
<br/>        陆子羡俊美的脸上迅速挂了彩，他用手背擦了一下嘴角的血渍，清冷的黑眸看着陆寒霆，对视里火花四溅，“那她知道你是从精神病院里跑出来的怪物么？”
<br/>
<br/>        陆寒霆舌尖舔了一下干燥的薄唇，“呵”一声，他又砸了陆子羡好几拳。
<br/>
<br/>        夏夕绾吓坏了，她第一次看到陆寒霆这样打人，她挣扎了两下，绵软无力的身体艰难的从地上爬了起来，然后扑上去，从后面抱住了陆寒霆精硕的腰身，“陆先生，够了，不要打了！”
<br/>
<br/>        后面有人抱了过来大大的限制了陆寒霆的动作，他挺直了英挺的脊背，嗓音森冷道，“放手，听到没？”
<br/>
<br/>        “我不放，陆先生，你误会了，刚才我们什么都没有做，你别打人了，不要激动！”
<br/>
<br/>        夏夕绾不敢放手，因为她知道情绪失控的他又有了发病的征兆。
<br/>
<br/>        陆寒霆精硕的胸膛上下起伏着，犹如野兽一样，英俊的眼睑重重的阖了一下，他骨关节拽到泛白的拳头缓缓松了下来。
<br/>
<br/>        他转身扣住了夏夕绾的皓腕，拽着她就走。
<br/>
<br/>        但是这时陆子羡快速的起身，拽住了夏夕绾另一只皓腕。
<br/>
<br/>        她同时被两个男人给拽住了。
