<br/>				当她正埋头在炼丹之时，飞船的出现也是引起了城中的轰动，尤其是看到那艘飞船落在凤府的门前时，更是让人忍不住猜测着：这又是谁来了？莫不是凤府大小姐？<br/>　　<br/>　　一些人因好奇而围了上来，然，当他们看到那从飞船上下来的那些人时，不由微怔了一下，那些人，一些是认识的，一些却是不认识的，但他们可以肯定，这些人当中并没有凤府大小姐的身影，也就是说，凤大小姐没回来？<br/>　　<br/>　　听到外面的动静，府里的人也出来看，尤其是凤萧等人，听说是飞船到来，便想到是轩辕墨泽他们来了，当下便来到外面一看，果然见是他们。<br/>　　<br/>　　“哈哈哈，墨泽啊！你们来啦！快进来快进来！”凤萧大笑着说着，请他进来。<br/>　　<br/>　　轩辕墨泽微点了下头唤了一声，便跟着他一同进了里面，来到厅中坐下时，他便问着：“萧叔，小九可有回来？”<br/>　　<br/>　　“哈哈哈，回来了回来了，她比你还先到，你不用担心，她很好。”<br/>　　<br/>　　凤萧朗声笑着，看着他松了一口气，这才道：“小九回来还问你到了没，我还问她怎么你们没一起来，她说是出了些意外，不过你不用担心，她是浑身上下没有半点损伤的回来的，这丫头，一回来一大早便扎进了丹房炼丹，这不，就在刚才还炸了个丹炉。”<br/>　　<br/>　　闻言，轩辕墨泽这才露出笑容来，提着的心也终于放了下来：“她回来就好。”<br/>　　<br/>　　站在他身后的灰狼也是咧嘴笑着，露出开心的笑容来：“我就知道鬼医一定会回来的，主子，幸好咱们一路赶过来了。”<br/>　　<br/>　　“对了，我听说你们捡了个孩子？有抱过来吗？我这小外孙子我们都得见一见啊！”他朗声笑说着，询问起孩子。<br/>　　<br/>　　“有，冷霜带着，可能还在后面。”轩辕墨泽示意着，让灰狼出去看看。<br/>　　<br/>　　不多时，灰狼回来，道：“主子，家主，小主子被老爷子抱去了，还有夫人她们都在逗着小主子。”<br/>　　<br/>　　听到这话，凤萧笑了笑，摆了摆手：“无妨，我们过去也是一样的。”说着对轩辕墨泽道：“小九现在在炼丹也还没出来，估计你就是想见也是见不到的，这样吧！我先带你去见见父亲他们，回头等小九炼好丹药出来了，咱们一家人再好好聚一聚。”<br/>　　<br/>　　“好。”轩辕墨泽站了起来应着，跟着他往外走去。<br/>　　<br/>　　一边走，凤萧一边道：“我们在商量着习凛的婚事，正准备挑个时间去叶家下聘，他们年轻人说，尽早把这婚事给办了，把叶菁娶过来。”<br/>　　<br/>　　凤萧边走边看着轩辕墨泽，笑道：“我问过小九了，她说你们的不急，你怎么看？当真不急？”<br/>　　<br/>　　闻言，轩辕墨泽唇角微勾，道：“我们商量过了，先一起去修炼提升实力，至于婚礼一事不急着办，其实对我们来说，婚礼只是一个形式，但纵是这样，我也希望到时可以给她一个盛大的婚礼，让普天同庆，而这，显然不是眼下能做到的。”<br/>　　<br/>　　：。：<br/>
