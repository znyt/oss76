　　“叶将军他……居然真的一个人，把珍宝岛海军基地给灭了！”
<br/>　　华夏一方的军舰之上，众人望着卫星传来的前方影响，所有人全都两眼瞪得老大，个个满脸不可思议的神色。
<br/>　　就连叶天歌，也同样满脸骇然，简直不敢相信，眼前所发生的一切是真的。
<br/>　　叶尘竟然真的以一己之力，灭掉了米国在太平洋战区的核心基地珍宝港！
<br/>　　要知道，哪怕是当年二战之时，岛国精心筹划，发动突然袭击，也仅仅只是让珍宝岛重创而已。
<br/>　　而今天，叶尘却是光明正大，孤身一人，独闯数万人的现代化军队，前后只用了不到一个小时时间，就将整个基地覆灭！
<br/>　　若非亲眼所见，这简直就像是在听神话故事一般。
<br/>　　“叶将军！恭喜！你生了一个好儿子！这可真是我华夏之幸啊！”
<br/>　　林震南冲着叶天歌，由衷的感叹道。
<br/>　　这个世界，说到底还是靠拳头来说话，今日叶尘一人扫灭了珍宝港，对于米国来说，将会是极大的震慑。
<br/>　　然而，叶天歌却轻轻一叹，脸上露出一抹担忧之色，缓缓道：
<br/>　　“米国称霸全球数十年，还从来没有受到过如此重创，我怕他们会狗急跳墙，直接动用核武！”
<br/>　　林震南听到这话，也不由得脸色微微一变，
<br/>　　“不错！米国所掌握的核武器，足以摧毁整个地球！”
<br/>　　“倘若米国真的动怒，只怕后果不堪设想，不过只要小叶将军，能够及时罢手，迅速返回我华夏境内。”
<br/>　　“米国就算是有天大的胆子，应该还不敢公然对我华夏投放核武，否则定会成为世界公敌！”
<br/>　　不料，叶天歌再次摇了摇头，叹息道：
<br/>　　“只怕小尘他，不会这么轻易善罢甘休……”
<br/>　　叶天歌的声音，才刚刚落下不久，顿时有人指着屏幕道：
<br/>　　“你们快看！小叶将军竟然越过瓦胡岛，继续朝东面去了！”
<br/>　　“什么！”
<br/>　　所有人全都一惊，不由得纷纷朝屏幕望去。
<br/>　　果不其然，那道璀璨的紫色剑芒，犹如暗夜中的流星，竟然朝米国本土的方向飞驰而去！
<br/>　　“他这是想干嘛？”
<br/>　　“难道他灭了珍宝港海军基地还不算，还想要一人对抗一国不成？”
<br/>　　“疯了！疯了！他简直疯了！”
<br/>　　“他这是逼着米国政府动用核武啊！”
<br/>　　……
<br/>　　米国国会大厦，
<br/>　　米国总统和所有国会议员，全部尽数到场，死死的盯着前方传来的影像，个个脸色铁青。
<br/>　　“这怎么可能！难道他是撒旦转世不成？”
<br/>　　“竟然连巡航导弹，都伤不了他分毫！”
<br/>　　“简直太强大了！”
<br/>　　……
<br/>　　看到前方传来的战报之后，整个国会都炸开了锅。
<br/>　　“诸位！现在可不是感慨的时候！根据卫星监测，那个叶狂仙，屠杀了我们整个海军基地之后，并没有返回华夏，而是朝我们的本土而来！”
<br/>　　一名白人军装老者，忽然敲了敲桌面，开口提醒道。
<br/>　　此人赫然正是米国的国防部长拉姆斯！
<br/>　　“什么！”
<br/>　　所有人听到这个消息，顿时一个个全傻眼了。
<br/>　　“他到底想干什么？”
<br/>　　“难道他还真想，一人灭我们一国不成？”
<br/>　　“简直狂妄至极！”
<br/>　　……
<br/>　　眼看众人又要再次争论起来，坐在首位上的米国总统普朗克，忽然重重咳了一声，众人这才重新安静下来，
<br/>　　“拉姆斯将军，你继续说！”
<br/>　　国防部长拉姆斯继续道：
<br/>　　“我军已经出动航空母舰战斗群，对叶狂仙进行拦截，不过情况不容乐观，所以……”
<br/>　　说到这里，拉姆斯微微一顿，接着又道：
<br/>　　“我提议，动用核武，不惜一切代价，诛杀此人！”
<br/>　　核武器可不是随便动用的，必须要经过国会同意，军方才能得到授权密码，从而动用大型核武。
<br/>　　国会众人闻言，顿时纷纷点头。
<br/>　　“不错！此人威胁太大！必须将其诛杀！”
<br/>　　“他若是立即返回华夏境内，我们还真不好动手，既然如今自投罗网，那就说什么也不能让他活着离开！”
<br/>　　“绝对不能让他踏入国土一步，否则我们米国可就丢人丢大了！”
<br/>　　……
<br/>　　国会众人一阵议论之后，显然也都意识到强大的危机，竟然全票同意动用核武。

<br/>　　叶尘带着梦幻之神苏珊，已经跨越了近三千公里的海洋，已经可以隐约看到米国本土的边缘地带。
<br/>　　“叶先生！我再最后提醒你一次，你这是在自寻死路！”
<br/>　　苏珊此时，已经从之前的震撼中，缓缓回过神来，不由得皱着眉头提醒道。

<br/>　　“我们米国所储存的核弹量，足以将整个地球摧毁几十次，你的实力就算再强，还能强的过核弹吗？”
<br/>　　“我知道，之前在北极狼王岛，你曾经抵挡过核弹而不死，可那仅仅只是一枚小型核弹罢了，真正的大型核弹，威力不知道要强大多少倍，绝对不是人力所能抵抗的！”
<br/>　　“所以，我劝你还是赶紧返回华夏吧！”
<br/>　　苏珊见叶尘一言不发，于是便苦口婆心的规劝起来。
<br/>　　除了不忍心看到叶尘被核弹轰炸至死以外，她自己也不想受到牵连。

<br/>　　前方忽然传来一阵阵轰鸣声，
<br/>　　很快两人发现，前方海面之上，出现了三只庞然大物，拦住了他们的去路。
<br/>　　“居然派出三只航母舰队，看来你们米国对我，还真是挺重视啊！”

<br/>　　这三艘航母舰队，恐怕已经是米国西海岸的全部战力了。
<br/>　　“可惜在我看来，也不过是三条小爬虫而已！”
<br/>　　叶尘冷冷一笑，直接身体一晃，将速度催动到极致，同时动用空间法则，瞬间便跨越了数十里，几乎一眨眼的功夫，就已经来到其中一只航母舰队的跟前，
<br/>　　此刻这只航母舰队的指挥官，都还完全没有反应过来，直到前方突然出现一道璀璨的紫色光芒，这才意识到情况不妙，
<br/>　　“敌袭！开火！快开火！”
