<br/>				天照神宫，<br/>　　<br/>　　这里不但是岛国精国社总部所在，也可以说是岛国人心中的圣地！<br/>　　<br/>　　只不过，寻常岛国人，只能在神宫外围的景点晃悠，根本没有资格进入其中，能够进入这里的，不光要有显赫的身份和地位，还必须要事先通报，只有获得允许之后，方才能进入，哪怕是岛国首相也不能例外！<br/>　　<br/>　　正是因为这个原因，这里平时极少有人来，十分的清净。<br/>　　<br/>　　不过今天，却是一个例外。<br/>　　<br/>　　不但岛国首相龟田三郎来了，就连岛国四大财团的掌舵人，三大武道世家家主，以及全岛国最顶级的几位武道强者和阴阳大师，尽数到场。<br/>　　<br/>　　甚至，连柳生家族唯一幸存的嫡系柳生正平，居然也来了。<br/>　　<br/>　　而在神宫周围，更是出动了特警部队，里三层外三层的包围起来，甚至整个景点都被直接封锁，禁制对外开放。<br/>　　<br/>　　至于这些岛国大人物前来的目的，不用问，自然全都是因为那个可怕的男人。<br/>　　<br/>　　……<br/>　　<br/>　　“雪姬小姐，麻烦您再去通报一下吧！这件事情关系到整个岛国的安危，如果羽鹤大仙再不出手，我岛国恐怕就要被那叶狂仙搅的天下大乱了！”<br/>　　<br/>　　岛国首相龟田三郎，冲着一名二十岁上下的美丽女子，卑躬屈膝的开口恳求道。<br/>　　<br/>　　倘若让普通岛国人看到这一幕，肯定大为震惊，无法想象他们的首相大人，竟然对一名年轻女子如此毕恭毕敬，这简直令人难以置信！<br/>　　<br/>　　可是周围四大财团的掌舵人，三大武道世家的家主们，却没有任何人觉得奇怪，反而觉得是理所应当，甚至连他们，也全都毕恭毕敬的站在一旁，脸上不敢有丝毫的不敬之色。<br/>　　<br/>　　原来，这个叫做雪姬的年轻女子，乃是羽鹤大仙的关门弟子，同时也是最得宠的弟子，专门负责外人与羽鹤大仙之间传话。<br/>　　<br/>　　羽鹤大仙在岛国的地位极为尊崇，不过自从数十年之前，就再也没有公开露过面，一直都是通过弟子和外人进行传话，大约从七、八前开始，传话之人就换成如今这位雪姬小姐。<br/>　　<br/>　　正所谓，宰相门前七品官，对于这样一位存在，众人自然不敢得罪。<br/>　　<br/>　　雪姬听到龟田三郎这话，顿时秀眉一蹙，竟然直接对着这位岛国首相冷声斥道：<br/>　　<br/>　　“是不是我刚刚说的不够清楚？此事老师他老人家自有论断，你们无须多说！请回吧！”<br/>　　<br/>　　龟田三郎听到这话，脸色顿时变得无比难看，望着眼前那张冰冷绝美的脸庞，心里气得牙根直痒痒，真恨不得立即将眼前这个女人拿下，然后狠狠蹂躏一番，可惜他没有这个胆子，只能忍气吞声。<br/>　　<br/>　　见雪姬这条路行不通，龟田三郎回过头，冲着柳生正平试了一个颜色，后者会意，当即冲上去，“噗通”一声，就跪倒在大殿门口，开始大声呼喊起来，<br/>　　<br/>　　“大仙救命啊！我是柳生家族的柳生正平！我哥哥是您的弟子柳生正龙！哥哥奉您的命令，前往华夏执行任务，却被那叶狂仙残忍杀害，如今这贼人又追杀到了京都，我们柳生家也惨遭他灭门！柳生家族就只剩下我一人了，请大仙看在哥哥的份上，为我柳生家做主啊！”<br/>　　<br/>　　柳生正平说到最后，声泪俱下，也算是真情流露了。<br/>　　<br/>　　他是真的吓坏了，倘若羽鹤大仙不出手，一旦他被叶狂仙找到，肯定死无葬身之地，所以只能用这个办法拼一下了。<br/>　　<br/>　　“大胆！”<br/>　　<br/>　　雪姬见状，顿时俏脸越发冰冷，正准备将这个大声咆哮的柳生正平，直接赶出天照神宫。<br/>　　<br/>　　不料，就在这个时候，神宫深处居然传出一道苍老的声音，<br/>　　<br/>　　“那叶狂仙当真杀了我那正龙徒儿？”<br/>　　<br/>　　声音虽然苍老，但是却依旧气势磅礴，众人顿时感觉耳朵一阵嗡嗡作响，心头又惊又喜。<br/>　　<br/>　　而柳生正平听到羽鹤大仙居然回应了自己，也是喜出望外，当即又痛哭流涕道：<br/>　　<br/>　　“启禀大仙！虽然到现在还未能找到我哥哥的尸体，可是随我哥哥一同前往的我那侄儿，已经可以确定被人杀害，而我哥哥却一直下落不明，估计也已经凶多吉少了！”<br/>　　<br/>　　“而放眼整个华夏，能够杀死我哥哥的，除了那叶狂仙，还能有谁？”<br/>　　<br/>　　宫殿深处，沉默了片刻之后，再次响起了羽鹤大仙充满愤怒的声音，<br/>　　<br/>　　“好个叶狂仙啊！老夫本念他是小辈，不想和他一般见识，没想到居然敢害我徒儿，夺我神器，还在我岛国肆意妄为！”<br/>　　<br/>　　“老夫若是再继续纵容他，他还真以为我岛国无人了！”<br/>　　<br/>　　龟田三郎等人听到这话，顿时纷纷精神一震，<br/>　　<br/>　　“大仙言之有理！眼下也只有您出马，方才能压制此人！”<br/>　　<br/>　　“恳请大仙出手，还我岛国安宁！”<br/>　　<br/>　　“必须严惩此贼！决不能让他活着离开岛国！”<br/>　　<br/>　　……<br/>　　<br/>　　众人纷纷高声呐喊起来。<br/>　　<br/>　　“神宫圣地！岂由你们在这里大声喧哗？都给我闭嘴！”<br/>　　<br/>　　台阶上的雪姬直接冷斥道。<br/>　　<br/>　　众人顿时一脸讪讪的闭上了嘴巴。<br/>　　<br/>　　在场的众人，全都是岛国各界最顶尖的人物，可是被这么一个小女子训斥，却愣是一点脾气也没有。<br/>　　<br/>　　神宫内，又沉默了片刻之后，羽鹤大仙那苍老的声音，再次缓缓响起，<br/>　　<br/>　　“我不管你们用什么办法，务必将姓叶的小子引到这天照神宫来！”<br/>　　<br/>　　“只要他敢来，老夫便有十足把握将其斩杀！”<br/>　　<br/>　　龟田三郎等人听到这话，先是一喜，继而面面相觑，纷纷露出一抹为难之色。<br/>　　<br/>　　以他们对叶狂仙的了解，此人狂傲无比，而且行事肆无忌惮，又岂是这么轻易受人摆布的？<br/>　　<br/>　　虽然他们都很好奇，羽鹤仙人为何非要在此处才能与叶狂仙决斗，可是这种问题任谁也不敢随意问出口。<br/>　　<br/>　　就在众人正一阵为难之际，天照神宫外忽然传来一道狂笑之声。<br/>　　<br/>　　“羽鹤老儿！你还真是大言不惭！叶狂仙在此，我今日倒要看看你有什么本事！”<br/>　　<br/>　　声音落下的同时，众人看到一道瘦弱的身影，陡然出现在前方，已经从门口的方向，朝他们这里缓缓走了过来。<br/>　　<br/>　　那道身影每跨出一步，身体就如同瞬移般前进数十米，几步迈出之后，人便已经来到宫殿之前，同时犹如雷鸣般的声音，已经再次响起，<br/>　　<br/>　　“羽鹤老儿，滚出来受死！！”<br/>　　<br/>　　//<br/>　　<br/>　　天才一秒记住本站地址：。手机版阅读网址：<br/>
