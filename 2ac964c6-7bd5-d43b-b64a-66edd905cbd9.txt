夏夕绾将纤长的羽捷安静的垂落了下来，有点伤心，还有点生气，他这是几个意思？

<br/>
<br/>        她嗅过药膏的成分，都是十分名贵的药材，是个人调制的，看样子应该是那个喜欢睡觉的男人给调的。

<br/>        在这个药房里，那个男人，还有双双，都卧虎藏龙。

<br/>        这三天她不敢回幽兰苑，一直在等身上的伤痕褪去，但是都三天过去了，陆寒霆竟然都不联系她。

<br/>        她又有点担心，她不在他的身边，也不知道他能不能入眠？

<br/>        如果又发病了该怎么办？

<br/>        夏夕绾恩…翎翎，我不回去了。
<br/>
<br/>        叶翎陪陆先生？（坏笑）

<br/>        夏夕绾真的不敢回叶翎那里了，虽然没有曼陀罗花，但是她研制了一个药丸，再加上她的针灸，想暂时先缓解一下陆寒霆的睡眠障碍。

<br/>        她的确要每天晚上陪着他，不敢再将他一个人丢下了。

<br/>        夏夕绾翎翎，前几天在酒吧你和顾少没发生什么故事吧？

<br/>        叶翎没有，我好得很！

<br/>        ……

<br/>

<br/>
<br/>        夏夕绾扒着窗户看着他一手缔结这个商界王国，突然有一种与有荣焉的感觉，恩，他太棒了。
<br/>
<br/>        “师傅，就在这里停车吧。”
<br/>
<br/>        “小姑娘，你不去幽兰路了？”
<br/>
<br/>        夏夕绾付了车费笑道，“嗯呐，我突然有点崇拜这位陆氏总裁，所以我现在进去找他。”
<br/>
<br/>        司机震惊的看着夏夕绾，这年头的姑娘怎么都这么的热情？

<br/>        ……

<br/>        “你好，我想找你们的总裁陆寒霆。”
