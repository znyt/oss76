<br/>?    街边一株梅花树下，一个身着破烂道袍的老人身前铺着一块布，上面有着两句话：你荣华富贵在我，我生死有命在天。
<br/>    
<br/>        老人拔出腰间的酒壶，饮了一口，脸上露出一抹醉酒的红，刚吃完烤鸡的手，满是油污。
<br/>    
<br/>        大街之上仿佛瞬间安静了下来，老人看向身前，那是一个身披黑羽长袍的人，脸上带着一张‘滑稽’面具，老人仔细看了一眼这张面具，总是觉得有一丝莫名的邪性。
<br/>    
<br/>        他可以确定两件事，第一件事就是这个人很有钱，因为他的身上穿着的黑羽长袍可不便宜。
<br/>    
<br/>        第二件事就是他今天可以饱餐一顿，说不定还可以去怡红院找相好的姑娘，因为他确定自己可以骗到这个年轻人的钱。
<br/>    
<br/>        他看向陈明，目光之中露出一抹感叹的神色，朝着陈明开口道：“我预见了你的死亡！”
<br/>    
<br/>        陈明伸手将老人身前的鸡骨头扒开，扫了一眼老人，危险指数4.5，战五渣都不如，在老人面前蹲下，“你确定你说的不是废话，人不是都会死吗？难道还有不死的人？”
<br/>    
<br/>        老人愣了愣，不是，这个家伙怎么不按套路出牌啊，正常人不是都应该问我是怎么死的吗？
<br/>    
<br/>        老人继续道：“那你是要算姻缘，那么我可以告诉你，远在天边，近在眼前。众里寻他千百度，那人却在灯火阑珊处。”
<br/>    
<br/>        陈明道：“你说的这又不是废话？远在天边就是说我现在没遇到，遇到之后不就是近在眼前，这不是哪个女人都可以算得上是你说的条件，说点有用的。”
<br/>    
<br/>        老人再次愣了愣，要骗这个家伙，有点难度啊。
<br/>    
<br/>        老人想了想道：“那么你是要算前程，前程坎坷，但是总有柳暗花明，拨开云雾见青天的时候。”
<br/>    
<br/>        陈明道：“你这不是又是废话,我是一个修行者，修行者之中打打杀杀都是常事，众所周知，活得久的一般都修为高深，只要活得久，总有柳暗花明，拨开云雾见青天的时候。”
<br/>    
<br/>        老人道：“那你要是死了呢？”
<br/>    
<br/>        陈明呵呵一笑道：“要是我死了，谁还来找你麻烦，这么一想，你还真可以算是算无遗策，毕竟算错了的，那些人都死了，没死的，注定被你算到了。”
<br/>    
<br/>        老人皱起眉头，终于是察觉到事情并不简单，短暂的疑惑之后，老人道：“你是自己人吧？我初来乍到，并不知道这个地方是你骗钱的地方，呸，卦师的事，怎么能算是骗呢，凭本事骗的钱，那都是赚的！这样，等我在这里赚够路费，我就离开你的风水宝地，如何？”
<br/>    
<br/>        陈明一脸黑线，不是，卦师都这么狠的吗？简直和假的神棍一毛一样！还真是融入了世俗啊。
<br/>    
<br/>        老人见陈明不回答，继续道：“你现在就要我走，我也走不了啊，没有盘缠啊，要不你给我一点，看你在这里混得不错的样子。”
<br/>    
<br/>        陈明道：“这样，你把你修行的经文给我看一眼，我就给你盘缠，让你换个地方，如何？”
<br/>    
<br/>        老人思虑片刻，道：“如果说要看我的修行经文，那么我就不会离开这里了。”
<br/>    
<br/>        陈明想了想，这个老家伙留在这里，恐怕是为了凌仙而来的，卦师这种诡异的职业，就是等一个有机会成为大人物的人，然后指点一番机缘所在，然后靠人情活着，这里将来能够出的大人物，也就是凌仙了。
<br/>    
<br/>        等等，这么说的话，自己还真的可以算是一个神棍啊，因为自己也能预测到谁能成就高，毕竟他看得到光环。
<br/>    
<br/>        陈明道：“你在这里，是要等凌仙？”
<br/>    
<br/>        老人再次吃了一惊，“你也算到了？既然凌仙的机缘已经被你寻到，想必凌仙的人情已经落在了你的身上，那我留在这里也没有意义了，我要去找另外三个人了。”
<br/>    
<br/>        对于老人所说的，陈明倒是没什么兴趣，反正他是来改剧本的，“到底要怎么样，你才能把你修行的经文给我看一眼。”
<br/>    
<br/>        老人道：“修行的经文乃是我宗的绝密，岂能轻易示人？”
<br/>    
<br/>        老人一副誓死不屈的模样，让陈明都有几分动容，看来，要用些特殊的手段了。
<br/>    
<br/>        陈明道：“黄金一万两！”
<br/>    
<br/>        老人挥手，“十万，绝对不能再少了！”
<br/>    
<br/>        “八千。”
<br/>    
<br/>        “五万！良心价！”
<br/>    
<br/>        “五千。”
<br/>    
<br/>        “三万，再少就不卖了！”
<br/>    
<br/>        “二十两。”
<br/>    
<br/>        “嗯，我们再商量商量五千的事......”
<br/>    
<br/>        ......
<br/>    
<br/>        经过一轮惊心动魄，你来我往，扣人心弦，唇枪舌剑，鸡飞狗跳，勾心斗角，高手之间的过招后，陈明最终以三百两黄金二十四两银子，三百二十八文半铜钱的价格，得到看一眼的机会。
<br/>    
<br/>        老人伸手从怀里摸出一本破旧的经文，放在陈明的面前，陈明拿起来扫了一眼，在心中道：“顿悟这本经文。”
<br/>    
<br/>        “叮，检测到这本经文没有任何可修行性，我比较担心你的智商，你怕不是被骗了吧。”
<br/>    
<br/>        陈明放下，再次看向老人，“这是假货。”
<br/>    
<br/>        老人的脸上露出一抹羞红之色，再次拍出来一本，陈明拿起来片刻之后放下，“这也是假货。”
<br/>    
<br/>        在老人从他那平平无奇的怀里掏出三十多本经书，陈明都有些疑惑这老家伙怎么装下了的，在系统的检测下，陈明终于是看到了一本《十文道》。
<br/>    
<br/>        “顿悟《十文道》，需要3000点师德。”
<br/>    
<br/>        陈明愣了愣，这《十文道》，还没有分层的，要么就是顿悟，要么就是没学会。云系的经文，还真是奇怪。
<br/>    
<br/>        陈明根本没有任何选择的机会，这个只能直接顿悟了，错过了这村，就没这店了，一阵肉痛，三千师德消失。
<br/>    
<br/>        《十文道》在陈明的手中飞快的翻动，陈明一目十行，将所有的经文看过一遍，已经明白了其中的东西，而他的双眼之中，也是露出了一抹精芒，而陈明的职业树，再添一种，卦师。
<br/>    
<br/>        陈明笑着看向梅山老人，“铜钱本半两，做得两半花。”
<br/>    
<br/>        梅山老人点点头，也没有露出什么疑惑之色，卦书这种东西，看得懂就是看得懂，看不懂就是看不懂，管他呢，“好，现在，把我的钱给我。”
<br/>    
<br/>        陈明，落下一张一千两的金票，“不用找了。”
<br/>    
<br/>        梅山老人接过，喜笑颜开，“不行，该找的零钱一定要找，要是不找，那我那番杀价，不是做了白用功，少半个铜子都不行。该是骗到多少钱，就是骗到多少钱！多了的，就不是骗到的了。”
<br/>    
<br/>        然后认真地将零钱找给陈明，连铜钱都找零，最后半个，摸出一个铜钱，咬开一半，将小的那一半给了陈明。
<br/>    
<br/>        陈明接过零钱，老家伙不过片刻就已经消失不见，不过陈明此刻已经学到了《十文道》，自然是知道梅山老人的去向，这老家伙施展幻术，已经是走了。看来这梅山老人，还不止一本《十文道》。
<br/>    
<br/>        群山峻林之中，梅山老人回头看向凌家的范围，“算卦这种事，都是讲究有真有假，我刚刚哪一卦是真的算了来着，好像是姻缘吧。”
