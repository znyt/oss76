        又过了几分钟，她收到了傅昼景发过来的关于大哥傅夜沉结婚的日期和地址的邮件。
<br/>
<br/>        不过，令童筱颖感到奇怪的是，那天在酒店里遇见的那个自称是大哥的“新婚妻子”的女人，又是怎么一回事？
<br/>
<br/>        大哥这是要结几次婚啊？
<br/>
<br/>        算了，不去想那么多了。
<br/>
<br/>        反正，大哥傅夜沉他人长得那么帅又有钱，有那么几个艳遇也很正常。
<br/>
<br/>        童筱颖收好手机，起身去了工具房取了镰刀，继续回园子里去干活。
<br/>
<br/>        瓜果蔬菜的种子她已经种完了，现在她在给自己的园子搭建竹篱笆，等竹篱笆搭建好后，她还可以在竹篱笆下种各色藤本蔷薇。
<br/>
<br/>        因为藤本蔷薇是四季都会开花，到时候，蔷薇花的蔓藤爬满竹篱笆，她的园子一定也会特别漂亮。
<br/>
<br/>        当童筱颖把她的瓜果蔬菜园搭建完后，想到那天又可以见到小尚轩了，她一时心血来潮，拿出笔和纸、几根木头、刻刀，坐在了园子外的茅草亭里，拿起手机查阅了一番资料后，开始自己制作孔明锁。
<br/>
<br/>        这孔明锁一做，就是一整夜。
<br/>
<br/>        孔明锁做完后，童筱颖不知不觉地趴在圆木桌上睡了过去。
<br/>
<br/>        夜深人静时，深山里只有她这儿的茅草亭是亮着的，周遭一片漆黑。
<br/>
<br/>        一只亮着绿光的萤火虫，飞到了童筱颖的鼻尖上，童筱颖偏着脑袋枕着手臂睡得太沉，压根就察觉不到这一丝轻微的异动。
<br/>
<br/>        时间一分一秒地过去，草丛间的萤火虫稀疏地飘了起来，绿光与黄光的交织，随后越飘越多，像星星一样忽闪忽闪。
<br/>
<br/>        流萤围绕在茅草亭的周遭，就像夜之精灵，守护着趴睡在茅草亭里圆木桌上的童筱颖。
<br/>
<br/>        可偏偏这样的画面，却飞入了某个人的梦中。
<br/>
<br/>        日有所思夜有所梦？
<br/>
<br/>        这句话，傅夜沉是信了。
<br/>
<br/>        “筱颖……”
<br/>
<br/>        一声深情似海的呓语，让傅夜沉从梦中惊醒。
<br/>
<br/>        最近这段时间，他觉得自己过得恍恍惚惚，而夜里还经常做梦。
<br/>
<br/>        而且他一做梦，就必定会梦到童筱颖。
<br/>
<br/>        这次与之前那两次他把筱颖压在自己身下承欢的梦不同的是，他这次的梦里，看到的是筱颖被一片绿光闪闪的流萤所包围了。
<br/>
<br/>        筱颖只身一人恬静地睡在那片流萤里……
<br/>
<br/>        任他怎么呼唤，都唤不醒沉睡中的她。
<br/>
<br/>        沉寂了三年，他又对筱颖起了欲念是怎么一回事？
<br/>
<br/>        傅夜沉坐起身来，按着额际从床上下来，趿上拖鞋，离开了房间去了楼下厨房里倒了杯水喝。
<br/>
<br/>        回来的时候，他路过隔壁房间，脚步不听使唤地停了下来。
<br/>
<br/>        这间房，筱颖曾经住过，他一直让刘婶留着，不作其它用途。
<br/>
<br/>        以前这间房里，还余留有筱颖的气息，现在那气息早已经消失殆尽。
<br/>
<br/>        “筱颖，我结婚的那天，你一定会来吧！”
<br/>
<br/>        傅夜沉抬起手来，大手按在了门板上，自言自语地问。
<br/>
<br/>        电子请帖他已经发给弟弟傅昼景了，想必筱颖应该也会看到。
<br/>
<br/>        傅昼景得知大哥傅夜沉结婚的对象是李忆莎，还特意发了条短信调侃他傅夜沉。
<br/>
<br/>        傅夜沉看到傅昼景回过来的短信，就像是“哑巴吃黄连”一样，有苦说不出。
<br/>
<br/>        傅昼景说：李忆莎终于嫁给了她喜欢了十四年的男人。大哥，你恐怕不知道吧！我们都知道李忆莎喜欢你，只有你一个人不知道而已。
<br/>
<br/>        看到弟弟傅昼景这样调侃自己，傅夜沉更想对傅昼景说：那是因为你娶走了我最爱的女人。不然，我的新娘压根就没她李忆莎什么事。
<br/>
<br/>        其实，这句话，他都已经在手机短信上全部打好了，但最后还是一个字一个字地全删了。
<br/>
<br/>        傅夜沉不知道李忆莎到底跟他爷爷傅炎廷说了什么，让爷爷深信了李忆莎的话，认为他那晚真对李忆莎做了那事。
<br/>
<br/>        李忆莎无中生有，把那晚的事情告知了爷爷傅炎廷后，爷爷放了狠话，非得让他娶了李忆莎，不然爷爷他讳疾忌医。
<br/>
<br/>        爷爷老了，身体经不起折腾了。
<br/>
<br/>        傅夜沉不希望爷爷拿身体来跟他赌气，他这当长孙的，自然也不会去跟爷爷主动负气。
<br/>
<br/>        最后，他只能妥协……
<br/>
<br/>        他的婚礼可以给李忆莎，但那本合法的结婚证，李忆莎想都别想拿。
<br/>
<br/>        这段时间，他心里真的过得很矛盾。
<br/>
<br/>        一面一直努力劝自己，要忘了筱颖，重新去开始另一段感情，一面又总是无时无刻地思念着筱颖。
<br/>
<br/>        结婚，也总算是给了他对筱颖的一个终结吧！
<br/>
<br/>        当初，他是为了娶筱颖，才答应爷爷退伍从商。
