<br/>　　忽然，他们的眼睛慢慢的睁大，眼神中甚至出现一丝古怪之色。
<br/>　　因为他们看到流露出恐惧的黑斑蟒眼前，不知何时出现了一条小小的金银蛇。
<br/>　　这蛇视地心引力如无物，诡异而轻飘飘的悬浮在巨大的黑斑蟒面前，那双蛇眸泛着妖治的异彩，冷淡无波的盯着黑斑蟒看。
<br/>　　如此小蛇，黑斑蟒一口就能给吞了，甚至还不够塞牙缝。
<br/>　　但黑斑蟒却在害怕，它甚至不敢正视这条小蛇的目光，十五米巨大的蛇躯不由自主的颤栗起来，那种害怕仿佛是来自于灵魂深处。

<br/>　　区区一条小蛇，能让黑斑蟒害怕，最匪夷所思的是，这条小蛇是悬浮在半空。
<br/>　　一条蛇飞着，深深盯着黑斑蟒看，每看一眼，黑斑蟒巨大的蛇躯不由自由的匍匐下来一分。
<br/>　　倏然，捕蛇人像是醍醐灌顶，楞着两只眼睛发痴地看着金银蛇，梦呓似的说：“是蛇王大人，是蛇王大人来了”
<br/>　　捕蛇人都知道证道的蛇王是可以将巍峨的体型化为一条很小的小蛇，小到精致，但是却上天入地，无所不能。
<br/>　　他们如梦初醒，诚惶诚恐的激动跪了起来：“叩见蛇王大人”
<br/>　　但凡是捕蛇人，无不激动的跪下来，唯有穿警察制服的人，一脸的懵逼，无神论的他们，如今经历的一幕幕都在摧毁他们的三观。
<br/>　　涂小安统治能力一开，王者之气散发，将这条黑斑蟒吓的不轻，没有一丝敢反抗的心理。
<br/>　　这条蟒，体型是大，但是等级太低，涂小安统治能力一开，具备蛇王之姿，低等级的蛇类都要无条件的臣服。
<br/>　　“吸血蛇在哪”
<br/>　　涂小安的话透过意识力传输到这条黑斑蟒的脑海中，带着王的睥睨。
<br/>　　“主人去蛇沟深处了”
<br/>　　黑斑蟒嘶嘶的叫唤了一下，诚惶诚恐的说出。
<br/>　　“蛇沟深处？”
<br/>　　涂小安目露深思之色，吸血蛇去蛇沟深处做什么，去找血脉高的毒蛇吃吗。
<br/>　　但这山洞中那么多人类，它根本不需要那么做吧。
<br/>　　要知道人乃万物之灵长，吞噬人，对自身有着明显的提升，涂小安要不是人类重生，都想吞一个人试。
<br/>　　想必会等级暴涨。
<br/>　　已经成为绝世蛇种的吸血蛇，实力跟灵智都今非昔比，它不会做没有意义的事，看来自己也要去蛇沟一看究竟才行。
<br/>　　“告诉我，你的体型是怎么变的那么大”
<br/>　　涂小安问出了心中的疑惑，这种黑斑蟒正常来说，长到十米都算过分了，可这家伙到好，矮人中出巨人吗，既然成长到了足足十五米。
<br/>　　要知道血脉基因决定一切，就好比一个亚洲成年男人一样，一米八跟一米九，看似很接近，但真正能达到一米九的能有几个。
<br/>　　更别说你长到二米了，那绝对是万中无一了。
<br/>　　蛇类也是如此，每一种蛇类的长度基本上都是有限制的。
<br/>　　蛇有蛇的长短，蟒有蟒的大小。
<br/>　　“我只是吃了蛇沟深处的一种植物，就慢慢的长成这么大了”
<br/>　　黑斑蟒瑟瑟发抖的说出，不敢隐瞒。
<br/>　　植物？
<br/>　　涂小安蛇眸猛然一亮，莫非蛇沟深处有什么天材地宝不成？
<br/>　　显然，这黑斑蟒是得到了什么机遇呀。
<br/>　　吸血蛇放任这些人类，去往了蛇沟深处，某非就是为了找这种植物不成？
<br/>　　此想法一出，涂小安不由自主的点头，这种解释非常的说的通。
<br/>　　“我到想看看是什么植物那般神奇”
<br/>　　倏然，涂小安强大的意识力骤然笼罩在黑斑蟒的身上，它瑟瑟发抖，水桶粗的蛇躯在颤栗。
<br/>　　“给我记忆浮现”
<br/>　　金银蛇的蛇眸浑然一眯，意识力钻入了黑斑蟒的脑海中。
<br/>　　瞬间，涂小安的眼前出现了很多不属于自己的记忆。
<br/>　　那是黑斑蟒的人生。
<br/>　　蛇的一生是非常的枯燥，不断的沉睡，不断的觅食，不重要的记忆全部忽略。
<br/>　　这时，涂小安看到了一条不过三米的小黑蟒，它像是追踪猎物，来到了一片冒着浓浓烟雾的沼泽外围。
<br/>　　沼泽烟雾弥漫，难以见到庐山真面目，到是里面隐隐散发着恐怖的气息，让小黑蟒不敢进去。

<br/>　　小黑蟒饥肠辘辘，不管三七二十一的将红彤彤的小果实给吃进了肚子。

<br/>　　从此之后，小黑蟒经常来到这块沼泽，不过都只是在外围觅食，运气好或许会碰到从泥地中冒出来红彤彤的小果实。
<br/>　　每次一遇见，它就会立刻就给吞吃肚子里。
<br/>　　就这样，小黑蟒的体型不断的暴涨，从三米暴涨到了十五米。

<br/>　　而且出现的方式也非常诡异，会转眼从泥地生长出来，神乎其神。
<br/>　　很显然，这就是传说中的天材地宝。
