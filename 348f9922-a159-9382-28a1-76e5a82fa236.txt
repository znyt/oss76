　　　　一条几万的蛇，说放生就放生，或许在外人的眼中是无法理解的。
<br/>　　甚至认为那是傻，钱烧的。
<br/>　　但白镇人民对蛇王是无比的信仰的，佘山所有蛇本来就是属于蛇王大人的。
<br/>　　上午十时许，新的蛇王雕像盖着一块大红布被四人一起抬了出来，蛇王大人惯例巡行开始，炮铳三声，锣鼓齐鸣。
<br/>　　随着蛇王出巡，人山人海的队伍鱼贯而出，队列由大锣开道。
<br/>　　隐隐间，很多人都非常的激动，不知道如此大的日子，蛇王大人会不会再次显化。
<br/>　　巨大蛇影金光灿灿，盘踞白镇上空的画面，让人终生难忘。
<br/>　　蛇王出巡，旗队紧随其后，写有“行雷”“连公”“肃静”“回避”的木牌并列在前，引领蛇王雕像前。
<br/>　　而跟着蛇王雕像除个四人大力士抬，还有一位身为道袍的中年男子，他便是李顺，一切事宜有他操办。
<br/>　　李顺跟着蛇王雕像旁，两边行人无数，大伙只能跟着，而他可是陪着蛇王大人一起出巡，满脸的荣光。
<br/>　　祖坟冒青烟，祖坟冒青烟。
<br/>　　大队伍俨然形成了一条长蛇阵。
<br/>　　但凡大队伍而至，沿街各户人家，手持香火燃鞭相迎，并与队列中人交换三支香火，名曰“分香”，以显示对蛇王大人沿街驱疫，降福闾里的共同敬仰。
<br/>　　不得不说白镇人民对于蛇王节太慎重，绝对不是敷衍了事。
<br/>　　如此阵仗，不知道的，还以为是国家大人物出巡。
<br/>　　长蛇形大部队中，很多人或绕脖颈或围腰间，或缠手臂，都有一蛇的存在，连儿童也不例外。
<br/>　　身上的蛇也是非常的安分，人蛇共襄盛举。
<br/>　　大部队中，此时有一位长像甜美，修长玉腿的美少女手中拿着一个手机杆，俏美的面容泛着满满的兴奋，正是进行手机直播。
<br/>　　“各位观众，你们看到没有，现在白镇正是举行蛇王出巡，你们看到没有，人多不多，壮不壮观”
<br/>　　一年一度的蛇王节，可谓声势浩大。
<br/>　　美少女手机屏幕冲着周围环境拍摄了一番，她直播间的观众立刻深深的被震惊了，呆若木鸡。
<br/>　　“乖乖，我的女神，你们这个地方要不要那么夸张啊，这人也太多了”
<br/>　　“我刚才没看错了，好像人人的身上都挂着一条毒蛇，不愧是白镇呀，真是百蛇出没”
<br/>　　“女神，告诉我你们的地址，那么热闹，我要赶过去看看”
<br/>　　白镇的蛇王节曾经可是登上央视的新闻联播，作为古老传下来的节日，非常的有文化底蕴。
<br/>　　今年的蛇王节参加的人数更是破纪录，每一个角落全都是人，黑压压的人头无法计算。
<br/>　　除了本地人，很多外来人口也来凑一凑这古老的节日。
<br/>　　这直播的美女正是李清照，她现在深深已自己是白镇人民为荣，直播要噱头，但什么噱头有蛇王节吸引人，光几十万人前扑后拥，就亮瞎你的眼。
<br/>　　更别说很多人都胸前挂着一条毒蛇，那画面是既怪异又新奇。
<br/>　　她此时不直播，更待何时。
<br/>　　这时，李清照化身采访的记者，用手机对着一位中年大叔开始采访。
<br/>　　“这位大叔，我要是没看错，你身上这条应该是银环蛇吧”
<br/>　　这中年大叔，典型的捕蛇人打扮，穿的很是朴实，不过他身上的毒蛇可是一点不朴实，是剧毒的罕见蛇种。
<br/>　　此蛇背有白环和黑环相间排列，白环较窄，尾细长，一米有余，眸射出冷冷的光。
<br/>　　“丫头，算你还有点眼力劲，我这可不是一般的银环蛇，你看它的头部有微微一触角，说明品种比一般的银环蛇都要高的多”
<br/>　　李清照手机对着，然后一看，还真别说，这银环蛇头顶真的有一小角，不过怎么看，都好像是头上长了一个包。
<br/>　　银环蛇算是本国最为毒性的毒蛇了，危险性极高，而且平时看似很温顺，使人放松警惕，但银环蛇咬人时从不警，咬住顿时呼吸麻痹，两小时不用血清，必死无疑。
<br/>　　这种蛇一般人看到，可是非常的害怕，但此蛇现在在中年大叔的身上非常的安分，纵然是李清照对着它又看又摸，也没什么大反应。
<br/>　　不怕生人，被驯服的很好。
<br/>　　“嘻嘻，各位观众，今天你们有福了，因为你们可以看到各式各样的毒蛇，我保证有绝大部分的蛇，你们别说看，恐怕连听都没听过”
<br/>　　李清照接着在人群中寻找不凡的毒蛇，旋即，他看到了一位胖大叔，挺着一个高高隆起的啤酒肚，吸引她注意的是胖大叔的脖子上环绕着一条黑蛇，看上去凶狠，霸道无比，非常有威势。
<br/>　　黑蛇浑身上下透着与众不同，细心看，它凶狠霸气间，还透着一股超级萌。
<br/>　　李清照完全看不出这是一条什么毒蛇。
<br/>　　“大叔大叔，你能告诉我，你身上的这条蛇是什么品种吗”
<br/>　　这位胖大叔穿着不像捕蛇人，到像是一位商人，由于身上的黑蛇，这位大叔走起路来，昂首挺胸，非常得意。
<br/>　　就差脸上写着，快来看我的蛇啊。
<br/>　　终于，是他的“嚣张”，还有身上的黑蛇，成功将李清照给吸引了。
<br/>　　李清照拿着手机对着胖大叔拍摄，这位胖大叔也不怕，反而更为的高兴，高声道：“我这蛇可来头不小，在我们佘山都找不到”
<br/>　　佘山找不到？
<br/>　　李清照闻言便有点诧异，要知道佘山出了名的蛇品种多，很多外国的品种，你都能在佘山看到。
<br/>　　“大叔，那你快说说，你这条是什么蛇”
<br/>　　李清照本来就是一个蛇迷，另外胖大叔身上的蛇通体黑的发亮，一点别的颜色都看不到，却楞是让人移不开眼。
<br/>　　而她的直播间，几十万的观众顿时也都在讨论这是一条什么品种的蛇。
<br/>　　一时之间，弹幕无数。
<br/>　　胖大叔笑呵呵，颇有点人来疯，让李清照的手机好好对着他，然后道：“我这条叫黑王蛇，是一条无毒蛇”
<br/>　　无毒蛇此言一出，李清照呆滞了，眨了眨修长的睫毛，有种被忽悠的感觉，对着胖大叔翻了一个大大的白眼：“大叔，人家现在可是在直播，几十万观众看着，你不带这样玩我的”
<br/>　　无毒蛇，还起一个那么牛掰的名字，合适吗。
<br/>　　另外无毒蛇跟有毒蛇，地位就完全不一样，价值也相差甚远，起码，无毒蛇连蛇毒都没有。
<br/>　　蛇毒可是人类非常重要的药用。
<br/>　　“哼...小丫头片子，懂什么”胖大叔看到李清照满满的失望，不悦的冷哼一声解释道：“你别看大叔我这条是无毒蛇，但它可是毒蛇的克星，本身无毒，但却有逆天级的抗毒属性，不仅性格温顺，而且颜值爆表，最牛之处在于它主要以其他毒蛇为食”
<br/>　　胖大叔抬着下巴说出来，无毒蛇怎么了，它可是已吃毒蛇为食。
