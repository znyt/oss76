这些人现在可谓是新仇旧恨一起算，恨死了夏夕绾，不知道是谁拿来了一个鸡蛋，直接砸在了夏夕绾的身上。

<br/>        她巴掌大的绝色小脸慢慢就冷了下来，一双犀利的澄眸往那些人的脸上扫射了过去。

<br/>        但是下一秒视线里突然闯入了一道高大英挺的身躯，有人来了，挡在了她的身前，将她护在了身后。

<br/>        夏夕绾纤长的羽捷一颤，竟然是…陆寒霆！

<br/>        陆寒霆来了！

<br/>        他怎么来了？

<br/>
<br/>        他看向那个丢青菜的人，目光如X光般洞穿过去，杀人的心都有了。

<br/>
<br/>        那些围观群众也丢下了手里的鸡蛋青菜小石头，转身就跟着跑了，好像后面有魔鬼在追他们。

<br/>        这些人都走了。

<br/>
<br/>        没想到霸总陆寒霆也有头顶青菜的这么一天，他头上绿油油的，很搞笑。

<br/>        他自己没手吗？

<br/>        陆寒霆没有直起身，而是保持着这个姿势，看着她巴掌大的绝色小脸。
