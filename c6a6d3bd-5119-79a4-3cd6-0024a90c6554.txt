　　慕清霄闻言，剑眉皱了皱，因为帮助药尘复活，对现在的他来说，极其困难。
<br/>　　药尘想要复活有三个条件，第一是需要一个修炼焚决的人，且拥有三种异火去帮他炼制一具可以容纳灵魂的躯体。
<br/>　　第一个条件，倒是简单，慕清霄只需要掌握炼制方法就行了，第二个条件是需要三种材料，七品颠峰丹药，生骨融血丹，七阶魔兽的精髓骨血。
<br/>　　而第三个条件却是一具斗宗强者的尸体，他现在才化神一重初期，也就是一星斗皇，越级战斗不难，但要杀斗宗强者，短期内肯定做不到。
<br/>　　“帮助你复活，对于我现在而言有些困难。”慕清霄皱眉说道。
<br/>　　药尘自然知道困难，但听他这么一说，心中也是松了口气，最起码得知慕清霄不是在骗他，若是满口答应，他反而会觉得虚伪。
<br/>　　毕竟，他经历的实在太多，这样的人见的也太多了。
<br/>　　“没事，我可以多等几年。”
<br/>　　药尘话语中充满了无奈，态度却不是很强硬，若是慕清霄不答应，他只好另想办法。
<br/>　　片刻后，慕清霄点了点头，道：“可以，但材料由你们准备。”
<br/>　　“好，我看你这娃娃也是有底线的人，如此年轻就成为斗皇强者，想必有自己的尊严，不会骗我这种老头子。”
<br/>　　“你不用试探我，答应过你的事情，我就会履行承诺。”
<br/>　　听到慕清霄的话，药尘的脸色也变得温和起来，随后不等他开口，便将骨灵冷火剥离灵魂。
<br/>　　异火玻璃药尘的灵魂后，立刻就想要逃走，但九霄琉璃焱的速度比它更快，瞬间将它重重包裹，慕清霄随手一卷就将骨灵冷火握在手中。
<br/>　　看到这一幕，药尘差点连下巴都惊掉在地上，虽然灵魂状态的他没有下巴，但心中着实惊骇的无以复加，手指指着慕清霄，语无伦次，道：“这这......”
<br/>　　活了这么大岁数，他还是头一次看到，有人能够不借用任何力量，徒手握住异火的，面前的青年简直是个怪物啊，可就在下一秒，更令他目瞪口呆的事情发生了。
<br/>　　只见慕清霄将骨灵冷火握在手中看了一圈，随后就丢入口中，就像孩童吃糖果一样，动作极其自然。
<br/>　　药尘深吸了口气，要知道他在年轻的时候，将骨灵冷火搞到手，不知道吃了多少苦，可面前的青年，用现实狠狠的将他打击了一番。
<br/>　　若非亲眼看到，打死他也不信，有人能够直接吞服异火。
<br/>　　异火下肚，慕清霄神情自然，他可是凤凰，又怎会惧怕火焰，而且有九霄琉璃焱在，骨灵冷火吓得都不敢动弹，只能乖乖待在他的腹中。
<br/>　　慕清霄也不急着炼化，道：“答应你的事情我会完成，你吸收了这小子三年斗之气，就由你补偿他吧，更何况，你想收集材料也需要靠他吧。”
<br/>　　药尘盯着萧炎看了许久，又看了一眼慕清霄，老实说他很想收慕清霄为徒，可后者如此逆天的资质，恐怕也看不上他啊。
<br/>　　不过，萧炎的心性与资质也让他颇为满意，最重要的是，他没有选择的权利，道：“萧炎，你可愿做我的徒弟？”
<br/>　　闻言，萧炎一愣，虽然这老家伙害的他被族人羞辱三年，但本身也是可怜人，被自己最信任的徒弟害到这步田地，连肉身都毁了。
<br/>　　虽然没了肉身，但知识都在啊，听两人的对话，这老家伙恐怕是一位超级强者，拜他为师貌似不亏啊。
<br/>　　萧炎的反应速度也是极快，沉吟几秒后，便跪地磕头，道：“师尊在上，请受徒儿一拜。”
<br/>　　待萧炎拜完师后，慕清霄，道：“我会在萧家暂住一段时间，往后你就跟着他修炼吧。”
<br/>　　“对了，焚决你就别教给他了，别问我为什么。”离开前，慕清霄丢下一句令萧炎莫名其妙的话。
<br/>　　药尘深深的望着慕清霄离去的背影，道：“刚才听你们的对话，好像有着某种关系，但你怎么会跟这么变态的小子扯上关系？”
<br/>　　萧炎并没有回到，来自地球的事情，他打算永远烂在心里，经过刚才的事情，他才惊醒过来，若慕清霄是敌人，他与药尘的性命可就交代在这里了。
<br/>　　正所谓，吃一堑长一智。
<br/>　　萧炎语气坚定，道：“师尊，这件事情是我唯一的秘密，莫怪徒儿不能告诉您。”
<br/>　　望着萧炎坚定的眼神，药尘微微点头，也没有继续逼问下去，慕清霄来自何处，又有何目的，对他来说都无所谓，他的目的只有杀了韩风，并且继续活下去。
<br/>　　“焚决是不可能教给你了，他似乎在收集异火，你先将基础打好，高级功法往后自然会有。”
<br/>　　离开萧家后山，慕清霄瞥了眼古薰儿所在的方向，随后便转身离开，那里确实有一股强悍的气息，根据气息来判断，估计是四星斗皇。
<br/>　　他并没有回到客房中，而是以极快的速度来到魔兽山脉中，找到一处隐蔽的场所，设立幻术结界后，便开始炼化骨灵冷火。
<br/>　　骨灵冷火想要反抗，但瞬间就被九霄琉璃焱吞噬，随着时间的推移，慕清霄的气息开始攀上，化神一重中期，化神一重巅峰...
<br/>　　数日清晨，慕清霄身躯一阵，一股无形的波动在空中荡起一阵涟漪，随后向四周扩散而出，极强的劲风将周围树木吹的东倒西歪。
<br/>　　睁开明亮的双眼，其中透露着兴奋之色。
<br/>　　“化神二重中期，修炼速度简直像是做火箭啊，异火榜排名第十一的骨灵冷火就让自己提升如此恐怖，排名更高的异火，简直不敢想象啊！”
<br/>　　捏了捏拳头，肉身的力量增强了太多，最主要的还是，九霄琉璃焱的威力有所提高，这可是他保命的杀招之一啊！
<br/>　　慕清霄其实很清楚，他现在能够如此快速的提高修为，且气息稳定无比，其中一部分可能是因为资质的原因。
<br/>　　但最关键的却是《御女心经》在前期给自己打下了牢固的基础。
<br/>　　由此可见，一本高级心法对日后发展的重要性。
