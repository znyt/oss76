　　一爪之下，落长恨便是被楚风眠，活活捏死在了手中。
<br/>　　看到落长恨就这么陨落，被楚风眠活活捏死。
<br/>　　在场无数人的眼神中，都是充满了震惊，不可置信。
<br/>　　落长恨，在北域中都算得上是有些名气的天才强者。
<br/>　　北邙学院的种子弟子，在北邙学院中都是地位极高的存在，更是修炼长恨之道，赫赫有名。
<br/>　　这样的一尊强者，如今居然被人活活捏死，这让他们无论如何也不敢相信。
<br/>　　“此人的实力，到底达到了哪一步，落长恨都被捏死了，毫无还手之力，就算是生死境四重，五重的强者，都不一定能做到这一点吧。”
<br/>　　沧澜霸看着这一幕，也是不由的吸了一口凉气，浑身都是无比冰冷。
<br/>　　北邙学院的种子弟子，可没有泛泛之辈，没有一位能够混到北邙学院种子弟子的，都是天才中的天才，北域闻名的天才。
<br/>　　单单以实力，落长恨的实力都不比许多生死境二重，三重的武者要差了。
<br/>　　可现在，落长恨却是直接死在了楚风眠的手中，楚风眠的实力，到底是可怕到了什么地步。
<br/>　　更何况楚风眠的境界，才区区神力境四重，未来还有着无数提升的空间。
<br/>　　要是让楚风眠再度成长起来，得是多么惊人。
<br/>　　楚风眠从这颜古龙帝的洞府中，已经是得到了无数的好处了，今日只要是楚风眠离开，可就是彻底的养虎为患了。
<br/>　　“不能让这小子今日活着离开！枯朽天，战魔尊者，动用全力，不惜一切代价也要杀了此人！”
<br/>　　沧澜霸大喝一声，他看向楚风眠的双眼中，都是深深的忌惮。
<br/>　　这样的天才，在北域中真的崛起，绝对是他们的噩梦。
<br/>　　更何况他们今日跟楚风眠为敌，无论如何，楚风眠也不会放了他们。
<br/>　　“武道极限！天意为武！”
<br/>　　沧澜霸狂吼起来，一身的灵力凝聚到了极致。
<br/>　　在他的身边，无数的宝物，无数灵丹，奇物，灵石，都被他的身躯，直接收取。
<br/>　　这些东西，全部都被沧澜霸瞬间吞噬，令他的力量，顿时暴涨十倍！
<br/>　　一股股力量，并没有提升沧澜霸的境界，而是全部融入到了他的武道之中。
<br/>　　令现在沧澜霸的武道，简直是到了一种不可思议的地步。
<br/>　　“武神降世！”
<br/>　　“轰！”
<br/>　　一道光芒，断降落在了沧澜霸的背后，他背后的光芒中，隐约可以看见一位人影的样子。
<br/>　　这人影，便是传说中的武神，曾经将武道，达到极致之人。
<br/>　　现在沧澜霸拼劲一切，便是将传说中的武神虚影，都给呼唤了出来，这并非是以秘术呼唤。
<br/>　　而是他的武道，已经是达到了临界点，武神虚影，自动凝聚。
<br/>　　比起以秘术凝聚出来的，不知道要强大无数倍，就跟楚风眠凝聚出的远古杀神虚影一样，极为类似。
<br/>　　“武道神拳！”
<br/>　　沧澜霸顿时咆哮一声，一拳猛然冲着楚风眠轰击而出。
<br/>　　这一拳中，简直便是蕴含了武道的精髓，玄妙无比，看似简单，却是根本无法躲避的开。
<br/>　　“沧澜霸动用全力了！”
<br/>　　“我们也出手！”
<br/>　　枯朽天，战魔尊者二人对视了一眼，也不顾一切，爆发出了自己的一切底牌。
<br/>　　今日落长恨都死在了楚风眠的手中了，他们在不动用全力，只怕会被楚风眠各个击破。
<br/>　　无数强劲灵力，疯狂爆发而出，在枯朽天的面前，突然化为了一根巨大无比的巨斧。
<br/>　　这巨斧上，蕴含这腐朽之际的力量。
<br/>　　仿佛一切沾染到这巨斧上的存在，都会被腐朽。
<br/>　　“腐朽巨斧！腐朽天地！”
<br/>　　这一道巨斧，顿时冲着楚风眠，猛然斩了过去。
<br/>　　“小子，腐朽吧，在这腐朽巨斧面前，天地都要腐朽，天地都终有灭亡，腐朽的一刻，小小生灵，又如何挣扎？”
<br/>　　枯朽天冷笑连连，催动这腐朽巨斧，斩了过去。
<br/>　　他的这一斧，要斩杀的可不是楚风眠的身躯，而是楚风眠身上的寿元。
<br/>　　他的腐朽巨斧，足以让世间一切都化为腐朽，无论是多么强大的武者，都有衰老，陨落，腐朽的一日。
<br/>　　他的这一斧，要斩杀的便是楚风眠的寿命。
<br/>　　一斧斩出，诡异无比，令许多人根本便是都不知道，如何可以抵挡。
<br/>　　“腐朽巨斧？腐朽之道，倒是不错，不过可惜，天地会腐朽，但是我不会，我将永生，永远不败！天地灭，我长存！你的腐朽之道在我的面前，根本就是一个笑话！”
<br/>　　楚风眠看着腐朽巨斧斩来，他大手一挥，直接徒手之下，挡下了这腐朽巨斧。
<br/>　　楚风眠的手，仅仅是握着腐朽巨斧，将其中的威力，全部抵挡住。
<br/>　　“徒手抵挡！你这是在找死！腐朽之道！让他腐朽陨落吧！你的寿元，都会被我直接剥夺！”
<br/>　　枯朽天看着楚风眠的举动，大笑起来，顿时将腐朽之道，催动到了极致。

<br/>　　但是楚风眠却是站在原地，任由着无数腐朽之气，围绕在他的身边。
<br/>　　楚风眠的嘴角，露出的表情，不屑到了极致。

<br/>　　楚风眠身躯上，顿时凝聚出了一股极为诡异的力量，正是来自他身躯剑胎之中的造化之力。
<br/>　　造化之力，乃是天地最为强大的力量。
<br/>　　腐朽之力，虽然强大，但是造化永生，这腐朽之力，在造化之力面前，也没有丝毫意义。
<br/>　　“统统给我吞噬！”
<br/>　　楚风眠大口一张，这无数腐朽之气，直接就是被楚风眠一口吞噬，全部吞入了口中。
<br/>　　一股股恐怖至极的腐朽之气，却是被楚风眠直接吞噬，楚风眠的身躯，非但是没有被腐朽，反而是身上的气息，越来越强。
