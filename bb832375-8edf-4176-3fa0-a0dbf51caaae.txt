　　望着云悠悠消失的背影，苏莫眸中闪过沉吟之色。
<br/>　　他没有去追对方，是要给对方消化的时间，他心中清楚，自己不能有什么要求。
<br/>　　失去了记忆的青璇，已经不是以前的青璇了，他不能逼的太紧，要给对方接受的时间
<br/>　　而且，苏莫相信，对方一定不会无动于衷，日后肯定会主动找他。
<br/>　　苏莫伫立在山巅之上，望着下方无际的山林，怔怔的发呆。
<br/>　　良久之后，苏莫在山巅上盘膝而坐了下来。
<br/>　　手臂一挥，两片残碎的尸身出现在半空之中，正是杨天威的尸身。
<br/>　　在杨天威的尸体周身，还有大量鲜血环绕，均是其身体中流落出的鲜血。
<br/>　　苏莫的目光，落在了杨天威的左手手指上，那里有一枚黑色的储物戒。
<br/>　　唰！
<br/>　　手掌一探，黑色的储物戒自行脱离了杨天威的手指，向苏莫飞了过来，被他一把抓在了手中。
<br/>　　“源石应该在其内吧？”拿着黑色的储物戒，苏莫心中有些期待，期待源石就在储物戒内。
<br/>　　这个可能性很大，毕竟源石是杨天威竞拍，应该不可能在扬天成的手中吧！
<br/>　　带着浓浓的期待，苏莫意念一探，直接进入了储物空间之中。
<br/>　　在储物空间内，苏莫看到了很多东西，中品灵晶足有七八千，各种宝器、玉简等也是不少。
<br/>　　而那枚劣质的源石，正静静的躺在角落中。
<br/>　　“果然在这里！”苏莫面上露出了一丝喜色，这源石可能对他有大用，而且就算无用，这么高的价值，也是一笔巨富。
<br/>　　也没有仔细查看，苏莫郑重的将黑色的储物戒收了起来，而后又看向了半空中的杨天威尸身。
<br/>　　吞噬战魂升腾而起，吞噬之力席卷，一瞬间，杨天威的尸身便被吞噬战魂了。
<br/>　　而后杨天威体内的一切，尽皆被苏莫吞噬了。
<br/>　　而后，苏莫身上的气息，立刻开始了增长。
<br/>　　虽然杨天威的神丹早已破碎，但其毕竟是武尊境三重的武者，肉身血气亦是非常的强悍。
<br/>　　与此同时，苏莫感觉到了一丝特殊的力量，进入了他的身体之中。
<br/>　　“天骨之体！”
<br/>　　苏莫低声自喃，这股特殊的力量，正是杨天威的天骨之体的血脉力量。
<br/>　　随即，苏莫将这股血脉力量，完全的吸收了，与他自身的血脉缓缓的融合在了一起。
<br/>　　至此，他再添一种血脉体质。
<br/>　　静坐了小半个时辰，苏莫停止了修炼，身上气息缓缓的归于平静。
<br/>　　他的修为增加了不少，达到了武尊境一重的中期，但距离武尊境一重的巅峰，还差距甚远。
<br/>　　少倾，苏莫长身而起，破空而去。
<br/>　　现在，他身上灵晶众多，不需要为资源发愁了，要返回道宗好好修炼，为接下来的古魔墓地做准备。
<br/>　　两日之后，苏莫回到了道宗，回到了自己的宫殿之中。
<br/>　　道宗相当平静，他特意的注意了一番，并没有因为他杀了杨天威，而发生什么大事。
<br/>　　既然没有什么大事，他也能相安无事的闭关了。
<br/>　　宫殿内，练功房之中，苏莫盘膝而坐，沉吟了片刻，他手掌一番，源石便出现在了手中。
<br/>　　轻轻的抚.摸手中的源石，一股淡淡的温热感传来。
<br/>　　“这源石，到底是有何奥妙？”苏莫低声自喃，少倾之后，他意念探出，钻入源石之中。
<br/>　　在源石之内，苏莫感受到了一丝玄妙的力量，这种力量是一种非常淡薄的气流。
<br/>　　这气流呈灰白之色，充满着玄妙的气息，仿佛来自于太古、又仿佛来自于异域世界。
<br/>　　苏莫皱眉思索了片刻，立刻催动了吞噬战魂，将源石中的这丝玄妙的气流吸了出来。
<br/>　　气流如发丝般纤细，在他的手掌中漂浮不定。
<br/>　　苏莫能真切的感受到，这丝玄妙的气流，蕴含着惊天伟地的力量，如不是他的手掌中蕴含大量的玄力，以及吞噬之力的牵制，他都控制不住这丝气流。
<br/>　　“这就是本源之力？”苏莫眉头微皱，他感觉这本源之力，和他的玄力之间，仿佛有一丝莫名的联系。
<br/>　　这让他心中疑惑不已，本源之力，到底是什么力量？
<br/>　　“这该不会就是混沌之力吧！”
<br/>　　突然，苏莫脑中闪过一道亮光，这个想法让他自己都激动了起来。
<br/>　　越是想下去，他越是觉得这有一丝可能。
<br/>　　不过，思索了良久，他都没有什么头绪，也无法证明自己的猜测。
<br/>　　“不知道真贤师兄在不在宗内？”片刻之后，苏莫将这丝本源之力吸纳进了丹田之中，便长身而起，走出了自己的宫殿。
<br/>　　他准备去找真贤道圣，详细的了解一番这本源之力，若真的如他猜想的这般，这说不定是他的大造化。
<br/>　　没过多久，苏莫便来到了问道山上方，真贤道圣的行宫之前。
<br/>　　“真贤长老！”苏莫站在门前，抱拳轻呼一声，但是，他等了良久，连续呼唤了三声，宫殿内也没有丝毫反应。
<br/>　　苏莫有些无奈，看来对方并不在宗内，只能改日再来了。
<br/>　　随即，苏莫只能无奈的返回了自己的宫殿。
<br/>　　既然真贤道圣不在，苏莫便也没有再研究这本源之力，而是开始修炼，还是先行提升实力要紧，毕竟古魔墓地开启的时日已经不远了。
<br/>　　他进入了玉镯空间之中，取出大量的灵晶，开始疯狂的吞噬了起来。
<br/>　　大量的灵晶被消耗，苏莫的修为不断的增长，修为到了武尊之境后，提升的速度大大的减缓了。
<br/>　　即便是吞噬灵晶修炼，提升的速度也是比以前暴降十倍。
<br/>　　五日之后，足足消耗了四千多块中品灵晶，苏莫的修为终于达到了武尊境一重的巅峰。
<br/>　　而他所用的时间，比武帝之境时，慢的太多太多了。
<br/>　　苏莫没有冲击武尊境二重境界，因为他心中心中清楚，瓶颈如桎梏，根本不可能冲破瓶颈。
<br/>　　不过，虽然没有达到武尊境二重，他的实力也是提升了一大截。
<br/>　　在武尊之境，即便是在小境界内的提升，跨度都是不小。
<br/>　　随后，苏莫有巩固了几日的修为，便再次离开了宫殿，前去找真贤道圣。
<br/>　　本源之力不弄清楚，他实在不甘心，这对他太重要了。
