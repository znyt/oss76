　　减兵增灶的意思其实很简单，就是把兵一点点的从营地里面撤出去，但是为了不让敌人看出其中的门道，兵撤走了却要再增加灶火。
<br/>　　毕竟有多少灶火就证明有多少人吃饭，灶火越多则证明吃饭的人也就越多，在古代这是一个判断敌军兵力多寡的一个重要依据。
<br/>　　黑齿常之所说的减兵增灶其实就是这个意思，一方面他们把十万靺鞨骑兵从军营之中撤出去，另一方面他们却要不断的增加灶火的个数。
<br/>　　倭人就在他们的对面，相对峙的情况下不可能对他们这十万大军不管不问，一定会派出探子在营地周围。这样一来越来越多的灶火会让那些倭人产生误判，会以为唐军等来了支援。
<br/>　　十万大军已经让倭人难以应付，如果再有援兵的话，估计倭人非得崩溃了不可，所以黑齿常之判断倭人在发现这一情况之后一定会派出主力部队对他们发起突袭。
<br/>　　而如果倭人真的发动了袭击，那就等于是将自己的尾巴暴露了出来，十万靺鞨铁骑就可以借机一拥而上，将那些倭人彻底消灭。
<br/>　　这样的办法其实他在对付大祚荣的时候已经用过一次，而且取得了很大的战果，这给了黑齿常之足够的信心。
<br/>　　钦陵也是一个心思机敏之辈，只听到‘减兵增灶’立刻就明白了黑齿常之的意思，笑着用手指虚点了他几下：“你这是扮猪吃老虎啊，明明有了主意却不说，非得逼的紧了才会露出只言片语。”
<br/>　　“大都督莫怪，末将这已经是最后的办法了，如果这样不行的话，末将可就一点招也没有了。”黑齿常之嘿嘿笑着说道。
<br/>　　“行了，别说那些没用的，本都督觉得你的这个办法很不错，可行。不过具体的细节还需要再考虑考虑。”钦陵摆了摆手，示意黑齿常之不要想的太多。
<br/>　　在他看来办法并没有什么好不好，只有实不实用，黑齿常之的办法很好，虽然有些老套，但只要能引出倭军就好。
<br/>　　要知道，以唐军本就强大无匹的实力来说，如果还有援兵在不断的加入，这对倭军士兵也是一个不小的压力，如果倭人不尽快做出应对，那么很可能他们的士兵还没等打仗自己就先崩溃了。
<br/>　　……
<br/>　　而事实上钦陵和黑齿常之的推测一点错都没有，倭人的确是藏着一支人数多达二十余万的尖兵，这些人装备着大唐的装备，有合身的皮甲，锋利的刀剑以及无数的长弓短弩。
<br/>　　这是一支可以决定战场胜负的强大力量，自从倭岛被攻击了之后，不管是山背大兄还是秦吉胜都没有再动用过这支力量。
<br/>　　现在两军合在一处，以他们为骨架辅以数十万的老弱残兵，倒也不是没有一战之力。
<br/>　　但是黑齿常之的减兵增灶却把秦吉胜和山背大兄推到了悬崖的边上，两人在听到探子的回报之后也有些惊慌失措。
<br/>　　尤其是已经交出兵权的山背大兄，这家伙已经堵上了自己的一切，如果这一次战败了，秦吉胜的死活他并不在乎，可是他自己却要面临失去一切的风险。
<br/>　　这是山背大兄无论如何也不能接受的，付出了这么多，如果依旧无法改变倭国的命运，依旧无法改变自己的命运，那此前的挣扎还有什么意义。
<br/>　　“服部将军，这场战事不能再拖下去了，朕以为我们现在就应该主动出击，在敌人立足未稳的时候打他们一个措手不及。”带着无限心事，山背大兄找到了秦吉胜，催促他尽快出兵。
<br/>　　而秦吉胜却并没有如他所想的那样或者反对或者答应，而是露出一脸愁容：“陛下，其实不用你来催我，我也想要与唐军决战，夜长梦多这句话可并不是在大唐才有的。”
<br/>　　“但是您想想，万一这一切都是假的呢？大唐兵法诡谲，不是我们可以度量的，如果他们用的是增灶之法我们应该如何应对？又或者这是唐军的埋伏怎么办？陛下，我们不能只考虑眼前看到的，那些没有看到的才是最重要的！”
<br/>　　不得不说，秦吉胜这家伙已经把大唐的兵法了解的差不多了，山背大兄就是十个八个绑在一起也不是他的对手。
<br/>　　可是有些时候形势比人强，这不是凭借个人的聪明就能弥补的，就好比一个人不可能像鸟一样飞是一个道理。
<br/>　　山背大兄当然也想过秦吉胜提到的问题，可是眼下的形势已经容不得他们再想那么多，这位手中已经没有了兵权的倭国皇帝在沉默了片刻之后幽幽叹了口气：“服部将军，你说的这些朕觉得很有道理，可就算如此，我们也还是必须出兵，毕竟我们知道这可能是唐军的障眼法，但是我们的士兵并不知道。”
<br/>　　“如果继续这样等下去，你知道士兵需要承受多大的心理压力？士气会受到多大的影响么？再说万一这一切是真的呢？万一唐军真的有支援呢？大唐的海上军事力量已经不是我们可以匹敌的了，他们完全有能力再输送一倍甚至十倍的兵力过来。”
<br/>　　“如果真是这样的话，我们再继续等下去只能是自掘坟墓，这些问题你想过没有？”
<br/>　　秦吉胜不说话了，似乎正在做一个十分艰难的决定一样。
<br/>　　出兵或者不出兵？赌上倭国或者自己的未来？到底应该怎么选？
<br/>　　这可不是在大街上买东西，到底买一斤还是两斤的选择，而是关于生存与死亡的选择。
<br/>　　“呯”一声，物部氏族长狠狠的拍了一下秦吉胜的桌子：“服部，你还在考虑什么？难道等下去就一定能够胜利么？早一天死与晚一天死有什么区别？”
<br/>　　“不错，打吧，拖下去没有任何意义，如果让大唐人在气势上占了上风，我们这仗真的就没法打了。”秦吉胜属下的将官也开始跟着吼道。
<br/>　　他们这些人全都已经将事情看的清清楚楚，知道不管是打还是不打，其实倭国都没有任何希望，在这样的情况下那就不如赌一次，杀一个够本，杀两个赚了。
