　　察觉到小医仙的一丝情绪波动，慕清霄搂着她的手掌不由的紧了紧：“不用担心，厄难毒体并非绝体，哪怕是绝体，只要有我在，就不会让你损伤丝毫。”
<br/>　　微微颔首，她那柔美至极的脸颊上露出难以掩饰的幸福笑容。
<br/>　　把玩着柔顺幽香的青丝，慕清霄再度沉吟起来。
<br/>　　原本，他并不打算这么早就来迦南学院，而是打算带小医仙前往出云帝国，那里，对于她的修行更有帮助。
<br/>　　但比起修行，更重要的是她的性命。
<br/>　　厄难毒体对于现在的他来说，终究是个难题，虽然能够用神火设下简单的封印，但终究只能封印一小段时间。
<br/>　　封印的同时，也在压制毒体，一旦压制时间过去，毒气冲破封印全力爆发，毒气扩散，到时候可就不是普通封印能解决的了。
<br/>　　毒气冲进脑海，佳人就会消香玉陨，画面慕清霄简直不敢想象，这也是他选择提前来迦南学院的原因。
<br/>　　因为在这里，有能够暂时封印厄难毒体的办法。
<br/>　　哪怕修炼速度慢上一些，也要想办法重设一道封印，完全压制住毒体，直到能完全控制。
<br/>　　...
<br/>　　...
<br/>　　数日清晨，两人早早的醒来。
<br/>　　临时营地偌大的空地上，导师霍德早已命人准备就绪，听到外面响起狮鹫低吼声，不再迟疑，两人转身走出房间。
<br/>　　此时，空地上正匍匐着一只高逾数丈的狮鹫，而狮鹫上面正站着一名青年，正是昨日与霍德导师在一起的执法队员。
<br/>　　“学弟学妹，别愣着了，快点上来吧，导师一早就出去巡逻了，由我护送你们去迦南城。”
<br/>　　“多谢。”
<br/>　　道谢一声后，慕清霄就跃上狮鹫，小医仙轻轻落在背上。
<br/>　　等两人站好，男子嘴中吹起一道哨声。
<br/>　　旋即，匍匐在地上的狮鹫翅膀一振，巨大的身体就离地而起，在高空盘旋几圈后，向着迦南城飞去。
<br/>　　望着越来越渺小的和平镇，慕清霄抬头望着碧空如洗的蓝天，低喃道：“迦南学院，真是有意思。”
<br/>　　......
<br/>　　......
<br/>　　狮鹫飞行了一夜，中途休息了两次，在第二天清晨，旭日东升，第一缕阳光倾洒而下。
<br/>　　闭目养神中，两人不约而同的睁开双眼，目光投向遥远的天边，在那里，一座大的令人咋舌的城市轮廓，隐藏在清晨的薄雾之中。
<br/>　　即便是站在高处，依旧未能将整座城市尽收眼底，由此可见，这座迦南城的面积，究竟有多么的庞大。
<br/>　　“学弟学妹，快要到迦南城了，等下了狮鹫，稍等片刻就有导师来迎接你们。”
<br/>　　慕清霄微微点头，昨日霍德已经将他们的信息传递给了学院，他们所表现出的天赋顶尖，有导师来迎接实属正常。
<br/>　　“多谢。”
<br/>　　“没事，等学弟学妹将来有了成就，到时候对学长照顾一二就好。”
<br/>　　青年摆了摆手，随口开了个玩笑，再次吹响口哨，狮鹫便低吼一声，双翅一展，便对着下面庞大的城市俯冲而下。
<br/>　　抵达迦南城飞行停留所后，交谈了几句，青年就命狮鹫腾空而起，朝着原路返回。
<br/>　　刚才青年提醒过，只要早停留所稍等片刻，便会有导师来迎接，所以慕清霄与小医仙也没有乱走动。
<br/>　　约莫十分钟后，一名身穿迦南学院导师制服的女子走了进来，女子容貌颇为俏丽，看上去虽普通了些，却是一名大斗师。
<br/>　　女子目光在停留所中环视一圈，当看到慕清霄与小医仙后，微微一愣，随后脸上就浮现一抹微笑，迎面走了过来。
<br/>　　“两位，是慕清霄同学，小医仙同学吧？”
<br/>　　“没错。”
<br/>　　上下打量两人一番，女导师微微一笑，道：“你们好，我是迦南学院导师若琳，霍德导师执法，就将你们安排给了我，也正好还了他的人情。”
<br/>　　若琳？
<br/>　　慕清霄记得，若琳不就是当初负责乌坦城招生的导师么，看来是没错了。
<br/>　　相互介绍一番后，若琳就带着两人离开停留所。
<br/>　　来到停留所外，街道路边停放着一辆，由一阶魔兽碧血兽拉着的车厢，车厢上刻有迦南学院的标志，正是迎接新生用的专属交通工具。
<br/>　　坐在车厢内，若琳就开始为两人介绍迦南城中的一些注意事项，比如迦南城中不能强行御空飞行，否则会遭到攻击等等。
<br/>　　当然，介绍归介绍，她可不认为面前的两位小年轻拥有这种实力。
<br/>　　将一些注意事项叮嘱完毕，若琳道：“比试选拔已经过去半个月，但你们两人的天赋，放在整个学院都算是尖子生，所以学院破例，让你们直接晋级......”
<br/>　　半小时后，碧血兽停下，跟随若琳下车后，映入眼中的就是占地面积极广大的迦南学院。
<br/>　　“这里就是迦南学院了，跟我来吧，我带你们转一圈，等时间差不多，就顺路带你们去另外一处地方。”
<br/>　　走进迦南学院中，道路上已经有许多穿着制服的学生，目光都炯炯有神。
<br/>　　两人跟在若琳身后，周围路过的学生都投来异样的目光，毕竟两人的容貌，不管在何处恐怕都会成为焦点。
<br/>　　当然，斗气大陆上实力为尊，容貌在优秀，没有实力，顶多也就是个花瓶。
<br/>　　学院占地非常的广，哪怕比地球上的高级学府都大的多，如果徒步的话，哪怕一整天的时间，恐怕都无法走完。
<br/>　　半个小时后，若琳带着两人经过教室大楼，抬头看了一眼天色，道：“时间也差不多了，我们走吧。”
<br/>　　几分钟后，跟在若琳身后，穿过林荫小道，最后来到迦南学院的西侧，四周建筑并不多，而那屹立在最中间的大楼，与周围显得格格不入。
<br/>　　大楼看上去已经有些年代，透露出古朴的气息，正面刻着三个大字，字体笔劲沧桑，长时间盯着，竟令人有些失神。
<br/>　　“图书馆！”
<br/>　　望着图书馆三个大字，慕清霄面色如常，但情绪却微微变化。
<br/>　　若是他记得没错，迦南学院图书馆深处，可是有着两位斗宗巅峰的老怪物，真是怕什么就来什么。
<br/>　　原本，他并不想与迦南学院高层接触，现在看来，不接触也得接触了，希望能蒙混过关吧，实在不行就只能另想办法了。
