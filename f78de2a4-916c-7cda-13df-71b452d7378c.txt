        &lt;h3&gt;第三卷第二百九十四章  推理&lt;/h3&gt;
<br/>
<br/>        陈子昂从四姐陈子乔的家里出来，心里松了一口气，陈子乔不是无药可救，她只不过是一些人手中的棋子，现在的这棵棋子可用可不用，那么洛城的项目可以加大力度，最起码要能动动夏颜，给四姐调动的机会，让四姐能够脱身。
<br/>
<br/>        夏颜频频来往于陆家，她一定会利用陆家来对付她陈子昂的，还有冯睿，她曾经的好二嫂，她们似乎是达成了一致，要和陈子昂不死不休。
<br/>
<br/>        呵，她陈子昂现在还真是四面楚歌，连自己的亲人都已经和她离心离德了。
<br/>
<br/>        陈伟冯现在在夏颜的手下工作，被夏颜管得服服帖帖的，就连爷爷生病他们都不曾来探望，亲情在他们的心里一文不值，还有陈天天，想起陈天天陈子昂有些心疼，小时候她也是经常带着陈天天玩的，可是她对陈天天又没有办法，谁会和自己的娘亲过不去？
<br/>
<br/>        这几天陆陆续续的有消息传来，守中在北京带着俩个孩子，他们实验的安防系统基本能够使用，守中已经将这一套系统装在了家里，陈虞和落妍俩个小东西正在帮忙完善中，陈子昂一直很庆幸她的工作从来都没有隐瞒过俩个孩子，家里的书籍他们随便看，她的电脑他们随便翻，导致俩个孩子对她电脑里的东西滚瓜烂熟，她写的东西陈虞和落妍都会拿出来分析，这一次的防护系统就是他们娘三个人的智慧结晶，为了生存，他们娘三都是拼了脑细胞，原本只是之前的构思，想着以后会不会作为一个项目加以推广，没有想到张函和他们相见之后带来的一系列反应，让他们娘三同心协力完成了一个奇迹，当然这也少不了守中的一个功劳，守中不但是一个保镖他还是一个信息收集能手。
<br/>
<br/>        想起孩子，陈子昂的心里满满的都是幸福，她从来都没有怨恨过张函，相反有时候她还很感激张函带给她俩个聪明可人的孩子。
<br/>
<br/>        陈子昂要去探望三姐，她边开车边给江俞轩打电话：“俞轩，我现在去医院看望我三姐，大概得一个小时，一个半小时后我们在榴花见。”
<br/>
<br/>        陈子昂将晚上吃饭的地方定在了榴花，也就是凌辰住的地方，这样对凌辰来说是很方便的，也没有必要让人接来送去。
<br/>
<br/>        江俞轩见陈昂让他提前过去，心里暗暗欢喜，他们终于有了俩个人相聚的时候，他要解释一下上次那句话，他要告诉她，他只是想气气陈子昂，他胡说的。
<br/>
<br/>        陈子昂到了医院，三姐能侧身了，偶尔能坐起来，不能坐太久。
<br/>
<br/>        陈子澜看着自己这个最小的妹妹，她现在已经褪去了青涩，显得成熟而稳重，娇媚而美好，脸上带着淡然的微笑，似乎有一种超脱与她这个年龄段的睿智和祥和。
<br/>
<br/>        她们是姊妹，而对于陈子澜来说，陈子昂就像是一个谜，她这些年的经历大家知之甚少，妹妹不说，她们都不问，不是不关心，而是尊重她，尊重她的生活方式，尊重她的选择。
<br/>
<br/>        陈子昂问候了陈子澜的身体情况，突然话题一转问到了甸城的情况。
<br/>
<br/>        “三姐，我很奇怪，为什么你会在甸城出车祸，我一直都没有问过你原因，你能告诉我吗？”
<br/>
<br/>        陈子澜知道陈子昂要去甸城，思虑再三，她觉得应该让陈子昂知道甸城并不是表面看到的那样。
<br/>
<br/>        “我去甸城，其实是去查案子的。我们达到甸城后前几天基本都在室内调取档案，哪里都不能去，说白了，我感觉我被变相的限制了自由。后来我就和同事们商量，我们俩个人出去，俩个人在档案室，我租了一辆车，和郑浩去了我们认为有问题的地方，果然，我们在哪里有所发现，但是我们只有俩个人，为了不打草惊蛇，我们原路返回，在返回的途中我们出车祸了。”
<br/>
<br/>        “你们去的那个地方是什么地方？”
<br/>
<br/>        “甸村，感觉比较偏远。”
<br/>
<br/>        “你们为什么会选择那个地方？”
<br/>
<br/>        “我们当时在档案里发现那个地方曾经被查过一次，是北京的人直接带人过去的，当时还带走了一个叫公孙楠的女人，将人带走之后便没有下文，后续收尾工作是当地部门进行的。”
<br/>
<br/>        “北京？公孙楠？时间？”陈子昂看着陈子澜。
<br/>
<br/>        “我和你姐夫怀疑是闫欢，而且张函也承认了是他带走了闫欢，回来后将闫欢的所有资料消除了，后来将闫欢送到了你五姐的超市。”
<br/>
<br/>        “闫欢已经去了甸城，她第一次去甸城是陆玉他们送过去，闫欢的青春在甸村消磨完了。张函带人过去将她带走，证明闫欢是离不开甸村的，能让一个人不能离开那个地方，那个村子是有问题的，谁有那么大的权利控制整个村庄？”陈子昂皱着眉头分析。
<br/>
<br/>        “子昂，这些不是你考虑的问题，你去甸城好好的做你的生意，不要接触一些不好的事情，切记！”陈子澜叮咛陈子昂。
<br/>
<br/>        “你确定你们碰到了不好的东西？”
<br/>
<br/>        “郑浩是这方面的专家，他的嗅觉是非常灵敏的，他在一个孤儿的院子里发现了不同寻常。”
<br/>
<br/>        “三姐，你们去甸城接待你们的应该是同系统的人吧？你们同系统的在甸城最高领导是谁？”陈子昂看了一眼陈子澜，她害怕陈子澜不给她说。
<br/>
<br/>        “秦寿！我们系统的榜样和英雄。”
<br/>
<br/>        陈子昂沉默了。
<br/>
<br/>        陈子昂刚开始怀疑谷强是她的哥哥，她认为谷强失忆了，后来谷强想和她一起做甸城的信息化，为了搞清楚谷强的身份她同意了，在他们电话联系的过程中，她坚信谷强就是陈子寒，后来祁峥也过去了，陈子昂便从祁峥的身上入手，隐隐约约中她知道了祁峥其实和陈子寒一样，都有着双重身份，当谷强犹豫不决，不知道该不该拉上陈子昂时，在一次通话的时候，谷强细细的了解信息化方案，并且对陈子昂说：“陈总，甸城不似别的地方，我其实又想你来，又不想你来，要不，你就不要来了吧，有我和祁峥就够了。”
<br/>
<br/>        “谷总，我们身上都流着同样的血，你能做的，我也能做，为什么要将我排斥在外？只是因为祁峥是男人吗？”从那之后谷强便同意她安排人过去做方案。
<br/>
<br/>        虽然她和谷强一起布局了很久，但是谷强从来都没有告诉过她甸城的情况。
<br/>
<br/>        当张函问她知道陈子寒多少的时候，她已经确认了谷强就是陈子寒，在当天晚上她又让守中通过别的途径确定了陈子寒的身份。陈子寒难道真的只是想做信息化？他隐藏了这么久的身份是为了什么？这些年甸城风平浪静，按说陈子寒已经没有必要再在甸城待下去了，为什么他还迟迟不回家？
<br/>
<br/>        陈子澜的话让陈子昂心里亮起了一道光，那久久不散的疑虑解开了。
<br/>
<br/>        张函告诉她，他是陆家的女婿，他会和陆家是一体，他们会成为对立面，那么是不是可以确认陆家是最后的操盘手？
<br/>
<br/>        陆家黑白通吃！西城的数据已经显示陆家其实已经盘根错节，掌握了大部分的资源，陈子昂以为他们只是贪婪，没有想到他们.........
<br/>
<br/>        原来，他的六哥只是用信息化作为诱饵！甸城信息化项目是西省总和的几倍！如此大的诱饵，后面藏着多么大的黑洞？
<br/>
<br/>        有一个村庄，那么是不是有两个村庄或者是三个村庄？甸城地处偏远，有多少个村庄隐藏在看不加的地方？
<br/>
<br/>        几年前陆玉能将闫欢送到那样的地方，致使闫欢不能回家，他们究竟到了什么地步？
<br/>
<br/>        陈子昂打了个寒颤！
