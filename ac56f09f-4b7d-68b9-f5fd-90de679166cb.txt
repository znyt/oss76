　　“如果内心觉得自己没错，就算是在再好的地方，依然觉得没错，若是有错，就算身在闹市，也一样可以静思己过。”
<br/>　　一个坚毅的声音，从亭子里面传来，两人转头看过去，才发现在凉亭靠外侧坐着一个人，因为被凉亭的立柱遮挡，过来的时候没有注意到。
<br/>　　一个身穿蓝色道袍的男子，坐在悬崖边上，没有面壁，而是盘膝坐在亭子的最外头，凌空而坐。
<br/>　　毛小方拱手笑道：“道兄所言极是。”
<br/>　　叶少阳走进凉亭，正好这个道士也转过头来打量他们。
<br/>　　“道渊祖师！”叶少阳一眼就认了出来，虽然面前这个男子看上去也只有二十多岁，浓眉大眼，但是脸型轮廓和五官，都跟老年的道渊真人一样，不是他能是谁。叶少阳可算是见到一个自己认识的人了，虽然在人间的时候跟他关系并不算太好，还在悬空观打过一架，不过叶少阳跟他并没有本质的闹矛盾，在这个世界见到他，内心十分地激动，忍不住这么叫了一声。
<br/>　　道渊真人愣了一下，上下打量他，发现不认识，冷着脸说道：“我好像不认识你，不过你这声‘祖师’切莫峦叫，我只是龙虎山二代弟子，当不起‘祖师’二字。”
<br/>　　叶少阳有点不好意思地笑了，自己刚才也是激动了，直接叫了百年后的称呼，当下嘿嘿一笑，说道：“你有二十多岁了吧？”
<br/>　　道渊真人一下子也愣住了，没想到叶少阳会突然这么问。
<br/>　　“别别，我不是要给你介绍对象。只是我一直都听说你有一百多岁，我以前是怀疑的，现在我真的信了。”
<br/>　　道渊真人看着他，像看一个神经病。这眼神，让叶少阳自己也觉得自己有点神经，当下干咳了两声，缓了缓，跟他介绍了毛小方和自己。
<br/>　　“茅山三十九代传人？”道渊真人看着他，嘴角带着一抹冷笑，说道：“我不知道你是谁，不知道你来找我干什么，但我很佩服你的勇气，我龙虎山与茅山世代相交，同理连枝，我对茅山的了解，比你想象的多的多，茅山现在的掌教是三十六代，三十九代，只怕要百年后去了。”
<br/>　　叶少阳耸了耸肩，道：“你说的没错，我还真是百年后的茅山传人。”
<br/>　　道渊真人当然不信，不过也没有再往下问，他已经断定叶少阳是个招摇撞骗的人，这样的人在江湖上很多，冒充茅山的，龙虎山的，麻衣神相的……比比皆是，他只是很好奇，眼前这个人为什么要到自己面前来行骗，不知道有什么目的，因此也没赶他走，转头对毛小方说道：“天道宗的道兄，来此何干？”
<br/>　　“我是陪叶天师来的。”毛小方拱了拱手。
<br/>　　“茅山的……叶天师。”道渊真人把“天师”两个字咬的很重，说道：“你来找我，便说个清楚，否则你打着茅山的招牌来这里招摇撞骗，我为了茅山的清誉，恐怕要让你不那么容易离开……”
<br/>　　他冰冷的眼神，让叶少阳想到了百年后那位不卑不亢、一身正气的道渊真人，神色都是一样的。
<br/>　　叶少阳在他对面坐下，望着悬崖对面的远山，用力叹了口气，说道：“从哪里开始说好呢，毛道长也在这，我也不瞒你们了，我叫叶少阳，我是大唐国师叶法善的后人，同时我也是茅山第三十九代掌教……”
<br/>　　毛小方和道渊真人听了，全都震住了。毛小方跟叶少阳一路走来，是信任他的为人的，听了他的话，还是忍不住有些怀疑。道渊真人更不用说，眼神中满满都是怀疑，甚至有点按捺不住。
<br/>　　“我师父是青云子，茅山第三十八代掌门——”
<br/>　　“满口胡言！”道渊真人猛然站起来，“你从一开始就跟我撒谎，我没法听下去了，我……嗯？这是什么？”
<br/>　　“天师牌，你能看得出这是真的假的。”叶少阳把自己的天师牌递了过去。
<br/>　　道渊真人接了过去，看了一会，灌入罡气，天师牌立刻亮起来，放出了柔和的明光，即使在白天也能看得见。
<br/>　　道渊真人大吃一惊，仔细看这块天师牌的材质，和上面附着的那股力量，他能感受到到这股灵力中的古朴之感。绝不是现代的东西。
<br/>　　这块天师牌的主人，绝对是道教一代宗师！
<br/>　　“这是谁的天师牌，你是怎么得到的？”
<br/>　　“这是我先祖叶法善传给我的，自然是我的，上面的一片枫叶，就是我家的姓氏。”叶少阳把天师牌拿回来，轻轻一握，立刻发出夺目的光彩，证明了自己主人的身份。
<br/>　　“你们先别震惊，我从头说起……我是从2016年来的，公元，今年是1922年……我是一个穿越者，你们可能不知道这是什么意思，我……”
<br/>　　叶少阳挠了挠头，想来想去，还是不要从头说了，免得泄露将来的事情，毛小方还好，自己那个世界没他，说明他没能活到九十多年后——这也正常，因为炼丹和懂得养身之道，虽然修道之人只要不被邪物弄死，一般都能比较长寿，但是能活到百岁以上的还是很少的，况且毛小方现在就三十多岁了——道渊真人的话，一百年后还没死，自己如果说出过多的未来世界的信息，对他来说，并不是一件好事。
<br/>　　叶少阳简单说了自己的经历，尽量不涉及别的人名，最后说到空界之战，自己被徐福用山海印带到了九十多年……
<br/>　　来到这个世界之后，叶少阳还是第一次对人吐露这些事情，而且听众有两个。叶少阳一口气说下来，说的那个畅快，说完之后，再看两人，简直目瞪口呆。
<br/>　　叶少阳看看毛小方，又看看道渊真人，说道：“我说完了，你们怎么看？”
<br/>　　道渊真人沉吟了一下，吐出两个字：“不信。”
<br/>　　叶少阳耸了耸肩，道：“怎么样你才会相信？”
<br/>　　“怎么样我都不信。我不相信会有这种事情发生，我也没听说过什么山海印。”
