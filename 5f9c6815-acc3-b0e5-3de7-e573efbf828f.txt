　　白尘这莫名其妙的一跪，还差点将李清照整的有点被感动起来，这白尘可是大少年脾气，心高气傲。
<br/>　　以前总是对她死缠烂打，每每讨好。
<br/>　　这次为了讨好她，居然跟别人下跪，牺牲那么大啊。
<br/>　　白尘可不是一般人，他的背景很深，在学校也是风云人物。
<br/>　　就是这样的一个人，为了她下跪，自己魅力那么大了？
<br/>　　李清照不禁摸了摸自己的小脸，有点怀疑起来。
<br/>　　“白尘，你快起来，你为我买条蛇还冲别人跪下，我多不好意思呀”
<br/>　　李清照忽闪忽闪眼眸，怪不好意思的，别人可是一片赤诚呀。
<br/>　　白尘闻言，真的想骂娘，小爷是心甘情愿的跪下来吗，不敢置信的看着眼前的少年，这家伙身上还带电？
<br/>　　电的自己浑身麻木，想站都站不起来。
<br/>　　“我...我...”
<br/>　　白尘张了张嘴，但是很是结巴，嘴巴中还冒出了白烟。
<br/>　　看的李清照一愣一愣的。
<br/>　　“蛇...蛇...蛇...”
<br/>　　白尘跪在地上结结巴巴，眼瞳在收缩，好像看到了什么。
<br/>　　李清照却道：“白尘，我跟你开玩笑的，你别当真，我虽然喜欢小蛇蛇，但我会通过自己的努力得到它，你快起来吧，别跪着了”
<br/>　　涂小安心里笑了笑，他是暂时起不来了：“丫头，你朋友喜欢跪着，就让他先跪着，你带我去前面看看”
<br/>　　哼，跟我玩阴的，还比力量大，我电不死你。
<br/>　　小小恶趣味，涂小安朝着前方一片草地走去。
<br/>　　“蛇...蛇...”
<br/>　　白尘依然结结巴巴，一只手指着某个地方颤抖。
<br/>　　李清照不好意思让白尘真的这样跪着，准备将他扶起来，忽然，一个余光下，顺着他的手指方向看去。
<br/>　　只见前面地面有一条鲜艳的小蛇在地上蜿蜒曲折的游曳。
<br/>　　咦？
<br/>　　“小哥哥，你又放出来一条蛇？”
<br/>　　李清照看着涂小安询问。
<br/>　　涂小安也注意到地上的蛇，微微诧异，这是一条赤链蛇，看着条纹鲜艳，头呈三角形，有点可怖，但却是一条无毒蛇，而且是非常普通的蛇。
<br/>　　“与我无关”
<br/>　　涂小安将赤链蛇抓起来，看了两眼，有点莫名其妙。
<br/>　　这校园怎么会出现一条赤链蛇，莫非谁在恶作剧放出来的。
<br/>　　涂小安这样想，李清照也早就四处找寻，看看周围有没有谁躲着恶作剧。
<br/>　　忽然，李清照清丽的面容满是错愕起来，但错愕之后，很是兴奋的指着前方：“哇...好多蛇”
<br/>　　嗯？
<br/>　　涂小安放眼望去，还真的是好多蛇，前面草坪上，很看到不少S型游曳的蛇，有黑斑条纹，有红褐条纹的蛇品种。
<br/>　　怎么回事呀？
<br/>　　涂小安一时之间有点懵逼，这个大学校园怎么会出现那么多条蛇。
<br/>　　莫非本蛇王出现，惹的很多蛇出来顶礼膜拜。
<br/>　　这不可能，涂小安根本没有散发蛇王的气息。
<br/>　　李清照这丫头疯了一样的朝着前方冲去，沿途路边三三二二的蛇不知道从那里钻了出来。
<br/>　　还有点络绎不绝的意思。
<br/>　　看的她是既激动又兴奋，不知道的还以为她中了大奖。
<br/>　　涂小安觉得事情有蹊跷，将手中的赤链蛇一扔，也跟了上去。
<br/>　　这一扔，也不知道是不是涂小安故意，既然端端正正的扔在了还全身麻木，跪在地上的白尘身上。
<br/>　　卧靠..．
<br/>　　白尘简直是一口老血要喷出来。
<br/>　　赤链蛇冒着冷冷的光，看着白尘，跟他来了一个眼神对视。
<br/>　　确认过眼神，你就是我要咬的人。
<br/>　　赤链蛇嘶叫一声，一口咬在了白尘的嘴唇上。
<br/>　　这一咬，简直打通了白尘的任督二脉，直接弹跳了起来，然后抓着赤链蛇狂甩，最后大力的扯成了几段。
<br/>　　此时，校园内传出了很多的尖叫声跟惊慌声。
<br/>　　不少新生泛着兴奋的心情在这知名的校园内闲逛，熟悉环境，空气中弥漫着怡静的气息，还夹着几许新鲜，楼屋，草地，老师，同学，操场，呃……还有皮皮蛇。
<br/>　　没错，逛着逛着，前方突然窜出一条鲜艳的蛇，朝着学子们热情的打招呼，顿时间，尖叫声四起。
<br/>　　女生宿舍，不少靓丽的少女推着手提箱，叽叽喳喳的搬进宿舍，床铺上一条一米多长的蛇泛着幽然的光，在欢迎她们。
<br/>　　好像在说：“美女，晚上我们一起睡觉觉”
<br/>　　啊...
<br/>　　尖叫声冲天而起。
<br/>　　男生宿舍，几位壮实的男生沿途欣赏了不少新生小姐姐的容颜，憧憬未来的校园生活，推开即将入住的宿舍门，还没迈进去。
<br/>　　一条黑褐色的蛇从天而降，掉到了他们的眼跟前。
<br/>　　毕竟是男生，难以置信的相互看一眼，既然嘎嘎笑了起来：“晚上有蛇羹可以吃了”
<br/>　　这许久没人住的宿舍，偷偷跑进来一条，也不算多新鲜。
<br/>　　一男生一把将这条蛇抓住，好像还很有经验，哈哈一笑：“这温城大学还真不一样呀，用蛇来迎接我们，莫非是考量我们的胆子？”
<br/>　　显然，他是想多了。
<br/>　　黑褐色的蛇不安分的扭动，好像在说：“我就皮一下而已，你们可千万不要扒了我的皮”
<br/>　　外面，校园内，操场，绿草坪，教学楼，走廊中，几乎是每个角落，你都能看到蛇的影子。
<br/>　　有胆大的，有胆小的，但校园忽然出现那么多的蛇，顿时间引起了恐慌。
<br/>　　涂小安跟李清照走到一个大花坛内，里面的花草树木排成美丽的图形，紧紧围绕着一棵茂盛的桂花树，每当桂花盛开的时候，校园里处处弥漫着清香，令人心旷神怡，花坛四周是绿油油的草坪。
<br/>　　环境非常的优美，空气透着清香，可就是这样的一个环境内，一条条蛇在里面成群结队的“玩耍”。
<br/>　　赤链蛇，王锦蛇，黄斑渔游蛇，短尾蝮。
<br/>　　涂小安一眼就将这些蛇的品种给看了出来，旋即发现这些蛇还都是一些无毒蛇，没有什么攻击性，就算咬到人，顶多是受一点皮肉之苦，完全不会有性命之忧。
<br/>　　这些蛇都没资格出现在佘山，出现了也是别的毒蛇的猎物。
<br/>　　前几种都是很常见的无毒蛇，到是后面的短尾蝮算是毒蛇，是国内最最最常见得毒蛇，但也是伤人率最高的毒蛇。
<br/>　　虽然致命率并不高，但危害度却一点不低。
<br/>　　这货夏天最爱栖息在背阳通风干燥的枯叶堆下，圈曲不同来避暑，很容易让人忽略它的存在。
