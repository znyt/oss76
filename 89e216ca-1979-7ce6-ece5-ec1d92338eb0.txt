　　不夜城内头，与别的大城，没有什么区别。
<br/>　　同样是酒肆茶楼、商行、建筑等等，应有尽有。
<br/>　　唯一不同的，恐怕便是那人来人往的街道上，没有半个普通人。
<br/>　　全是武者。
<br/>　　而且最弱都在洞玄九重修为。
<br/>　　破玄境武者，随处可见。
<br/>　　地元境、天元境武者，也不在少数。
<br/>　　极寒之地虽危险，但同样是许多武者历练的不二之选。
<br/>　　且数量庞大的妖兽下，也吸引了众多猎妖师、独行武者等等。
<br/>　　另外，这边宗门林立，同样存在着城池。
<br/>　　只不过，城池是由这里的宗派，联手创建。
<br/>　　整个极寒之地，大大小小，有数十座城池，共数百个宗派。
<br/>　　当然了，数十座城池，都极其巨大。
<br/>　　但在这无边无际的极寒之地中，又显得如同沧海一粟，彼此间的间隔极其远。
<br/>　　这各座城池，便是从全大陆各地而来的武者的唯一可安全休息之地。
<br/>　　其中，又以不夜城最为著名。
<br/>　　不夜城中的十数个宗派，以不夜冰宫为首。
<br/>　　这次的冰缘大会，亦是由不夜冰宫牵头，其余极寒之地各势力配合所举行。
<br/>　　整个极寒之地，最强的势力，便是一宫一门二教了。
<br/>　　不夜冰宫、雪刀门、冰河教、冰武教，四大势力。
<br/>　　言归正传。
<br/>　　萧逸与白冰雪进入不夜城后。
<br/>　　首先去的，便是不夜冰宫。
<br/>　　不夜冰宫，其实是不夜城变相地城主府了。
<br/>　　两人在不夜冰宫做好了登记，取了参赛凭证。
<br/>　　所谓参赛凭证，是两个不夜冰宫的令牌。
<br/>　　萧逸的令牌，是82号，白冰雪的，是83号。
<br/>　　令牌号码，只是代表前去不夜冰宫登记的先后顺序，并无特殊意义。
<br/>　　之后，两人离开了不夜冰宫，随意在附近寻了个客栈住下。
<br/>　　距离冰缘大会开始，还有一个月多些。
<br/>　　还陆续有各方天骄和武者前来。
<br/>　　两人只能暂时只能等待。
<br/>　　不过，这种等待，却并非没有意义。
<br/>　　极寒之地，之所以能让如此多宗派在此开宗立派，不是没有原因的。
<br/>　　这片无边无际的辽阔之地，上承极寒罡风，下临千万年不化之积雪。
<br/>　　冰属性灵气，在这里极其充裕。
<br/>　　冰属性武者，在这里修炼，一日，可抵其它地方数日之功。
<br/>　　也就是说，极寒之地，乃是天生的一处冰属性武者修炼宝地。
<br/>　　当然了，在这里开宗立派的，几乎都是修行冰雪一道的武者。
<br/>　　此时，萧逸二人寻了客栈，并住下。
<br/>　　客栈的消费极高，一晚便要十万两。
<br/>　　作为极寒之地中，唯一能给武者安全休息的地方，每座城池内的消费，都不低。
<br/>　　而在这高昂的费用下，城池给予武者的，是绝对的安全。
<br/>　　毕竟是这里的宗派势力所建城池，武者力量还是有保护的。
<br/>　　极寒之地虽然妖兽横行，但从未出现过能攻破一座城池的情况。
<br/>　　武者能在安全的环境下休息，并安然修炼，区区十万两，便也值了。
<br/>　　另外，如果说，极寒之地是一片冰属性武者的修炼宝地。
<br/>　　那么，不夜城，便是这里宝地中的宝地。
<br/>　　不夜城之下，乃是一处小型雪脉之地。
<br/>　　所谓雪脉之地，便是能产出寒冰之髓的地方。
<br/>　　如果说极寒之地这里的充裕冰属性天地灵气，能让武者修炼速度提升数倍的话。
<br/>　　那么，寒冰之髓的存在，将能让这个速度再翻几倍。
<br/>　　加起来，达到8倍至10倍，这是一个相当可观，甚至惊人的速度了。
<br/>　　此时，萧逸身处的客栈房间内，便有一聚灵阵。
<br/>　　不夜城内的客栈酒楼等等，都是不夜冰宫等宗派势力的产业。
<br/>　　聚灵阵，也是不夜冰宫所设。
<br/>　　武者催动之，便可开始修炼。
<br/>　　萧逸试了试，果然，澎湃的冰属性灵气，正不断从聚灵阵中释放。
<br/>　　随后被萧逸快速吸收。
<br/>　　当这些灵器被吸入体内后，萧逸脸上蓦地一喜。
<br/>　　“修炼速度，果真达到8倍。”
<br/>　　“且这里的冰属性灵器，极其精纯。”
<br/>　　这所谓的精纯，指的是质量比正常的冰属性灵气要高级。
<br/>　　萧逸吸入体内，转过为元力后，正不断从‘冰山’中溢出，源源不断地被冰鸾剑吸收着。
<br/>　　……
<br/>　　所谓山中无岁月，修炼不知时。
<br/>　　在这般高速地修炼速度下，时间，渐渐过去了一个多月。
<br/>　　冰缘大会，只剩一天便开始了。
<br/>　　萧逸从修炼中醒来，一抹精光从眼中迸发而出。
<br/>　　他早便在客栈中缴纳了数百万两，加上白冰雪那份，近八百万两，一个多月来，修炼不曾停歇过。
<br/>　　修为虽未增加多少，大部分力量被冰鸾剑所夺。
<br/>　　但明显的，体内气泉中，冰山火海下方的气态元力，渐渐开始有液化的迹象了。
<br/>　　估计突破天元，也要不了多久。
<br/>　　吱呀一声。
<br/>　　萧逸推开房门，明日便是冰缘大会了，他并不打算再修炼下去。
<br/>　　巧的是，同一时间，对面一房门，也同样打开。
<br/>　　白冰雪从房中缓缓走了出来。
<br/>　　“好巧。”白冰雪清冷一笑。
<br/>　　萧逸点了点头。
<br/>　　“明日便是冰缘大会了，来不夜城一月有余，还不曾看过这边的风光，要一道吗？”白冰雪问道。
<br/>　　萧逸摇摇头，笑道，“不了，我打算随便走走，松下筋骨，回来继续修炼。”
<br/>　　事实上，萧逸也打算看看这不夜城别有风味的景色。
<br/>　　只不过，他更喜欢一人独处。
<br/>　　“随你。”白冰雪清冷回答了一声，自顾离开。
<br/>　　……
<br/>　　不夜城，城墙之上。
<br/>　　萧逸自顾靠墙假寐。
<br/>　　这里，是不夜城最高的地方，最接近高空的地方。
<br/>　　这里，也是最接近极光的地方。
<br/>　　那氤氲极光，照耀下来，沐浴其中，让人浑身暖洋洋的。
<br/>　　在这极寒之地的冰冷气息中，这种暖洋洋的感觉，极为难得，也极为舒服。
<br/>　　城内不少武者，都喜欢来这里欣赏极光美色，享受这种舒服之感。
<br/>　　萧逸自然也是慕名而来。
<br/>　　只不过，对比周遭其他武者的享受和惬意。
<br/>　　他的脸上，却写满了复杂。
<br/>　　他很喜欢独处，却又很讨厌独处。
<br/>　　他觉得一个人的独处，很静，可以陶醉在自己的心念中。
<br/>　　只是，每当这种独处，他不是处于修炼状态时，又每每都会情不自禁想到过往的种种。
<br/>　　他总是想着那人儿，总是盼着找到她，却又始终找不到，甚至没有半分线索。
<br/>　　他的手上，正捏着一令牌，令牌锃白，背面写着‘圣月’二字。
<br/>　　他的手，捏得很紧，哪怕捏得生疼，也不为所觉。
<br/>　　以他现今实力，全力之下，竟也撼动不了这块令牌分毫。
<br/>　　“丫头，你到底在哪。”萧逸凝望着天际，凝望着这极光的尽头。
<br/>　　恰在此时。
<br/>　　不远处，一剧烈的争吵声，极其刺耳地传来。
<br/>　　“嗯？”萧逸皱了皱眉头，双眼一眯。
<br/>　　远处，白冰雪同样在欣赏着极光。
<br/>　　只不过，此刻正与某群人起着争执。
<br/>　　看其服饰，竟是雪刀门中人。
