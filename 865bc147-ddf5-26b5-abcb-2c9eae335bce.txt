　　另外一边，
<br/>　　在浪潮般的嘲讽声之中，叶尘缓步来到了千劫神塔跟前。
<br/>　　巧的是，那凌嚣，刚好就在叶尘相邻的位置。
<br/>　　“哈哈！姓叶的，我发现我都开始佩服你了！你的脸皮，究竟是用什么做的？怎么可以厚到如此程度！”
<br/>　　叶尘没有理会，凌嚣继续开口嘲讽道：
<br/>　　“我要是你，就赶紧一头撞死得了！还1000层，你特么怎么不说自己是元始大帝转世呢？这样说或许还真有傻子会信！哈哈哈！”
<br/>　　面对凌嚣的嘲讽，叶尘始终没有说话，不过心里，已经动了杀意。
<br/>　　只要他待会赢了赌约，就可以将这只讨厌的苍蝇，光明正大的杀死！
<br/>　　而眼下最重要的，是尽全力登上顶层，得到混沌法则的奥义！
<br/>　　尽管叶尘之前放出了豪言，可是他同样也无比清楚，
<br/>　　连昊天大帝那样的横贯古今的超级妖孽，都未曾做到的事情，那难度之大，可想而知！
<br/>　　他必须要全力以赴，方才有那么一丝可能，绝对容不得半点含糊！
<br/>　　“开始！”
<br/>　　随着主持人一声暴喝，叶尘深吸了一口气，将自己的状态调整到了最佳，方才缓步朝神塔的入口处走去。
<br/>　　“嘿！装的还挺像！我倒要看看你还能死撑到什么时候！”
<br/>　　凌嚣再次出言嘲讽了一句，同时身体一晃，已经窜进入口之中。
<br/>　　很快，
<br/>　　叶尘也进入千劫神塔，
<br/>　　嗡~
<br/>　　周围空间一阵晃动，
<br/>　　下一刻，
<br/>　　叶尘发现自己，已经出现在一个神秘的空间之内。
<br/>　　而在他前方，出现了一道蜿蜒向上的石梯，每一阶石梯，大约有十米宽，一共近万米长。
<br/>　　叶尘微微一愣，很快便明白了过来，
<br/>　　“莫非每一阶石梯，相当于是一层？”
<br/>　　一念及此，
<br/>　　叶尘并没有急于冲上去，而是缓步迈到第一道石阶之上。
<br/>　　嗡~
<br/>　　周围重力陡然一下子增加了十倍！
<br/>　　紧接着，
<br/>　　咔嚓！
<br/>　　空中突然一道神雷劈落，就好像是凭空出现的一般。
<br/>　　即便是叶尘，都完全没有反应过来，那道神雷，就已经落在了他的身上。
<br/>　　“嗯？这道神雷之中，似乎包含着规则之力？而且被这神雷劈过之后，我的法则之力，似乎更加的凝实，甚至隐隐已经开始凝聚成符文！”
<br/>　　这一发现，顿时让叶尘大喜过望，
<br/>　　自从他在轮回路中，领悟了四十九条法则之力后，其实就一直琢磨着，如何将法则之力凝聚成符文，可是却始终未能成功。
<br/>　　而眼下的这神雷，居然还有这等功效，实在大大出乎叶尘的意料之外！
<br/>　　叶尘心里正想着呢，
<br/>　　咔嚓！！
<br/>　　又有一道神雷劈落了下来，再次轰在了叶尘的身上。
<br/>　　叶尘不惊反喜，心念一动，
<br/>　　“造化法则！”
<br/>　　嗡~~
<br/>　　一股充满造化气息的强大能量，笼罩在了叶尘的身上，隐隐已经开始浮现出符文的形态。
<br/>　　叶尘见状，顿时满脸狂喜之色，
<br/>　　“看来我猜的没错！我若是能借助这神雷之力，将四十九条法则凝聚成符文，即便是不能得到最后一条混沌法则，我的实力，也必然能大大提升！！”
<br/>　　一念及此，叶尘倒也不再着急前进，足足被劈了九道神雷之后，不再有神雷落下，他这才缓步迈上第二道台阶。
<br/>　　咔嚓！
<br/>　　等到叶尘登上第二道台阶之后，神雷果然再次落下，而且比第一道台阶上神雷的威力，强大了近十倍！
<br/>　　又是一连九道神雷，叶尘全部坦然受了，造化法则符文，也开始出现雏形。
<br/>　　叶尘没有犹豫，当即又越到第三道石阶之上，开始继续用神雷，锻造自己的法则之力。
<br/>　　对于其他闯关者来说，肯定是尽可能不要被神雷劈中，所以闯关的速度越快越好！
<br/>　　但是叶尘不同，他巴不得每一道神雷都劈在自己身上，用来锻造自己的法则之力，所以自然是要多慢有多慢，足足将每一层石阶上九道神雷，全部劈落之后，这才会去冲击下一层。
<br/>　　……
<br/>　　千劫神塔之外，
<br/>　　众人看到大屏幕上，十六人的闯关情况，早已经哄笑一片。
<br/>　　只见速度最快的，自然要数擎苍宗的凌嚣，如今还不到半个小时，已经突破了100层，仅仅就速度而言，丝毫不在之前的绝无敌之下！
<br/>　　而同样令众人瞩目的，还有叶尘，只不过，叶尘之所以会引人注目，并不是因为他的成绩，像凌嚣一样好，恰恰相反，是很差，极差！
<br/>　　快半个小时过去了，叶尘居然才只达到第8层！
<br/>　　甚至连那位负责主持元始门长老，都一度都以为，是不是自己的阵法出了问题？
<br/>　　在经过一番排查之后，元始门长老确认一切正常，数据完全没有问题！
<br/>　　叶尘真的才只闯到了第8层……
<br/>　　一时间，众人早已经都笑弯了腰，整个大会场中，都充斥着漫天的嘲笑和谩骂。
<br/>　　“哈哈！我说什么来着！这小子，就是一个跳梁小丑！”
<br/>　　“按照他的这个闯关速度，最多30层，不能再高了！”
<br/>　　“这家伙极有可能打破千劫神塔有史以来的纪录，不过确实最差纪录！”
<br/>　　“他将会成为天骄战历史上，最大的一个笑话！”
<br/>　　“嘿！真不知道，他之前是怎么通过千劫山考验的？”
<br/>　　……
<br/>　　落霞宗阵营中，
<br/>　　琼碧落脸色铁青无比，
<br/>　　“看来我还真是高估他了！本以为他至少也应该有中等水平的实力，没想到竟然废物到这等程度！！”
<br/>　　说完这话，琼碧落望向一旁的曦月，冷冷道：
<br/>　　“傻丫头！这下你总该死心了吧？看看！这就是你选的男人！”
<br/>　　一旁的鹤飞，也跟着嘲讽道：
<br/>　　“是呀，曦月，宗主大人说的对，这样的废物，根本不值得你喜欢！”
<br/>　　不料，
<br/>　　曦月深吸了一口气，抬起头，直视着琼碧落，道：
<br/>　　“师尊！你这话我不能同意，我喜欢他，与他的实力强弱无关，就算他真的是废物，我也喜欢他！”
<br/>　　“而且，结果还没有出来，您现在就下结论，未免太早了些吧？”
<br/>　　琼碧落顿时双眸一瞪，显然没有想到，一向乖巧温顺的爱徒，竟然敢公然顶撞她，
<br/>　　“好好！既然你不死心，那我们就等那小子的成绩出来，到时候看你还有何话说！”
