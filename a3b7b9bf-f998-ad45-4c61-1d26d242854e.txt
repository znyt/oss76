　　对于后续任务出来，刘哔并没有感到惊奇，今天是第一轮任务的最后一天，此时已经过了午夜，那么时限自然应该结束了。
<br/>　　但让他没有想到的是，这个任务的第二轮，居然要应对一个不知实力几何的鬼修，这就让他感到意外了。
<br/>　　事情已经发生，没给他任何选择的空间，任务上明确表明，他必须要在五个小时之内，也就是天亮之前，将这名来袭的鬼修给解决掉。
<br/>　　不能逃，只能战，好不容易才将第一轮任务完成，还没看到一根毛的奖励，如果要刘哔在此时放弃，他是万万做不到的。
<br/>　　静坐在竹床上，刘哔手边放着一柄古旧的桃木剑，虽然不知道会不会有作用，但总比赤手空拳对敌要好一些。
<br/>　　时间缓缓流逝，已经过了半个多钟的功夫，鬼修依旧是不见踪影，刘哔心中有些急切，任务的时限只有五个小时，在这样拖下去，情况对他可是十分不利。
<br/>　　就在他即将忍受不住，想要出门查看的时候，义庄的大门却是自主打开了，一阵彻骨的阴风袭过，让刘哔身体微微一颤。
<br/>　　身为丧尸，他是感受不到寒冷的，但是体表温度发生变化，他还是可以分辨出来的。
<br/>　　“你来了。”将眼睛睁开一条缝隙，刘哔看着空无一人的停尸间，嘴角露出一抹微笑。
<br/>　　像是时间静止了一般，木门停止了晃动，先前足以让人身体冻伤的阴风也停止了呼啸，一个身穿灰色道袍的拘楼老人从门外走了进来。
<br/>　　看到敌人出现，刘哔缓缓睁开了双目，猩红色的神光，宛若实质一般，从其眼眶中射了出来。
<br/>　　眼前的这名老人很普通，左手颤颤巍巍的拄着一根龙头拐杖，身上没有任何灵气波动，一身灰色的破旧道袍，上面缝满了大大小小的补丁，苍白的脸上，堆满了密密麻麻的皱褶。
<br/>　　唯一证明其身份的，是他身后漂浮着的五颗人头，有男有女，有老有少，但无一不是满脸怨毒，狠狠的瞪着刘哔，像是在面对不共戴天的仇敌一般。
<br/>　　“你怎么知道我要来？”一道沙哑难听的声音响起，鬼修老人站在停尸间中央，饶有兴致的看着刘哔。
<br/>　　冷哼一声，刘哔身影一动，抄起手边的桃木短剑，如同一柄利刃般向鬼修老人袭来。
<br/>　　他只有四个多小时的时间，故此他并没有任何多余的动作话语，出手就是自己全部的实力，他想尽快结束这场战斗。
<br/>　　“嗬，小娃娃，偷袭可不是什么好习惯。”咳了一下，将一口浓痰吐在地上，鬼修老人拐杖一跺，地面被砸的寸寸碎裂，一股肉眼可见的冲击波向四周扩散开来。
<br/>　　一阵杂乱的声音响起，冲击波经过之处，皆是变的一片狼藉，正在向鬼修老人疾冲的刘哔，也经受不住这股力道，反向倒飞出去。
<br/>　　“咔嚓……咔嚓……咔嚓咔嚓咔嚓……砰！”
<br/>　　摔落在房间尽头的墙壁上，巨大的力道差点将刘哔从房间内咋了出去，背后的墙壁，出现了一个蛛网般的裂缝。
<br/>　　挣扎了几下，刘哔有些费力的站了起来，一口黑红色的鲜血，从其口中涌了出来。
<br/>　　“咦，不是人类？”鬼修老人抽动了几下鼻子，有些奇怪的开口，一边拄着拐杖，向刘哔缓缓走了过去。
<br/>　　怎么会这么强！
<br/>　　身体四处传来乱棉棉的感觉，让他清楚的知道，自己体内的骨头大概已经断了，这也不过只是鬼修老人的随手一击而已。
<br/>　　“小娃娃，你到底是个什么物种，这股味道，我怎么从来没见过？”走到刘哔面前不远处战定，鬼修老人咧开了嘴巴，露出了一口黑黄的牙齿。
<br/>　　完了，没得打。
<br/>　　心中有些微凉，刘哔强撑起海绵般的身体，警戒着鬼修老人的靠近。
<br/>　　实力相差实在是太过于悬殊，他根本不敢主动攻击，就连自保，在此刻都成了问题。
<br/>　　“桀桀，小仔，知道我要来，还不快跑，是在关心我老人家饿了，着急送口粮吗。”
<br/>　　怪笑一声，鬼修老人抬起了手上的拐杖，轻飘飘的向着刘哔头顶点来。
<br/>　　虽然那根拐杖上面没有一丝能量波动，但刘哔依旧感受到一股莫大的压力，像是整个人都陷入了泥潭一般，连举起木剑都无比艰难。
<br/>　　似乎是受到生死威胁的激发，刘哔体内刚刚凝结不久能灵种疯狂旋转，将全身的灵气，都汇率到了右手上。
<br/>　　随后，刘哔勉强举起了木剑，挡在了鬼修老人的拐杖前。
<br/>　　“哗啦！”
<br/>　　一阵铁链抖动的声音响起，在刘哔错愕的目光下，鬼修老人的拐杖从把手处断开，露出了其内隐藏的铁链。
<br/>　　突如其来的变故，让刘哔一时间没有反应过来，铁链闪电般的向他脖颈处传来，上面尖锐狰狞的倒钩，在烛光下反射着金属的光芒。
<br/>　　看着眼前如同死神般的锁链，刘哔大脑一片空白，他想要闪躲，但因为之前的伤势，身体根本就进行不了任何闪躲的动作！
<br/>　　“呲……”
<br/>　　一阵冰冷的触感从脖颈处传来，刘哔觉得好像有什么东西从皮肤外面刺进去了，随后，四周的场景一阵天旋地转，再次恢复的时候，整个世界仿佛都倾斜了一般。
<br/>　　“真是不堪一击啊，就这样的实力，也敢妄图染指我的东西！”阴冷的轻笑一声，鬼修老人将手上的铁鞭重新恢复成拐杖，抬脚走了过去。
<br/>　　不对，不是世界倾斜了，
<br/>　　刘哔眼珠转动了一下，用眼角的余光，看见了一旁倒在地上的无头尸体！
<br/>　　那是自己的身体！
<br/>　　是他的脑袋被人砍掉了！
<br/>　　那脖子处有碗大的一道疤痕，清晰可见其中的细小血管和边上肉芽，似乎在妄图寻找着头颅一般，不停蠕动着。
<br/>　　“虽然很垃圾，但也算入门修士了，带回去给阿方那兔崽子，重新做个头颅吧，总不能老是让他用普通人的头。”嘀咕了一声，鬼修老人蹲下身体，干枯的鬼手冲着刘哔散落在地上的头颅抓来。
<br/>　　一抹恐惧从刘哔眼眸逐渐扩散，末世十年，无数次他面临生死危机，却从没像这次一般，距离死亡这么近过。
<br/>　　他见过无数死人，但是，他从来都没有想过，那个死去的人会是自己！
