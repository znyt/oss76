　　“灭。”
<br/>　　钟无忧冷喝一声。
<br/>　　他是暗影楼少楼主，暗影楼未来的接班人；更是北山榜第十的超级天才；整个北山郡最备受瞩目，也是最出色的天才武者。
<br/>　　他有着傲视一切同龄武者的底气。
<br/>　　澎湃的真气从他体内发出，而后传入参天树武魂。
<br/>　　只是，紫炎并没有消失，仍旧快速地烧毁着他的参天树。
<br/>　　“怎么可能？这紫炎到底怎么回事？”钟无忧头一次露出了惊讶之色。
<br/>　　“灭。”他再次喝了一声。
<br/>　　体内真气加大了几分。
<br/>　　这时，紫炎才慢慢消失。
<br/>　　萧逸冷冷看着，心头也是一阵惊讶，如此大范围的紫炎，竟然被他轻松抵消了，这钟无忧看来也并非浪得虚名。”
<br/>　　萧逸心头惊讶着，孰不知，钟无忧心头的惊讶一点儿都不比他小。
<br/>　　“一般的火焰，我轻松就能化去。可这些紫炎，竟足足耗费了我体内大半真气才全部散去。”
<br/>　　钟无忧心里暗暗想着，脸上明显涌现出了战意。
<br/>　　“易霄，本公子本想再留你半个时辰的性命，既然你自己找不痛快，非要自寻死路，那便别怪本公子了。”
<br/>　　钟无忧冷声说罢，真气挥扬而出，原本被紫炎烧得满目苍痍的参天树恢复了原状。
<br/>　　漫天须根再次飞舞，九条灰色长龙也再次凝聚而出。
<br/>　　嗖嗖嗖....无数须根绞杀而来。
<br/>　　沿途拦路的碎石、花草、尽皆被切割成齑粉。
<br/>　　“噬火百刃。”萧逸冷喝一声，数十道火刃齐齐打出。
<br/>　　他没有用紫炎，而是用普通火焰。
<br/>　　对付这些普通须根，他并不想浪费太多真气。
<br/>　　嘭、嘭、、
<br/>　　一阵阵爆炸声响起，须根尽皆被炸成粉碎。
<br/>　　钟无忧见着自己的须根被毁，却丝毫不在意，反而冷声道，“易霄，别告诉我你只有这丁点儿本事。”
<br/>　　“观你刚才释放的火刃，起码是玄阶武技吧。以你的修为，能释放多少次？”
<br/>　　“而我参天树的须根，足有千万，你毁得完？”
<br/>　　钟无忧虽然有些战意，但仍旧没将萧逸放在眼中。
<br/>　　萧逸冷笑道，“钟无忧，纵然须根千万，可若打不到我，又有何用？”
<br/>　　话音刚落，萧逸的身影凭空消失。
<br/>　　千万须根，密密麻麻打来。
<br/>　　可他的身影，却始终在缝隙中不断穿插，须根连他的身子都碰不到。
<br/>　　钟无忧脸色一变，“好快的速度，好灵敏的战斗反应。”
<br/>　　这时，钟无忧大手一挥，九条灰色长龙倾巢而出。
<br/>　　......
<br/>　　另一边，数百武者愣愣地看着战斗。
<br/>　　“无忧公子好强，他那些须根，哪怕一条，就足以将先天武者绞成粉碎。”
<br/>　　“千万条须根，怕是先天九重武者也难以幸免。若我等武者在那须根之中，怕是人数再多也必死无疑。”
<br/>　　“那易霄虽然不弱，速度极快，身体反应灵敏。但，在那九条须柱之下，怕是再难以抵挡了。”
<br/>　　.....
<br/>　　萧逸这边，看着攻来的九条灰色长龙，手上紫炎再次凝聚而出。
<br/>　　这九条灰色长龙，乃是参天树无数须根所化，应该是这武魂的最强攻击手段了。一记砸下，连先天九重武者都要被拍死，半步洞玄都得重伤。
<br/>　　他自然不敢轻敌。
<br/>　　嘶...紫炎再次环绕上九条灰色长龙。
<br/>　　这一次，钟无忧同样没有大意，立刻用真气抵消掉。
<br/>　　之前，萧逸释放的紫炎并不多，只不过是烧在参天树上，快速蔓延，才越烧越烈，从零星之火变为烈火燎原。
<br/>　　上一次，钟无忧足足耗费了大半真气才完全抵消掉这么大范围的紫炎。
<br/>　　这一次，他自然不会放任下去。
<br/>　　否则，体内真气耗光，自己将不战而败。
<br/>　　紫炎消散，九条灰色长龙带着摧毁一切之势，快速袭来。如无意外，将轻松吞噬萧逸，将萧逸击杀。
<br/>　　“哼。”萧逸冷哼一声，手中烈焰手套出现。
<br/>　　顿时，漫天紫炎凭空而现，包裹了九条灰色长龙，将其烧成灰烬。
<br/>　　若是单凭自身真气，绝对无法立刻凝聚这么多的紫炎。
<br/>　　但烈焰手套内的力量，却可以。
<br/>　　钟无忧毕竟是钟无忧，单凭修为已是半步洞玄。
<br/>　　可萧逸，真正的修为不过是先天三重，只是靠着各种手段才达到半步洞玄的实力。
<br/>　　他此时也只能使用灵器了。
<br/>　　同时，他也暗暗警惕，自离开萧家以来，除了天星雷鳄外，这钟无忧绝对是自己所见过的敌人中最强的一个。
<br/>　　当然，不是萧逸的天赋差。毕竟，真正算起来，他来这个世界不过数月，真正修炼的时间也不过数月。
<br/>　　而钟无忧，可是从小就在暗影楼的大量资源下培养起来的，自然厉害。
<br/>　　.....
<br/>　　另一边，一众武者惊呼道，“无忧公子的攻击被化去了，是灵器，易霄竟然拥有灵器。”
<br/>　　....
<br/>　　钟无忧这边，看着九条灰色长龙被灭，却笑了出来，“易霄，终于要使用灵器了吗？”
<br/>　　“你以为就只有你有灵器吗？”
<br/>　　话语刚落，无数须根再次凭空而出。
<br/>　　只不过，这一次，每条须根之上，却出现了一把锋利长剑。
<br/>　　千条须根舞动，便是千把长剑挥舞。
<br/>　　萧逸脸上写满了凝重，这些长剑上的气息，每一把都给他极大的压力。
<br/>　　甚至于，每一条须根的舞动，竟都不亚于刚才灰色长龙的全力一击。
<br/>　　钟无忧单凭这一手，已经可以碾压一切半步洞玄武者。
<br/>　　就算是之前的天残地缺、黑白无常四大杀手，在这千把长剑面前，也绝对撑不过十秒钟就会被绞杀成一堆肉泥。
<br/>　　钟无忧自傲道，“此剑名唤千影，乃中品灵器。你能死在本公子这一招之下，也算你易霄的荣幸了。”
<br/>　　“绞。”
<br/>　　钟无忧大手一挥，千把长剑在须根的舞动下绞杀而来。
<br/>　　每一把灵器，都有其特殊的功效。
<br/>　　像烈焰手套，便是大幅度增加火焰威力。
<br/>　　而千影剑，则是能身化万千。
<br/>　　中品灵器，又是在青色武魂参天树的控制下。这千道长剑的绞杀，威力强得可怕。
<br/>　　只不过，萧逸仍旧不惧。
<br/>　　“传闻你钟无忧狂妄自大，目空一切，如今看来，果真如此。”
<br/>　　萧逸冷笑一声，手中漫天紫炎倾斜而出。
<br/>　　紫炎形成了一个百丈之大的光幕，抵挡在自己面前。
<br/>　　千道长剑，被紫炎光幕挡下。
<br/>　　须根顷刻间就被烧成虚无，哪怕是长剑，也在紫炎下快速融化，最终消散。
<br/>　　“怎么可能？”钟无忧脸色大变，“千影剑乃是中品灵器，怎么会被紫炎烧得融化。”
<br/>　　......
<br/>　　另一边，观战的数百武者也是大吃一惊，“紫炎，又是易霄的紫炎。刚才就是这一手，逼得无忧公子必须应战。”
<br/>　　“如今，竟连无忧公子那等强悍的攻击，也在紫炎面前无可奈何吗？”
<br/>　　.....
<br/>　　“钟无忧，我早便说过，你那半个时辰不必等，只会浪费我的时间。”萧逸淡淡地说着，大手一挥，漫天紫炎将参天树彻底烧毁。
<br/>　　而后转过身，准备离去。
<br/>　　紫晶灵炎，焚烧万物，乃是地元境武者都不敢硬抗的神火。
<br/>　　中品灵器虽强，却也抵不住它。
<br/>　　萧逸弱就弱在修为，凭他自身的真气，顶多能释放十来道紫炎。
<br/>　　若是哪一日，他能无穷无尽地释放紫炎，必将横行北山郡。
<br/>　　当然，想要到那个程度，起码要达到破玄境以上的修为。
<br/>　　说起来，也不知当年狂血玄君是凭何将紫炎狮王这等强大妖兽击杀掉，取了精血的。
<br/>　　要知道，紫炎狮王可是在地元境那个层次都横着走的狠角色。
<br/>　　言归正传，借助烈焰手套的力量，萧逸能释放出这漫天紫炎。
<br/>　　但这并不代表着他现在就能无穷无尽地释放紫炎。
<br/>　　灵器的力量也是会耗尽的，而且这不过是下品灵器罢了。
<br/>　　只不过，很显然，现在对付钟无忧，就算不必耗尽灵器的力量，也足以战胜。
<br/>　　“站住。”钟无忧冷喝一声。
<br/>　　“怎么，还有事？”萧逸头也不回，淡淡地问道。
<br/>　　钟无忧咬了咬牙，冷声道，“易霄，你以为你赢了？而且还能赢得这么轻松？”
<br/>　　“整个北山郡，没人敢如此轻视我。就算是北山榜前十的其他天才，也不敢说稳赢我。”
<br/>　　“你这个本来籍籍无名的小子，忽然才声名鹊起的猎妖师，有何资格在我面前说赢。”
<br/>　　萧逸皱了皱眉头，忽然感受到背后一股恐怖的力量正在快速凝聚。
<br/>　　“千影剑，合。”
<br/>　　钟无忧大喝一声，千把本来已经融化的长剑凭空出现，而后融合成一把真正的千影剑。
<br/>　　“地阶武技，暗影千仞。”
