　　叶尘听到这道声音，顿时身体微微一震，
<br/>　　“这声音，是洛冰玄！”
<br/>　　与此同时，一道阴测测的声音，也跟着响起，
<br/>　　“洛冰玄，你还真是天真的可以啊！人不为己，天诛地灭！你连这么简单的道理，都不知道吗？真是个蠢女人！桀桀！”
<br/>　　叶尘此时，已经悄无声息的靠近那群人百米之内。
<br/>　　场中一共有五个人，四男一女，那唯一的一名女子，被其他四个男人，围在中央的位置。
<br/>　　这女子，气质冰冷，一袭白裙，赫然正是那位天骄榜第二的洛冰玄。
<br/>　　只不过，此时的洛冰玄，却显得有些狼狈，头发散乱，绝美的脸庞惨白无比，后背之上，赫然有着一道极长的伤痕，而那一袭雪白长裙之上，此刻也沾满了鲜血，几乎变成了一件血衣……
<br/>　　至于另外一边，那四名男子中，为首的那个人，身材高大，器宇轩昂，赫然正是天骄榜排名第三的云破天！
<br/>　　不过，此时的云破天，眼眸中却透着一抹浓浓的阴毒之色，和之前在初始星上，那个如沐春风的模样，简直判若两人！
<br/>　　“洛冰玄，你怎么也想不到吧？他们三个，都是我无极宗的人！连我也没想到，我们竟然会出现在同一个星球之上，或许这就是天意吧！哈哈哈！”
<br/>　　云破天得意大笑了一阵之后，脸色猛然一肃，冷冷盯着洛冰玄，嘴角微微一扬，挂着一抹嘲弄的笑容，道：
<br/>　　“怎么？到了这个地步，你还不赶快捏碎你的传送玉简？难道你真的以为，本公子还会怜香惜玉，不会杀你不成？”
<br/>　　“你可要知道，只要杀了你，从今之后，百年天骄榜第二的位置，可就是本公子的了！”
<br/>　　说话之间，云破天的身上，已经开始升起一股浓浓的杀意！
<br/>　　洛冰玄见状，秀眉顿时一蹙，微微思量了片刻之后，右手一翻，已经将传送玉简取出，便要准备将其捏碎。
<br/>　　没办法，虽然她很不想放弃这次难得的机会，可是眼下这个局面，她已经没有任何胜算，而云破天也绝对不会放过她！
<br/>　　留得青山在，不愁没柴烧，只能先离开这里再说了。
<br/>　　然而，
<br/>　　就在洛冰玄准备忍痛捏碎传送玉简的时候，忽然一道突兀的声音，陡然响了起来，
<br/>　　“哟，挺热闹啊！”
<br/>　　说话之间，叶尘已经从一旁的丛林后面，缓缓走了出来。
<br/>　　“什么人！！”
<br/>　　云破天等人，听到这道突兀的声音，顿时纷纷脸色大变。
<br/>　　他们刚刚，竟然丝毫没有察觉到，有外人靠近！
<br/>　　“是你！”
<br/>　　等到云破天，看清楚是叶尘，而且只有他一个人之后，脸色顿时缓和了许多。
<br/>　　他在之前，曾经目睹过，叶尘和绝无敌之间的战斗。
<br/>　　在他看来，眼前这个小子，虽然也算是一个不弱的天才人物，可是毕竟修为太弱，和绝无敌，还尚有一定的差距，更遑论他云破天，可是天骄榜排名前三的存在！
<br/>　　而且，在眼下这处秘境之中，所有人的修为，都被压制的很低很低，他这边有四个人，具备绝对的优势！
<br/>　　“怎么？小子，莫非你想英雄救美？”
<br/>　　云破天似笑非笑的望着叶尘，脸上尽是嘲讽之色。
<br/>　　叶尘先看了一眼洛冰玄，发现她身上的伤势，着实不轻！
<br/>　　眼眸之中，顿时闪过一抹冰冷之色，冷然道：
<br/>　　“你们若是现在滚的话，我可以饶尔等不死！！”
<br/>　　云破天等四人，顿时先是一愣，继而纷纷哄笑了起来，
<br/>　　“云师兄，你听到了吗？这小子让我们滚呢！”
<br/>　　“哇，他的眼神好冷漠，好恐怖啊！我都快要被他吓尿了！”
<br/>　　“是呀，好霸气！好强势啊！我忍不住要跪下了呢！”
<br/>　　……
<br/>　　四人仿佛听到了天大的笑话，顿时纷纷对叶尘肆无忌惮的嘲讽起来。
<br/>　　不过这也难怪！
<br/>　　他们不但有着人数上的优势，还有云破天这样的绝世天骄，无论是在外面，还是在里面，应该说都没有他们惹不起的人。
<br/>　　洛冰玄如何？那可是百年天骄榜排名第二的存在！
<br/>　　如今还不是被他们打的毫无还手之力！
<br/>　　现在，一个名不见经传的无名小卒，竟然也敢说出如此狂言，简直就是不知所谓！不知死活！
<br/>　　面对四人的嘲笑，叶尘一言不发，直接朝洛冰玄缓缓走了过去。
<br/>　　“不知死活的小子！你找死！”
<br/>　　其中一名无极宗的天才，看到叶尘竟然完全不把他们放在眼里，还敢往前走，顿时怒不可遏，当即纵身一跃，然后狠狠一拳便朝叶尘砸落下来！
<br/>　　叶尘却连看都没有看他一眼，随手一掌挥出，
<br/>　　那人如今才刚刚恢复到筑基境一重，肉身强度也很一般，而叶尘如今已经恢复到金丹境，两人足足相差了一个大境界！
<br/>　　“噗！！”
<br/>　　没有任何悬念，男人直接狂喷了一口鲜血，犹如断了线的风筝一般，倒飞到数十米开外，继而重重砸落在地面上，身体一阵颤粟之后，彻底没有了气息。
<br/>　　“什么！！”
<br/>　　其余三人看到这一幕，顿时个个全都傻眼了。
<br/>　　要知道，如今大家的修为，都被压制到了一个极低的层次。
<br/>　　刚刚被叶尘一掌轰杀的这个人，可是无极宗少有的天才人物，即便比不上云破天，那也绝对是顶级水准！
<br/>　　即便是云破天和洛冰玄，这样的绝世天骄，也不敢说，可以一掌将气轰杀啊！
<br/>　　“不可能！这怎么可能！他竟然有这等实力？”
<br/>　　云破天是真的惊住了。
<br/>　　他自问天赋绝顶，放眼整个修真界的天骄，也只有龙辰和洛冰玄，可以压他一筹，而且他心中，其实还一直并不太服气。
<br/>　　可如今，又冒出来一个无名小子，居然不在他之下，这让他如何能够忍受？
<br/>　　而叶尘此时，一掌轰杀那人之后，已经缓缓走到了洛冰玄跟前，微微一笑，
<br/>　　“没事吧？”
<br/>　　洛冰玄此刻，也尚未从方才的震惊中恢复过来，听到叶尘这温和的话语，还有那灿烂的笑容，心中不由得微微一阵悸动，连她也说不上这种感觉，就好像眼前这个人，和她很早就认识了一般。
