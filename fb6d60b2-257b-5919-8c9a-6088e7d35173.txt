　　“晓寒，这不是配不配得上的问题，喜欢一个人，或不喜欢一个人，都是没什理由的，如果……你真的需要理由，我是不喜欢你这种性格，这就不多说了，你明白的。”
<br/>　　张晓寒咧开嘴，冷笑一声，道：“所以呢，你今天才说，当着这么多人的面，是要出我的丑吗？”
<br/>　　“我之前就告诉过你，但是你不相信，或者内心相信，但还是一心要娶我……我本来也想认命嫁给你算了，但是……现在事情闹成这个样子，全是因为我，我不能再沉默下去了。晓寒，你是我师兄，你而今放他走，我感恩你一辈子。”
<br/>　　说完转头瞪了叶少阳一眼，道：“你还不走等什么！”
<br/>　　之前，她是冲动了一把，想要跟叶少阳一起离开，但是现在，理智恢复之后，她知道这无论如何是做不到的：不止一个张晓寒，还有鬼姨、法术公会的师兄弟们，他们绝不会看着叶少阳把自己带走。
<br/>　　叶少阳只有一个人……就算加上毛小方等人，人数上一只手也数得过来，而现场参加婚宴的宾客，九成就是法术公会的附庸，只要鬼婆一声令下，他们肯定会帮忙擒拿叶少阳一伙人。因此，芮冷玉现在只盼望叶少阳能全身而退，至于她自己……也只好先完成婚礼，其余将来再说了。
<br/>　　叶少阳自然知道她的想法，笑了笑道：“我不是跟你说了，我要带你一起走！”
<br/>　　梧桐气不打一处来，跺脚说道：“再不走，你就走不了了！”
<br/>　　“他已经走不了了。”张晓寒往前走了一步，望着梧桐，缓慢地说道：“到这个时候，你还在想着保全他，我只问你一次，你……是不是要跟他走？”
<br/>　　梧桐叹了口气。张晓寒的态度，让她知道是没什么希望了，于是摇了摇头，“我跟谁走不重要，但是……我的确不想嫁给你。”
<br/>　　说完，将手中的红盖头丢在了地上。
<br/>　　一片死寂。所有宾客都傻傻地看着梧桐，谁也没想到，一场婚宴会演变到眼前这个局面。新娘子婚礼上突然就说不嫁了……就算是对于普通人而言，这也是奇耻大辱，更不要说张晓寒是法术界最牛逼的年轻人。
<br/>　　张晓寒的一张脸逐渐变红，又慢慢变黑。
<br/>　　“梧桐！！”鬼姨大喝一声，“你知道自己在做什么！”
<br/>　　“鬼姨，今天嫁人，也是你逼我我的，我说我不嫁，你却是不同意，擅作主张，安排下今天的婚宴，今日之事，你也有责任！”
<br/>　　梧桐知道没有退路，对鬼姨也不再有什么敬畏之心，一股脑吐露出了自己的心声，听得鬼姨目瞪口呆，一时间竟然不知道该怎么去反驳。
<br/>　　“你要跟他走！”张晓寒重复了一遍之前梧桐说的话。
<br/>　　梧桐刚要开口，张晓寒猛然欺身而上，双手一把掐住了她的脖子，两之通红的眼睛，加上狰狞的五官，让他看上去像是一头发狂的猛兽。
<br/>　　“你让我今天丢尽了颜面，贱女人，今天只有你死，方能补偿一二！”
<br/>　　梧桐想要开口，但是窒息让她说不出话，也无法呼吸，只能拼命挣扎，但是张晓寒已经发狂，一心想要掐死她，自然不会松手。
<br/>　　就在梧桐感到越来越窒息的时候，一道人影飞来，举手打向张晓寒的头顶。
<br/>　　“草泥马！放开她！”情急中叶少阳连脏话都飙出来了。
<br/>　　张晓寒感受到了这一掌的威力，只好分出一只手来，结印抵挡。
<br/>　　这一掌，张晓寒用上了大约五成法力，对自己信心满满的他以为肯定能将叶少阳挡回去，不了双掌相交，他自己差点被震飞。
<br/>　　叶少阳却不会给他任何反抗之机，一个箭步上前，左手捏决，食中二指竖起，迅速地点在他抓住梧桐的手腕上。手腕一麻，张晓寒不由自主地松开了手，梧桐挣脱出来，深吸了一口气，转身去拉叶少阳：“快走！”
<br/>　　叶少阳却反手一把抱住她，头也不回地吹了一声口哨。
<br/>　　众人不解其意，然后下一秒钟，四宝、吴嘉伟、凤兮三人立刻就离开桌子，冲了出去。他们是捉鬼联盟的成员，多次的并肩作战，让彼此间形成了强烈的默契，叶少阳一吹口哨，他们就知道该上了。
<br/>　　“我们呢……”毛小方跟道渊真人互相看了看。
<br/>　　“动手！”
<br/>　　道渊真人也冲了出去，毛小方紧随其后。
<br/>　　“愣着干什么，过来呀！”凤兮转身招呼崔颖，崔颖也赶紧跟上。
<br/>　　餐桌前只剩下妙心一个，愣了半天，也走到广场上去，对叶少阳道：“你这么做，会坑了大家的。”
<br/>　　“那你还来？”
<br/>　　“我……哎，被你害惨了。”妙心走到梧桐身边，挽住她的胳膊，说道：“梧桐，你没事吧？”
<br/>　　梧桐咳嗽了一会，已经缓过来，抬起头，有些不忍地对叶少阳道：“你要找死，我陪你一场就是了，你又何必把被人都拉扯进来。”
<br/>　　叶少阳笑笑，对妙心道：“你带梧桐先走，我们顶着。”
<br/>　　妙心一怔，道：“叶少阳你会死的！”
<br/>　　“快走！”
<br/>　　“叶少阳，你真当我不存在吗？”张晓寒突然动身，挡住了梧桐和妙心的退路，冲叶少阳冷笑，意思我就看你怎么让他们走。
<br/>　　叶少阳没有啰嗦，直接冲上去，左手结印，打向张晓寒。张晓寒出手抵挡，将叶少阳这一手法术的威力尽数化尽，刚要反攻，突然一道灵符凌空飞向自己。
<br/>　　“九曜顺行，元始徘徊，华精茔明，元灵散开；流盼无穷，降我光辉，诛尽妖邪，九星齐明！”
<br/>　　叶少阳念咒的时候，张晓寒正在结印，想要将灵符震开。
<br/>　　咒语念完，他掌心焕发出的灵力，正好也打在了灵符上。
<br/>　　灵符上金粉流动，汇聚成九点星光，一起落向张晓寒的身上。
<br/>　　九星神符！叶少阳所掌握的几大暗金神符之中，唯一能对人类施展的，之前在下面吃饭的时候，叶少阳就偷偷画好了灵符。
<br/>　　金粉还是老郭之前帮他研磨的，撞在一个密封的小瓶子里，已经所剩无几，叶少阳画了三张暗金神符，其中一张就是这九星神符。
