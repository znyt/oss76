　　阁楼中，苏莫盘膝而坐。
<br/>　　他本想出去走动走动，好好参观一番这神武学府，但目前身份牌还未办妥，他暂时还不是神武学府的弟子，便放弃了这个打算，静等明日那名木璃导师将身份牌送来。
<br/>　　沉吟了片刻，苏莫准备尝试冲击准帝之境。
<br/>　　准帝之境，便是要武皇境武者将神丹提升到帝级，或是将武魂进化成战魂，这两种只要能做到一种便可踏入准帝之境。
<br/>　　现在苏莫武魂无法进化，便只能先将神丹突破了，只要神丹得意突破，他便能立刻晋升准帝之境。
<br/>　　他的修为，已经沉淀了足够长的时间，完全可以尝试冲击准帝之境了。
<br/>　　少顷之后，苏莫手掌一番，掌中出现了一枚玉镯。
<br/>　　这枚玉镯，便是巫森的空间宝器，其内有十数万名武者，有黄泉魔宗的强者，有阴阴神宫的强者，还有北堂世家的强者。
<br/>　　下一息，苏莫意念一动，身形瞬间消失，进入了玉镯空间之中。
<br/>　　玉镯空间中。
<br/>　　一座宫殿耸立，宫殿的四周围拢着大量的武者。
<br/>　　“怎么回事？这都快一年了，宗主怎么还不放我们出去？”
<br/>　　“是啊！按理说，现在应该早已离开了苍穹世界！”
<br/>　　“不会出什么事情吧！”
<br/>　　众人议论纷纷，嘈杂的声音此起彼伏，无论是黄泉魔宗之人，还是阴阳神宫之人，亦或是北堂世家之人，全部都急了。
<br/>　　因为从试炼者降临苍穹世界，到现在都快一年了，巫森也没有放他们出去，而且没有任何消息。
<br/>　　这让所有人都担心了起来，担心遭遇意外。
<br/>　　宫殿之中。
<br/>　　三大势力的高层人物，已经在大殿中坐了很久，个个眉头紧皱。
<br/>　　“肯定出事了！”阳鼎天满脸的愁容，这么久了，定然有变故，不然的话，根本要不了这么久的时间。
<br/>　　“有什么变故？难道苍穹世界之中，还有人能威胁到试炼者?”二魔老脸色阴沉，强自冷笑道。
<br/>　　虽然他心中也有不好的预感，但他不相信会出事。
<br/>　　试炼者的实力毋庸置疑，绝对是无敌的存在，就算那逆天的苏莫，也不可能会翻出什么浪花。
<br/>　　除非，试炼者不愿意带宗主离开苍穹世界。
<br/>　　所以，即便这么长时间了，即便心中隐隐有不好的预感，二魔老也不愿意相信出事了。
<br/>　　玉镯空间内的众人，根本不知道外界的情况，他们哪里知道，试炼者早已一败涂地，巫森也已生死魂灭。
<br/>　　“可是，为何这么长时间，巫宗主也不放我们出去？”
<br/>　　殷棕眉头紧皱，继续道：“而且，巫山应该也是出去了，居然这么久都没有回来！”
<br/>　　所有人都是沉默了，他们实在是等急了，没有半点消息，一直这样干等，让他们心中担忧不已。
<br/>　　“回来了！”
<br/>　　就在此时，一道冷漠的声音响起，大殿的中央，突兀的出现了一名青年。
<br/>　　众人闻言，先是微微一怔，随即个个眼眸一瞪，几乎瞪出了眼眶之外。
<br/>　　“苏莫！”
<br/>　　“苏莫！”
<br/>　　“怎么可能！”
<br/>　　惊恐的呼声，顿时便在宫殿中炸开，宫殿中的众强者瞬间便沸腾了起来。
<br/>　　因为，这突然出现的青年，居然是苏莫！
<br/>　　“诸位，好久不见！”苏莫环视宫殿中的众人，微微一笑，露出一口白皙的牙齿。
<br/>　　嗖！嗖！嗖！！
<br/>　　回答苏莫的，是一道道呼啸的破空声，宫殿中的众人，几乎如同老鼠见到猫一般，个个惊骇欲绝，疯狂向外逃窜。
<br/>　　几乎是眨眼之间，宫殿中的所有人，便全部消失无踪。
<br/>　　“呵呵！”苏莫冷笑一声，无语的摇了摇头，他也没有出手阻拦众人。
<br/>　　因为这里是空间宝器之中，才方圆三百里的空间，还没有虚界神图中的空间大，这些人又能逃到哪里呢！
<br/>　　随即，苏莫大步而出，走到了宫殿之外。
<br/>　　“是苏莫！”
<br/>　　“什么？苏莫怎么进来了？”
<br/>　　“天啊！快跑！”
<br/>　　宫殿之外，那些黄泉魔宗、阴阳神宫和北堂世家的武者，先是见到从宫殿中逃出的众人，然后见到从宫殿中走出的苏莫。
<br/>　　同样是纷纷惊恐的大叫了起来，没有人敢出手，纷纷向四方逃窜。
<br/>　　破空声不绝于耳，无数的人影四散而开，片刻之间，十几万人全部逃窜一空。
<br/>　　而此刻，只有四人没有逃走，站在宫殿之外，静静的望着苏莫。
<br/>　　这四人，分别是君无惜、风少羽、云非、花碎羽！
<br/>　　苏莫的目光同样望着四人，望着这四位曾经的师兄师姐，眸中蕴含感慨之色。
<br/>　　曾经，他和几人关系极好，以诚相待。
<br/>　　而风少羽在角魔境之时，也曾为了保护他们，而以命相搏。
<br/>　　“五师兄、七师姐、八师兄、九师兄！”苏莫目光一一扫过四人，逐个喊道。
<br/>　　此刻，他不禁心中无奈，自己居然将四人给忘了，到现在才想起来。
<br/>　　“十一师弟！”四人轻声呼喊，个个面色复杂，作为曾经的师兄弟，他们曾经共同进退，现在却成了这番局面。
<br/>　　“几位师兄师姐，能看到你们安然无恙，我也就放心了！”苏莫叹了口气道。
<br/>　　他不准备为难四人，不管四人这几年如何，不管四人是否与黄泉魔宗有关系，也不管四人是什么想法。
<br/>　　四人闻言，心中也是稍松了一口气，看来苏莫还记得同门之宜，不会对付他们。
<br/>　　“十一师弟，二师姐和十师弟她们？”花碎羽轻声问道。
<br/>　　“她们很好，都加入了苍穹神宫！”
<br/>　　苏莫轻声说道，言罢，他略一沉吟，又道：“你们若是愿意，也可以加入！”
<br/>　　四人闻言沉默了，安暖和天辰无事，他们自然也就放心了。
<br/>　　但加入苍穹神宫，对于四人来说，也是心中有些芥蒂。
<br/>　　他们毕竟在黄泉魔宗待了很久，也基本上融入了黄泉魔宗，而且皇甫擎待他们不薄。
<br/>　　“十一师弟，这处空间里的人你怎么处理？”君无惜凝声问道，他知道既然苏莫来到了此地，那巫森肯定已经死了。
<br/>　　虽然他不知道外界到底如何，又发生了什么事，但现在在这处空间中，苏莫就是主宰般的存在。
<br/>　　“除了你们四人之外，所有人都要死，包括皇甫擎！”苏莫沉声说道，之前在宫殿中，他已经看到了皇甫擎。
<br/>　　四人闻言，个个神情默然，虽然皇甫擎所作之事有些不近人情，但对他们四人可是关爱有佳。
<br/>　　说实话，他们都不希望皇甫擎死！
<br/>　　“十一师弟……！”风少羽刚要开口，为皇甫擎求情，却被苏莫摆手打断了。
<br/>　　“我已经给过他机会，是他自己不知悔改！”
<br/>　　苏莫淡淡的说道，他当初没杀对方，只是废了其修为，没想到对方还是在黄泉魔宗与他为敌，真是不知死活！
<br/>　　四人闻言心中暗叹，他们当然清楚皇甫擎对苏莫的迫害，想让苏莫网开一面实在太难了。
<br/>　　“四位，我先送你们出去，稍后再为你们安排！”苏莫轻声说了一句，便意念一动，将四人放出了玉镯空间。
<br/>　　而后，苏莫眼眸如电，扫视四方，眸中尽是冷意。
<br/>　　那些黄泉魔宗、阴阳神宫以及北堂世家的武者，都躲在远处，个个惊恐不已。
<br/>　　面对苏莫，现在他们连一战的勇气都没有了。
<br/>　　苏莫冷笑一声，随即直接盘膝而坐，释放出了吞噬武魂。
