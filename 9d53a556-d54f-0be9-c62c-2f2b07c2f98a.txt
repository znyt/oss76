　　<br/>　　    林汶骐更干脆，直接提起洛阳铲照着脚下面的兜子就砸了几下，里面传来好几声“咔嚓”的动静，镯子肯定是碎了，那卷竹简上也肯定有划痕了，至于那个青铜酒碗是啥样就不知道了。<br/>　　<br/>　　    宁为玉碎不为瓦全，林汶骐的意思就是，我们折了，东西你们也别想要。<br/>　　<br/>　　    “嘎吱”<br/>　　<br/>　　    这时，奔驰和GL8也停了下来，吴洋，周深还有其他人全都推开车门下来，然后围了过去。<br/>　　<br/>　　    林汶骐和北北拎着洛阳铲下了车，咬牙看向围拢过来的人，小东则是刚推开车门就明显挺不住了，半边身子依靠在座椅上，一脸惨笑。<br/>　　<br/>　　    吴洋拎着五连发，皱眉说道：“你们这是何苦呢？刚才把东西卖给我们不就得了，非得还再费一回事，让我们来了个千里奔袭当街拦路，值得么？”<br/>　　<br/>　　    北北看了眼倒在座椅上的小东，无助的吼道：“就为了三件死货，你们至于么？”<br/>　　<br/>　　    “东西我们很想要，有什么至于不至于的？本来我也没想着动手，我们的人追上来之后，你们乖乖交出来不就得了，本来挺小的一件事非得搞出流血事件来……”吴洋也挺无语的摇了摇头，说道：“光是善后就挺让人头疼的了，我比你们操心呢，毕竟，你们要是死了就啥也管不着了”<br/>　　<br/>　　    “北北，跟他们干了！”林汶骐下车之后就没吭声，他突然招呼了一声，举起洛阳铲就要奔着吴洋那边拍过去。<br/>　　<br/>　　    “哗啦”<br/>　　<br/>　　    “嘭”<br/>　　<br/>　　    吴洋见状，往后退了两步之后，快速的撸着枪栓，压低枪口朝着林汶骐的脚下就崩了一枪：“还挣扎个啥，你分不清楚状况啊，要人要枪你什么都没有，你干的起么？”<br/>　　<br/>　　    当林汶骐和北北已经陷入绝境时，他们这两伙人的后方突然有一束车灯透了过来，大灯晃了两下后，就定格在了这边。
