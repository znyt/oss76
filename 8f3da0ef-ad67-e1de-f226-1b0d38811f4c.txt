　　盘古神威，一斧之间，便劈开妖族周天星辰大阵。洪荒集体失声，久久无语。
<br/>　　众人十分凝重的看着天地间的那尊盘古真身，脸色变得十分严肃，巫族有如此地牌，除了天地间的圣人，又有谁能抵挡呢？
<br/>　　元始眼睛盯着远处的盘古真身，眼中闪烁一丝诡异之色，默默道：
<br/>　　“混元初期巅峰了，就快要突破混元中期了，盘古，你真的死了吗？”
<br/>　　之后，他看着那苍茫的巨汉，沉默无语。
<br/>　　“哈哈！再来！”
<br/>　　此时，帝江一声朗笑，虽然他们也面色变得苍白，但他们眼中此刻却闪过无比的兴奋之色，携带着破灭妖族星辰大阵的无尽威势，盘古真身再次举起开天斧影，向着妖族剩下的强者劈去。
<br/>　　斧影带着无穷威势，破开重重空间，划破苍茫，劈向妖族。
<br/>　　就在妖族众人快要绝望之际，陡然间，一只芊芊玉手，穿过虚空，带着无穷伟力，迎向盘古真身所劈出的这一斧。
<br/>　　这是女娲圣人出手了！
<br/>　　女娲从一开始就非常惊讶了，她想不到巫族为何会出现召唤盘古真身的方法？更想不到这仅仅只是一个盘古真身的虚影，便能劈开妖族无数大罗妖神组成的周天星斗大阵。尽管她很惊讶，但她不能眼睁睁地看着妖族被灭，虽然之前她与后土有过约定，但她管不了那么多了，对于她来说，妖族还不能灭。
<br/>　　圣人出手，自然是手段非凡，玉手划破重重空间，带着毁天灭地般的力量，造化法则不仅能造，还能化。化尽天地一切，迎向开天斧虚影。
<br/>　　两道攻击，具皆朴实无华，并无其他玄妙，大道至简，一举一动，都带着莫大的威势。
<br/>　　轰！
<br/>　　两道强烈的光芒相触碰，绽放出璀璨耀眼的光芒，大象无形，********，天地巨震，宛若出现一道道波纹，在虚空之中震荡起来，周遭出现剧烈爆炸，山石毁灭，星辰崩碎。
<br/>　　天地失色，日月暗淡，乾坤动摇，风暴不断。
<br/>　　这一刻，时间仿佛停止了流逝，在众人的眼中，女娲那只手还在与开天斧影相撞，并没有出现哪方破碎的痕迹。
<br/>　　“咔擦！”
<br/>　　突然，女娲那幻化出来的巨大玉手出现一阵破碎声响，紧接着，声响不断，不过瞬息之间，女娲那只手便布满裂痕，紧接着化作碎片，瞬间便化为飞灰湮灭。
<br/>　　盘古真身也是一阵晃动，倒退数十里之远，才停了下来，十二祖巫们脸色变得十分苍白，显然这次交手，也不是众人看到的那般容易，这对于他们还是有很大的负荷的。
<br/>　　再看女娲，女娲的圣人的法相被破，连着本体也遭受到反噬，她的嘴角出现点点殷红，显然是此次交手，她受伤了。
<br/>　　以圣人手段，对抗盘古真身，一个回合，女娲落败。
<br/>　　“嘶！”
<br/>　　这一刻，天地间所有关注此战的大神通者们，全部倒吸一口气，他们眼中带着浓浓的惊骇之色，他们一直以为，圣人就很厉害了，但今天的交手，让他们知道，圣人也不是最厉害的。
<br/>　　“这就是盘古真身的威能吗？圣人都不可挡啊！”
<br/>　　“盘古啊，能开天辟地的存在，仅仅一道真身，便能与天道圣人抗衡！”
<br/>　　“不愧是盘古，巫族这是要逆天吗？”
<br/>　　“以后巫族要一家独大吗，就连妖族都不能阻挡，还有什么可以阻挡他们？”
<br/>　　“巫族啊，所有人都小看了他们，谁能想到他们还有如此手段？”
<br/>　　“……”
<br/>　　这一刻，天地间的大神通者们不能平静了，看着巫族，他们仿佛看到了以后的洪荒，巫族称霸天下的情景。若是真是如此，那到时候，他们又该如何？
<br/>　　昆仑山，元始将目光投向三十三天之外，瞟了一眼，默默道：
<br/>　　“巫族不可能一家独大，天道需要平衡。鸿钧啊，我突然间明白了，你为何那么想后土化轮回了，祖巫少了一人，便不能使出十二都天神煞大阵，妖族便能与之抗衡了。
<br/>　　呵呵！不仅如此，后土化轮回，更能弥补天地法则之失，使洪荒变得更加完善。鸿钧，你真的那么伟大，愿意以身合道，弥补天道之失？我辈修道者，哪有那般无私之人？”
<br/>　　“如今，鸿钧，也该出现了。”
<br/>　　果然，在那十二祖巫演化出来的盘古真身想要更近一步，踏步上前，将砍杀帝俊等人之时，忽见天边彩霞四起，异象弥漫，金光漫道，半空显出一道身影，足踏金光大道，缓缓而来。
<br/>　　“盘古开天有大功德，造化万物，尔等为盘古精血所化，自该守护洪荒生灵，完善盘古遗愿，为何做此无妄之争，毁灭无数生灵，造无数因果，还是退去吧。”
<br/>　　鸿钧道祖真身降临凡尘，眼神淡漠，一件丝毫情绪，开口第一句话，便是规劝巫族退出战斗。
<br/>　　盘古真身一晃，声若洪雷，道：
<br/>　　“鸿钧，我尊你为道祖，但此事是我巫族与妖族之事，两族积怨已深，彼此之间不可和解，早晚都有一战，你还是不要多管闲事为好，如若不然，连你一起杀了！”
<br/>　　鸿钧面无表情，淡淡道：“贫道与你打个赌，如何？”
<br/>　　“什么赌？”
<br/>　　盘古真身依旧威势不减，但直面鸿钧道祖，他们没有一点胜算，若非如此，他们早就开打了，哪还有那么多废话？
<br/>　　“一击定胜负！”
<br/>　　鸿钧老祖淡然出声道：“我败，妖族任由尔等处置，我胜，尔等便就此离去，此后，妖族管天，巫族管地，不可再有征战，如何？”
<br/>　　“可以！”
<br/>　　十二祖巫面对鸿钧老祖虽然没有把握，但是眼前情况容不得他们退缩，与鸿钧道祖交手，却是无法避免了。
<br/>　　只见盘古真身抬起右手，聚之全身之力，煞气与法则凝聚成的开天斧影高高举起，以一种无可匹敌之势，破碎层层空间，地水火风涌动，看得妖族众人胆战心惊。
<br/>　　道祖面色不变，只见那斧影到了鸿钧道祖面前，却如同打在了棉花之中，毫无着力，鸿钧面前的空间如水纹一般荡漾几圈，那斧头便不能再进一步。这极为矛盾的感觉，让人看得心中郁闷，极为难难受。
<br/>　　鸿钧道祖淡然无比，一挥袖袍，虚空一阵波动，仿佛有一股不可阻挡的力量，将盘古真身击退。
<br/>　　那盘古真身一阵模糊，又分为十二祖巫，他们此刻的脸色变得苍白无比，脸上带有不服之状，只是面对鸿钧道祖，他们不敢再有异议，鸿钧道祖可不是一般的天道圣人，不是他人可以比拟的，从之前的交手便可以看出，如女娲这般圣人，远远比不上鸿钧这等圣人。
<br/>　　“如何？”
<br/>　　鸿钧道祖还是如之前那般，面无表情，口中淡淡道：“尔等，还是就此退去吧！”
<br/>　　巫族愿赌服输，点头称是，之后在众祖巫的带领之下，巫族大军如同潮水般退去。
