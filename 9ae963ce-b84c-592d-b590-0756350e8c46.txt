　　苏莫一路疾行，向终极神王殿赶去，不过，他远远的落后于人德神王等人。
<br/>　　就在他进入天池神朝境内不久之后，人德宫主率领玄天神王等四位神王，已经赶到了终极神王城。
<br/>　　五位神王，身形如电，飞到了终极神王城的上空，强大的气势如同亿万火山，爆发而出。
<br/>　　一瞬间，天穹轰鸣，大地震颤，虚空破碎。
<br/>　　五位神王，仿佛是屹立在天穹之上的五座大山，恐怖的气势让得天地色变。
<br/>　　“怎么回事？”
<br/>　　“是人德神王，人王宫的人！”
<br/>　　“他们要干什么？”
<br/>　　一瞬间，整个终极神王城之中，响起无数的惊呼之声，无数终极神王殿的弟子为之色变。
<br/>　　因为，所有人都看出来了，人德宫主五人，来者不善。
<br/>　　“紫霄，滚出来！”
<br/>　　一声怒喝，从人德神王的口中传出，犹如凭空炸雷，瞬间将终极神王城方圆数千里之内的虚空和大地震的崩碎开来。
<br/>　　终极神王城之上，顿时白光刺目，出现了一个巨大的阵法。
<br/>　　白光之中，隐约可见万千长剑在呼啸游走，霎时惊人。
<br/>　　唰唰唰！
<br/>　　终极神王城之中，瞬间掠出了大量强大的身影，终极神王、风雪神王、毁灭剑王、紫霄、绫裳等等终极神王殿的高层，尽皆在其中。
<br/>　　“人德，你们来此所谓何事？”终极神王伫立在低空之中，目光透过阵法，看向上空的人德神王等人。
<br/>　　这阵法倒不是他开启的，而是受到强大的外力，而自信开启的。
<br/>　　此时此刻，终极神王的脸色有些难看，他根本没有料到，人王宫居然会气势汹汹而来，点明让紫霄滚出来。
<br/>　　难道，紫霄得罪了人王宫？
<br/>　　终极神王有些不解。
<br/>　　而此刻，紫霄同样脸色铁青的凝声着人德神王，他心中有些懵逼，这人德什么想要干什么？
<br/>　　“终极神王，紫霄本宫主要带走。”人德神王面色郑重的说道。
<br/>　　“人德，到底有何事？我终极神王殿的一宫之主，岂能让你说带走就带走？”终极神王阴沉着脸说道，对方不说清楚具体所谓何事，他不可能交出紫霄。
<br/>　　“终极神王，我人王宫弟子叶凌天失踪了，此事与紫霄有关，本宫需要亲自审问于他！”人德宫主沉声说道，他知道终极神王不会交人，他这只是借口而已。
<br/>　　“一派胡言！”紫霄闻言大怒，立刻怒视着人德宫主大声道：“什么叶凌天，我从来就没有见过。”
<br/>　　紫霄是真满心的怒火，这些日子，因为秦陨之事，他本就极为郁闷，没想到人王宫居然也来找茬。
<br/>　　最主要的是，这简直就是扯淡，完全是无中生有。
<br/>　　那所谓的叶凌天，他倒是知晓，是一位万古天骄，并且是顶级的万古天骄，修为已经接近神王。
<br/>　　但是，这个也凌天他只听过其名，从未见过，完全的素不相识。
<br/>　　“也凌天？”终极神王亦是眉头一皱，大感头疼。
<br/>　　“紫霄，你还敢狡辩？据我们调查，就是你杀了叶凌天！”人德宫主眸光极为凌厉，如同两柄天刀，犀利骇人。
<br/>　　“人德宫主，你莫要血口喷人，叶凌天失踪与否，完全与我无关！”紫霄大声咆哮道。
<br/>　　洪亮的声音，响彻整个终极神王城，让得下方无数的弟子，都听出了紫霄心中的愤怒。
<br/>　　紫霄的身边不远处，绫裳面无表情，她知道，这可能又是苏莫计划中的一步。
<br/>　　“无关？”
<br/>　　人德神王闻言，顿时冷笑一声，道：“你当然不会承认。”
<br/>　　紫霄的脸色，阴沉的可怕，心中也在思索，这到底是怎么回事？
<br/>　　是人德宫主在故意诬陷于他，还是另有其人？又是为了什么在害他？是因为终极剑道和终极神剑吗？
<br/>　　终极神王的脸色，阴沉如水，他看了看紫霄，又看了看人德神王，道：“人德，紫霄说了，与叶凌天失踪无关，我相信不是虚言？”
<br/>　　“终极殿主，紫霄是你终极神王殿之人，你自然相信，但是本宫主不信，所以需要验证。”人德宫主朗声说道。
<br/>　　“如何验证？”终极神王问道。
<br/>　　“很简单，搜魂！”人德神王面色冷漠。
<br/>　　“这不可能！”
<br/>　　人德神王话音未落，紫霄和终极神王异口同声的开口了。
<br/>　　紫霄彻底暴怒了起来，搜魂？这简直是欺人太甚。
<br/>　　被人搜魂，自己的一切隐秘都将被别人知晓，这对被搜魂之人而言是巨大的耻辱，而对于他这种神王级别的强者而言，更是不可接受的事情。
<br/>　　所以，即便与人德神王一战，紫霄也不会同意搜魂。
<br/>　　而终极神王，也差不多是同样的想法，他都不敢对于紫霄搜魂。
<br/>　　毕竟，他若是对紫霄搜魂，等于是和紫霄彻底决裂了。
<br/>　　“叶凌天失踪不久，本宫主只搜你近三年的记忆，这不算过分吧？”人德宫主微眯着眼眸说道。
<br/>　　“这不可能，人德宫主，我紫霄不是泥人，会任你拿捏！”紫霄咆哮说道，一副不惧一战的模样。
<br/>　　紫霄此刻，心中的愤怒，已经无法抑制。
<br/>　　“终极神王，你的意思呢？”
<br/>　　人德宫主看向终极神王，提醒道：“今日没有结果，那就只能兵戎相见了！”
<br/>　　他的言语之间，威胁之意不言而喻，同意一切都好说，不同样他们就会出手。
<br/>　　人德神王完全是按照苏莫的计划在执行，今日他的目的，是要查探出芊寻月在哪里？是在终极神王的手中？还是在紫霄的手中？
<br/>　　若是能完成这个目标，那目的就算完成了，当然，若是能救走芊寻月，那更好不过了。
<br/>　　不过，就算救不走芊寻月，他也不会担心，因为还有后续的一系列的计划。
<br/>　　为了不提早暴露目标，他只能以叶凌天失踪，给紫霄强行扣上帽子。
<br/>　　毕竟，除此之外，别无它法。
<br/>　　他们可没有理由直接索要芊寻月，就算有理由，对方也不会给。
<br/>　　而且，芊寻月牵扯到苏莫的身份，直接索要的话，很可能会让苏莫和他们的关系，暴露给天王宫和地王宫。
<br/>　　终极神王闻言，眼眸微眯，他作为终极神王殿的殿主，自然不惧威胁，也不会被轻易欺骗。
<br/>　　他感觉的到，此事有问题，人德宫主如此气势汹汹而来，估计不只是叶凌天一事。
<br/>　　终极神王沉默不语，陷入了短暂的沉声之中。
<br/>　　“贵殿的剑阵，不知威力如何？”
<br/>　　不待终极神王回答，人德宫主眸中精光一闪，陡然就出手了，一掌狠狠的按了下去。
<br/>　　就在人德宫主出手之时，遥远的轮回神朝，也同样掀起了大战的硝烟。
<br/>　　地王宫没有多考虑，便拒绝了天王宫的要求。
<br/>　　对此，天王宫大批的强者，浩浩荡荡向地王宫而去。
