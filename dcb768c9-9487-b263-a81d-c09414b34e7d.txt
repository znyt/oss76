　　剑源融入神魂，苏莫闭上了双眸，静静的感知。
<br/>　　这剑源极为特殊，虽然融入了他的神魂之中，但是却是完全独立存在的。
<br/>　　换言之，剑源好似只是寄存在他的神魂之中，属于他却也不完全属于他。
<br/>　　现在，通过神魂他能轻易的控制剑源，催动剑源，便能动用出终极剑道。
<br/>　　并且，通过剑源，他还能感知到一剑兵器的所在。
<br/>　　剑源和那件神器，同根而生，有某种神秘的联系。
<br/>　　“这剑源，只能算是一件宝器而已！”苏莫心中暗道，终极剑道只能算是一件宝器。
<br/>　　失去了剑源，也就失去了施展终极剑道的能力。
<br/>　　换言之，这剑源始终只是外物，不是自身修炼的实力。
<br/>　　但不管如何，能掌控终极剑道，就是一种巨大的机遇。
<br/>　　那终极剑尊，能创造出终极剑道，果然是万古奇才，只是不清楚其去了哪里？
<br/>　　据说，荒界只是三千大世界之一，那么终极剑尊是去了其它大世界吗？
<br/>　　可能是去了，也可能是在找寻去其它世界的路，更有可能已经死了。
<br/>　　“该想办法离开了！”
<br/>　　苏莫睁开了眼眸，长长的出了口气，但是，下一刻，他立刻神情一怔。
<br/>　　因为此时此刻，他居然已经离开了那片空间，出现在了一片荒凉的平原之上。
<br/>　　他，居然在不知不觉之间，已经自行的离开了万道剑冢。
<br/>　　“是玄妙的空间之力！”
<br/>　　苏莫心中暗道，刚才在他融合剑源之时，有玄妙的空间之力，将他无声无息的转移了出去，由于他一心感知体内的剑源，所以才没有察觉。
<br/>　　“这里不知道距离万道剑冢有多远？”
<br/>　　苏莫腾空而起，神识弥漫八方，荒原辽阔无垠，宽广不知几许，没有任何的人烟。
<br/>　　“正好试验一下终极剑道的威力！”
<br/>　　苏莫见此，心中大动，好不容易得到了终极剑道，他自然要试试其威力。
<br/>　　随即，苏莫抬起一根手指，以指代剑，手指之上混沌之力环绕。
<br/>　　没有动用任何元神之力，也没有动用任何血脉之力，苏莫陡然催动了剑源。
<br/>　　刹那之间，一股浩大恐怖的剑威，便从他身上冲天而起。
<br/>　　这股剑威傲世天穹，包容一切，毁灭一切，镇压一切，气息无比的浩大玄妙。
<br/>　　剑威一出，方圆十里之内的空间和大地，全部消融，被剑威彻底的毁灭了。
<br/>　　随即，剑威消散，苏莫手指之上的混沌之力，随之涌入了体内。
<br/>　　他没有试着攻击，因为，仅凭这终极剑道的剑威，他变得大概的知晓了终极剑道的威力。
<br/>　　强大！
<br/>　　无比的强大！
<br/>　　没有动用元神之力和血脉之力，仅凭剑威，就做到这一步，非常的恐怖。
<br/>　　要知道，神域的空间和大地，不是四方星域可比，无比的凝实。
<br/>　　这终极剑道的威势，让得他的剑道威力，提升数倍都不止。
<br/>　　这让得他的总体实力，呈现爆炸式的增长。
<br/>　　若是没有终极剑道，以他现在的修为，爆发全部元神、血脉之力，也超不多就是这种威势了。
<br/>　　……
<br/>　　万道剑冢，此时此刻，剑冢的入口早已消失无踪。
<br/>　　但是，拓跋流明、云沐，以及其他大量的天才妖孽，均是没有离开。
<br/>　　原剑冢入口之处，聚集了约莫三千人之中，嘈杂的声浪冲销而起。
<br/>　　“你们说，那个秦陨，会不会得到终极剑道的秘密？”
<br/>　　“不一定，据说通过了剑山一关，是最后一关，但是最后一关到底是什么，还有没有其它的考验，没有人知晓！”
<br/>　　“应该不会有其它的考验了，剑山之巅如此的困难，这么多年都无人登上，后面若是还有考验，那根本不可能有谁能通过！”
<br/>　　“也许秦陨已经得到了终极剑道的秘密，不然剑冢不会关闭，当然，这也只是猜测！”
<br/>　　无数的人，议论纷纷，没有人离开，所有人都想要等待，看看能不能等到秦陨出现。
<br/>　　终极剑道是一件天大的事，他们必须要确定，秦陨有没有得到终极剑道的秘密。
<br/>　　轰隆隆！
<br/>　　虚空爆炸，苍穹之上，有两道流光撕裂虚空，如同天雷滚滚，急速而来。
<br/>　　呼吸之间，两道流光来到了近前，化为了一男一女。
<br/>　　“见过紫霄宫主和绫裳宫主！”
<br/>　　“见过紫霄宫主和绫裳宫主！”
<br/>　　“见过紫霄宫主和绫裳宫主！”
<br/>　　韩生和一众终极神王殿的弟子，一见这一男一女，立刻纷纷行礼，因为来人正是紫霄和绫裳。

<br/>　　紫霄伫立在天穹之上，目光环视众人，随即落在了韩生的身上，目光如电，摄人心魄。
<br/>　　至于绫裳，面色清冷，但目光同样犀利，她也想要之效，苏莫是否获得了终极剑道。

<br/>　　“消失了！”
<br/>　　紫霄低声自喃，他目光打量四方，见万道剑冢的入口果然消失了，心中有些惊疑不定。
<br/>　　不过，他倒是没有太过担忧，因为，那个死亡的女人在他的手上，秦陨会回来的。

<br/>　　环视了一圈，没有见到慕容雷利，紫霄神王疑惑的万道，因为他知道慕容雷利也来了。

<br/>　　她不希望紫霄再次，因为，若是苏莫一旦出现，必然会被紫霄掌控。

<br/>　　终极剑道关系到终极神王殿的终极传承，这么重要的秘密，两位宫主居然好似不太在意，就这么走了。

<br/>　　随后，韩生便招呼一众终极神王殿的弟子，一起返回神剑山。
