　　佛修，便是来自与远古时代的一大神秘宗门，如今早已消失。
<br/>　　厉害的佛修，便是可以化身天地佛神，如今这牛峰所化身的，便是远古佛修之中的一尊强者，烈火金刚。
<br/>　　“早就听闻，这牛峰曾经得到过不小的奇遇，没想到居然是早已失传许久的佛修。”
<br/>　　“据说这烈火金刚身，一旦修成，便是就会刀枪不入，灵器的轰击都不惧，身上的火焰，更是足以焚尽一切，远超寻常火焰。”
<br/>　　“同境界的武者中，这可是近乎无敌的人物，得到如此大机缘，天生便是超凡与寻常人。”
<br/>　　无数围观武者，看向牛峰的眼神中，都尽是忌惮。
<br/>　　毕竟他们他们这些外门弟子之间，本身也蕴含有着无数争斗，需要为了许多资源而互相争抢。
<br/>　　这样一尊强大人物，放在外门，简直便是称王称霸一般的人物。
<br/>　　“哼，这小子今天算是完了，惹到了牛峰，他在北邙学院也别想好好混下去了。”
<br/>　　马允在一旁，看着楚风眠表情上露出几分冷笑。
<br/>　　牛峰的性格，本身便是瑕疵必报之人，这一次楚风眠招惹到了牛峰，将来楚风眠在北邙学院中，也别想好好混下去了。
<br/>　　“小子，你败在我的烈火金刚身上，也是你的荣耀了，这烈火金刚身，我本来还打算隐藏一些的，如今因为你暴露出来，你也要付出代价才行！”
<br/>　　牛峰看着楚风眠，打量了楚风眠一眼，眼神中满是贪婪。
<br/>　　“你的血脉，很特殊，我在你的力量中，察觉到了几分龙族的气息，你的血脉应该是有着真龙血脉，所以才会有着如此力量！”
<br/>　　“我就将你的血脉抽出来吧！免得这真龙血脉，被你埋没！”
<br/>　　牛峰开口之际，他的身形也是动了。
<br/>　　他大步一动，脚下的岩石都被灼烧起来，留下了一个火脚印。
<br/>　　如同一尊真正的烈火金刚一般，猛然便是向着楚风眠大步走来，一举一动之间，已经是跟一尊盛怒的烈火金刚，没有丝毫区别。
<br/>　　“烈火金刚身？在远古佛道之中，只能算是旁枝末节的功法罢了，居然敢在我的面前嚣张？”
<br/>　　楚风眠不屑的看了一眼那牛峰，冷哼一声道。
<br/>　　就在这时，楚风眠动了，他身躯之中的战龙血脉，微微催动，在楚风眠的身上，气息爆发而出。
<br/>　　这战龙血脉，被楚风眠压制了大半，看起来只是寻常真龙血脉一般。
<br/>　　无尽龙力，在楚风眠的手心中凝聚，令楚风眠的右手直接化为了一道龙爪，散发这滂湃灵力。
<br/>　　“什么烈火金刚身！给我碎！在真龙面前，烈火金刚又算得了什么！”
<br/>　　楚风眠大笑一声，一爪猛然抓了过去。
<br/>　　这一爪，直接抓在了牛峰的身上，看到楚风眠出手，牛峰脸上露出几分得意笑容道。
<br/>　　“小子，你还真是愚蠢，我的烈火金刚身上，可是有着佛火，你敢抓过来，小心不要惹祸上身！”
<br/>　　牛峰的言语中，尽是讥笑。
<br/>　　烈火金刚身，最为强大的，便是其中身躯的坚韧之处，而是上面加持的无数金色佛火。
<br/>　　一旦稍稍沾染这佛火，便是就永远无法熄灭，必须要彻底燃烧殆尽才行。
<br/>　　敢徒手对付这烈火金刚身，简直就是在自寻死路。
<br/>　　“你以为就这等小小火焰，可能伤的到我？可笑！”
<br/>　　楚风眠冷哼一声，右手化为龙爪，直接爪了过去，这围绕在烈火金刚身上的无数佛火，顿时向着龙爪席卷过去。
<br/>　　看到这一幕，牛峰脸上满是不屑笑容。
<br/>　　可是下一刻，牛峰脸上的表情，却是完全僵住了。
<br/>　　这些佛火，燃烧在龙爪之上，居然是完全无法对这龙爪造成任何一点伤害。
<br/>　　无数火焰灼烧过来，对楚风眠的龙爪，甚至还不如挠痒痒一般。
<br/>　　“这小小佛火，还不如上九天中最弱的紫阳真火，紫阳真火都无法灼烧我的身躯，这小小佛火居然妄想？”
<br/>　　楚风眠大手一挥，那龙爪之上顿时又是爆发出一股滂湃灵力来，直接疯狂席卷。
<br/>　　“灭！”
<br/>　　这无数佛火，直接是在一瞬之间，全然熄灭，在楚风眠的力量之下，全然熄灭。
<br/>　　如今这牛峰的烈火金刚身，便是已经失去了其中的佛火，看起来光秃秃的，简直滑稽。

<br/>　　牛峰身上的佛火被楚风眠直接熄灭，简直是不可置信的尖叫起来。

<br/>　　楚风眠冷笑一声，右手龙爪再次发力，一抓拍在了牛峰的身上，令他身上的烈火金刚身，顿时开始节节破碎，一下子出现无数裂痕。
