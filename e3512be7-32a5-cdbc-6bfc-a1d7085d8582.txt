　　“现在就是看天雷会有几道了”牛妖王仰望赤炎如红的天，紧促的呼吸起来。
<br/>　　虽然都不它牛妖王渡劫，但是看到如此可怕异象，就不由自主的紧张起来。
<br/>　　巨大的天外陨石衍生一股无与伦比的火焰出来，火焰有三种颜色变化，高端上档次。
<br/>　　紧接着巨大陨石开始分化，同样道生一，一生二，二生三，三生万物，整整三十六块天外陨石带着太阳一般的温度，笼罩在涂小安的上空，随时落下，恐怖如斯。
<br/>　　天劫的事先准备工作完成，接下来就是准备将渡劫之人给彻底粉身碎骨。
<br/>　　修行本就逆天而行，与天要力量，抢寿命，天劫说是洗礼，其实就是毁灭，但还可以算是九死一生。

<br/>　　猴儿山的上空一片天三十六朵紫色劫云诞生，另一片天天外陨石般的天雷三十六凝结，天威浩荡，天雷震震，轰天霹雳，天火滚滚，焚一切魑魅魍魉。
<br/>　　猴儿山陷入无比的恐惧之中，那种无形的压迫感，侵袭窒息感，难以言喻，很多弱小的生物直接无声息的死去。
<br/>　　而此时，猿猴王跟赤阳巨猿站在一起，表情要多难看，就有多难看，简直杀涂小安这条蛇精一万次的心都有。
<br/>　　多缺德的一条蛇啊，渡劫跑到他们的地盘来，更加无语的是，这条蛇真是逆天了，渡劫形成的异象既然恐怖到这种地步。
<br/>　　这还仅仅只是显化，还没真正的开始呢。
<br/>　　不得不说这条蛇的蛇品真的不怎么样，老天都不想他活下去。
<br/>　　三十六朵劫云，三十六块陨石，这是传说中的七十二天劫啊，唯有惊才绝艳之辈，或者是十恶不赦的人，才有资格被天地法则如此的“照顾”。
<br/>　　正常的生物渡劫，都是降下四九小天劫，四道天雷，九道天火，这是最小的天劫，也有六九小劫，就是牛妖王经历的六道天雷，九道天火。
<br/>　　“大哥，你当初渡劫由于是超等生物，好像降下的是十二道天雷，十二道天火吧，是二十四中劫”赤阳巨猿惊讶的说：“这区区一道蛇精，难道血脉比大哥还要高贵不成，既然是传中说的七十二天劫，简直闻所未闻，不给活路啊”
<br/>　　天劫的制度也是天地间的渡劫生物自己琢磨出来的，四九小劫，六六小劫，最为普遍，随后就是十二十二，二十四中劫，二十四二十四，四十八大劫难了。
<br/>　　至于三十六，三十六，七十二是传说级别的，号称必死天劫，历史上好像也找不出几个。
<br/>　　猿猴王沉默不语，看着涂小安那道悬空的身影，就像在看一个死人。
<br/>　　这是必死劫，天雷天雷一起轰炸，就算现在换一个妖王层次的来，也必死无疑。
<br/>　　更别说，对方还仅仅是一条蛇精层次，拿来去抗，抗的过去，才没天理。

<br/>　　这样的天劫，别说渡了，光看都能看的人心生绝望。
<br/>　　青竹林领域。
<br/>　　这里一眼望去，有无数个数不胜数的小山头，随处可见任何觉醒生物，它们就算不是飞禽也能一跃十几丈，它们形态娇小却可生吞大象。
<br/>　　精怪级别的数量几乎快到了满大街的地步。
<br/>　　这里是妖灵山脉五大深山老妖之一白妖王的地盘，有无数妖精投靠在他的麾下。
<br/>　　势力之大，难以言喻。
<br/>　　青竹林中央地带有一座巨大的古堡，壮观绝伦却妖气冲天，古堡的大殿中正上方慵懒的半躺着一位白衣男子，男子衣冠楚楚，面带笑容，只是笑容给人的感觉却很冷冽，他的周身有一团黑气缭绕，妖气强盛，普通的生灵光站在他身边就会立刻散失神智。
<br/>　　这是彻彻底底的资深妖王级别，毁天灭地的存在。

<br/>　　好像长在地层之中的龟道：“如若这等天劫被渡过去，将来成就会比白妖王您还要高，起码最少是妖王层次了”
