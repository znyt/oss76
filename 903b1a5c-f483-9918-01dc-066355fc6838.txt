　　小九立刻抱起叶少阳，朝山下冲去，捉鬼联盟所有人都毫不迟疑地跟了上去。
<br/>　　事情的结局、还有什么东岳火精的下落，在这一时间都不重要了。叶少阳的安危，才是他们最关心的事。
<br/>　　叶少阳昏迷了，醒来的时候，大家都在身边，左右看看，好像是医院病房，自己右边肩膀上挂着一道绷带，体内有点隐隐作痛。
<br/>　　“我怎么了？”叶少阳试图坐起来，立刻被小九扶起。
<br/>　　“你肺部被洞穿了，造成肺出血，还好小九给你送来医院了，没啥事了，不过要休养一阵子才能出院。”老郭解释道，是他刚才跟医生沟通的，虽然一屋子都是人，但医生只能看到他一个。
<br/>　　“少阳……”
<br/>　　芮冷玉挤过来，抓住他的手，眼睛红红的，泪水几乎夺眶而出。
<br/>　　叶少阳在她脸上抹了一把，道：“你怎么也来了，你不是去追女魃的吗，结果呢？”
<br/>　　“没追上，跑了。”
<br/>　　叶少阳愕然。
<br/>　　“那东岳火精被她拿走了？”
<br/>　　“也不一定，我去的时候，她跟鬼车还在打，后来右君开辟虚空，将她一起拉进去了，我实力有限，不敢跟进去，就赶紧追着大伙来了这里。不知道那东岳火精最后是女魃还是右君抢走了。”
<br/>　　这结果，也是叶少阳完全没想过的，想了下又问大伙，黎山老母那一帮人后来怎样了，林三生告诉他，既然东岳火精不知去向，黎山老母他们也只好先回空界，再慢慢追查，这件事就算过去了。慈心师太等一干人间法师，也都各自回去了。
<br/>　　“她没有怪你？”叶少阳问。
<br/>　　“怪是要怪的，好在双方也没死伤，再说就算没有我们，尸族和太阴山也早就埋伏好了，他们现在一心想要找回东岳火精，哪里有什么工夫怪我，再说为了空界平衡，他们也不会再闹事的。”
<br/>　　叶少阳沉吟片刻，问道：“那东岳火精落在尸族或太阴山手中，会有什么后果？”
<br/>　　林三生摇了摇头，“我原先以为他们是要请回东岳大帝，不知道是什么东岳火精，对这物事一无所知，更没想到却是让尸族和太阴山捡了便宜，也只有将来再看了。好在没有让黎山老母得到。”
<br/>　　叶少阳心中一动，道：“你这话说得有问题啊，黎山老母怎么说也是友军，太阴山和尸族都是敌方，你宁愿把这好东西给他们？”
<br/>　　林三生笑道：“东岳火精，是阐教的物事，在右君或女魃手中，未必就能发挥作用，如果被黎山老母所得，那才是了不得，没有它，我们还是友军，能并肩作战，有了它，可能友军也要变敌军了。”
<br/>　　抬头看了叶少阳一眼，迟疑道：“少阳，你是怎么遇袭的？”
<br/>　　叶少阳回想了下，道：“我当时被摇光仙子抱着，根本没提防，嗯，是我大意了，我以为右君不会对我下手，没想到这么狠。”
<br/>　　想起自己受伤经过，叶少阳也是感慨，肉身对于人类来说，是优势也是最大的拖累，像自己当时，真气充沛，状态也很好，但只是被偷袭了一下，就险些丧命。
<br/>　　好像自己跟邪物能大战三百回合、吊打人间一切不服，令邪物闻名色变，但随便遇到个什么意外，或者被人从后面放个黑枪，都能把自己打死。
<br/>　　人类就是这么脆弱。
<br/>　　叶少阳正乱七八糟地想着，芮冷玉突然说道：“右君没有理由对你下手啊。”其余人也纷纷点头，最后随着叶少阳一起，把目光都落在了林三生脸上，他的智商是大家都认可的。
<br/>　　林三生沉吟着，说道：“少阳，你确定是右君对你下的手？”
<br/>　　叶少阳一怔，“这有什么疑问，当时就他在我身后，而且洞穿我身体的，是一股实质化的鬼气，正是他的手段。”
<br/>　　芮冷玉接过话头，点着头说道：“当时你受伤后，我转头看你，右君正好逃走，不是他还能是谁？”
<br/>　　林三生道：“以右君的实力，还用得着逃走？”
<br/>　　芮冷玉道：“他目的是偷袭，一击得手，我们都注意到了，他不可能再有机会，还留下做什么。”
<br/>　　林三生捋着胡须，“你说的有道理，在别人眼中，这一切顺其自然，但前提是，他为什么要对付少阳？”
<br/>　　一屋人互相看着，都说不上来。
<br/>　　叶少阳自己也觉得蹊跷，按说自己的身份现在是半公开了，右君作为太阴山高层，他肯定知道自己的真实身份，不是什么天选之子，之前当面也说过会保护自己，为何转眼就对自己下手？
<br/>　　但当时也没别人在场，除了他，难道还能有别人？
<br/>　　林三生突然道：“通玄那厮呢？”
<br/>　　叶少阳猛然想起这茬，拿出阴阳镜，手指在上面画了几下，阴阳镜逐渐变化，好像一块透明玻璃，能看到里头一团雾气之中，掩映着房舍和树丛，小桥流水。
<br/>　　“出来！”
<br/>　　叶少阳在镜面上弹了两下，一只脑袋凑了上来，正是通玄道人本来的模样，小眼睛到处转着。
<br/>　　“叶天师好，诸位好。”
<br/>　　他跟广宗天师长的一模一样，看到他的一瞬间，林三生激动了下，随后才反应过来不是师父，心中有些怅然若失。
<br/>　　“木子呢？”叶少阳一直惦记着这件事，劈头问道。
<br/>　　“他……他去阴司了，这当口只怕是已轮回了。”
<br/>　　叶少阳一听就怒了，骂道：“你特么骗鬼呢，木子是邪灵，三魂七魄不全，怎么去轮回！是不是被你害了！”
<br/>　　当下就要把他从阴阳镜中拽出来，通玄道人急忙讨饶，“是真的，我原先也巴不得把他给炼了，但这小子八成是继承了你的神念，顽固地要命，我虽然占据了肉身，把他挤到一边去，但根本无法炼化他，而且每次趁我意念耸动，他就来抢夺身体，老祖我是烦不胜烦，最后与他商量，让他放弃肉身，前去阴司轮回，他不同意，
<br/>　　后来有一次，你前往天弃山对付后卿，我知你危难，与他商议，我暗中救你一回，条件是他乖乖去阴司，把肉身让给我……他见我真的救了你——这件事我暗中所为，你并不知道。之后又答应他，占了他的身体之后，便不再纠缠你，他这才乖乖去了阴司……”
<br/>(本章完)
