        &lt;h3&gt;第三卷第八十一章  探讨&lt;/h3&gt;
<br/>
<br/>        陈子昂没有回北京，财叔带着孩子回去上学了。西城的三个项目已经接近尾声。
<br/>
<br/>        另外，九曦的项目已经在做方案中，现在是百花齐放百家争鸣，像他们的这种软件厂商也是如雨后春笋，到处都有，基本都能用，相对来说，他们的竞争力也越来越大。
<br/>
<br/>        社会在发展，你看到有商机的地方，别人也会看到有商机的地方。
<br/>
<br/>        呼延寰宇和李凌菲他们也越来越忙。
<br/>
<br/>        陈子昂的耗材已经成批量的在生产了，几个试用的单位反馈都还不错，陈子昂准备将产品推向市场，西城也是个不可忽视的地方。
<br/>
<br/>        陈子昂又去找了乔小东。
<br/>
<br/>        “乔哥，我最近想上一种新产品，之前一直都没有关注过集采中的耗材情况。。”
<br/>
<br/>        “什么产品？”
<br/>
<br/>        “耗材。之前的电脑有在进入集采，耗材不知道有没有？”
<br/>
<br/>        “耗材明年刚好要招供货商，你的是什么品牌？”
<br/>
<br/>        “品牌我还没有想好，都是自己的公司研发的，”
<br/>
<br/>        “质量呢？”
<br/>
<br/>        “质量杠杠的。”他们的产品经过几个单位的试用，合格率在百分之九十八。送检和抽样的产品都满足行业标准和要求，几个认证证书只要确定品牌名称就好了。
<br/>
<br/>        “新的产品需要一个推广期，集采的东西一般都是成熟的产品，参加协议供货商也需要有前期供货的合同作为评分标准中的一项，我建议你今年先打开市场，为明年进入协议供货商做准备。”乔小东建议。
<br/>
<br/>        “行，那我先在市面上进行推广，然后经过多方面的测试和反馈，等到明年进集采。”陈子昂算是知道了西城集采对于耗材的态度。
<br/>
<br/>        “多少有点知名度，大家都知道了，会很顺利。”乔小东所说的就是品牌效应。
<br/>
<br/>        “你说的很对，我对这方面还是不太了解。”乔小东就是一个给她指路的人。
<br/>
<br/>        “每个市场都有一定的规则，掌握好规则才能稳步前进。九曦那个项目你们也在参加？”乔小东永远都是一个有原则的人。
<br/>
<br/>        “呼延寰宇他们已经准备很久了。”
<br/>
<br/>        “估计你们得有一场硬仗要大，某公司新开发的软件据说集成了你们和别的厂家软件总和，功能强大，而且还有国外的一家好像也会参与。”乔小东分析陈子昂他们面对的对手。
<br/>
<br/>        “嗯，他们都有过了解，新崛起的那家公司产品还不是很成熟，他们能做的我们也能做，而且我们早都将各个厂家的接口预留了，只要客户有需求我们都会对接的。国外的那一家我们不太了解，在别的地区有过几个案例，我还说让人去考察一下。”陈子昂对她所面临的竞争对手还是了解的。
<br/>
<br/>        “知己知彼方能百战不殆，希望你能稳步前进。”乔小东对陈子昂这样做事业的人都充满着期望，希望他们都越来越好，每一个社会的发展其实是离不开他们这样的人的配合。
<br/>
<br/>        这个世界是相互协作、互惠互利、共同成长的。
<br/>
<br/>        “谢谢乔大哥，我一定会踏踏实实的做事，安安稳稳的做人，努力的稳步前进。”陈子昂心存感激。
<br/>
<br/>        陈子昂告别乔小东，十月底的天气有些凉，她紧了紧衣服，西城现在到处在搞修建，这个城市已经在做着巨大的变化，而且开始修地铁了，再过几年，地铁通了出门就更方便了。
<br/>
<br/>        陈子昂回到西城分公司，大家都在忙着，她去了陈子霖的办公室，陈子霖正在教训一个小姑娘。
<br/>
<br/>        看见陈子昂来了之后挥了挥手，让小姑娘走了。
<br/>
<br/>        “怎么了，今天这么大的火气?”陈子昂感觉哥哥有事情。
<br/>
<br/>        “二嫂打电话让我们下午去她家里吃饭。”陈子霖不耐烦的说。
<br/>
<br/>        “你要是不想去就不去了呗。”他们真的很少去二哥家。
<br/>
<br/>        “去看看吧，毕竟是二哥家。”陈子霖又不想二哥难过。
<br/>
<br/>        “好，我随你。”陈子昂并不知道陈子霖心里装着事情。
<br/>
<br/>        “哥，我准备给西城铺一些耗材，估计大家也是要辛苦一些了。”三句话不离工作的陈子昂又开始和哥哥商量工作上的事情。
<br/>
<br/>        “准备什么时候铺？”陈子霖也从心烦意乱中回过神。
<br/>
<br/>        “元旦过后吧，开个新闻发布会。”陈子昂将她的想法告诉哥哥。
<br/>
<br/>        江俞轩这几个月毕竟忙，这两天正在做招标的事情，她想等耗材发布会的时候由江俞轩来主持，而且想让江俞轩来定品牌名称。
<br/>
<br/>        “行。”
<br/>
<br/>        “我最近这一段时间会跑一些经销商，亲自去各地看一看，然后等元旦过后全国各地同时联动，做一次活动。”
<br/>
<br/>        “各地同时做？”
<br/>
<br/>        “是的。明天我就要走了，这里你好好照顾着，另外九曦的项目也注意点，竞争比较大。”
<br/>
<br/>        “九曦月底招标，你放心，寰宇都弄好了。”陈子霖看着自家的妹妹，心里疼痛，要不是二嫂，她也不会这么辛苦吧？到现在还是独自一人。
<br/>
<br/>        如果子昂知道是二嫂伙同别人算计她不知道该有多难受。
<br/>
<br/>        陈子昂对哥哥的能力是放心的，虽然哥哥书读的不多，但是为人不错。
<br/>
<br/>        西城分公司这几年的实力也不容小觑。
<br/>
<br/>        呼延寰宇在西城信息化项目上的作用不可小觑。
<br/>
<br/>        还有林凤轩，那可是从南方城市挖过来的信息化专家啊。
<br/>
<br/>        裴榕溪是西电的高材生，博导！
<br/>
<br/>        刘清影是外语系的教授，修得是管理。
<br/>
<br/>        白鹭、党清寒、王涛等都是她高薪聘请过来的，他们各负其责，这一段时间在每一个岗位上都兢兢业业，尤其是他们几个的配合度，让她省了很多事。
<br/>
<br/>        她只管带领着一帮人在前面奔跑，后面自会有他们安心的守护。
<br/>
<br/>        但是，他们的年龄都比她大，生活阅历也很多，按照俗话说：他们吃的盐都比陈子昂走的路多。
<br/>
<br/>        经过上一次的改制，她是将他们深深的和她捆绑在一起了，公司与他们每个人都是荣辱与共的，息息相关的。
<br/>
<br/>        “哥，我打算明年三月份去甸城，谷强那边已经在做准备工作了，我们和千千鸟的祁峥一起合作。拿下甸城整个信息化项目。”
<br/>
<br/>        “整个信息化项目？”陈子霖并不知道甸城的项目有多大。
<br/>
<br/>        “对，谷强已经在做前期的工作了，如果都拿下来的话，份额顶得上两个西省的整个项目。不过甸城是山区，项目进度恐怕时间比较长。”陈子昂说，在陈子昂的心中她是想去看谷强的。
<br/>
<br/>        谷强说让她和祁峥一起合作，她也同意了，原本两家公司也有过合作，现在的社会只能是利益共享，虽然他们也有过竞争，但是不妨碍俩个人共同合作。
<br/>
<br/>        “子昂，我听三姐夫说甸城比较乱，你过去我实在是不放心，要不我们一起去吧？”
<br/>
<br/>        “我们是过去做生意的，怕什么，现在是朗朗乾坤，又不是古代。”陈子昂不以为然。
<br/>
<br/>        “你一个女孩子啊。”
<br/>
<br/>        “跑了这么多年，还不是我一个人闯出来的。”
<br/>
<br/>        “子昂，我们都亏欠你了。”陈子霖忧伤的说，几个大的都是各忙各的。老六已经没有音讯这么多年了，只有陈子昂一个人还在锲而不舍的找着。
<br/>
<br/>        “说什么话呢，对了七哥，寒伯安他们交给了我一个文案，我最近在想着做软件的事情，这个软件做下来也很了不得呢，赚不赚钱就另说了，反正是一件有意思的事情。”陈子昂的语言里透露着自豪。
<br/>
<br/>        “你做的哪一件事情不是有意思的事情？”陈子霖还在想着晚上去二哥家的事情，幽幽的回答。
<br/>
<br/>        二嫂所做的事情让陈子霖心里悲疼万分，却又纠结不已。
<br/>
<br/>        但是，兄弟姐妹之间还要往来啊。
