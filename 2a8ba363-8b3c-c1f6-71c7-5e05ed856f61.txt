        &lt;h3&gt;第三卷第二百章  孤独的新年&lt;/h3&gt;
<br/>
<br/>        冯睿这个新年哪里都没有去，娘家已经搬迁了，平时不是回娘家就是回陈家，和陈子建离婚之后一直都没有给陈子建联系过，这个男人真的是心狠，他们在一起生活了那么多年，说分开就分开，一点也不留恋，看着空荡荡的房间，感觉冷冰冰的，虽然家里的暖气很足，虽然家里的绿植还散发着生机勃勃的生命力，但是那种刺骨的寒冷让冯睿心生冷意。
<br/>
<br/>        孩子有家了去陪丈母娘去了，留下冯睿一个人过新年，看着偌大的房间，孤零零的一个人，冯睿连吃饭的心思都没有。
<br/>
<br/>        别人家都是团团圆圆的，她家就她一个人，原本想着一个人也得好好的给自己做点吃的，打开冰箱看着里面的菜，就没有了做饭的欲望。
<br/>
<br/>        拿起手机拨拉着电话号码，想着给几个相熟的人打个电话，拜个年，可是到了陈子建的电话上就再也不动了，“老公”两个字让她的眼泪扑梭梭的往下掉。
<br/>
<br/>        她就不明白了，平时送东西陈子建根本就不管，他家里的那个破手镯有什么好的，当着江俞轩的面动手打她，想想她心里都是恨，此时，心里不但有恨还有说不出的眷恋：“子建，如果你回来我就原谅你。”
<br/>
<br/>        冯睿心事烦乱，终于是忍不住拿起电话给陈子建拨打电话，一声又一声，只是传来冰冷的：“你所拨打的电话已经关机，请你稍后在拨。”
<br/>
<br/>        冯睿将电话仍在沙发上，头埋在两腿之间，过了很久，电话铃声响起来，冯睿拿起电话看了一下，是夏颜打的。
<br/>
<br/>        “冯妈，新年快乐。”夏颜的语气欢快。
<br/>
<br/>        “颜颜啊，新年快乐，你在哪里呢？”冯睿像是抓住了一根救命稻草，在这样一个孤寂的新年，夏颜的电话给她带来了温暖和希望，冲淡了家里的那份寒冷。
<br/>
<br/>        “我在泉城呢，和我妈在一起，你呢？”
<br/>
<br/>        “我在西城呢，颜颜你什么时候回来？冯妈想你了。”冯睿此时是真的想念夏颜，因为只有夏颜记得给她打电话。
<br/>
<br/>        “过两天，在家里要陪陪我妈和孩子，等过两天我就去西城陪你。”
<br/>
<br/>        “你妈还好吧？”
<br/>
<br/>        “挺好，能吃能睡能干。”
<br/>
<br/>        “嗯，那就好，对了颜颜，你早点过来，我们就这几天放假的时候可以去趟北京。我带你见见我的那两个闺蜜。”
<br/>
<br/>        “呀，真的呀，那太好了，我后天出来吧，我将家里安顿一下。”夏颜听说冯睿带她去北京，心里莫名的兴奋。
<br/>
<br/>        “那你就后天出来，对了你的那个同学会一起出来吗？”冯睿现在除了自己和陈子建的工资之外基本就没有别的经济来源。
<br/>
<br/>        夏颜一听就明白了冯睿是什么意思，她微一沉吟：“我让他开车送我吧，带着他也好给我们提提包。”
<br/>
<br/>        “呵呵呵，你这孩子，人家不是大老板吗？”冯睿心情好了很多，她似乎也没有多少乐趣了。
<br/>
<br/>        “再大的老板不也是应该给我们提包？”夏颜语气轻松。
<br/>
<br/>        “行，你厉害，谁都能给你提包。”冯睿阴转多云。
<br/>
<br/>        “你说这话没错，只要我想，谁都可以给我提包。”夏颜开心的大笑，现在的马总都给她提包，谁敢不给她提包？
<br/>
<br/>        “冯妈，不说了啊，对了，我后天出来，你得让我多了解一下你那两个闺蜜的信息，要不然我什么都不知道。”
<br/>
<br/>        “你想知道什么我都告诉你，这又不是什么大事。”冯睿大大方方的答应了。
<br/>
<br/>        这边挂了电话，冯睿给弥青云打电话，可是弥青云的电话无人接听，于是她就打了蔡少芬的电话。
<br/>
<br/>        电话没有想几声就被蔡少芬接起来了：“冯睿，新年快乐。”
<br/>
<br/>        “蔡姐，新年快乐。给你拜年了。”
<br/>
<br/>        “呵呵呵，你到是有心，怎么样年过得好吧？”蔡少芬出奇的态度好。
<br/>
<br/>        “挺好的，你呢？”
<br/>
<br/>        “我们也好啊，年年都这样，不过今年就是女儿没有回来。”
<br/>
<br/>        “过年都没有回家，工作一定是很忙的。”
<br/>
<br/>        “哎，我们家女孩子当男孩子用，也是她有能力，再努力努力，以后前景也是不错的。”
<br/>
<br/>        “恭喜你生了一对好儿女啊。”
<br/>
<br/>        “冯睿啊，你上次说的陈子昂孩子的事情是不是真的？我看最近张家也没有什么动静。”
<br/>
<br/>        “是真的啊，江俞轩亲口给我们说的呢。”
<br/>
<br/>        “哦。”蔡少芬长长的“哦”了一声。
<br/>
<br/>        “蔡姐，我想这两天去北京探望你，你方便不方便？”冯睿小心翼翼的问。
<br/>
<br/>        “放假了，没有什么方便不方便的，要来就来吧，对了给青云说了吗？”
<br/>
<br/>        “没有呢，我还没有给她打电话。”
<br/>
<br/>        “那你就先不要给她打电话了，先来我这里，我到时候告诉她。”张家最近没有任何动静，按说江俞轩有孩子的事情是大事情，前几天江家还去了张家，似乎没有任何问题，这就不得不让人怀疑了，况且他们也不希望别的家族势力能比他们大。
<br/>
<br/>        如果江家和张家不能结亲，对他们来说未尝不是一件好事情。
<br/>
<br/>        .......
<br/>
<br/>        夏颜带着呼延临庆到了冯睿的家里，他们没有想到冯睿新年是一个人过，家里有些荒凉，一股子暮气沉沉的感觉。
<br/>
<br/>        呼延临庆那胖胖的圆脸，一笑眼睛就眯了缝：“冯妈，你想要什么尽管给我说，我呢和颜颜就是你的儿女。”
<br/>
<br/>        “哎呀，你们这两个孩子，我哪里敢当啊。”冯睿心里还是很高兴，夏颜和呼延临庆年轻有为，好好利用也是有奔头的。
<br/>
<br/>        “这个是我取的现，你看你是买东西呢，还是就拿这？我寻思啊，买的东西不合人意，还不如拿点现完全，人家想买啥就买啥。”呼延临庆也不打马虎眼，拉开手提包。
<br/>
<br/>        冯睿心里一惊，平时拿的都是古玩啥的，哪能和现钞相比？这明晃晃的老人头真的是太让人心动了。
<br/>
<br/>        “这个我也不知道拿什么好，只是这些？”看这些东西也不少啊。
<br/>
<br/>        “你不是要去见俩个人吗，多带点没有坏处，给颜颜牵个线搭个桥，颜颜还年轻着呢。这个是给你的，你平时没事了自己去买个首饰化妆品啥的。”
<br/>
<br/>        “我要它干嘛啊，我什么都有。”冯睿假装推辞。
<br/>
<br/>        “冯妈，你和我有什么客气的呢，我们以后的来源还需要这小子呢，再说你也不是白拿，他是一个生意人，以后会碰见各种大老总，你只要将他们的信息少少的透露给这小子一点点就好了。”夏颜摇着冯睿的胳膊。
<br/>
<br/>        冯睿思考了一下并没有推辞，收下了呼延临庆送的东西，她虽然已经退居二线了，但是该知道的她都知道，不该知道的她也能知道。
<br/>
<br/>        “冯妈，我想知道陈子昂之前男朋友张函家的信息，你能给我吗？”夏颜靠着冯睿完全就是一个乖巧女儿。
<br/>
<br/>        冯睿：“我知道的也不是很多，想要知道详细的只能去查。”
<br/>
<br/>        “那今天能查吗？”
<br/>
<br/>        夏颜和冯睿从冯睿的办公室里出来时，心里一个计划已经形成了。
