　　李霄力毕竟在国外待过一段时间，对于什么口径的子弹会造成什么样的创口，还是有些了解的。所以看到两个马仔尸体上皮肉外翻，狰狞恐怖的创口，再联想到刚才在货轮上所看到的无人机，立刻就意识到这不是缉毒警或者是特警的行动，警察不会装备这种杀伤力如此强大的子弹。
<br/>　　额头上的冷汗滑落下来，李霄力手中握着贝雷塔92F手枪，身体靠在汽车，不敢冒头，心脏“嗵嗵”跳的很快，脑子飞快的转动着，思考着脱身的办法。
<br/>　　尽管李霄力自己也知道，要想从特种部队的手里逃跑很难，可是这也并不意味着李霄力就会这么乖乖束手就擒。
<br/>　　司徒空左手握着复合反曲弓，右手从箭筒中抽出一支箭，张弓拉弦，瞄准了一个手中拿着一把“化隆造”的马仔，就在司徒空松开弓弦的时候，那个马仔手里的“化隆造”突然炸膛了，马仔的半个手掌直接被炸飞，马仔看着自己鲜血淋淋，只剩下一半的右手掌，张嘴正要大叫，弓箭穿过了他的脖子。
<br/>　　马仔瞪大着眼睛，嘴里不断有着鲜血往外涌出，身体晃了晃，栽倒在地上。
<br/>　　“浪费了。”
<br/>　　司徒空有些可惜的摇了摇头，早知道，那个马仔手中的“化隆造”会炸膛，司徒空绝对不会浪费这支箭去射杀他。
<br/>　　炸膛，这就是“化隆造”最大的特点，手中握着“化隆造”就要做好被炸断手掌的准备，因为这他妈根本就是一颗极不稳定的炸弹啊。运气好，可能把膛线都磨平了也不会炸，运气不好，可能还是新枪第一发子弹就炸了。
<br/>　　司徒空脚步不断移动的时候，反手又从箭筒中取出一支箭，搭在弓弦上，把弓弦拉满，瞄准了一个手中拿着土制猎枪打得正欢的马仔。
<br/>　　随着距离不断被拉近，很快就会进入土制猎枪的有效杀伤距离。土制猎枪的做工虽然很粗糙，精度什么的更是跟正规武器差了十万八千里，但是有一点是正规武器所比不了的。
<br/>　　那就是子弹！
<br/>　　正规武器的子弹都是从生产线上下来的，子弹的装药量、弹丸数量都是有着明确标准的；可是土制猎枪的子弹也都是黑作坊里自己做的，为了使威力更大，黑作坊在制作子弹的时候，会加大装药量和弹丸数量。
<br/>　　尽管战术中队队员身上的战术背心中都有防弹插板，可是土制猎枪呈扇形的火力覆盖面，还是有可能会打中腿和手臂这些没有受到保护的部位。
<br/>　　嗖！
<br/>　　噗嗤！
<br/>　　弓箭一头扎进马仔的胸口，弓箭直接把马仔穿了个透心凉，马仔气绝倒地。
<br/>　　继续推进，距离拉近，复合反曲弓就不再像之前那样的灵活了，司徒空索性把复合反曲弓挂在身上，用手一拨，反背到身后，双手食指勾住大腿两侧刀囊中的飞刀刀柄末端的圆环，将两把飞刀拉出刀囊，在手中打了个转，反握在手中，朝着自己右侧握着土制猎枪的马仔冲去。
<br/>　　马仔看到冲向自己的司徒空，立刻就把枪口瞄准了他；可就在马仔扣动扳机的瞬间，司徒空双脚一屈，接着惯性，身体向前滑去，左手飞刀刺进马仔膝关节，一刀切断了对方的十字韧带，抓住马仔膝盖弯曲，身体失去平衡的瞬间，右手飞刀从对方左侧第二、第三肋骨间穿过，刺进马仔的心脏，然后迅速拔出，带起一道血珠。
<br/>　　一个合格特战队员，他对于人体构造的掌握程度绝对是顶级外科手术医生的级别，只有清楚了解人体构造，才能知道在近身搏击过程中攻击什么位置是最致命的，可以用最快的速度、最省力的方式，造成最致命的攻击！
<br/>　　马仔想要用手按住自己胸口的刀口，阻止血液的流出；可是司徒空这一刀已经是伤及了他的主动脉，马仔的任何补救都是徒劳的，留给这个马仔的时间，最多不会超过三十秒！
<br/>　　司徒空手中那两把沾着血的飞刀，就像是猛兽尖锐的利齿，每一次挥出都会对港口上这些马仔造成最致命的伤害。
<br/>　　周柏左手边，一个马仔端着土制猎枪从那辆银灰色五菱宏光面包车背后冒了出来，想要给周柏一个偷袭，可是还没等到他扣下扳机，司徒空已经率先对着马仔甩出了左手的飞刀。
<br/>　　飞刀旋转着向马仔飞去，沾在飞刀上的血珠也是被甩了出去。
<br/>　　那名试图偷袭的马仔刚从面包车背后探出头来的时候，周柏眼角的余光便是已经发现了对方；可是就在周柏脚步移动，做出战术规避动作，并且准备反杀的时候，司徒空的飞刀竟是抢在他的面前，扎进了马仔的喉咙中。
<br/>　　“速度还真快！”
<br/>　　周柏看着喉咙上插着一把飞刀，仰头到底的马仔，也是忍不住对司徒空的飞刀称赞了一句。司徒空现在的飞刀技术已经是彻底得到了王小虎的真传，再加上司徒空本身就是练武之人，他对各种冷兵器的使用，其实要比热兵器更加得心应手。
<br/>　　“不用感谢后，回头请我吃顿好的就行了。”
<br/>　　周柏的耳麦中突然传来了司徒空的声音，周柏听着耳麦中的声音，嘴角忍不住抽了抽，嘀咕了道：“还真是不客气啊。”
<br/>　　指挥车中，周宋捷通过罗昊携带在身上的战场视频系统，也是了解到了港口中战斗的情况，虽然画面因为罗昊各种战术动作的关系而抖动，但是现场的战斗过程还是让周宋捷忍不住咽了一口口水；除了周宋捷外，还有一个人被发生在港口中的战斗深深震惊，那就是伪装成港口工作人员的刘炯，刘炯看到的要比周宋捷更加直观。
<br/>　　这场战斗，如果需要用一个词来形容的话，那就是“砍瓜切菜”。给刘炯的感觉，就像是一群穿着顶级神装的土豪级玩家去刷低级副本，李霄力手下那些马仔看似凶狠，可是面对罗昊他们的时候，根本就阻止不起来有效的反击。
