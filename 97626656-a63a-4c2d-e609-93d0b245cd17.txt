　　第27章
<br/>　　“大哥，说实话，菜做的是真地道，盐也给得足。”高飞豹一边剔牙一边道：“我还偷偷顺了他们一包盐，哈哈，他们一定没有发现。”
<br/>　　高飞虎盯着自己弟弟，哭笑不得道：“你就只惦记着饭做得好吃嘛，我让你去是做什么，如果想打下蛤蟆寨，究竟该怎么做？”
<br/>　　“其实也没有什么好看的，蛤蟆寨一共六个人，我主要顾忌的是程大雷的五雷法，也不知道是真是假。”
<br/>　　“我让你去的主要目的就是这个，你今天看出来什么没有？”
<br/>　　“没有，他一天都没有露面。”
<br/>　　“那你明天继续去，多带几个人，一定要摸清楚程大雷底细。”
<br/>　　“嗯，我也想着明天早点去，明天早上我就不吃饭了，多吃他们些。大哥，要不要你也去，顿顿都有白面啊。”
<br/>　　高飞虎脸上肌肉抖动，口中吐出一个字：“滚！”
<br/>　　……
<br/>　　第二天来的人是第一天的好几倍，足足有上千人。黄三元都被这一幕吓着了，他是不太了解青牛山的山贼饿到什么程度。
<br/>　　当然，人越多，干活的进度就越快。黄三元不愧是给一城之主做过管家的人，即便是一千人也被指挥得井井有条。
<br/>　　有人用山石筑墙，有人挖土烧砖，也有人在山坡上砍伐大树，整个山寨都是一副热火朝天的架势。
<br/>　　土屋内，程大雷和苏樱大眼瞪小眼，苏樱被程大雷看得有些不自然，默默扭过头去。
<br/>　　“你待在这里做什么？”
<br/>　　“保持神秘感。”
<br/>　　“神秘感？”
<br/>　　程大雷躺在炕上，道：“现在外面都说我和神仙学了五雷法，日日在房间中呼吸吐纳，修炼长生之术。我要是经常出去晃，被他们看见高人也需要上厕所怎么办，偶像崇拜会崩塌的。”
<br/>　　现在程大雷躺在炕上，枕着两条胳膊，苏樱坐在床角。这一幕，莫名让苏樱有种小媳妇陪着丈夫的感觉，可是她也不好说什么。毕竟身为一个人质，程大雷对自己不打不骂，也没有强迫自己做什么，自己实在不能得寸进尺。
<br/>　　“那你为什么一定要待在我的房间？”苏樱。
<br/>　　“因为你好看啊。”程大雷自然而然道。
<br/>　　苏樱紧绷着脸，眉头微颦有些怒气。
<br/>　　“你别生气，我这样待着很无聊的好不好，整个山寨，我也就觉得你好看。”
<br/>　　土屋外，众人皆在忙碌，一副热火朝天的架势。
<br/>　　秦蛮扛着铁枪，拖着一只野猪缓缓从外面走进来，丢在做饭的厨房前。
<br/>　　“秦大哥，今天又打到野猪，我们可是有口福了。”
<br/>　　“嚯，这只野猪足有三百斤重，秦大哥一人打到的。”
<br/>　　山贼之中，自然强者为尊，经过这两天接触，青牛山这些山贼，就没有不服秦蛮的。秦蛮性子单纯，直来直往，如此也最得山贼喜欢，更何况，大家每天吃得肉都要靠秦蛮打猎打回来。
<br/>　　“秦大哥，你这两臂有三四百斤力气吧。”
<br/>　　“哈，没有那么夸张。”秦蛮爽朗的笑着。
<br/>　　“秦大哥，露一手给我们瞧瞧呗，也让我们开开眼。”一帮山贼起哄道。
<br/>　　秦蛮满面通红，像是喝了烈酒一般。他却不过，看到不远处有一石碾，笑着道：“那就试试吧。”
<br/>　　“秦大哥，这石碾可是有四五百斤，您悠着点。”有人担心道。
<br/>　　“试试，也就是试试。”
<br/>　　秦蛮扎紧腰带，屈膝下蹲，双手抱住石碾。所有人都睁大眼睛，仔细盯着这一幕，不肯错过任何一个细节。
<br/>　　咔嚓，咔嚓！
<br/>　　就像全身骨骼都爆炸开来一般，秦蛮满面通红，情景暴露，忽然口中呵斥一声，那石碾被他一点点抱着举起来。哈呀一声，竟然双手举过了头顶。
<br/>　　哗！
<br/>　　万籁寂静，落针可闻。所有人都张大嘴巴，心脏在砰砰打鼓。
<br/>　　“秦大哥，您……您真是天神下凡呐！”
<br/>　　“力举石碾，这力气超过五百斤呐。”
<br/>　　秦蛮把石碾撂下来，哈哈笑道：“算不得什么，我可不是这个山寨力气最大的。”
<br/>　　有几个走到石碾前，也试了试，结果莫说举起来，便是推都推不动。
<br/>　　“那么，谁是这山寨力气最大的？”有人好奇的问，目光已经看向林少羽。
<br/>　　“嘿，他说的可不是我，是我们大当家，秦蛮的力气连我们大当家一半都没有。”
<br/>　　嘶！
<br/>　　一片倒吸气的声音，已经有人暗中揣测着程大雷的力气。
<br/>　　土屋内，程大雷透过窗户纸看着这一幕，嘴角抿起：“好啦，该我表演了。”
<br/>　　苏樱看着他，目光追随着他的背影在房间中消失。然后也凑到窗户前，盯着外面的动静。
<br/>　　程大雷一走出屋子，所有人的目光都搁在他身上。目光有好奇，有畏惧，有忐忑。
<br/>　　秦蛮大步走过来：“大当家，和我过过手吧。”
<br/>　　众人都来了精神，心道：耳听为虚眼见为实，这人究竟有多大本事，还得看过才知道。
<br/>　　“过什么手，干活，干活。”
<br/>　　程大雷对秦蛮的请战不管不顾，迈步向前走。秦蛮站在他身后，忽然大叫一声：“你看不起俺秦某人么！”
<br/>　　握紧拳就照着程大雷背影砸来，出拳如猛虎，大叫丝毫不怀疑，这一拳如果砸中，能直接砸碎石头。
<br/>　　程大雷突然回头，不急不缓的接住秦蛮的石头，看情形不像是他挡住拳头，反倒是像拳头往他手掌送。
<br/>　　然后就是角力，秦蛮脸上的青筋都站起来，黑黝黝的脸膛变成红色，显然已经使出了全身力气。可程大雷脸上却仍然一滴汗珠都没有，显得风轻云淡。
<br/>　　“滚！”
<br/>　　程大雷突然爆喝一声，手臂猛地发力，秦蛮的身体就像是断线的风筝一般，一直砸出七八米远。
<br/>　　程大雷迈步走过去，将他从地上拉起：“你没事吧。”
<br/>　　“没事，多谢大当家手下留情。”
<br/>　　“好啦，以后不要这么冲动，我今天只出了六分力，不然的话，你就真的要受伤了。”
<br/>　　哗！
<br/>　　所有人都震惊了，呆呆的看着程大雷，就如同见到鬼一般。
<br/>　　这样，也才六分力气！
<br/>　　在大家半是崇拜，半是畏惧的目光中，程大雷走回房间。苏樱赶忙坐直身体，奇怪看到程大雷脸色苍白，冷汗滴落。
<br/>　　“你怎么了？”
<br/>　　“扭着腰了。”
<br/>　　
<br/>(本章完)
