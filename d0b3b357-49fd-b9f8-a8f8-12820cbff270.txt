　　只听“咔嚓”一声，士兵的脖颈立刻就被咬断，女尸趴在他脖子上，哼哧哼哧地吸起血来，喝了几大口之后，将脖子后面的大筋咬了出来，在嘴里嚼动。
<br/>　　这士兵起初还尖叫着挣扎，很快就倒在了地上自己的血泊里。
<br/>　　周围几人之前被这突然发生的一幕惊呆了，完全没反应过来，这时候直到这士兵倒下，才相继回过神来，大声叫着，逃命似的往外冲。
<br/>　　小三没有跑，神情反而坚毅起来，把枪口对着了女尸，突突地打出了一梭子子弹。
<br/>　　子弹打在女尸的身上，女尸浑身哆嗦起来，但是等子弹打完了，女尸除了身上的旗袍多了几个窟窿，并没有什么变化，放开那具尸体，转过头，阴恻恻地看着小三，朝他扑了过去。
<br/>　　小三躲过了第一下，转头就往外跑，女尸立刻冲了过来。她现在是鬼尸，受限于肉身，之前鬼魂的那些进攻手段完全发挥不出来，只能是抓、咬这种，不过速度上比活人要迅捷太多。
<br/>　　小三刚冲出去，就被女尸抓住了，对着他后颈咬下去。
<br/>　　我要死了……
<br/>　　在女尸口中吐出的热气喷吐在他后颈上的时候，小三心中产生这个念头，然而女尸咬下去的瞬间，从他的脖子上突然放出一道银光，女尸惨叫一声，竟然从他身上被震了下去。
<br/>　　小三转头看了一眼被震开的女尸，女尸的表情跟他一样，也是一下子愣住了，不知道发生了什么事。
<br/>　　“三爷快走！”一个士兵拉了小三一把，小三回过神来，跑了出去，反手关上门，招呼几个手下过来一起把门顶住，只听见里面砰砰的撞门声，几个大汉死命才顶住。
<br/>　　小三转头外面一看，很多土匪正在聚涌着朝这边赶来，为首的是铁算盘和几大金刚，都是鬼见愁的心腹。
<br/>　　小三扫视了一眼，计上心头，对他们说道：“大当家的在里头被新娘子杀了，你们快来！”
<br/>　　所有人大吃一惊，但也没有停止脚步，一口气冲到了门前。
<br/>　　“撤！”
<br/>　　小三喝了一声，几个士兵一起放开手，门呼啦一下被撞开了，然后在众目睽睽之下，女尸冲了出来。
<br/>　　因为披头散发的缘故，加上天黑，女尸根本也看不出有什么异样，出门之后，立刻朝对面的人扑过去。
<br/>　　二龙山有四大金刚，这会儿自然是冲在最前面，其中一个彪形大汉并不知道怎么回事，上去一把抓住女尸，用力拉了一把，想把女尸摔倒。
<br/>　　结果女尸对着他的手腕就是一口。
<br/>　　“啊！”这彪形大汉被咬这一口，居然大声惨叫起来，疼的用手握拳去击打女尸的脑袋，打到第三下的时候，女尸突然抬头，一口又咬住了他的拳头，直接从手臂上撕了一块皮下去。
<br/>　　彪形大汉惨叫不止。
<br/>　　“你们快把二金刚救下来啊，一起上！”小三站在一边大声喊。
<br/>　　附近几人这才回过神来，一起上，有的用拳头，有的用枪托去砸女尸的脑袋，女尸猛然松开大汉，扑在了一个人身上，照着脖子就咬。
<br/>　　而那彪形大汉，却倒在地上抽搐起来，有小土匪想要把他扶起来，送去医治，结果那彪形大汉在地上抽抽了一会，居然死了。
<br/>　　围观的土匪之中，有人发出了一声细不可闻的叹息，正是叶少阳。
<br/>　　在众多土匪朝鬼见愁的房间赶过去的时候，他也夹杂在人群中赶了过来。现在所有人的注意力都在新娘子的深山个，已经换上土匪衣服的他，自信不会被人认出来。
<br/>　　眼前这个新娘子，已经成为鬼尸，鬼煞合一，体内血液流淌的尽是煞气，普通人被她咬上一口，必然会被煞侵入身体，自然是没救的。
<br/>　　现在的自己，如果冲上去对付这具刚化形的女尸，凭借手中的七星龙泉剑等法器，绝对不成问题，不过那样的话自己也就暴露了，只怕是清理了这女尸，自己也会被土匪们打死。
<br/>　　为了这些土匪的性命，肯定是不值得的。叶少阳只能等待机会。
<br/>　　“鬼，不，这不是人！”当第四个人被鬼尸扑到脸上咬的时候，总算是有人看到了她口中的獠牙，惊声叫起来。
<br/>　　这时候也有人掏出了盒子枪，对着鬼尸的脑袋连开几枪，鬼尸只是脑袋晃了晃，连血都没流出来，迎着枪声冲上去，扑到放枪的人身上，啃食起他的面皮。
<br/>　　更多地人拿出手枪，对着鬼尸一通乱打，女尸除了抽搐，没有别的反应。
<br/>　　叶少阳在后面冷眼看着，任何僵尸，就算有尸气护体，毕竟也是肉身，如果是一百年后的手枪，例如大名鼎鼎的沙漠之鹰，绝对可以轰碎鬼尸的脑袋。如同一切僵尸一样，只要脑袋碎了，就不可能再控制身体，尸气散了也就死了。但是这个时代的手枪，威力实在太差，对付这种人形煞肯定是不行的。
<br/>　　鬼尸在被子弹不断袭击之下，变得更加的疯狂，见人就咬。因为人太多，总有目标可以抓，不一会的工夫，已经有数十个人死在女尸的口下。
<br/>　　不过小三带着自己几个手下，一直都在附近冷眼旁观，并没有前去送死。
<br/>　　叶少阳站在外围，想到了一个对付这具鬼尸的办法，于是用指甲划破手掌，用自己的血，在掌心涂抹出了一道符文，然后等待机会。
<br/>　　在连子弹都打不死这女尸的情况之下，众多土匪也完全相信了她不是人，再加上有土匪不断被咬死，大伙纷纷后退，不过这鬼尸被鲜血刺激的狂性大发，到处追着人咬。
<br/>　　被咬的立刻就会死。
<br/>　　这些土匪虽然平时凶悍，但看到同伴们一个个倒下去，生怕自己成为下一个目标，一个个到处寻找躲避的地方，有的跑出了寨子，有的跑进了卧房，更多的则跑进了之前吃饭的大厅，然后一起把门堵上。
<br/>　　偌大的空地上，除了有一些在远处偷偷观察的，基本上都跑光了。
<br/>　　叶少阳眼看机会到了，对着鬼尸吹了一声口哨，先吸引她的注意，然后默念了一遍咒语，把手掌对她伸过去。
